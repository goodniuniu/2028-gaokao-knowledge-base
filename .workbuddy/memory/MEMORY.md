# 项目长期记忆

## 项目目标
2028广东新高考（语数英物化生）650分知识库。当前259张卡片（2026-07-22 阶段2.3收尾），目标260+。

## 质量标准约定
- 每张核心知识网络卡必须有：题型识别标志 + 标准化解题路径 + 至少1道可追溯真题（年份+题号）
- 真题来源：语数英=新课标Ⅰ卷，物化生=广东选择性考试
- 真题不能编造，每道题需搜索验证后嵌入

## 工作流
1. 改卡片md → 跑 `python build_site.py`（确认无断链） → 跑 `python kb_audit.py`
2. 新卡片必须严格遵循 `01_知识卡片模板.md` 格式规范
3. 统计数字唯一来源：`kb_audit.py` 输出，只在 `00_知识库总索引.md` 展示
4. **MD 是唯一源**：`build_site.py` 的 `card_page` 已由 md 重新生成卡片正文（2026-07-22 修复，此前曾错误地从旧 HTML 提取导致内容漂移）
5. 改动大量卡片后跑 `python check_html_md_consistency.py` 确认 HTML 与 MD 一致（0=一致、1=不一致）
