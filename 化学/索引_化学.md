## 五、重点专题规划（预设计）

### 核心知识网络（建议建设）
- [x] 化学计量：物质的量、阿伏伽德罗常数、气体摩尔体积
- [x] 元素周期律：位-构-性关系、半径/电负性/金属性比较
- [x] 化学反应与能量：焓变、热化学方程式、盖斯定律
- [x] 化学反应速率与平衡：速率计算、平衡常数、勒夏特列原理
- [x] 水溶液中的离子平衡：三大守恒、沉淀溶解平衡、Ksp
- [x] 电化学：原电池、电解池、金属腐蚀与防护
- [x] 有机化学：官能团性质、同分异构体书写、合成路线
- [x] 结构化学：杂化轨道、分子构型、晶体计算

### 典型题型与方法（建议建设）
- [ ] 化学计量：以物质的量为中心的计算通法
- [ ] 元素推断：周期表位构性综合分析
- [ ] 化学平衡：等效平衡、转化率计算、图像分析
- [ ] 离子平衡：溶液中离子浓度大小比较
- [ ] 电化学：电极反应式书写、电子转移计算
- [ ] 有机推断：官能团特征反应与结构推导
- [ ] 化学实验：仪器选择、装置连接、定量分析
- [ ] **广东工艺流程**：矿石提取、回收利用综合分析（广东卷高频）
- [ ] **广东实验探究**：真实情境下的实验设计与评价
- [ ] **广东有机合成**：药物/材料合成路线分析

### 易错警示与辨析（建议建设）
- [ ] 阿伏伽德罗常数：八大陷阱（标况、状态、水解等）
- [ ] 氧化还原：双线桥与单线桥、氧化性强弱判断
- [ ] 离子反应：少量/过量问题、离子共存条件
- [ ] 化学平衡：转化率与起始量关系、等效平衡判断
- [ ] 电化学：原电池vs电解池、阴阳极/正负极判断
- [ ] 有机化学：同分异构体漏写/重复、反应条件混淆
- [ ] 实验：除杂顺序、检验干扰、定量误差分析
- [ ] **广东工艺陷阱**：流程图中信息提取、循环物质判断
- [ ] **等级赋分**：原始分与赋分的差异、基础题不丢分策略

### 素材与拓展（建议建设）
- [ ] 化学前沿：碳中和、新能源电池、催化技术
- [ ] **广东工业素材**：石油化工（茂名/广州）、盐化工（珠海）、稀土（粤北）
- [ ] 化学史：门捷列夫、侯德榜、拉瓦锡等科学家
- [ ] 生活化学：食品、药品、环境化学常识
- [ ] 工业化学：化工生产实际流程（合成氨、硫酸、氯碱）
- [ ] 诺贝尔奖：近年化学奖成果简介
- [ ] **大湾区环保**：珠江口水质治理、大气污染监测、固废处理
