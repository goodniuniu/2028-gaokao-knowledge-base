## 核心内容

> **本质理解**：少量/过量问题的本质是"**反应分步进行，量决定停在哪一步**"。通法只有一句：**"少定多变，以少定一"**——把量少的物质系数定为1，按它"需要什么、需要多少"来配量多的物质；量多者有剩余，剩余部分不进方程式。

---

### 一、"少定多变"书写四步法

1. **定一**：量少的反应物系数定为1（或其化学式中离子按组成比例齐全写出）。
2. **按需配多**：量多的物质按"少量者的需求"取用——少量者要几个就给几个，多余的不要写。
3. **查守恒**：原子守恒 + 电荷守恒双双核验。
4. **查拆分**：沉淀、气体、水、弱电解质、氧化物不拆；浓硫酸不拆。

> 反向自检：写完把"少量物质"系数化成1，看另一种物质系数是否正好满足需求且**比例合理**（如Ca(OH)2少量时Ca2+与OH-必须是1:2同时出现）。

### 二、高频体系分步方程式对照表

#### 体系1：CO2 / SO2 通入碱液（酸性氧化物 + 碱）

| 体系 | 碱过量（CO2少量） | 酸气过量（CO2过量） |
|------|-------------------|---------------------|
| CO2 → NaOH | CO2 + 2OH- = CO3^2- + H2O | CO2 + OH- = HCO3- |
| SO2 → NaOH | SO2 + 2OH- = SO3^2- + H2O | SO2 + OH- = HSO3- |
| CO2 → 澄清石灰水 | CO2 + Ca2+ + 2OH- = CaCO3↓ + H2O | 2CO2 + Ca(OH)2过量时沉淀溶解：CaCO3 + CO2 + H2O = Ca2+ + 2HCO3-（现象：先浑后清） |

> 分界点：n(CO2) : n(NaOH) = 1:2 时恰生成 CO3^2-；= 1:1 时恰生成 HCO3-；介于两者之间两者共存（计算题考法）。

#### 体系2：NaOH 与铝盐互滴（铝三角，最高频）

| 滴加方式 | 离子方程式 | 现象 |
|----------|-----------|------|
| NaOH 少量（滴入AlCl3） | Al3+ + 3OH- = Al(OH)3↓ | 立即产生白色沉淀 |
| NaOH 过量 | Al3+ + 4OH- = AlO2- + 2H2O | 沉淀先生成后溶解 |
| 反向：AlCl3滴入NaOH中 | 先 Al3+ + 4OH- = AlO2- + 2H2O；继续加：Al3+ + 3AlO2- + 6H2O = 4Al(OH)3↓ | 开始无沉淀，一段时间后出现沉淀且不消失 |

> **互滴可鉴别**：NaOH与AlCl3互相滴加现象不同，不用其他试剂即可鉴别。
> ⚠️ 写法提示：新教材（2019人教版）将 AlO2- 写作 [Al(OH)4]-（Al3+ + 4OH- = [Al(OH)4]-），两种写法本质相同，以所用教材为准、全卷统一。

#### 体系3：NaAlO2 溶液与盐酸互滴

| 盐酸用量 | 离子方程式 |
|----------|-----------|
| 盐酸少量 | AlO2- + H+ + H2O = Al(OH)3↓ |
| 盐酸过量 | AlO2- + 4H+ = Al3+ + 2H2O |

> 与体系2合称"铝三角四式"，是广东卷离子方程式判断的钉子户考点。

#### 体系4：Na2CO3 与盐酸互滴（现象鉴别高频）

| 滴加方式 | 分步 | 离子方程式 | 现象 |
|----------|------|-----------|------|
| 盐酸滴入 Na2CO3（盐酸少量） | 第一步 | CO3^2- + H+ = HCO3- | **开始无气泡** |
| （继续加盐酸） | 第二步 | HCO3- + H+ = CO2↑ + H2O | 后产生气泡 |
| Na2CO3 滴入盐酸（盐酸过量） | 一步 | CO3^2- + 2H+ = CO2↑ + H2O | **立即产生气泡** |

> **互滴可鉴别**：盐酸与Na2CO3溶液互滴现象不同（先无泡vs立即有泡）。
> ⚠️ 易错：盐酸少量时直接写 CO3^2- + 2H+ = CO2↑+H2O —— 错，少量H+只够把CO3^2-推到HCO3-。

#### 体系5：Fe 与稀硝酸（产物看铁的量）

| 谁过量 | 化学/离子方程式 | 原因 |
|--------|----------------|------|
| HNO3 过量（Fe少量） | Fe + 4H+ + NO3- = Fe3+ + NO↑ + 2H2O | 硝酸足量，Fe被氧化到+3 |
| Fe 过量 | 3Fe + 8H+ + 2NO3- = 3Fe2+ + 2NO↑ + 4H2O | 过量的Fe把Fe3+还原：Fe + 2Fe3+ = 3Fe2+ |

> 两步合成：Fe过量 = 先生成Fe3+，过量Fe再与之反应，合并后即上式。

#### 体系6：Cl2 通入 FeBr2 溶液（竞争氧化，还原性排队）

还原性顺序：**Fe2+ > Br-**（Cl2先氧化Fe2+，Fe2+耗尽才轮到Br-）

| Cl2用量 | 离子方程式 |
|---------|-----------|
| Cl2 少量 | Cl2 + 2Fe2+ = 2Cl- + 2Fe3+ |
| Cl2 与 FeBr2 等量（1:1） | 2Cl2 + 2Fe2+ + 2Br- = 4Cl- + 2Fe3+ + Br2 |
| Cl2 过量 | 3Cl2 + 2Fe2+ + 4Br- = 6Cl- + 2Fe3+ + 2Br2 |

> **避坑**：Cl2少量时方程式中出现Br-被氧化 → 必错（Fe2+还没氧化完）。
> 同类：Cl2通入FeI2（还原性 I- > Fe2+，先氧化I-）。

#### 体系7：NaHCO3 与 Ca(OH)2 互滴（"少定多变"最经典的演练场）

| 谁少量 | 分析 | 离子方程式 |
|--------|------|-----------|
| NaHCO3 少量（Ca(OH)2过量） | HCO3-定为1，需1个OH-中和，Ca2+沉淀CO3^2- | HCO3- + Ca2+ + OH- = CaCO3↓ + H2O |
| Ca(OH)2 少量（NaHCO3过量） | Ca(OH)2定为1 → 1个Ca2+带2个OH-，需2个HCO3-，生成1个CO3^2-沉淀、另1个CO3^2-游离 | 2HCO3- + Ca2+ + 2OH- = CaCO3↓ + CO3^2- + 2H2O |

> **判错技巧**：Ca(OH)2少量时若方程式中Ca2+与OH-不是1:2配比出现 → 必错（少量者必须"整建制"参加）。

#### 体系8：Ba(OH)2 与 NaHSO4（两种"恰好"）

| 目标 | 配比 | 离子方程式 |
|------|------|-----------|
| 恰好中和（溶液呈中性） | Ba(OH)2 : NaHSO4 = 1:2 | 2H+ + SO4^2- + Ba2+ + 2OH- = BaSO4↓ + 2H2O |
| 恰好沉淀完全 | 1:1 | H+ + SO4^2- + Ba2+ + OH- = BaSO4↓ + H2O |

> 两句话记牢："**中和看氢氧，沉淀看钡硫**"——至中性则H+与OH-相等；至沉淀完全则Ba2+与SO4^2-相等。

### 三、常见判错套路速查（选择题秒杀清单）

| 题面 | 秒杀判断 |
|------|----------|
| "少量CO2通入NaOH生成HCO3-" | 错，少量CO2应生成CO3^2- |
| "NaOH滴入AlCl3：Al3+ + 4OH- = AlO2- + 2H2O" | 错，滴入=碱少量，应生成Al(OH)3 |
| "盐酸滴入Na2CO3立即写CO2↑" | 错，盐酸少量先生成HCO3- |
| "少量Cl2氧化FeBr2中的Br-" | 错，先氧化Fe2+ |
| "Ca(OH)2少量时OH-系数与Ca2+相同" | 错，必须是1:2 |
| "Fe过量与稀硝酸生成Fe3+" | 错，生成Fe2+ |

## 备注

- "少定多变"的本质是把少量物质当成"需求方"：它有多少组成离子就全部出动，需要对方多少就拿多少。凡是写出的方程式中少量物质"缺胳膊少腿"（如Ca(OH)2少量时只来1个OH-），必是错的。
- AlO2- 与 [Al(OH)4]- 两种写法：以现行教材为准，答题卡上全文统一用一种，不要混写。
- 广东卷选择题四个选项常把"量"藏在文字里（"少量""过量""滴入""恰好中和""恰好沉淀完全"），读题时先把这些词圈出来再判断。
- 赋分提示：离子方程式题3分一档，属中低档题；等级赋分制下，中低档题的失误对赋分的伤害远大于压轴题，此类题目标是100%正确率。
