## 核心内容

> **本质理解**：电化学所有判断从一个字出发——**"氧"**：失电子、被氧化的一极，在原电池叫**负极**，在电解池叫**阳极**。记住"**负氧、阳氧**"（负极氧化、阳极氧化），其余全部由此推出：得电子被还原的一极，原电池叫正极、电解池叫阴极。

---

### 一、判定总流程（先看有没有外接电源）

```
装置中是否有外接电源？
├── 无电源 → 原电池（本身发电）
│     负极：失电子、氧化反应（通常是较活泼金属）
│     正极：得电子、还原反应
│     离子迁移：阳离子→正极，阴离子→负极（"正正负负"）
│     电子流向：负极 → 外电路 → 正极
└── 有电源 → 电解池（被驱动）
      阳极：与电源【正极】相连，失电子、氧化反应
      阴极：与电源【负极】相连，得电子、还原反应
      离子迁移：阳离子→阴极，阴离子→阳极（"阴阳相吸"）
      电子流向：电源负极 → 阴极；阳极 → 电源正极
```

> **口诀**："**负氧阳氧、正还阴还；原电池正正负负，电解池阴阳相吸。**"
> （"正正负负"指原电池中阳离子向正极移动、阴离子向负极移动——离子找"名异实同"的电极：阳离子去的极发生还原，阳离子自己被还原的备选。）

#### 原电池正负极判定的两个经典特例（活泼金属不一定作负极）

| 电池 | 负极 | 原因 |
|------|------|------|
| Mg-Al 插入 NaOH 溶液 | **Al** | Mg不与NaOH反应，Al能与NaOH反应失电子：Al - 3e- + 4OH- = AlO2- + 2H2O |
| Fe-Cu 插入浓硝酸（常温） | **Cu** | Fe常温下在浓硝酸中钝化，Cu持续失电子 |

> **判负极的真正标准**：谁失电子谁是负极，"活泼金属作负极"只是常见情形，遇特殊电解质必须看实际反应。

### 二、放电顺序（电解池判断产物的前提）

#### 阴极（阳离子得电子，氧化性强者先放电）

Ag+ > Fe3+ > Cu2+ > H+(酸) > Pb2+ > Sn2+ > Fe2+ > Zn2+ > H+(水) > Al3+ > Mg2+ > Na+ > Ca2+ > K+

> 水溶液中 **Al3+及更活泼金属离子永不放电**（轮到水电离的H+）——电解NaCl、Na2SO4水溶液阴极都出H2。

#### 阳极（失电子，分两类）

1. **活性电极**（除Pt、Au、石墨外的金属电极）：**电极本身溶解**，M - ne- = M^n+，溶液中离子不放电。
2. **惰性电极**（Pt、石墨）：阴离子放电，顺序 S2- > I- > Br- > Cl- > OH- > 含氧酸根（SO4^2-、NO3-等不放电，由OH-放电）。

> **口诀**："**阳极先看电极料，活性自身先溶掉；惰性才排阴离子，硫碘溴氯氢氧根。**"

---

### 三、电极反应式书写四步法

1. **定电极、定反应类型**：阳氧/负氧（失电子，-ne-在左），阴还/正还（得电子，+ne-在左）。
2. **定放电物质**：按放电顺序确定谁反应、生成什么。
3. **按介质配平**：先配变价原子与电子，再用介质粒子配电荷、配原子——
   - 酸性介质：用 H+ 和 H2O 调平（**禁止出现OH-**）；
   - 碱性介质：用 OH- 和 H2O 调平（**禁止出现H+**）；
   - 熔融/固体电解质：用传导离子调平（如熔融碳酸盐用CO3^2-，固体氧化物用O2-）。
4. **三查**：查电子守恒（两极得失电子相等）、查电荷守恒（方程式两侧净电荷相等）、查原子守恒；气体标↑、沉淀标↓（电极反应式中生成气体通常标↑）。

#### 示范：碱性甲烷燃料电池负极（介质最容易写错的一例）

1. 负极氧化：CH4 失电子，C由-4 → +4，失8e-。
2. 碱性介质中碳产物是 **CO3^2-**，不是CO2。
3. 配平：**CH4 - 8e- + 10OH- = CO3^2- + 7H2O**。
   - ❌ 错写：CH4 - 8e- + 2H2O = CO2↑ + 8H+（碱性介质中出现H+和CO2，双重错误）。

---

### 四、常见错误清单（逐条对照自查）

| 错误类型 | 错例 | 正确做法 |
|----------|------|----------|
| 介质不写/错写 | 碱性电池电极式中出现H+；酸性电池中出现OH- | 看题给电解质：酸补H+/H2O，碱补OH-/H2O |
| 电荷不守恒 | Cu - 2e- = Cu+（左-2右+1） | 两侧净电荷必须相等：左 0-(-2)=+2... 正确为 Cu - 2e- = Cu2+ |
| 电子方向写反 | 阴极写成 M - ne- | 阴极/正极得电子：M^n+ + ne- = M |
| 气体符号漏标 | 电解饱和食盐水总式漏 H2↑、Cl2↑ | 溶液中的反应生成气体标↑ |
| 总反应与两极式不匹配 | 两极式电子数不相等就直接相加 | 先乘以最小公倍数使得失电子相等再相加 |
| 水溶液电解阴极写"2H+ + 2e- = H2↑" | 中性溶液中H+来自水 | 规范写法：**2H2O + 2e- = H2↑ + 2OH-** |
| Al3+/Na+在水溶液中"被电解出来" | 电解NaCl溶液得金属Na | 水溶液中活泼金属离子不放电，得Na须电解**熔融**NaCl |

### 五、充电电池："负对负、正对正"

- 放电时是原电池；充电时变电解池，电极角色反转：放电时的**负极**充电时作**阴极**（接外电源**负极**），放电时的**正极**充电时作**阳极**（接外电源**正极**）。
- **接线口诀**："**负对负、正对正**"——电池负极接电源负极，电池正极接电源正极。
- 充电反应 = 放电反应的逆过程；充电时电极反应式 = 放电时对应电极反应式倒写。

#### 铅蓄电池示范

- 放电：负极 Pb - 2e- + SO4^2- = PbSO4；正极 PbO2 + 2e- + 4H+ + SO4^2- = PbSO4 + 2H2O。
- 充电：原负极接电源负极作阴极，反应倒写：PbSO4 + 2e- = Pb + SO4^2-。
- ❌ 易错：充电时把"阳极"与"正极"混为一谈，误认为正极仍发生还原——充电时原正极接电源正极，作**阳极**，发生**氧化**。

### 六、电镀与精炼铜特例（对比记忆）

| 项目 | 电镀（铁件镀铜） | 电解精炼铜 |
|------|------------------|-----------|
| 阳极 | **镀层金属** Cu：Cu - 2e- = Cu2+ | **粗铜**：Cu - 2e- = Cu2+（Zn、Fe、Ni也随之溶解） |
| 阴极 | **镀件** Fe：Cu2+ + 2e- = Cu | **纯铜**：Cu2+ + 2e- = Cu |
| 电解质 | 含镀层金属离子的溶液（CuSO4溶液） | CuSO4溶液 |
| 溶液浓度 | **不变**（溶多少析多少） | **Cu2+浓度减小**（阳极溶解的不全是Cu，阴极只析Cu） |
| 特殊产物 | — | **阳极泥**（Ag、Au等不如Cu活泼，不失电子，沉在阳极下方） |

#### 高频易错

- ❌ "精炼铜时阳极减少的质量 = 阴极增加的质量"——错。阳极溶解的有Zn/Fe/Ni等轻杂质（且部分变成阳极泥不溶），阴极只析出Cu，**阳极减重 > 阴极增重** 一般不成立，两者无固定关系；电子守恒才是唯一可靠等量关系。
- ❌ 电镀时把镀件作阳极——镀件必须作阴极（被"镀上"，发生还原析出）。
- ❌ 精炼铜时认为溶液中Cu2+浓度不变——杂质放电拖走电子，Cu2+浓度变小，需定期补充。

### 题型识别标志（陷阱信号）

| 题干/选项信号 | 触发陷阱 | 立即动作 |
|------|------|------|
| "装置有无外接电源" | 原电池 / 电解池判反 | 无电源→原电池（负氧）；有电源→电解池（阳氧） |
| "Mg-Al/NaOH、Fe-Cu/浓硝酸" | 活泼金属必作负极 | 看实际失电子者（Al、Cu 反常），谁失电子谁负极 |
| "充电时电极 a / b 是阴 / 阳极" | 充放电角色混淆 | 负对负正对正：放电负极充电作阴极，放电正极充电作阳极 |
| "阴极 / 阳极产物" | 忘放电顺序 | 阴极按氧化性：Ag⁺>…>H⁺>…>Al³⁺不放电；阳极活性电极先溶 |
| "电极反应式出现 H⁺ / OH⁻" | 介质写错 | 酸补 H⁺/H₂O，碱补 OH⁻/H₂O，禁止出现相斥粒子 |
| "精炼铜阳极质量" | 以为阳极减重 = 阴极增重 | 阳极溶杂（Fe/Zn/Ni）+ 阳极泥，仅电子守恒可靠 |

### 标准化纠错路径（阴阳极判定四步法）

拿到电化学题，强制走四步：
1. **判池型**：有外接电源→电解池；无电源→原电池。
2. **定电极**：原电池失电子者负极（活泼金属不一定）；电解池与电源正极连→阳极，负极连→阴极。
3. **写反应**：负氧/阳氧失电子（−ne⁻ 在左），正还/阴还得电子（+ne⁻ 在左）；按放电顺序定物质。
4. **配介质 + 三查**：酸/碱/熔融补对应粒子，查电子守恒、电荷守恒、原子守恒；气体标 ↑。

### 真题母题（2022 广东卷 T16）

**题**：科学家基于 Cl₂ 易溶于 CCl₄ 的性质，发展了一种无需离子交换膜的新型氯流电池，可作储能设备（如图）。充电时电极 a 的反应为：NaTi₂(PO₄)₃ + 2Na⁺ + 2e⁻ = Na₃Ti₂(PO₄)₃。下列说法正确的是（ ）
A. 充电时电极 b 是阴极
B. 放电时 NaCl 溶液的 pH 减小
C. 放电时 NaCl 溶液的浓度增大
D. 每生成 1 mol Cl₂，电极 a 质量理论上增加 23 g

**错解**：把充电时电极 a 当"阳极"（见"充电"二字直觉误判，实际 a 得电子=阴极，故 b 是阳极，A 错）；或放电时以为 NaCl 被消耗而 pH 减小（B 错）；或算电极 a 增重时漏乘 2 mol Na⁺，得 23 g（D 错，应为 46 g）。
**正解**：充电时电极 a 得电子→阴极，则电极 b 为阳极，A 错；放电时负极 Na₃Ti₂(PO₄)₃ − 2e⁻ = NaTi₂(PO₄)₃ + 2Na⁺，正极 Cl₂ + 2e⁻ = 2Cl⁻，Na⁺、Cl⁻ 均增多→NaCl 浓度增大、pH 不变，B 错 C 对；充电时阳极 2Cl⁻ − 2e⁻ = Cl₂↑，每生 1 mol Cl₂ 转移 2 mol e⁻，电极 a 对应 2 mol Na⁺ 迁入增重 2×23 = 46 g，D 错。答案选 **C**。
**避错要点**：本题命中本卡两大核心——①"有无外接电源 / 充放电角色"（充电时 a 得电子=阴极，对应放电时 a 为负极，正是"负对负、正对正"）；②"介质 / 电荷守恒"（放电产物为 NaCl，pH 不变，增重按 2 mol Na⁺ 计）。先标电极再读选项，可避免 A、D 两类方向性失误。
