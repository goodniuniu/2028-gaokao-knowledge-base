## 核心内容

### 关键概念
- **位-构-性**：元素在周期表中的位置（位）→ 原子结构（构）→ 元素性质（性）的相互关系
- **原子半径**：同周期从左到右递减，同主族从上到下递增
- **电离能**：气态原子失去电子所需能量，同周期递增（半满/全满异常）
- **电负性**：原子吸引键合电子的能力，同周期递增，同主族递减
- **金属性/非金属性**：元素失电子/得电子能力的相对强弱

### 核心规律与公式

#### 1. 周期律核心规律表
| 性质 | 同周期（左→右） | 同主族（上→下） |
|------|----------------|----------------|
| 原子半径 | 逐渐减小 | 逐渐增大 |
| 核电荷数 | 逐渐增大 | 逐渐增大 |
| 电子层数 | 相同 | 逐渐增多 |
| 最外层电子数 | 逐渐增多（1→8） | 相同 |
| 金属性 | 逐渐减弱 | 逐渐增强 |
| 非金属性 | 逐渐增强 | 逐渐减弱 |
| 最高价氧化物水化物 | 碱性减弱，酸性增强 | 碱性增强，酸性减弱 |
| 气态氢化物稳定性 | 逐渐增强 | 逐渐减弱 |
| 第一电离能I₁ | 总体增大（ⅡA、ⅤA异常） | 逐渐减小 |
| 电负性 | 逐渐增大 | 逐渐减小 |

#### 2. 电离能异常规律
```
ⅡA > ⅢA（ns²全满稳定，ns²np¹易失）
ⅤA > ⅥA（ns²np³半满稳定，ns²np⁴易失一对）
```
> 典型例子：I₁(Mg) > I₁(Al)；I₁(N) > I₁(O)

#### 3. 对角线规则
```
Li 与 Mg 性质相似
Be 与 Al 性质相似（两性）
B  与 Si 性质相似（半导体、含氧酸弱酸）
```

#### 4. 微粒半径比较三步法
```
① 比层数：层多径大（电子层数越多，半径越大）
② 比核电荷：同层时，核电荷数越大，半径越小
③ 比电荷：同元素，电子越多半径越大（r_阴离子 > r_原子 > r_阳离子）
```

#### 5. 金属性与非金属性判断依据
| 金属性强弱 | 非金属性强弱 |
|-----------|-----------|
| 单质与水/酸反应置换H₂越容易 | 单质与H₂化合越容易 |
| 最高价氧化物水化物碱性越强 | 最高价氧化物水化物酸性越强 |
| 单质还原性越强 | 单质氧化性越强 |
| 阳离子氧化性越弱 | 阴离子还原性越弱 |
| 置换反应：活泼金属置换不活泼 | 置换反应：活泼非金属置换不活泼 |

### 方法步骤

#### 元素推断题解题框架
1. **突破口定位**：
   - 地壳中含量最高：O（非金属）、Al（金属）
   - 最轻元素：H；最外层电子=层数：H、Be、Al
   - 短周期中半径最小：H；最大：Na
   - 特殊电子排布：全满（ⅡA）、半满（ⅤA）、全空
2. **结构验证**：根据原子序数差推位置，验证半径/电负性/化合价
3. **性质确认**：用周期律预测未知性质，判断选项正误

#### 周期表片段推断速查
| 已知条件 | 推断方向 |
|----------|----------|
| A、B、C三种短周期元素，原子序数依次增大 | 先看周期位置，再比性质 |
| X的氢化物是10电子分子 | X可能是N（NH₃）、O（H₂O）、F（HF） |
| Y的最高价+最低价的绝对值=8 | Y是第ⅥA族（S等） |
| Z的氧化物对应水化物是两性 | Z是Al |
| W的气态氢化物最稳定 | W是F（短周期）或同周期最右非金属 |

### 记忆口诀/技巧
> **半径口诀**：层多径大，核大径小，阴大阳小（同元素）。
> **电离能异常**：半满全满更稳定，失去电子要更多能量。
> **金属性口诀**：左下强，右上弱；非金属性反过来。
> **广东情境提示**：广东卷元素推断题常以广东本地科技为情境（如稀土元素、锂电池材料、半导体材料），注意提取元素符号和位置信息。

## 备注

- 周期律是广东卷化学选择题高频考点，常以"下列说法正确/错误的是"形式出现
- 陷阱：
  - 比较酸性/碱性时必须用"最高价"氧化物对应水化物
  - H₂O和HF的稳定性不能说明O和F的非金属性（必须用气态氢化物）
  - 离子半径比较时，先比层数再比核电荷，Fe²⁺ > Fe³⁺（电子数越多半径越大）
- 广东卷特色：元素推断题常结合新材料（如GaN半导体、钙钛矿太阳能电池），关注题目中的陌生元素
