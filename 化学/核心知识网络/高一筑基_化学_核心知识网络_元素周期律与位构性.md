## 核心内容

### 关键概念
- **位-构-性**：元素在周期表中的位置（位）→ 原子结构（构）→ 元素性质（性）的相互关系
- **原子半径**：同周期从左到右递减，同主族从上到下递增
- **电离能**：气态原子失去电子所需能量，同周期递增（半满/全满异常）
- **电负性**：原子吸引键合电子的能力，同周期递增，同主族递减
- **金属性/非金属性**：元素失电子/得电子能力的相对强弱

### 核心规律与公式

#### 1. 周期律核心规律表
| 性质 | 同周期（左→右） | 同主族（上→下） |
|------|----------------|----------------|
| 原子半径 | 逐渐减小 | 逐渐增大 |
| 核电荷数 | 逐渐增大 | 逐渐增大 |
| 电子层数 | 相同 | 逐渐增多 |
| 最外层电子数 | 逐渐增多（1→8） | 相同 |
| 金属性 | 逐渐减弱 | 逐渐增强 |
| 非金属性 | 逐渐增强 | 逐渐减弱 |
| 最高价氧化物水化物 | 碱性减弱，酸性增强 | 碱性增强，酸性减弱 |
| 气态氢化物稳定性 | 逐渐增强 | 逐渐减弱 |
| 第一电离能I₁ | 总体增大（ⅡA、ⅤA异常） | 逐渐减小 |
| 电负性 | 逐渐增大 | 逐渐减小 |

#### 2. 电离能异常规律
```
ⅡA > ⅢA（ns²全满稳定，ns²np¹易失）
ⅤA > ⅥA（ns²np³半满稳定，ns²np⁴易失一对）
```
> 典型例子：I₁(Mg) > I₁(Al)；I₁(N) > I₁(O)

#### 3. 对角线规则
```
Li 与 Mg 性质相似
Be 与 Al 性质相似（两性）
B  与 Si 性质相似（半导体、含氧酸弱酸）
```

#### 4. 微粒半径比较三步法
```
① 比层数：层多径大（电子层数越多，半径越大）
② 比核电荷：同层时，核电荷数越大，半径越小
③ 比电荷：同元素，电子越多半径越大（r_阴离子 > r_原子 > r_阳离子）
```

#### 5. 金属性与非金属性判断依据
| 金属性强弱 | 非金属性强弱 |
|-----------|-----------|
| 单质与水/酸反应置换H₂越容易 | 单质与H₂化合越容易 |
| 最高价氧化物水化物碱性越强 | 最高价氧化物水化物酸性越强 |
| 单质还原性越强 | 单质氧化性越强 |
| 阳离子氧化性越弱 | 阴离子还原性越弱 |
| 置换反应：活泼金属置换不活泼 | 置换反应：活泼非金属置换不活泼 |

### 方法步骤

#### 元素推断题解题框架
1. **突破口定位**：
   - 地壳中含量最高：O（非金属）、Al（金属）
   - 最轻元素：H；最外层电子=层数：H、Be、Al
   - 短周期中半径最小：H；最大：Na
   - 特殊电子排布：全满（ⅡA）、半满（ⅤA）、全空
2. **结构验证**：根据原子序数差推位置，验证半径/电负性/化合价
3. **性质确认**：用周期律预测未知性质，判断选项正误

#### 周期表片段推断速查
| 已知条件 | 推断方向 |
|----------|----------|
| A、B、C三种短周期元素，原子序数依次增大 | 先看周期位置，再比性质 |
| X的氢化物是10电子分子 | X可能是N（NH₃）、O（H₂O）、F（HF） |
| Y的最高价+最低价的绝对值=8 | Y是第ⅥA族（S等） |
| Z的氧化物对应水化物是两性 | Z是Al |
| W的气态氢化物最稳定 | W是F（短周期）或同周期最右非金属 |

### 记忆口诀/技巧
> **半径口诀**：层多径大，核大径小，阴大阳小（同元素）。
> **电离能异常**：半满全满更稳定，失去电子要更多能量。
> **金属性口诀**：左下强，右上弱；非金属性反过来。
> **广东情境提示**：广东卷元素推断题常以广东本地科技为情境（如稀土元素、锂电池材料、半导体材料），注意提取元素符号和位置信息。

## 题型识别标志

> **看到什么条件 → 立刻想到什么方法/模型**

| 题干关键条件 | 识别为 | 首选方法 |
|-------------|--------|----------|
| "短周期主族元素""原子序数依次增大" | 元素推断基础情境 | 位构性推断框架：先定周期族，再验性质 |
| "价层 p 轨道半充满" | N 或 P（ⅤA族） | ⅤA 特征 ns²np³，第一电离能异常偏高 |
| "价层电子排布式 nsⁿ⁻¹" | Li(2s¹) 或 Mg(3s²) | 代 n 讨论，结合电中性定元素 |
| "地壳中含量最多" | O 元素 | 直接锁定 E=O |
| "比较电负性" | 位→性 性质推断 | 同周期右大、同主族上大，H 最小 |
| "比较氢化物沸点" | 含氢键判断 | 先看状态(液>气)，再看氢键(NH₃/H₂O/HF) |
| "比较第一电离能" | 异常点考查 | ⅡA>ⅢA、ⅤA>ⅥA；电子层数大则小 |
| "空间结构为三角锥/平面三角形" | VSEPR 考查 | 价层电子对数→杂化→构型(有无孤对) |

## 解题路径（位构性推断四步法）

### 第一步：抓突破口，锁定已知元素
- 找"绝对条件"：地壳最多→O；最轻→H；半满/全满→ⅡA/ⅤA；原子序数依次增大等
- 对 nsⁿ⁻¹ 型排布：n=2→Li(2s¹)，n=3→Mg(3s²)

### 第二步：建立元素映射并验证
- 据"每个短周期均有分布""仅 Y 和 M 同族"等约束逐一确定 Z/M/X/E/Y
- 代入化合价代数和为 0（如 XYZ₄ME₄）验证：各元素化合价之和为零

### 第三步：用周期律判断选项
- 电负性：O>N>H（同周期核电荷越大越强，H 最弱）
- 第一电离能：ⅤA(N)>ⅥA(O)>ⅡA(Mg)（半满/全满稳定）
- 氢化物沸点：H₂O(液)>NH₃(氢键)>PH₃(无氢键)

### 第四步：用 VSEPR 定空间结构
- NH₃：N 价层 4 对(3 键+1 孤对)→三角锥形
- NO₃⁻：N 价层 3 对(无孤对)→平面三角形
- 易错：分子构型 ≠ 电子对构型（需去掉孤对所占位置）

## 母题（2023 广东选择性考试·第14题，4分）

> 广东卷经典"位构性"推断题，突破口为 p 轨道半充满与电中性约束。

**题目**：化合物 XYZ₄ME₄ 可作肥料，所含的 5 种元素位于主族，在每个短周期均有分布，仅有 Y 和 M 同族。Y 的基态原子价层 p 轨道半充满，X 的基态原子价层电子排布式为 nsⁿ⁻¹，X 与 M 同周期，E 在地壳中含量最多。下列说法正确的是（  ）
A. 元素电负性：E>Y>Z
B. 氢化物沸点：M>Y>E
C. 第一电离能：X>E>Y
D. YZ₃ 和 YE₃⁻ 的空间结构均为三角锥形

**解**：
- E 在地壳中含量最多 → **E=O**；Y 价层 p 半充满 → 2s²2p³ 或 3s²3p³，即 N 或 P；Y、M 同族
- X 价层 nsⁿ⁻¹：n=2→2s¹(Li)，n=3→3s²(Mg)；结合 X 与 M 同周期、XYZ₄ME₄ 化合价代数和为 0，得 **Z=H、M=P、X=Mg、E=O、Y=N**
- A：同周期核电荷越大电负性越强，且 H 显+1 价电负性最小，故电负性 O>N>H，即 E>Y>Z，**A 正确**
- B：氢化物 PH₃、NH₃、H₂O 中 H₂O 为液态沸点最高，NH₃ 有氢键高于 PH₃，故 E>Y>M，**B 错**
- C：Mg 电子层数多、第一电离能最小；N 的 2p 半满最稳定，故 Y>E>X，**C 错**
- D：NH₃ 为三角锥形，但 NO₃⁻ 中心 N 价层 3 对无孤对 → 平面三角形，**D 错**

**答**：A

> 💡 关键技巧：先由"电中性约束"确定 Z=H、再据周期分布定 X=Mg，是本题最快突破口；ⅤA 第一电离能高于同周期ⅥA 是高频设陷阱点。

## 备注

- 周期律是广东卷化学选择题高频考点，常以"下列说法正确/错误的是"形式出现
- 陷阱：
  - 比较酸性/碱性时必须用"最高价"氧化物对应水化物
  - H₂O和HF的稳定性不能说明O和F的非金属性（必须用气态氢化物）
  - 离子半径比较时，先比层数再比核电荷，Fe²⁺ > Fe³⁺（电子数越多半径越大）
- 广东卷特色：元素推断题常结合新材料（如GaN半导体、钙钛矿太阳能电池），关注题目中的陌生元素
