化学反应速率与化学平衡

⚠️待强化★★★★☆#高二深化#广东选择性考试来源：人教版选择性必修第一册 / 2023-2025广东选择性考试·化学速率平衡专题
广东考情：考查频率：高频（近5年广东卷每年必考，原理综合大题的核心板块）；难度定位：中档~压轴（K值计算、转化率计算和图像分析常为全卷难点）；特色描述：广东卷常将速率平衡与工业生产工艺情境结合（如合成氨、废气处理），图表信息量大，重视平衡常数表达、三段式计算和勒夏特列原理的规范表述；赋分提示：原理大题分步赋分，三段式列式正确即有过程分，平衡移动方向与原因的文字表述要注意因果链完整，是拉开赋分档次的关键板块

核心内容

关键概念

  • 反应速率：单位时间内反应物或生成物浓度的变化量，$v = \Delta c / \Delta t$
  • 化学平衡：可逆反应中正逆反应速率相等，各组分浓度保持不变的状态
  • 平衡常数K：平衡时生成物浓度幂之积与反应物浓度幂之积的比值
  • 转化率：已转化的反应物占起始反应物的百分比
  • 勒夏特列原理：改变影响平衡的一个条件，平衡向减弱这种改变的方向移动
  • 等效平衡：不同起始状态达到相同平衡状态（各组分百分含量相同）

核心公式/定理

1. 反应速率计算

v = Δc/Δt（单位：mol·L⁻¹·s⁻¹ 或 mol·L⁻¹·min⁻¹）
同一反应：v(A):v(B):v(C):v(D) = m:n:p:q（化学计量数之比）
注意事项：速率取正值；固体和纯液体浓度视为常数，不代入速率表达式；v表示的是平均速率，非瞬时速率

2. 平衡常数K

一般表达式：mA(g) + nB(g) ⇌ pC(g) + qD(g)
K = [C]^p[D]^q / ([A]^m[B]^n)
适用条件：只与温度有关，与浓度、压强、催化剂无关

注意事项：

  • 纯固体和纯液体不写入表达式
  • K值越大，反应进行程度越大
  • 正反应K与逆反应K'互为倒数：K_正 = 1/K_逆
  • 反应相加，K相乘；反应乘以系数n，K变为n次方

3. 平衡转化率

α = (已转化的浓度/起始浓度) × 100%
注意事项：多反应物时，增加一种反应物浓度，可提高另一种反应物的转化率，但自身转化率降低

4. 勒夏特列原理速判表

改变条件平衡移动方向对速率的影响K值变化
增大反应物浓度正向移动v_正突增，v_逆渐变不变
减小生成物浓度正向移动v_逆突减不变
增大压强（缩体）向气体分子数减小方向均增大不变
减小压强（扩体）向气体分子数增大方向均减小不变
升高温度向吸热方向均增大吸热反应K增大
降低温度向放热方向均减小放热反应K增大
加入催化剂不移动同等倍数增大不变
压强只对气体分子数有变化的反应有影响；恒容充惰性气体，各组分浓度不变，平衡不移动

5. 平衡状态判断标志

本质标志：v_正 = v_逆 ≠ 0
宏观标志：各组分浓度/百分含量保持不变
条件判断是否平衡
各组分浓度不变是（本质标志）
混合气体总压不变（Δn≠0）是
混合气体平均摩尔质量不变（Δn≠0或全气体）是
混合气体密度不变（恒容/恒压需具体分析）不一定
颜色不变（有有色物质）是
温度不变（绝热体系）是

方法步骤

三段式计算法（核心工具）

         mA(g) + nB(g) ⇌ pC(g) + qD(g)
起始     a        b        0        0
转化     mx       nx       px       qx
平衡     a-mx     b-nx     px       qx
步骤：

  1. 列出起始浓度（mol/L）
  2. 设转化量为x，按比例写出转化量和平衡量
  3. 根据已知条件列方程（如转化率、K值、压强比）
  4. 求解x，进而求K、转化率、平衡浓度

等效平衡判断

条件等效平衡类型判断标准
恒温恒容完全等效起始投料换算成同侧后完全相同
恒温恒容比例等效（Δn=0）起始投料换算成同侧后成比例
恒温恒压比例等效起始投料换算成同侧后成比例
换算方法：一边倒法——将生成物全部转化为反应物，看是否相同/成比例

记忆口诀/技巧

勒夏特列原理口诀："增谁减谁，减谁增谁；升温向吸，降温向放；加压向少，减压向多。"

K值温度关系：吸热升温K增大，放热升温K减小——与平衡移动方向一致。

等效平衡速记：恒温恒容要相同，恒温恒压成比例；一边倒法是通法。

广东情境提示：广东卷速率平衡题常以广东本地化工为情境（如茂名石化裂化、珠海盐化工、锂电池生产），注意从情境中提取反应方程式和条件变化。

关联卡片

  • 元素周期律与位构性 — 反应速率与元素性质关联
  • 电化学体系 — 电极电势与氧化还原平衡
  • [水溶液中的离子平衡] — 弱电解质平衡、水解平衡（待补充）

备注

  • 平衡图像分析是广东卷高频考点，注意：
  • v-t图：突变点分析条件变化（浓度突变、压强渐变、温度影响两条曲线）
  • c-t图：拐点即平衡，斜率即速率
  • 转化率-温度/压强图：先找最佳条件，再判断ΔH和Δn
  • 平衡计算注意：K的单位有时需写出（取决于表达式）；若题目给的是分压，则用Kp计算
  • 多平衡体系中，注意联立关系（如CO₂溶于水，涉及溶解平衡、电离平衡、水解平衡）
  • 催化剂只改变速率，不改变平衡，不能提高转化率，只能缩短达到平衡的时间
