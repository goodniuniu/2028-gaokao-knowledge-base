## 核心内容

### 一、知识网络

```
沉淀溶解平衡：AₘBₙ(s) ⇌ mAⁿ⁺(aq) + nBᵐ⁻(aq)
        │
        ├─ 溶度积 Ksp = cᵐ(Aⁿ⁺)·cⁿ(Bᵐ⁻)  （只与温度有关）
        ├─ 离子积 Q = cᵐ(Aⁿ⁺)·cⁿ(Bᵐ⁻)（任意时刻）
        │     Q > Ksp → 沉淀生成
        │     Q = Ksp → 饱和共存
        │     Q < Ksp → 沉淀溶解
        ├─ 沉淀生成/溶解/转化
        └─ 分步沉淀：Ksp小的先沉淀（但需结合离子浓度）
```

### 二、核心公式/结论

```
1. 溶度积与溶解度（AB型，如AgCl）：Ksp=s²，s=√Ksp
2. 分步沉淀：当多种离子可沉淀时，先满足 Q>Ksp 者先沉淀
3. 沉淀完全：一般认为 c(离子)<1×10⁻⁵ mol/L 时沉淀完全
4. 沉淀转化：AgCl(s)+I⁻⇌AgI(s)+Cl⁻，K=Ksp(AgCl)/Ksp(AgI)>>1
5. 竞争沉淀临界：两沉淀刚共存时，c(Ba²⁺)=Ksp(沉淀1)/c(离子1)=Ksp(沉淀2)/c(离子2)
```

### 三、易混点

| 易混点 | 澄清 |
|--------|------|
| Ksp越小越易沉淀？ | 不一定！需结合离子浓度，浓度差异大时Ksp大的也可能先沉淀 |
| 同离子效应 vs 盐效应 | 加相同离子→溶解度降(同离子效应)；加无关盐→微弱升(盐效应) |
| 沉淀完全=Ksp=0？ | 错，沉淀完全指浓度<10⁻⁵ mol/L，平衡仍存在 |
| 温度变化Ksp | 溶解吸热则升温Ksp增大，需具体看过程 |

### 四、Ksp计算标准步骤

```
Step 1：写出沉淀溶解平衡式与Ksp表达式（注意系数幂次）
Step 2：由已知Ksp与某离子浓度，求另一离子平衡浓度
Step 3：竞争沉淀时，先求"共存临界"的c(共同离子)
Step 4：结合Ka/Kw等求相关型体浓度比，列出算式
Step 5：代温度条件（25℃常用），检查单位与数量级
```

### 五、真题母题（2021 广东选择性考试·第18题，节选）

> 广东卷"废催化剂回收"流程中，沉钼步骤考查BaMoO₄与BaCO₃的竞争沉淀临界计算。

**已知**：25℃时 Ksp(BaMoO₄)=3.5×10⁻⁸，Ksp(BaCO₃)=2.6×10⁻⁹；H₂CO₃的 Ka₂=4.7×10⁻¹¹；pH>6.0时Mo以MoO₄²⁻存在。

(3)"沉钼"中pH=7.0。②若条件控制不当，BaCO₃也会沉淀。为避免BaMoO₄中混入BaCO₃沉淀，溶液中 c(HCO₃⁻):c(MoO₄²⁻)= ______（列出算式）时，应停止加入BaCl₂溶液。

**解**：
- 刚析出BaMoO₄：c(Ba²⁺)=Ksp(BaMoO₄)/c(MoO₄²⁻)
- 不析出BaCO₃需 c(Ba²⁺)·c(CO₃²⁻) ≤ Ksp(BaCO₃)
  → c(CO₃²⁻) ≤ Ksp(BaCO₃)·c(MoO₄²⁻)/Ksp(BaMoO₄)
- 由 Ka₂=c(CO₃²⁻)·c(H⁺)/c(HCO₃⁻)，pH=7.0即 c(H⁺)=10⁻⁷
  → c(HCO₃⁻)/c(MoO₄²⁻) ≤ Ksp(BaCO₃)·c(H⁺)/[Ksp(BaMoO₄)·Ka₂]

**答**：$\dfrac{2.6\times10^{-9}\times10^{-7}}{3.5\times10^{-8}\times4.7\times10^{-11}}$ （与官方答案一致）

> 💡 关键技巧：竞争沉淀题核心是"两沉淀刚共存"——用共同离子(Ba²⁺)搭桥，把两个不等式连成等式，再借Ka₂换算HCO₃⁻/MoO₄²⁻之比。
