## 核心内容

> **素材卡定位**：稀土分离与锂电池回收是广东卷工艺流程题中最"硬核"的两类情境——流程长、综合性强、计算与表达并重。本卡的目标：看到"稀土矿""废旧锂电池"开头的题目时，能迅速套用"酸浸→除杂→分离→产品"的通用流程骨架，并锁定每步背后的化学原理。

### 素材情境一：粤北稀土——"工业维生素"的提取与分离

#### 【情境背景】

- 粤北地区（韶关、清远一带）蕴藏**离子吸附型稀土矿**，稀土元素以离子形态吸附在黏土矿物表面，可用电解质溶液浸取。
- 稀土共 17 种元素（15 种镧系元素 + 钪、钇），被称为"工业维生素"，广泛用于永磁材料、发光材料、催化剂、储氢材料等高新技术领域，是战略资源。
- 稀土元素化学性质**极其相似**（镧系收缩导致相邻元素原子半径、离子半径相近），单种稀土的分离提纯是流程工业的难题——这正是命题人设置"分离方法"设问的素材来源。
- 典型产业链：稀土矿 → 酸浸/盐浸 → 除杂 → 萃取分组 → 沉淀（草酸盐/碳酸盐）→ 灼烧得稀土氧化物。

#### 【学科原理】

- **浸取（酸浸）**：用稀酸（H₂SO₄/HCl）把矿石中的金属离子溶解转入溶液，实现"固→液"转化，与脉石（SiO₂ 等不溶物）分离。矿石**粉碎**可增大接触面积、加快浸取速率、提高浸取率。
- **调 pH 除杂**：利用不同金属氢氧化物 **Ksp 不同、沉淀 pH 不同**，通过调节 pH 使杂质离子（如 Fe³⁺、Al³⁺）优先水解沉淀，而目标金属离子留在溶液中——**分步沉淀**思想。
- **萃取分离**：利用溶质在两种互不相溶溶剂中**溶解度（分配比）不同**，加入萃取剂（如有机膦类）把目标金属离子从水相转移到有机相，再反萃取回水相，实现性质相近组分的分离。离子交换法则利用树脂对离子吸附能力差异。
- **沉淀与灼烧**：加入草酸等沉淀剂使稀土离子转化为草酸盐沉淀，过滤洗涤后灼烧分解为稀土氧化物。

#### 【考点链接】

- 酸浸反应示例（以铁杂质为例）：Fe₂O₃ + 6H⁺ = 2Fe³⁺ + 3H₂O；Al₂O₃ + 6H⁺ = 2Al³⁺ + 3H₂O。
- 水解沉淀原理（写离子方程式）：
  ```
  Fe³⁺ + 3H₂O ⇌ Fe(OH)₃ + 3H⁺
  ```
  > 加入碱性物质（如氨水、目标金属的氧化物/碳酸盐）消耗 H⁺，平衡右移，Fe³⁺ 沉淀完全。
- Ksp 定量判断：
  ```
  开始沉淀：c(Mⁿ⁺)·cⁿ(OH⁻) > Ksp[M(OH)ₙ]
  沉淀完全：通常规定离子浓度 ≤ 1.0×10⁻⁵ mol·L⁻¹ 时视为完全
  由 Ksp 反推所需最低 pH：c(OH⁻) = ⁿ√(Ksp / c(Mⁿ⁺))
  ```
  > 典型结论：Fe³⁺ 在 pH≈3 左右即可沉淀完全，而 Mg²⁺、Ni²⁺ 等需 pH 更高——"调 pH 至 3~4 除 Fe³⁺、Al³⁺"是流程题标准操作。
- 草酸盐沉淀示例：2RE³⁺ + 3H₂C₂O₄ = RE₂(C₂O₄)₃↓ + 6H⁺（RE 表示稀土元素）。
- 萃取操作要点：分液漏斗、振荡放气、静置分层、下层从下口放出/上层从上口倒出；判断萃取剂要求（与原溶剂互不相溶、溶质在其中溶解度更大、不与溶质反应）。

#### 【命题角度】

- **流程框架填空**：给出"稀土矿→粉碎→酸浸→调 pH 除杂→萃取→反萃取→草酸沉淀→灼烧"流程，设问粉碎目的、酸浸用酸的选择（为什么不能引入杂质离子）、滤渣成分（SiO₂ 等）。
- **Ksp 计算**：给出 Fe(OH)₃、Al(OH)₃ 的 Ksp，计算"Fe³⁺ 沉淀完全而 RE³⁺ 不沉淀"的 pH 控制范围——广东卷常见小计算，注意用 10⁻⁵ mol/L 判据。
- **条件控制分析**："调 pH 时为什么用稀土氧化物/碳酸盐而不用 NaOH？"（不引入新的杂质阳离子 Na⁺）；"酸浸温度为什么不能过高？"（盐酸挥发/某些物质分解/能耗）——这类"目的与原因"设问是流程题踩分密集区。
- **分离方法辨析**：性质相近的稀土离子之间用萃取/离子交换（而非沉淀），考查"为什么不能用调 pH 直接分离各稀土"（Ksp 相近、沉淀 pH 重叠）。

### 素材情境二：广东锂电池产业——从正极材料到废旧电池回收

#### 【情境背景】

- 广东是全国新能源汽车与锂电池产业重镇：深圳比亚迪从电池起家发展为全球新能源汽车头部企业，珠三角聚集了完整的动力电池与储能电池产业链。
- 锂电池主流正极材料包括钴酸锂（LiCoO₂）、磷酸铁锂（LiFePO₄）、三元材料（LiNiₓCoᵧMn₁₋ₓ₋ᵧO₂）等；负极常用石墨，电解质为锂盐有机溶液。
- 动力电池大规模退役后，**废旧锂电池回收**成为新兴产业：回收 Li、Co、Ni、Mn 等有价金属，既是资源循环利用，也是绿色化学、碳中和主题的命题富矿。
- 典型回收流程：放电处理 → 拆解破碎 → 酸浸（加 H₂SO₄ + H₂O₂）→ 调 pH 除铁铝 → 萃取/沉淀分离钴镍锰 → 回收锂（碳酸盐沉淀）→ 再生正极材料。

#### 【学科原理】

- **充放电的氧化还原本质**：放电时负极 Li 失电子（氧化），Li⁺ 经电解质迁移到正极嵌入；充电时外加电源强制逆向进行，原负极变阴极。Li⁺ 在两极之间往返嵌入/脱出，故称"摇椅式电池"。
- **浸出时的氧化还原**：LiCoO₂ 中 Co 为 +3 价，直接用酸难以溶出，加入 **H₂O₂ 作还原剂**把 Co(Ⅲ) 还原为易溶的 Co²⁺（H₂O₂ 同时被氧化为 O₂，不引入杂质离子——"绿色还原剂"）。
- **除杂与分离**：浸出液中的 Fe³⁺、Al³⁺ 用调 pH 沉淀除去；Co²⁺、Ni²⁺、Li⁺ 之间用萃取或选择性沉淀分离；Li⁺ 常用饱和 Na₂CO₃ 溶液沉淀为 Li₂CO₃（溶解度较小）回收。
- **金属活动性与冶炼方法**：金属活动性顺序决定冶炼方法——K、Ca、Na、Mg、Al 用电解法（Li 亦用电解熔融 LiCl 制取），Zn~Cu 用热还原法，Hg、Ag 用热分解法。

#### 【考点链接】

- 放电时电极反应（以 LiCoO₂/石墨电池为例）：
  ```
  负极（氧化）：Li − e⁻ = Li⁺   （更严谨写法：LiC₆ − e⁻ = Li⁺ + C₆）
  正极（还原）：Li₁₋ₓCoO₂ + xLi⁺ + xe⁻ = LiCoO₂
  ```
  > 充电时阴阳极反应为放电时的逆过程；口诀"负失氧、正得还"。
- 酸浸钴酸锂（H₂O₂ 作还原剂）：
  ```
  2LiCoO₂ + H₂O₂ + 3H₂SO₄ = Li₂SO₄ + 2CoSO₄ + O₂↑ + 4H₂O
  ```
  > 离子形式：2LiCoO₂ + H₂O₂ + 6H⁺ = 2Li⁺ + 2Co²⁺ + O₂↑ + 4H₂O。配平检查：Co 由 +3 → +2，H₂O₂ 中 O 由 −1 → 0。
- 除杂与沉锂：
  ```
  Fe³⁺ + 3OH⁻ = Fe(OH)₃↓（调 pH 除去，Al³⁺ 同理）
  2Li⁺ + CO₃²⁻ = Li₂CO₃↓（饱和碳酸钠溶液，趁热过滤减少溶解损失）
  ```
- 金属冶炼方法对照：
  ```
  电解法：2NaCl(熔融) ──通电──→ 2Na + Cl₂↑；2Al₂O₃(熔融) ──通电、冰晶石──→ 4Al + 3O₂↑
  热还原法：Fe₂O₃ + 3CO ──高温──→ 2Fe + 3CO₂
  热分解法：2HgO ──△──→ 2Hg + O₂↑
  ```
- 废旧电池"放电处理"目的：使残留电能释放，防止拆解短路起火（安全），同时使 Li⁺ 嵌入正极、便于后续浸出（原理）。

#### 【命题角度】

- **流程大题设问链**（广东卷标志性考法）：
  1. "放电处理的目的是什么？"（安全 + 富集 Li 于正极，双答法）
  2. "酸浸时加入 H₂O₂ 的作用？写出离子方程式。"（还原剂，把 Co(Ⅲ) 还原为 Co²⁺；方程式如上）
  3. "调 pH 范围的选择依据？"（使 Fe³⁺/Al³⁺ 沉淀完全而 Co²⁺/Ni²⁺ 不沉淀，用 Ksp 数据说明）
  4. "沉锂时为什么用饱和 Na₂CO₃ 且趁热过滤？"（Li₂CO₃ 溶解度随温度升高而减小、用饱和溶液增大 CO₃²⁻ 浓度促进沉淀完全）
  5. "该流程体现的绿色化学思想？"（资源循环利用、原子经济性、减少重金属污染）
- **电化学选择题**：给锂电池充放电总反应，判断充电时某电极名称（原负极接电源负极作阴极）、Li⁺ 迁移方向（放电时向正极迁移）、电极质量变化。
- **氧化还原配平**：酸浸反应、焙烧预处理反应（如废旧正极材料与碳酸盐焙烧）的陌生方程式配平，用"得失电子守恒 + 电荷守恒 + 原子守恒"三步法。

### 两大素材的通用流程骨架（考场提取模板）

```
原料（矿/废料）→ 预处理（粉碎/焙烧/放电，目的：增大接触面、转化形态、安全）
   → 浸出（酸浸/碱浸，核心：固→液，写方程式）
   → 除杂（调 pH 分步沉淀 / 氧化还原调价态，核心：Ksp 与 pH 计算）
   → 分离提纯（萃取 / 离子交换 / 选择性沉淀，核心：性质差异）
   → 产品转化（沉淀→灼烧 / 电解 / 再生，核心：目标产物形态）
```

| 流程环节 | 粤北稀土线的体现 | 锂电池回收线的体现 |
|---------|----------------|------------------|
| 预处理 | 矿石粉碎 | 放电、拆解破碎 |
| 浸出 | 酸浸溶出 RE³⁺ | H₂SO₄+H₂O₂ 还原浸出 Co/Li |
| 除杂 | 调 pH 除 Fe³⁺、Al³⁺ | 调 pH 除 Fe³⁺、Al³⁺ |
| 分离 | 萃取分组稀土 | 萃取分离 Co/Ni，沉锂 |
| 产品 | 草酸盐沉淀→灼烧得氧化物 | Li₂CO₃ / 再生正极材料 |

### 例题风格描述（自测用）

- **风格一·全流程大题**："粤北某稀土企业以离子型稀土矿为原料……"或"某企业回收废旧磷酸铁锂电池……"开头，附流程图与 Ksp 数据表，6~8 个设问覆盖方程式、pH 计算、操作目的、循环利用——这是广东卷工艺流程题的标准长相。
- **风格二·陌生方程式书写**：题干给出反应物与部分产物信息（如"LiCoO₂ 与 H₂O₂ 在硫酸中反应，有气体生成"），要求写出完整离子方程式——先定价态变化，再守恒配平。
- **风格三·数据图表融合**：给出"金属氢氧化物沉淀 pH 范围图"或"浸出率—温度曲线"，要求选择最佳工艺条件并说明理由（浸出率提升趋缓 + 能耗/副反应增加的权衡）。

## 备注

- 本卡为化学"素材与拓展"维度破零批次之二，难度定位高于石化/盐化工卡：Ksp 计算与电化学内容属高二知识，高一阶段先建立"流程骨架 + 情境识别"的直觉，计算部分随选修一课程逐步补全。
- 易错提醒：
  1. 锂电池**充电时**原正极接电源正极作**阳极**（失电子），"正负极"与"阴阳极"别混用：放电讲正负、充电讲阴阳。
  2. H₂O₂ 在浸出中作**还原剂**（产物 O₂），别惯性思维写成氧化剂；判断依据是 Co 价态从 +3 降为 +2。
  3. "调 pH 除杂"加入的试剂优先选目标金属的氧化物/氢氧化物/碳酸盐，避免引入新杂质离子——这是流程题高频踩分点。
  4. 萃取是**物理过程**（分配），但流程中常与化学反应（络合萃取）结合，答题按题干信息为准。
  5. Li₂CO₃ 溶解度"随温度升高而减小"是反常规特性，"趁热过滤"的理由必须写全。
- 广东卷情境化命题高频素材（整体表述，不对应具体题号）：稀土、锂电池、芯片用高纯试剂等"新材料+新能源"情境近年持续出现，建议把教材必修二"化学与可持续发展"与选择性必修一 Ksp 章节跟本卡对照复习。
