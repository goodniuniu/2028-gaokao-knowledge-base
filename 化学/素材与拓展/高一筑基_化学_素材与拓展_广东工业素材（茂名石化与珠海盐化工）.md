### 素材情境一：茂名石化——从石油到"工业血液"的全链条

#### 【情境背景】

- 茂名石化位于广东西部茂名市，是中国石化旗下大型炼化一体化基地，也是华南地区重要的炼油与乙烯生产基地。
- 原油进厂后经**常减压分馏**得到液化石油气、汽油、煤油、柴油、润滑油、重油等馏分；重油再经**催化裂化**提高轻质油产量；馏分油进一步**裂解**生产乙烯、丙烯等基础化工原料。
- 乙烯是衡量一个国家石油化工发展水平的标志，乙烯下游衍生出聚乙烯塑料、合成纤维、合成橡胶等"三大合成材料"。
- 茂名石化是广东卷情境化命题的天然素材：它同时覆盖"物理变化与化学变化辨析""有机反应类型""流程图分析"三类考点。

#### 【学科原理】

- **分馏（物理变化）**：利用石油中各组分**沸点不同**，加热汽化后在分馏塔内分段冷凝。沸点低的组分在塔的上部（低温区）收集，沸点高的在下部收集。整个过程没有新物质生成，是**物理变化**。
- **催化裂化（化学变化）**：在催化剂、加热条件下，把相对分子质量大、沸点高的烃断裂为相对分子质量小、沸点低的烃，产物为**烷烃 + 烯烃**。
- **裂解（深度裂化，化学变化）**：在更高温度下将烃进一步断裂，目的是获得**乙烯、丙烯**等短链不饱和烃。
- **加聚反应**：乙烯等含碳碳双键的单体，在催化剂作用下相互加成形成高分子化合物（如聚乙烯）。

#### 【考点链接】

- 物理变化 vs 化学变化辨析：分馏/蒸馏/干馏/裂化/裂解，只有**分馏、蒸馏是物理变化**，煤的干馏、石油裂化裂解都是化学变化。
- 催化裂化示例（产物不唯一，注意守恒即可）：
  ```
  C₁₆H₃₄ ──催化剂、加热──→ C₈H₁₈ + C₈H₁₆
  ```
- 裂解示例：
  ```
  C₈H₁₈ ──高温──→ C₄H₁₀ + C₄H₈
  C₄H₁₀ ──高温──→ CH₄ + C₃H₆   或   C₄H₁₀ ──高温──→ C₂H₆ + C₂H₄
  ```
- 加聚反应：n CH₂=CH₂ ──催化剂──→ [—CH₂—CH₂—]ₙ（聚乙烯，括号外下角标 n，双键打开变单键）。
- 乙烯的结构与性质：含 C=C 双键，能使溴水（加成，CH₂=CH₂+Br₂→CH₂BrCH₂Br）和酸性 KMnO₄ 溶液（氧化）褪色——**鉴别烷烃与烯烃**的经典方法。

#### 【命题角度】

- **选择题**：给出"分馏、裂化、裂解、干馏、风化、潮解"等过程，判断属于物理变化的有几个——陷阱是把干馏（化学变化）和分馏混在一起。
- **流程题**：给出"原油 → 分馏塔 → 重油 → 催化裂化装置 → 裂解炉 → 乙烯 → 聚乙烯"流程图，设问：
  1. 分馏塔中温度自下而上如何变化？（逐渐降低，轻组分在上部蒸出）
  2. 写出 C₁₆H₃₄ 催化裂化的一种化学方程式（产物组合合理、原子守恒即可）。
  3. 裂解产物中乙烯的检验方法及现象（通入溴的 CCl₄ 溶液或溴水，褪色）。
- **实验题变体**：石蜡油（液态烷烃混合物）在碎瓷片（催化剂/蓄热）作用下分解，产物通入溴水褪色——必修教材原型实验，常考碎瓷片作用、褪色原理、防倒吸。

### 素材情境二：珠海及广东沿海盐化工——大海里的"白色产业"

#### 【情境背景】

- 广东海岸线绵长，海水综合利用是沿海化工的重要方向：海水晒盐（湛江、珠海一带历史上盐场密布）、以食盐为原料的氯碱工业、从海水中提取镁和溴。
- 氯碱工业是基础化工的支柱：电解饱和食盐水同时得到 **NaOH（烧碱）、Cl₂、H₂** 三种产品，是"一电三得"的经典工业。
- 海水中镁、溴储量巨大但浓度低，"**富集**"思想贯穿整个海洋化工流程——这是流程题的核心命题思想。

#### 【学科原理】

- **海水晒盐**：利用日光和风力蒸发水分，使 NaCl 结晶析出，属于**蒸发结晶（物理变化）**。粗盐含 MgCl₂、CaCl₂ 等杂质，需提纯。
- **氯碱工业**：电解饱和食盐水，Cl⁻ 在阳极失电子生成 Cl₂，水中的 H⁺ 在阴极得电子生成 H₂ 并留下 OH⁻。工业上用**阳离子交换膜**隔开两极，防止 Cl₂ 与 NaOH、H₂ 混合反应。
- **海水提镁**：加碱使 Mg²⁺ 沉淀为 Mg(OH)₂（富集）→ 加盐酸溶解得 MgCl₂ 溶液 → 在 HCl 气氛中蒸发脱水得无水 MgCl₂ → **电解熔融 MgCl₂** 得金属镁。
- **海水提溴**：通 Cl₂ 氧化 Br⁻ 得 Br₂ → 用热空气或水蒸气**吹出** Br₂ → 用 SO₂ 水溶液吸收（还原为 Br⁻，再次富集）→ 再通 Cl₂ 氧化得高浓度 Br₂。

#### 【考点链接】

- 氯碱工业总方程式：
  ```
  2NaCl + 2H₂O ──通电──→ 2NaOH + H₂↑ + Cl₂↑
  ```
  - 阳极（石墨）：2Cl⁻ − 2e⁻ = Cl₂↑
  - 阴极（铁网）：2H₂O + 2e⁻ = H₂↑ + 2OH⁻（或写 2H⁺ + 2e⁻ = H₂↑）
  - 检验：阳极产物 Cl₂ 用湿润的淀粉-KI 试纸（变蓝）；阴极产物 H₂ 用爆鸣实验；阴极区溶液使酚酞变红。
- 粗盐提纯除杂顺序（沉淀法）：依次加入过量 **BaCl₂（除 SO₄²⁻）→ NaOH（除 Mg²⁺）→ Na₂CO₃（除 Ca²⁺ 和过量 Ba²⁺）**，过滤后加适量盐酸（调 pH，除过量 OH⁻ 和 CO₃²⁻）。**关键**：Na₂CO₃ 必须在 BaCl₂ 之后加入，盐酸在过滤之后加入。
- 海水提镁核心方程式：
  ```
  MgCl₂ + Ca(OH)₂ = Mg(OH)₂↓ + CaCl₂   （用廉价石灰乳作沉淀剂）
  Mg(OH)₂ + 2HCl = MgCl₂ + 2H₂O
  MgCl₂(熔融) ──通电──→ Mg + Cl₂↑        （工业制镁）
  ```
  > 注意：直接加热 MgCl₂ 溶液会因水解（MgCl₂ + 2H₂O ⇌ Mg(OH)₂ + 2HCl，HCl 挥发）得不到无水 MgCl₂，须在 **HCl 气流中**加热脱水。镁比铝活泼，不能电解水溶液（H⁺ 优先放电），只能电解熔融盐。
- 海水提溴核心方程式：
  ```
  Cl₂ + 2Br⁻ = 2Cl⁻ + Br₂                （氧化：Cl₂ 氧化性 > Br₂）
  Br₂ + SO₂ + 2H₂O = 4H⁺ + SO₄²⁻ + 2Br⁻  （吸收富集：SO₂ 作还原剂）
  Cl₂ + 2Br⁻ = 2Cl⁻ + Br₂                （再次氧化，得到高浓度 Br₂）
  ```

#### 【命题角度】

- **流程题标准设问链**（广东卷高频）：
  1. "加入石灰乳的目的是什么？"（使 Mg²⁺ 转化为 Mg(OH)₂ 沉淀，实现富集与分离；石灰乳原料为贝壳/石灰石，成本低廉）
  2. "为什么要先酸浸/酸溶？"（将沉淀转化为可溶盐溶液，便于后续操作）
  3. "电解时为什么用熔融 MgCl₂ 而不用 MgCl₂ 溶液？"（溶液中 H⁺ 得电子能力强于 Mg²⁺，阴极析出 H₂ 而得不到 Mg）
  4. "提溴流程中 SO₂ 的作用？"（作还原剂，把吹出的 Br₂ 吸收并还原为 Br⁻，实现溴的富集）
- **电化学选择题**：氯碱工业装置图判断阴阳极、产物、离子移动方向；阳离子交换膜"只允许阳离子通过"的作用（防止 Cl₂ 与 OH⁻ 反应、防止 H₂ 与 Cl₂ 混合爆炸）。
- **绿色化学角度**：提镁副产的 Cl₂ 可循环用于制盐酸或氧化 Br⁻；流程题常设问"该工艺中可循环利用的物质是____"。

### 例题风格描述（自测用）

- **风格一·情境选择题**："茂名某石化基地以原油为原料生产乙烯，下列说法正确的是____"，选项分别设置"分馏是化学变化""裂化可提高汽油产量""乙烯能使酸性 KMnO₄ 褪色是因为发生加成反应"等干扰项——每个选项对应一个基础概念的精准辨析。
- **风格二·流程推断题**："广东沿海某化工厂利用海水资源设计了如下流程……"，给出"海水→晒盐→母液→(通 Cl₂)→吹出→(SO₂ 吸收)→(再通 Cl₂)→Br₂"箭头链，要求补全中间物质、书写两处离子方程式、回答"吹出"步骤利用 Br₂ 的什么性质（易挥发、沸点低）。
- **风格三·电化学融合题**：以氯碱工业离子交换膜电解槽示意图为载体，综合考查电极判断、电极反应式、膜的作用、产品出口位置——把工业装置图翻译成"电解池模型"是破题关键。

### 两条素材线的共性解题思维

| 思维模型 | 茂名石化线的体现 | 盐化工线的体现 |
|---------|----------------|----------------|
| **富集思想** | 裂解把分散在重油中的化工原料集中为乙烯 | 海水中 Mg²⁺/Br⁻ 浓度低，先沉淀/先氧化吹出再吸收 |
| **物理/化学过程区分** | 分馏物理，裂化裂解化学 | 晒盐物理，电解化学 |
| **原料→产品主链** | 原油→馏分→乙烯→聚乙烯 | 海水→食盐→NaOH/Cl₂/H₂；海水→Mg；海水→Br₂ |
| **绿色化学** | 催化裂化催化剂再生、废气处理 | Cl₂ 循环、母液回收 |

## 备注

- 本卡为化学"素材与拓展"维度破零批次之一，建议与《广东化学工艺流程题通用框架》配套使用：框架给"怎么答"，本卡给"见到什么情境想什么模型"。
- 易错提醒：
  1. **干馏 ≠ 分馏**：煤的干馏是化学变化（隔绝空气加强热），石油分馏是物理变化。
  2. 写氯碱工业方程式别漏"通电"条件；阴极区显碱性是因为 H₂O 提供的 H⁺ 放电后留下 OH⁻。
  3. 粗盐提纯中 BaCl₂ 与 Na₂CO₃ 顺序不可颠倒，否则过量的 Ba²⁺ 无法除去。
  4. 海水提溴要通 **两次** Cl₂，第一次是氧化低浓度 Br⁻，第二次是氧化富集后的 Br⁻——目的不同，答题时写清"富集"二字。
