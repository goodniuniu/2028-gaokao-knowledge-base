## 核心内容

### 一、工艺流程题的本质

> **本质**：用化学反应原理解决工业生产中的实际问题。
>
> **核心思想**：**一切反应或操作都是为获得产品而服务。**

---

### 二、通用解题框架（四步通法）

```
Step 1：读题干 → 明确最终产品、原料成分、目标元素
Step 2：析流程 → 每一步"进什么→发生什么→出什么"
Step 3：想原理 → 反应条件控制（速率、平衡、沉淀、氧化还原）
Step 4：答问题 → 规范表述（用化学术语，避免口语化）
```

### 三、流程分析核心要点

#### 3.1 常见原料预处理

| 操作 | 目的 | 涉及反应/原理 |
|------|------|--------------|
| **粉碎/研磨** | 增大接触面积，加快反应速率 | 固体反应速率影响因素 |
| **灼烧/煅烧** | 改变结构、除去有机物、氧化还原 | 如菱铁矿煅烧生成Fe₂O₃ |
| **酸浸** | 溶解金属氧化物，除去不溶物 | 金属氧化物 + H⁺ → 盐 + H₂O |
| **碱浸** | 溶解Al、SiO₂、两性氧化物 | 2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑ |
| **水浸** | 溶解可溶性盐 | 物理溶解过程 |

#### 3.2 核心反应类型

| 反应类型 | 常见考法 | 示例 |
|----------|----------|------|
| **氧化还原** | 加入氧化剂/还原剂的目的 | 将Fe²⁺氧化为Fe³⁺，便于沉淀分离 |
| **调节pH** | 控制金属离子沉淀 | Fe³⁺在pH≈3沉淀，Cu²⁺在pH≈5-6沉淀 |
| **沉淀转化** | Ksp小的沉淀更稳定 | AgCl + I⁻ → AgI + Cl⁻ |
| **络合反应** | 溶解难溶物 | AgCl + 2NH₃ → [Ag(NH₃)₂]⁺ + Cl⁻ |
| **电解** | 提纯或提取金属 | 电解精炼铜、电解法制备铝 |

#### 3.3 分离提纯操作

| 操作 | 适用场景 | 注意事项 |
|------|----------|----------|
| **过滤** | 固液分离 | 趁热过滤（防止结晶析出） |
| **蒸发结晶** | 溶解度随温度变化不大的物质 | NaCl：蒸发结晶 |
| **冷却结晶** | 溶解度随温度变化大的物质 | KNO₃：冷却结晶 |
| **萃取分液** | 溶质在不同溶剂中溶解度差异大 | 注意萃取剂选择（不互溶、不反应） |
| **蒸馏** | 沸点差异大的液体分离 | 注意温度计位置、冷凝水流向 |
| **重结晶** | 提纯含少量杂质的固体 | 溶解→蒸发浓缩→冷却结晶→过滤 |

### 四、条件控制原理

#### 4.1 温度控制

| 温度影响 | 答题角度 |
|----------|----------|
| **升高温度** | 加快反应速率、促进溶解、使平衡向吸热方向移动 |
| **降低温度** | 防止物质分解/挥发、使平衡向放热方向移动、减少能耗 |
| **控制温度** | 防止副反应、控制晶体大小、节能 |

#### 4.2 pH控制

| 目的 | 答题角度 | 示例 |
|------|----------|------|
| **促进沉淀** | 使某金属离子完全沉淀（离子浓度<10⁻⁵ mol/L） | Fe³⁺：pH≥3.2 |
| **防止沉淀** | 使目标离子留在溶液中 | Cu²⁺：pH<5 |
| **抑制水解** | 防止金属离子水解 | FeCl₃溶液中加HCl |
| **促进水解** | 制备氢氧化物沉淀 | 加热促进TiCl₄水解 |

#### 4.3 试剂选择

| 试剂类型 | 常见选择 | 选择原则 |
|----------|----------|----------|
| **氧化剂** | H₂O₂、Cl₂、NaClO、KMnO₄、空气 | 不引入新杂质或易除去 |
| **还原剂** | Fe粉、SO₂、H₂S、Na₂SO₃ | 同上 |
| **调节pH** | CuO、Cu(OH)₂、NH₃·H₂O、Na₂CO₃ | 不引入新杂质，消耗H⁺ |
| **沉淀剂** | NaOH、NH₃·H₂O、Na₂CO₃、H₂S | 使目标离子沉淀完全 |

> **绿色化学原则**：优先选择H₂O₂（产物是水）、空气（廉价易得）、不引入新杂质的试剂。

### 五、广东本地产业情境素材

| 产业/地区 | 涉及元素 | 典型流程 | 可能考点 |
|-----------|----------|----------|----------|
| **粤北稀土** | 稀土元素（La、Ce、Nd） | 稀土矿→酸浸→萃取分离→沉淀 | 氧化还原、沉淀pH、Ksp计算 |
| **茂名石化** | C、H、S | 石油裂化→脱硫→催化重整 | 化学反应、催化剂、绿色化学 |
| **珠海盐化工** | Na、Cl、Mg | 海水晒盐→电解→制备镁 | 电解原理、蒸发结晶、萃取 |
| **粤东锂电** | Li、Co、Ni | 废旧电池→酸浸→萃取→电沉积 | 氧化还原、调节pH、电化学 |
| **佛山陶瓷** | Al、Si、Ca | 高岭土→提纯→烧结 | 酸浸、过滤、SiO₂分离 |
| **广东冶炼** | Cu、Fe、Zn、Pb | 多金属矿→焙烧→浸取→电解 | 氧化还原、电解精炼、Ksp |
| **新能源材料** | Li、Co、Ni、Mn | 三元前驱体→共沉淀→烧结 | 沉淀控制、氧化还原、晶体结构 |

### 六、答题规范模板

#### 6.1 "目的/原因"类

> **模板**：为了达到...目的，采取...措施，因为...（原理）。

| 问题类型 | 答题模板 |
|----------|----------|
| 为什么粉碎？ | 增大固体与液体的接触面积，加快反应速率，提高浸取率 |
| 为什么控制温度在X℃？ | 低于X℃反应速率慢；高于X℃XX会分解/挥发/促进副反应 |
| 为什么调节pH到X？ | 使Fe³⁺（或其他杂质）完全沉淀，而目标离子仍留在溶液中 |
| 为什么加H₂O₂？ | 将Fe²⁺氧化为Fe³⁺，便于后续调节pH沉淀除去，且H₂O₂产物为水，不引入新杂质 |
| 为什么用XX调节pH？ | 消耗H⁺，使pH升高，且不引入新杂质（如CuO反应生成Cu²⁺，即目标离子） |

#### 6.2 "方程式书写"类

- 注意：看清是**离子方程式**还是**化学方程式**
- 酸性环境：可补H⁺和H₂O；碱性环境：可补OH⁻和H₂O
- 配平：先电子守恒，再电荷守恒，最后原子守恒

#### 6.3 "计算"类

- Ksp计算：注意离子浓度是否乘以系数幂次
- 产率计算：产率 = (实际产量/理论产量) × 100%
- 纯度计算：纯度 = (纯物质质量/样品总质量) × 100%

### 七、2024广东卷第18题拆解（示范）

> **题目背景**：从多金属精矿中提取Fe、Cu、Ni，涉及"酸浸→高压加热→萃取→电解"流程。

| 小题 | 考查点 | 涉及原理 |
|------|--------|----------|
| (1)提高浸取速率 | 反应速率影响因素 | 温度、浓度、接触面积 |
| (2)写离子方程式 | Fe²⁺氧化为Fe₂O₃ | 氧化还原 + 方程式配平 |
| (3)调节pH分离 | Fe³⁺、Al³⁺、Cu²⁺、Ni²⁺ | Ksp + 沉淀pH控制 |
| (4)绿色化学思想 | 新型冶铁与传统高炉对比 | 环保、节能、原子经济 |

## 备注

- **广东卷特色**：工艺流程题是广东化学的"定海神针"，第18题固定位置、固定分值。近年趋向"真实工业流程+绿色化学理念"，如2024题涉及"新型冶铁方法与传统高炉对比"，体现绿色化学思想。
- **易错点**：
  1. "不引入新杂质"是选择试剂的重要原则，如除去Cu²⁺溶液中的Fe³⁺，用CuO或Cu(OH)₂调pH，不能用NaOH（引入Na⁺）
  2. 趁热过滤 vs 冷却过滤：防止目标物质析出或促进杂质析出
  3. Ksp计算时注意离子浓度的系数幂次（如Fe(OH)₃的Ksp = [Fe³⁺][OH⁻]³）
- **等级赋分提示**：工艺流程题是化学的压轴题型，分值高但步骤多。建议训练"分步得分"意识，即使不能完全解出，写出每步的原理和分析也能拿部分分。
- **推荐练习**：2023广东化学第18题、2024广东化学第18题、2024广州一模工艺流程题。
