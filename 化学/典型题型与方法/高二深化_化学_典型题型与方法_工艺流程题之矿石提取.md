## 核心内容

### 题型识别标志
> **看到什么条件 → 立刻想到什么方法/模型**

| 题干关键条件 | 识别为 | 首选方法 |
|-------------|--------|----------|
| "从某矿石/硝酸浸取液中提取金属Ni、Co" | 矿石提取类 | 先圈出核心金属与杂质元素，标注各步价态变化 |
| "氨浸/酸浸/碱浸" | 浸取方式判断 | 判断是否配位溶解（NH₃配位Ni²⁺/Co²⁺）或酸碱溶解 |
| "调pH除杂" + 沉淀pH表 | 氢氧化物沉淀分离 | 查Ksp/沉淀pH表，写离子方程式控制pH区间 |
| "配离子 [M(NH₃)₆]²⁺" | 配位选择性浸出 | 利用配合物稳定性让核心金属留溶液、杂质沉淀 |
| "析晶/热解/结晶纯化" | 产品分离与试剂循环 | 通HCl气体促结晶、热解再生试剂（如HNO₃、MgO） |
| "立方晶胞/体心/最近O个数" | 流程中嵌入结构化学 | 晶胞粒子计数与配位数计算 |

### 解题路径（矿石提取四步法）

#### 第一步：粗读流程图，锁定"核心元素"与"杂质元素"
- 核心金属=目标产品（如Ni、Co）；杂质=Fe、Al、Mg等。
- 标注每一步操作：焙烧→浸取→调pH→沉杂→析晶→热解。

#### 第二步：分析核心元素价态，判断是否需氧化还原
- 若核心金属以高价态存在（如Co³⁺），常用还原剂（SO₃²⁻、Fe²⁺）还原为易溶低价（Co²⁺）。
- 写方程式时配平电子转移，注意介质（碱性/氨性用NH₃·H₂O、OH⁻）。

#### 第三步：除杂——依据沉淀pH表控制分离
- 调pH使杂质氢氧化物先沉淀（沉淀完全pH），核心金属因配位或pH未到而留在溶液。
- 区间写法：使杂质沉淀完全的pH ≤ 控制pH < 核心金属开始沉淀的pH。

#### 第四步：分离提纯 + 试剂循环
- 析晶：通入酸性气体（如HCl）增大同离子效应促进结晶。
- 热解：分解中间产物再生试剂（如Mg(NO₃)₂热解→MgO+HNO₃循环）。
- 化学方程式务必配平，注意配离子写法 [Co(NH₃)₆]²⁺。

### 母题（2023 广东选择性考试·第18题，14分）

> 2023年广东卷工艺流程题，以"氨浸工艺从矿石硝酸浸取液提取Ni、Co"为背景，融合氧化还原、配位、Ksp与晶胞。

**题目**（节选核心设问）：Ni、Co均是重要的战略性金属。从处理后的矿石硝酸浸取液（含 Ni²⁺、Co²⁺、Al³⁺、Mg²⁺）中，利用氨浸工艺可提取Ni、Co，并获得高附加值化工产品。工艺流程如下（活性MgO调pH→过滤得硝酸镁→结晶纯化→热解；氨浸→分离→析晶→…）。已知：氨性溶液由 NH₃·H₂O、(NH₄)₂SO₄、(NH₄)₂CO₃ 配制；Ni²⁺、Co²⁺、Co³⁺ 与 NH₃ 形成可溶配离子；Co(OH)₃ 易被空气氧化为 Co(OH)₂；部分氢氧化物Ksp见下表。

| 氢氧化物 | Ksp |
|---------|-----|
| Co(OH)₂ | 10⁻¹⁵.⁹ |
| Co(OH)₃ | 10⁻⁴⁴.¹ |
| Ni(OH)₂ | 10⁻¹⁶.⁵ |
| Al(OH)₃ | 10⁻³³.¹ |
| Mg(OH)₂ | 10⁻¹².⁵ |

(1) 活性MgO可与水反应，化学方程式为 ______ 。
(2) 常温下，pH=9.9的氨性溶液中，c(NH₃·H₂O) ______ c(NH₄⁺)（填">""<"或"="）。
(3) "氨浸"时，Co(OH)₃ 转化为 [Co(NH₃)₆]²⁺ 的离子方程式为 ______ 。
(4) (NH₄)₂CO₃ 使胶状物质转化为疏松棒状颗粒物，滤渣X射线衍射出现明锐衍射峰，说明NH₄Al(OH)₂CO₃属于 ______（晶体/非晶体）；其提高浸取速率的原因是 ______ 。
(5) ①"析晶"过程中通入的酸性气体A为 ______ ；②由AlₓCoₒᵧ可制备晶体，其立方晶胞如图，Co位于 ______ ，晶体中一个Co周围与其最近的O的个数为 ______ 。

**解**：

**(1)** 活性MgO与水化合：
$$\mathrm{MgO + H_2O = Mg(OH)_2}$$

**(2)** 氨性溶液：NH₃·H₂O ⇌ NH₄⁺ + OH⁻，$K_b = 10^{-4.7}$。pH=9.9 ⇒ pOH=4.1 ⇒ $[\mathrm{OH^-}] = 10^{-4.1}$。
$$\frac{c(\mathrm{NH_4^+})}{c(\mathrm{NH_3\cdot H_2O})} = \frac{K_b}{[\mathrm{OH^-}]} = \frac{10^{-4.7}}{10^{-4.1}} = 10^{-0.6} \approx 0.25 < 1$$
故 **c(NH₃·H₂O) > c(NH₄⁺)**，填 ">"。

**(3)** Co(OH)₃ 中Co为+3价，产物 [Co(NH₃)₆]²⁺ 中Co为+2价（被还原）；SO₃²⁻ 被氧化为 SO₄²⁻。配平（碱性/氨性介质）：
$$2\mathrm{Co(OH)_3} + 12\mathrm{NH_3\cdot H_2O} + \mathrm{SO_3^{2-}} = 2[\mathrm{Co(NH_3)_6}]^{2+} + \mathrm{SO_4^{2-}} + 13\mathrm{H_2O} + 4\mathrm{OH^-}$$
（电荷：左 −2，右 +4−2−4 = −2，配平✓）

**(4)** 出现明锐衍射峰 ⇒ **晶体**；(NH₄)₂CO₃ 将胶状物转为疏松棒状，减少胶状物质对Ni、Co氧化物的包裹，增大滤泥与氨性溶液接触面积 ⇒ 浸取速率提高。

**(5)** ① 析晶通入HCl增大Cl⁻同离子效应促NiCl₂结晶 ⇒ **HCl**；② 立方晶胞中Co原子个数为1，位于 **体心**；一个Co周围最近O的个数为 **12**。

> 💡 **关键技巧**：矿石提取题"调pH除杂"必须依据沉淀pH/Ksp表确定区间；配位浸出（NH₃配Ni²⁺/Co²⁺）是实现选择性分离的核心，方程式配平别漏NH₃配体。
