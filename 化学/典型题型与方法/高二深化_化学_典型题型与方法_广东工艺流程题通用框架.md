## 核心内容

### 一、题型识别标志

> 看到"矿石/废催化剂/海水/稀土 → 酸浸/碱浸 → 除杂 → 沉淀/电解 → 产品"的箭头流程图 → 立刻识别为工艺流程题。

| 题干关键特征 | 识别为 | 首选切入 |
|-------------|--------|----------|
| "焙烧/煅烧/酸浸/碱浸" | 原料预处理 | 写出溶解反应，关注价态变化 |
| "调pH除杂/沉X" | 沉淀分离 | 用Ksp与沉淀pH判断 |
| "加入氧化剂/还原剂目的" | 氧化还原 | 把某元素氧化/还原到指定价态 |
| "电解/电沉积" | 电化学分离 | 电极反应 + 电子守恒 |
| "循环物质/滤液回用" | 流程优化 | 找可重复利用的试剂 |

### 二、通用解题框架（四步通法）

```
Step 1：读题干 → 锁定最终产品、原料成分、目标元素（常是金属）
Step 2：析流程 → 每一步"进什么 → 发生什么反应 → 出什么"
Step 3：想原理 → 速率 / 平衡 / 沉淀(Ksp) / 氧化还原 / 电化学
Step 4：答问题 → 用化学术语，分点规范表述（目的/原因/方程式/计算）
```

### 三、流程核心反应类型速查

| 类型 | 常见考法 | 关键判断 |
|------|----------|----------|
| 溶解 | 酸浸/碱浸 | 金属氧化物+酸→盐；两性氧化物+碱→盐 |
| 氧化还原 | 加氧化剂除Fe²⁺ | 强制弱、价态规律 |
| 调节pH沉淀 | 除Fe³⁺/Al³⁺ | Ksp + 离子浓度<10⁻⁵视为完全 |
| 沉淀转化 | BaMoO₄ vs BaCO₃ | Ksp小的优先沉淀，需算共存条件 |
| 电解 | 电沉积金属 | 阴极还原，电子守恒 |

### 四、答题规范模板

| 问题类型 | 答题模板 |
|----------|----------|
| 为什么粉碎？ | 增大接触面积，加快反应速率，提高浸取率 |
| 为什么加H₂O₂？ | 将Fe²⁺氧化为Fe³⁺便于沉淀除去，且产物为水不引入新杂质 |
| 为什么调pH到X？ | 使杂质离子完全沉淀，而目标离子仍留在溶液中 |
| 用XX调pH？ | 消耗H⁺且不引入新杂质（如CuO反应生成目标离子Cu²⁺） |

### 五、真题母题（2021 广东选择性考试·第18题，节选）

> 广东卷以"废催化剂(含Al、Mo、Ni氧化物)回收"考查流程中的沉淀分离与Ksp控制。

**已知**：25℃时 Ksp(BaMoO₄)=3.5×10⁻⁸，Ksp(BaCO₃)=2.6×10⁻⁹；H₂CO₃的 Ka₂=4.7×10⁻¹¹；pH>6.0时Mo以MoO₄²⁻存在。

(3)"沉钼"中pH=7.0。②若控制不当BaCO₃也会沉淀。为避免BaMoO₄中混入BaCO₃，溶液中 c(HCO₃⁻):c(MoO₄²⁻)= ______（列出算式）时，应停止加入BaCl₂溶液。

**解**：
- 刚析出BaMoO₄时：c(Ba²⁺)=Ksp(BaMoO₄)/c(MoO₄²⁻)
- 不析出BaCO₃需：c(Ba²⁺)·c(CO₃²⁻) ≤ Ksp(BaCO₃)
  → c(CO₃²⁻) ≤ Ksp(BaCO₃)·c(MoO₄²⁻)/Ksp(BaMoO₄)
- 由 Ka₂=c(CO₃²⁻)·c(H⁺)/c(HCO₃⁻)，pH=7.0时 c(H⁺)=10⁻⁷
  → c(HCO₃⁻)/c(MoO₄²⁻) ≤ Ksp(BaCO₃)·c(H⁺) / [Ksp(BaMoO₄)·Ka₂]

**答**：$\dfrac{2.6\times10^{-9}\times10^{-7}}{3.5\times10^{-8}\times4.7\times10^{-11}}$ （与官方答案一致）

> 💡 技巧：沉钼与沉碳酸钡是"Ksp竞争沉淀"，停止加BaCl₂的临界条件 = 两沉淀刚共存；先不急于代数字，先把不等式列对。
