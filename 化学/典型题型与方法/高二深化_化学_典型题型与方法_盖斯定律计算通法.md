## 核心内容

### 一、题型识别标志

| 题干关键条件 | 识别为 | 首选方法 |
|-------------|--------|----------|
| "已知①②③反应，求目标反应ΔH" | 盖斯定律方程组 | 倒转变号、乘系数、加和消去 |
| "反应历程/能量变化图求总焓变" | 盖斯定律图像法 | 分步焓变相加：ΔH总=Σ各步ΔH |
| "键能数据求ΔH" | 键能法 | ΔH=Σ反应物键能−Σ生成物键能 |
| "正逆反应活化能求ΔH" | 活化能法 | ΔH=Eₐ(正)−Eₐ(逆) |

### 二、核心公式

```
盖斯定律：ΔH只与始态和终态有关，与途径无关
  若 ③ = ① + ②，则 ΔH₃ = ΔH₁ + ΔH₂
  若 ③ = ① − ②，则 ΔH₃ = ΔH₁ − ΔH₂
  若 ③ = n×①，则 ΔH₃ = n×ΔH₁

键能法：ΔH = Σ(反应物键能) − Σ(生成物键能)
图像法：ΔH = E(生成物总能量) − E(反应物总能量) = Eₐ(正) − Eₐ(逆)
```
> 注意事项：① 热化学方程式须标状态(s/l/g/aq)；② ΔH带符号，放热为负；③ 系数变n倍，ΔH变n倍；④ 方程式颠倒，ΔH变号。

### 三、标准化解题路径（方程组四步法）

```
Step 1：写目标 → 写出目标反应的化学方程式（状态标全）
Step 2：选已知 → 从已知反应中找出含目标物质与中间产物的式子
Step 3：调系数 → 颠倒(ΔH变号)/乘n(ΔH等倍)，使中间产物相消
Step 4：加和算 → 方程式相加消去中间物，ΔH对应代数相加
```

> 图像法提示：图像法本质"分段焓变代数和"，按 E(终)−E(始) 取符号；反应物能级在上、生成物在下为放热(ΔH<0)。

### 四、真题母题（2021 广东选择性考试·第19题，节选）

> 广东卷以"CH₄与CO₂重整"反应体系考查盖斯定律代数式书写（碳中和情境）。

**题目**：该重整反应体系主要涉及以下反应：
(a) CH₄(g)+CO₂(g)=2CO(g)+2H₂(g)  ΔH₁
(b) CO₂(g)+H₂(g)=CO(g)+H₂O(g)  ΔH₂
(c) CH₄(g)=C(s)+2H₂(g)  ΔH₃
(d) 2CO(g)=CO₂(g)+C(s)  ΔH₄
(e) CO(g)+H₂(g)=H₂O(g)+C(s)  ΔH₅
(1)根据盖斯定律，反应a的ΔH₁ = ______（写出一个代数式即可）。

**解**：
- 观察：a = b + c − e 的变形，或直接 a = c − d（因 d: 2CO→CO₂+C，则 c−d: CH₄−[2CO→CO₂+C] → CH₄+CO₂→2CO+2H₂ = a）
  → ΔH₁ = ΔH₃ − ΔH₄
- 另一组：a = b + c − e  → ΔH₁ = ΔH₂ + ΔH₃ − ΔH₅
- 官方答案给出两种等价式：**ΔH₁ = ΔH₂ + ΔH₃ − ΔH₅** 或 **ΔH₁ = ΔH₃ − ΔH₄**

**答**：ΔH₂+ΔH₃−ΔH₅（或 ΔH₃−ΔH₄）

> 💡 关键技巧：盖斯定律"拼图法"——把目标反应当作拼图，用已知反应正着加、倒着减，消去所有中间产物(C、CO、H₂O等)即可；不必死记，先消C(s)最稳。

## 备注

- 广东卷盖斯定律常以"反应体系多反应+能量图"呈现，重点不是算数值而是"列代数式"，务必写全状态。
- 易错：忘记方程式颠倒时ΔH变号；系数乘n时ΔH未同步乘n；把中间产物漏消。
- 等级赋分提示：反应原理题第(1)问是"稳定得分区"，背熟四步法可保底拿分。
