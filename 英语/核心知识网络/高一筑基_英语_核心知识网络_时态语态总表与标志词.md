## 核心内容

> **本质理解**：时态 = 时间（现在/过去/将来）× 状态（一般/进行/完成/完成进行）。做题先找**时间标志词**，再看**语境动作关系**，两步定位。

---

### 一、8大核心时态构成总表

| 时态 | 主动构成 | 被动构成 | 基本用法 | 例句 |
|------|----------|----------|----------|------|
| **一般现在时** | do/does | am/is/are done | 习惯、真理、客观事实 | She **teaches** English. / English **is taught** in China. |
| **一般过去时** | did | was/were done | 过去发生的动作（与现在无关） | He **visited** Beijing last year. / The bridge **was built** in 1990. |
| **一般将来时** | will do / be going to do | will be done | 将来要发生的动作 | I **will call** you tomorrow. / The meeting **will be held** next week. |
| **现在进行时** | am/is/are doing | am/is/are being done | 此刻正在进行；近期安排 | Look! They **are playing** football. / A new library **is being built**. |
| **过去进行时** | was/were doing | was/were being done | 过去某时正在进行；叙事背景 | I **was reading** when he came in. / The road **was being repaired** then. |
| **现在完成时** | have/has done | have/has been done | 过去动作对现在的影响；持续至今 | I **have finished** my homework. / Many trees **have been planted**. |
| **过去完成时** | had done | had been done | "过去的过去"，先于另一过去动作 | The train **had left** before I arrived. / The work **had been done** by then. |
| **现在完成进行时** | have/has been doing | （极少用被动） | 从过去持续到现在并仍在继续 | She **has been learning** piano for ten years. |

> **记忆口诀**："一般进行完成三状态，现在过去将来三时间；三三得九减将来完成进行，高考核心八大件。"

### 二、时间标志词速查表（做题第一线索）

| 时态 | 高频标志词 | 语法填空典型信号 |
|------|-----------|------------------|
| **一般现在时** | always, usually, often, every day, on Sundays | 客观真理、格言、说明文陈述事实 |
| **一般过去时** | yesterday, last week, ...ago, in 2019, just now, the other day | 句中出现具体过去时间；叙事情节推进 |
| **一般将来时** | tomorrow, next week, soon, in the future, in + 一段时间 | 计划、预测、安排 |
| **现在进行时** | now, right now, at the moment, look, listen | Look! Listen! 开头必用 |
| **过去进行时** | at that time/moment, at 8 last night, when/while 引导的过去背景 | "when + 一般过去时, 主句过去进行时"结构 |
| **现在完成时** | already, yet, just, ever, never, so far, up to now, recently, since..., for..., in the past few years | since/for 几乎是送分信号 |
| **过去完成时** | by/before + 过去时间, by the end of last year | 主句一般过去时，从句动作更早 → had done |
| **现在完成进行时** | for + 时段, since..., all day, these days | 强调"一直在做"，常带抱怨/赞叹语气 |

> **避坑**：标志词只是线索，不是铁律。"for three years" 也可配一般过去时（He lived here for three years. 现已搬走），最终看语境是否延续到现在。

### 三、高频易混时态对比

#### 1. 一般过去时 vs 现在完成时（高考最爱考）

| 对比项 | 一般过去时 | 现在完成时 |
|--------|-----------|-----------|
| 与现在的关系 | 无关，纯过去 | 有影响/延续到现在 |
| 时间状语 | 具体过去时间（ago, in 2020） | 模糊/延续时间（already, since, so far） |
| 例句 | I **saw** the film yesterday.（只说看过这件事） | I **have seen** the film.（现在知道内容了） |

#### 2. 过去完成时的"跳板原则"

> 过去完成时不能单独存在，必须有**一般过去时作参照**。口诀："没有过去，哪有过去的过去？"

- When I got to the station, the train **had left**.（got 是跳板）
- By the end of last term, we **had learned** 2,000 words.（by + 过去时间 = 隐形跳板）

#### 3. 现在完成时 vs 现在完成进行时

| 对比项 | 现在完成时 | 现在完成进行时 |
|--------|-----------|----------------|
| 侧重点 | 结果（做完/做过多少） | 过程（一直在做） |
| 例句 | I **have written** three letters.（成果：三封） | I **have been writing** letters all morning.（过程：一上午） |

### 四、被动语态：构成与特例

#### 1. 构成公式

```
被动语态 = be + 过去分词（be 随时态/人称变化，done 永远不变）
```

| 时态 | 被动结构 | 例句 |
|------|----------|------|
| 一般现在 | am/is/are done | The classroom **is cleaned** every day. |
| 一般过去 | was/were done | The window **was broken** yesterday. |
| 一般将来 | will be done / be going to be done | More roads **will be built** next year. |
| 现在进行 | am/is/are being done | The problem **is being discussed** now. |
| 过去进行 | was/were being done | The patient **was being operated** on at that time. |
| 现在完成 | have/has been done | The task **has been completed**. |
| 过去完成 | had been done | The money **had been stolen** before the police came. |
| 含情态动词 | 情态动词 + be done | The rules must **be obeyed**. |

#### 2. 被动语态六大特例（语法填空陷阱区）

| 特例 | 规则 | 例句 |
|------|------|------|
| **主动表被动①** | 系动词（look, sound, taste, smell, feel）无被动 | The soup **tastes** good.（≠ is tasted） |
| **主动表被动②** | sell, read, wash, write, open 等 + well/easily 表性质 | The book **sells** well. / The pen **writes** smoothly. |
| **主动表被动③** | need/want/require + doing = to be done | The room needs **cleaning** = needs **to be cleaned**. |
| **主动表被动④** | be worth doing | The film is worth **seeing** twice. |
| **无被动①** | 不及物动词/短语：happen, take place, occur, break out, belong to, come out | Great changes **have taken place**.（≠ have been taken place） |
| **无被动②** | 表度量的动词：weigh, measure, cost, last | The bag **weighs** 5 kilos. |

#### 3. 双宾语与短语动词的被动

- 双宾语：He gave **me** a book. → **I was given** a book. / **A book was given** to me.（介词 to/for 不可丢）
- 短语动词：They laughed **at** him. → He **was laughed at**.（介词 at 必须保留）

### 五、做题三步法（语法填空时态语态题）

1. **判语态**：主语是动作的发出者还是承受者？承受者 → 被动。
2. **定时态**：找标志词 → 无标志词看上下文谓语动词的"时态统一战线"（叙事文=过去时家族，说明文=现在时家族）。
3. **定主谓一致**：主语单复数决定 be/have 形式（there be、倒装句最易错）。

> **真题风格示例**（语法填空）：
> - So far, the program ______ (help) thousands of children. → **has helped**（so far → 现在完成时）
> - When the fire broke out, most residents ______ (sleep). → **were sleeping**（过去某时正在进行）
> - The museum, which ______ (build) in the Ming Dynasty, attracts many visitors. → **was built**（过去 + 被动）

---

### 六、避坑清单

- ❌ 瞬间动词（die, leave, join, buy）+ for/since：He has died for three years. → 改为 **has been dead** for three years。
- ❌ have gone to（去了未回）与 have been to（去过已回）混用。
- ❌ 主将从现遗忘：时间/条件状语从句用一般现在时表将来 —— I will call you when he **comes** back.（≠ will come）
- ❌ 读后续写中时态漂移：开头过去时，写到激动处跳回现在时，全文时态混乱直接降档。
- ❌ 被动语态漏 be 或错用过去式：It was happened. → **It happened.**（happen 无被动）

## 备注

- 时态语态是英语语法的"地基中的地基"，高一上学期必须过关，否则高二语法填空和读后续写会持续失分。
- 新高考Ⅰ卷语法填空的时态题越来越少给明显标志词，更强调**语境判断**——做完后把句子放回段落，检查与上下句谓语动词是否同一"时间家族"。
- 广东听说考试Part A模仿朗读虽不考时态产出，但Part C复述时必须自觉用过去时，本卡第二部分标志词表同样适用。
