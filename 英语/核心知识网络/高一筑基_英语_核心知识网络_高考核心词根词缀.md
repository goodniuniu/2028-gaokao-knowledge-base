## 核心内容

> **本质理解**：英语单词 = 前缀（改词义）+ 词根（定词义）+ 后缀（定词性）。掌握约50个高频词根词缀，阅读中生词猜测正确率可从30%提升到70%以上，词汇量"一生二、二生三"式扩展。

### 一、高频前缀（改变词义）

#### 1. 否定类前缀（最重要，阅读态度题命门）

| 前缀 | 含义 | 高考词例 |
|------|------|----------|
| un- | 不、无 | unfair, unbelievable, unforgettable, unemployment |
| in-/im-/il-/ir- | 不（按词根首字母变形） | incorrect, impossible, illegal, irregular |
| dis- | 不、相反 | disagree, disappear, dishonest, disadvantage |
| non- | 非、无 | non-smoking, nonfiction, nonstop |
| mis- | 错误地 | misunderstand, mislead, misjudge |
| anti- | 反对、抗 | anti-war, antibody |

> **阅读应用**：看到含否定前缀的态度词，先翻转词义再判断作者态度——"The results were **unconvincing**." = 结果不能令人信服（负态度）。

### 四、生词猜测实战四步法（阅读词义猜测题）

1. **拆构词**：找熟悉的前缀/词根/后缀 → unpredictable = un-(不) + predict(预测) + -able(可……的) = 不可预测的。
2. **看上下文逻辑**：同义复现（or, that is）、反义对比（but, unlike, while）、因果解释（because, so）、举例（such as）。
3. **代入验证**：把猜出的词义代回原文，读通即可。
4. **排除干扰**：选项中与熟词"望文生义"的字面义多为陷阱（如 book 在文中可能作动词"预订"）。

> **真题风格示例**：The disease is largely **preventable** through vaccination. → prevent(预防) + -able(能够……的) = 可预防的。

### 五、避坑清单

- ❌ in- 只当否定：in- 还有"向内"义（import, include），看词根判断。
- ❌ -ly 一律当副词：friendly, lonely, lovely, lively, costly, daily 是形容词。
- ❌ hard/hardly 混淆：hardly = 几乎不（否定副词），不是"努力地"。
- ❌ 词性转换题漏双写/去e：true → truly；wide → width；happy → happiness；begin → beginning。
- ❌ 靠背词根猜一切：词根猜测是辅助手段，必须代入语境验证，约20%派生词词义已引申（如 respect 不止"再看一眼"，已固化为"尊敬"）。

### 六、训练计划（8周，每天10分钟）

| 周次 | 任务 | 检验标准 |
|------|------|----------|
| 第1-2周 | 背诵否定类+方向类前缀（约25个），每个前缀记3个例词 | 随机抽查能说出词义+2个例词 |
| 第3-4周 | 背诵四类后缀，结合语法填空词性转换题训练 | 词性判断正确率≥90% |
| 第5-7周 | 每周背10个词根，用"词根+前后缀"自行组词 | 30个词根全掌握 |
| 第8周 | 每天精读1段高考阅读，圈出所有派生词并拆解 | 养成"见生词先拆词"的习惯 |

## 备注

- 词根词缀是"性价比最高"的词汇学习方法：记住 spect 一个词根，inspect/respect/expect/prospect 一串词全部打通，且终身受用（大学四六级、考研同样适用）。
