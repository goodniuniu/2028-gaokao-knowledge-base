# 📈 2028广东高考650分知识库完善计划

> **制定时间**：2025年6月
> **校准时间**：2026年7月20日（阶段日期按2028高考学制顺延修正，现状诊断更新为全库审计数据）
> **目标**：2028年广东新高考总分650+（全省前1%-2%）
> **当前阶段**：高一结束 → 高二深化起步
> **策略**：以"高频考点优先、广东特色优先、薄弱环节优先"为原则，分阶段、有节奏地完善知识库

---

## 一、现状诊断总结

| 维度 | 诊断结果 |
|------|----------|
| **卡片总量** | 2026-07审计：**92张**，目标200+张，缺口约 **54%** |
| **掌握状态** | 92张中91张⚠️待强化、1张❌未理解、**0张✅已掌握**，实际掌握度仍未破零 |
| **四维度结构** | 仍失衡：易错警示仅3张（数学2+物理1），语文/英语/化学/生物空白；素材拓展仅数学1张+语文2张 |
| **广东特色** | 含广东来源卡片已达80张，但广东特色专项标签卡（情境/工艺/长句/思辨/听说）仍需扩充 |
| **方法体系** | 6张方法卡片有基础，仍缺错题管理、时间管理、等级赋分专项策略 |

## 三、分阶段完善路线图

### 第一阶段：高二上（2026.9-2027.1）—— 核心知识筑基

**目标**：新增约80张卡片，总卡片数达110+

#### 数学（新增20张，累计37张）
- [ ] 核心知识网络（6张）：数列（等差/等比/递推）、直线与圆、椭圆/双曲线/抛物线定义、导数基础（求导法则/极值）、排列组合基础、二项式定理
- [ ] 典型题型与方法（6张）：数列求和通法（裂项/错位）、解析几何韦达定理联立模板、导数含参讨论通法、排列组合计数策略、新高考Ⅰ卷开放性题目通法、选填速解策略
- [ ] 易错警示与辨析（4张）：数列n=1验证、等比q=1讨论、解析几何焦点位置不确定、导数定义域端点
- [ ] 素材与拓展（4张）：数学思想（数形结合/分类讨论）、竞赛思维（高考压轴中的竞赛背景）、数学建模案例、新高考Ⅰ卷创新题型收集

#### 语文（新增20张，累计26张）
- [ ] 核心知识网络（6张）：文言文虚词体系（之/其/而/以/于/则）、古诗词术语分类（意象/意境/手法/情感）、现代文阅读术语体系、语言文字运用病句类型、作文结构模型（并列/递进/对比）、论述类文本论证方法
- [ ] 典型题型与方法（6张）：文言文断句通法、古诗词情感分析三步法、小说阅读人物形象分析、信息类文本图文转换、作文审题立意速判法、语言文字运用情境化题通法
- [ ] 易错警示与辨析（4张）：常见成语误用（第一辑）、文言文实词一词多义、病句搭配不当与成分残缺、作文偏题常见原因
- [ ] 素材与拓展（4张）：人物素材（时代楷模）、科技素材（AI/新能源/生物医药）、文化素材（传统与当代）、金句素材分类

#### 英语（新增20张，累计22张）
- [ ] 核心知识网络（6张）：语法时态语态总表、定语/名词/状语从句连词、非谓语功能对比、特殊句式（倒装/强调/虚拟）、高考3500核心词根词缀、高级句型结构
- [ ] 典型题型与方法（6张）：阅读理解四题型通法、七选五逻辑衔接法、完形填空语境线索、语法填空无提示词口诀、应用文模板（书信/通知/演讲稿）、读后续写情节构建
- [ ] 易错警示与辨析（4张）：易混动词（rise/raise/lie/lay）、介词搭配陷阱、时态语境辨析、定语从句vs同位语从句
- [ ] 素材与拓展（4张）：主题词汇（环保/科技/教育/文化/健康）、高级替换表达、名言积累、外刊精选

#### 物理（新增12张，累计14张）
- [ ] 核心知识网络（6张）：电场公式体系、磁场公式体系、电磁感应定律、动量定理与守恒、功和能关系、热学/光学/原子物理选修
- [ ] 典型题型与方法（3张）：带电粒子在电场中运动、带电粒子在磁场中圆周运动、电磁感应单杆/双杆导轨
- [ ] 易错警示与辨析（2张）：电场线方向与电势关系、电磁感应感应电量与电流区别
- [ ] 素材与拓展（1张）：广东科技素材（华为5G/比亚迪/大疆/港珠澳大桥）

#### 化学（新增12张，累计14张）
- [ ] 核心知识网络（6张）：化学计量体系、元素周期律（位构性）、反应速率与平衡、水溶液离子平衡、电化学体系、有机化学官能团
- [ ] 典型题型与方法（3张）：化学平衡图像分析、电化学电极反应式书写、有机推断官能团特征反应
- [ ] 易错警示与辨析（2张）：阿伏伽德罗常数八大陷阱、离子反应少量/过量问题
- [ ] 素材与拓展（1张）：广东工业素材（茂名石化/珠海盐化工/粤北稀土）

#### 生物（新增12张，累计14张）

- [ ] 素材与拓展（1张）：广东科技素材（生物医药/农业育种/海洋生态）

#### 方法（新增5张，累计10张）
