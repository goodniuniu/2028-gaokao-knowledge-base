# 📝 知识卡片标准模板

> 每条知识卡片必须严格按以下格式编写，确保信息完整、检索高效、贴合广东考情。

---

## 模板格式

```markdown
# 【卡片标题】知识点的精准命名

| 字段 | 内容 |
|------|------|
| **来源** | 教材章节 / 真题年份 / 试卷类型 / 题号 |
| **时间标签** | #高一筑基 / #高二深化 / #高三冲刺 |
| **难度** | ★☆☆☆☆ ~ ★★★★★ |
| **状态** | ✅已掌握 / ⚠️待强化 / ❌未理解 |
| **试卷来源** | #新高考Ⅰ卷·广东 / #广东选择性考试 / 无 |
| **广东考情** | 该知识点在广东卷中的考查频率、难度、特色说明 |

---

- 概念2：...
```

## 标签体系说明

### 1. 科目标签（6个）
`#语文` `#数学` `#英语` `#物理` `#化学` `#生物`

### 2. 维度标签（4个）
`#核心知识网络` — 概念、公式、定理的体系化整理
`#典型题型与方法` — 高考真题、名校模拟的题型归纳与通法提炼
`#易错警示与辨析` — 常见错误、易混概念、审题陷阱
`#素材与拓展` — 作文素材、前沿科技、实验案例、文化常识

### 3. 时间标签（3个）
`#高一筑基` — 2025.9-2026.7，教材同步、基础概念
`#高二深化` — 2026.9-2027.7，专题突破、题型归纳
`#高三冲刺` — 2027.9-2028.5，综合卷、查漏补缺、应试技巧

### 4. 难度标签（5级）
`#难度1` ★☆☆☆☆ — 基础识记
`#难度2` ★★☆☆☆ — 理解应用
`#难度3` ★★★☆☆ — 综合运用
`#难度4` ★★★★☆ — 复杂分析
`#难度5` ★★★★★ — 压轴创新

### 5. 状态标签（3个）
`#已掌握` ✅ — 完全理解，能独立解题
`#待强化` ⚠️ — 基本理解，偶有失误
`#未理解` ❌ — 概念不清，需要重新学习

### 6. 试卷来源标签（必须标注）

#### 语数英（新高考Ⅰ卷）
`#新高考Ⅰ卷·广东` — 广东考区真题
`#新高考Ⅰ卷·江苏` — 江苏考区真题（教育强省）
`#新高考Ⅰ卷·山东` — 山东考区真题
`#新高考Ⅰ卷·福建` — 福建考区真题
`#新高考Ⅰ卷·河北` — 河北考区真题
`#新高考Ⅰ卷·湖北` — 湖北考区真题
`#新高考Ⅰ卷·湖南` — 湖南考区真题

> **语数英收录原则**：优先广东卷，其次兼顾江苏、山东等教育强省优质题目。

#### 物化生（广东省卷）
`#广东选择性考试` — 广东省自主命题真题
`#广州一模` — 广州名校模拟题
`#深圳一模` — 深圳名校模拟题
`#佛山一模` — 佛山名校模拟题
`#广东名校模拟` — 其他省内名校模拟题

> **物化生收录原则**：优先广东省卷真题，其次广州/深圳/佛山等省内名校模拟题。

### 7. 广东特色标签（物化生重点关注）
`#广东特色` — 广东卷情境化命题、生活化素材、科技前沿融合
`#广东情境` — 物理情境化命题（如生活场景、科技应用）
`#广东工艺` — 化学工艺流程类题目（如矿石提取、回收利用）
`#广东长句` — 生物长句表达类题目（原因类、依据类、措施类）
`#广东听说` — 英语人机对话听说考试专项
`#广东思辨` — 语文作文思辨性、批判性思维

## 广东考情标注规范

每条知识卡片的「广东考情」字段必须包含以下信息：

| 维度 | 说明 | 示例 |
|------|------|------|
| **考查频率** | 高频/中频/低频/未考 | 高频（近5年广东卷4次考查） |
| **难度定位** | 基础/中档/压轴 | 中档（计算量适中，物理模型较新） |
| **特色描述** | 广东卷的命题特点 | 常以"生活情境+科技前沿"为背景，需从情境中提取物理模型 |
| **赋分提示** | 等级赋分制下的注意点（仅物化生） | 原始分差距在赋分后可能放大，需确保基础题不丢分 |

## 文件命名规范

```
[时间标签]_[科目]_[维度]_[知识点名称].md
```

**示例：**
- `高一筑基_数学_核心知识网络_三角函数诱导公式.md`
- `高二深化_物理_典型题型与方法_板块模型通法.md`
- `高三冲刺_化学_易错警示与辨析_阿伏伽德罗常数常见陷阱.md`
- `高一筑基_语文_素材与拓展_人物素材_张桂梅.md`
- `高三冲刺_英语_典型题型与方法_广东听说考试题型通法.md`
- `高二深化_生物_典型题型与方法_广东长句表达答题模板.md`

> 命名规则确保按文件名排序时，同阶段同科目的卡片会聚集在一起，便于浏览。
