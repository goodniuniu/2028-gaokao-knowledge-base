# ⚙️ 使用规则与检索说明

> 本文件定义知识库的使用规范、检索方式、入库规则及维护规则，请在使用前仔细阅读。

---

## 一、广东高考背景速查

| 科目 | 试卷类型 | 命题方式 | 特殊说明 |
|------|----------|----------|----------|
| **语文** | 新高考Ⅰ卷 | 教育部统一命题 | 思辨性强、贴近时代 |
| **数学** | 新高考Ⅰ卷 | 教育部统一命题 | 综合性强、创新思维 |
| **英语** | 新高考Ⅰ卷 | 教育部统一命题 | 笔试不含听力；广东单独听说考试（人机对话，20分） |
| **物理** | 广东省卷 | 自主命题 | 等级赋分，情境化命题 |
| **化学** | 广东省卷 | 自主命题 | 等级赋分，工艺流程、实验探究 |
| **生物** | 广东省卷 | 自主命题 | 等级赋分，长句表达、实验设计 |

> **核心差异**：语数英对标"新高考Ⅰ卷"（多省共用），物化生对标"广东卷"（自主命题，特色鲜明）。

## 二、入库规则（AI 执行）

当您提供具体学科内容时，AI 将按以下流程执行入库：

### 入库步骤
1. **识别科目**：从指令中提取目标科目（语/数/英/物/化/生）
2. **判断维度**：根据内容性质归入四个二级维度之一
   - 概念/公式/定理 → **核心知识网络**
   - 真题/模拟/题型 → **典型题型与方法**
   - 错误/陷阱/辨析 → **易错警示与辨析**
   - 素材/拓展/常识 → **素材与拓展**
3. **确定时间标签**：根据当前学习阶段或内容深度标注 `#高一筑基` / `#高二深化` / `#高三冲刺`
4. **评估难度**：按 ★-★★★★★ 标注难度等级
5. **标注试卷来源**：
   - **语数英**：标注 `#新高考Ⅰ卷·广东` 或 `#新高考Ⅰ卷·江苏/山东/福建` 等
   - **物化生**：标注 `#广东选择性考试` 或 `#广州一模` / `#深圳一模` 等
   - 教材同步内容标注具体教材章节
6. **填写广东考情**：说明该知识点在广东卷（或新高考Ⅰ卷广东考区）的考查频率、难度、特色
7. **生成卡片**：按 [01_知识卡片模板.md](01_知识卡片模板.md) 格式编写
8. **文件命名**：按 `[时间标签]_[科目]_[维度]_[知识点名称].md` 命名
9. **放置目录**：存入对应科目/维度的文件夹中
10. **更新索引**：在对应科目索引和总索引中更新卡片数量和状态统计

### 真题收录优先级（强制执行）

| 科目 | 第一优先级 | 第二优先级 | 第三优先级 |
|------|-----------|-----------|-----------|
| **语文** | 新高考Ⅰ卷·广东 | 新高考Ⅰ卷·江苏/山东 | 新高考Ⅰ卷·其他省份 |
| **数学** | 新高考Ⅰ卷·广东 | 新高考Ⅰ卷·江苏/山东 | 新高考Ⅰ卷·其他省份 |
| **英语** | 新高考Ⅰ卷·广东 + 广东听说 | 新高考Ⅰ卷·其他省份 | — |
| **物理** | 广东选择性考试 | 广州一模 | 深圳/佛山一模 |
| **化学** | 广东选择性考试 | 广州一模 | 深圳/佛山一模 |
| **生物** | 广东选择性考试 | 广州一模 | 深圳/佛山一模 |

> **广东特色标签**：物化生卡片如涉及广东卷的情境化命题、生活化素材、科技前沿融合，必须标注 `#广东特色`。

## 三、检索规则

### 快速检索（推荐）
对 AI 说出以下格式指令，即可快速检索：

```
查[科目]+[关键词]
```

**示例：**
| 指令 | 检索范围 | 返回内容 |
|------|----------|----------|
| `查数学+三角函数` | 数学科所有卡片 | 含"三角函数"的卡片列表及摘要 |
| `查物理+牛顿定律` | 物理科所有卡片 | 含"牛顿定律"的卡片列表及摘要 |
| `查语文+作文素材` | 语文科素材与拓展 | 作文素材类卡片 |
| `查化学+易错` | 化学科易错警示 | 易错警示类卡片 |
| `查英语+阅读理解` | 英语科典型题型 | 阅读理解题型卡片 |
| `查生物+必修一` | 生物科所有卡片 | 含"必修一"的卡片 |
| `查所有+高一筑基` | 全库所有卡片 | 高一筑基阶段的所有卡片 |
| `查所有+广东卷` | 全库所有卡片 | 标注"广东卷"来源的所有卡片 |
| `查物理+广东特色` | 物理科所有卡片 | 广东情境化命题相关卡片 |
| `查英语+广东听说` | 英语科所有卡片 | 英语听说考试专项卡片 |

### 高级检索
支持组合条件检索：

```
查[科目]+[维度]+[时间标签]+[关键词]
```

**示例：**
- `查数学+典型题型与方法+高一筑基+函数` — 数学高一阶段函数类题型卡片
- `查物理+易错警示与辨析+高二深化+电磁` — 物理高二电磁易错点
- `查所有+素材与拓展+高三冲刺+科技` — 高三科技类拓展素材
- `查化学+典型题型与方法+高三冲刺+广东工艺` — 化学高三工艺流程类卡片
- `查生物+典型题型与方法+高二深化+广东长句` — 生物高二长句表达类卡片

## 四、复盘规则

### 每周复盘流程
1. **打开** [复盘追踪/知识掌握状态表.md](复盘追踪/知识掌握状态表.md)
2. **浏览** 本周新增/复习的卡片
3. **标记状态**：每张卡片标记为 ✅已掌握 / ⚠️待强化 / ❌未理解
4. **关注广东特色**：重点检查 `#广东特色` 和 `#广东卷` 来源的卡片掌握情况
5. **更新统计**：在总索引中更新各科状态统计数字
6. **制定计划**：针对 ❌未理解 和 ⚠️待强化 的卡片制定下周复习计划
7. **英语听说**：英语科目额外关注广东听说考试专项卡片

### 状态标记规范
- **✅ 已掌握**：能独立、准确、快速完成相关题目，无知识盲区
- **⚠️ 待强化**：基本理解，但做题时偶有失误或速度较慢，需要定期复习
- **❌ 未理解**：概念模糊，无法独立解题，需要重新学习或请教老师

## 五、维护规则

### 卡片更新
- 发现卡片内容有误或需要补充 → 直接修改原文件，保留修改记录
- 同一知识点不同阶段有不同理解 → 创建新卡片，旧卡片标记为 `#历史版本`
- 关联卡片新增 → 在原卡片「关联卡片」部分追加链接
- 广东卷新真题发布后 → 及时补充 `#广东选择性考试` / `#新高考Ⅰ卷·广东` 来源卡片

### 定期整理（建议每月）
- [ ] 检查各科目文件夹是否有未分类的卡片
- [ ] 检查卡片标签是否完整（科目+维度+时间+难度+试卷来源+广东特色）
- [ ] 更新各科索引文件中的卡片清单
- [ ] 清理无效或重复的卡片
- [ ] 检查物化生卡片是否优先收录了广东卷及省内名校模拟题

## 六、常见问题

### Q1：如何添加新的知识卡片？
**A**：直接告诉 AI 具体内容，例如：
> "请入库：2025广东选择性考试·物理，第12题，关于电磁感应的导轨模型..."

AI 将自动按规则分类、标注试卷来源、填写广东考情、格式化并保存到正确位置。

### Q2：如何修改已有卡片？
**A**：告诉 AI：
> "请修改 [卡片标题]，将...改为..."

### Q3：如何查看某科所有卡片？
**A**：使用指令：
> "查[科目]+所有" 或 "列出[科目]的全部卡片"

### Q4：如何查看广东卷相关的所有卡片？
**A**：使用指令：
> "查所有+广东卷" 或 "查物理+广东特色"

### Q5：英语听说考试的内容放哪里？
**A**：英语听说考试内容统一归入英语科目的「典型题型与方法」或「易错警示与辨析」维度，必须标注 `#广东听说` 标签。

### Q6：复盘提醒如何关闭/调整？
**A**：告诉 AI 需要调整复盘频率或时间即可。

---

> 📌 **核心原则**：知识库是活的，不是一次性工程。贵在持续积累、定期复盘、动态更新。物化生优先广东卷，语数英兼顾新高考Ⅰ卷教育强省题目，关注广东特色命题风格。

## 七、标准使用命令模板

以下是与 AI 交互的**三大高频命令模板**，可直接复制使用：

### 模板一：知识点检索

```
查[科目]+[知识点关键词]
```

| 使用场景 | 示例指令 | AI 返回 |
|----------|----------|---------|
| 查具体知识点 | `查物理+传送带模型` | 物理科含"传送带"的卡片列表+核心公式摘要 |
| 查题型方法 | `查化学+工艺流程` | 化学科工艺流程相关卡片+通用解题框架 |
| 查广东特色 | `查生物+广东长句` | 生物科广东长句表达类卡片+答题模板 |
| 查阶段内容 | `查所有+高一筑基` | 全库高一筑基阶段的所有卡片清单 |
| 查试卷来源 | `查所有+广东卷` | 全库标注广东卷来源的所有卡片 |

---

### 模板二：知识整理入库

```
我要整理[科目]的[维度]，内容是：...
```

| 使用场景 | 示例指令 | AI 执行 |
|----------|----------|---------|
| 整理核心概念 | `我要整理物理的核心知识网络，内容是：牛顿三大定律体系...` | 自动分类到「物理/核心知识网络」，生成标准卡片 |
| 整理题型方法 | `我要整理数学的典型题型与方法，内容是：导数恒成立问题通法...` | 自动分类到「数学/典型题型与方法」，标注难度和来源 |
| 整理易错点 | `我要整理化学的易错警示，内容是：阿伏伽德罗常数八大陷阱...` | 自动分类到「化学/易错警示与辨析」 |
| 整理素材 | `我要整理语文的素材，内容是：2025年热点人物素材...` | 自动分类到「语文/素材与拓展」 |

> **简化版**：也可以直接说 `请入库：...`，AI 会自动识别科目和维度。

---

### 模板三：出题练习

```
帮我出[科目]的[知识点]练习题，要求：...
```

| 使用场景 | 示例指令 | AI 返回 |
|----------|----------|---------|
| 基础巩固 | `帮我出物理的板块模型练习题，要求：3道选择题+2道计算题，难度中等，附答案解析` | 原创/改编练习题+详细解析 |
| 专题突破 | `帮我出化学的氧化还原方程式配平练习题，要求：10道缺项配平，含广东卷风格` | 针对性练习题+配平步骤 |
| 真题演练 | `帮我出生物的长句表达练习题，要求：5道原因类+3道依据类，仿2024广东卷风格` | 仿真题+评分标准+参考答案 |
| 错题变式 | `帮我出数学的恒成立问题变式题，要求：基于卡片内容，难度递增，每题标注考点` | 变式题串+考点对应卡片 |

> **进阶用法**：出题后可要求 `将上述练习题也作为知识卡片入库`，实现"练习即积累"。

---

> 💡 **提示**：三个模板可组合使用。例如先 `查物理+力学三大模型` 复习，再 `帮我出物理的力学三大模型练习题` 巩固，最后 `将错题整理入库` 形成闭环。
