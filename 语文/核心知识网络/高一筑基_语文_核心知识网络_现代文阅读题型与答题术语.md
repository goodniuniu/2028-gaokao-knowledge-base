## 核心内容

### 一、论述类/信息类文本阅读（19分）

#### 题型1：内容理解与分析（3分×3=9分）

| 设错类型 | 标志词/特征 | 识别方法 |
|----------|------------|----------|
| **偷换概念** | 改变修饰语、替换关键词 | 比对选项与原文核心概念 |
| **以偏概全** | "所有""都""必然"等绝对化表述 | 关注范围词、程度词 |
| **无中生有** | 原文未提及，选项凭空捏造 | 逐字比对原文 |
| **张冠李戴** | 甲的观点放到乙身上 | 关注观点归属 |
| **因果混乱** | 因果倒置、强加因果 | 定位原文因果关系词 |
| **时态混淆** | 已然vs未然，或然vs必然 | 关注"了""将""可能"等 |

**答题术语**：
- "选项A将原文的'XX'偷换为'YY'，属于偷换概念"
- "选项B扩大了原文的范围，原文仅指……，而选项说……，属于以偏概全"

---

#### 题型2：论证分析（3分×1=3分）

| 考查角度 | 答题术语 |
|----------|----------|
| **论证结构** | 总分式/并列式/层进式/对照式/总分总式 |
| **论证思路** | 首先提出……观点；接着从……角度论证；最后总结/升华…… |
| **论证方法** | 举例论证/对比论证/比喻论证/引用论证/因果论证/假设论证 |
| **行文脉络** | 围绕……核心话题，先……再……最后…… |

---

#### 题型3：材料比较/观点概括（4分）

**答题模板**：
```
相同点：两则材料都围绕……展开论述，都强调了……
不同点：材料一侧重……，从……角度论证；材料二侧重……，采用……方法论证
```

---

### 二、文学类文本阅读（小说/散文，16分）

#### 小说阅读题型框架

| 题型 | 答题角度 | 核心术语 |
|------|----------|----------|
| **人物形象** | 外貌/语言/动作/心理/神态/侧面烘托 | 性格特征：善良坚韧、精明能干、懦弱自私…… |
| **情节作用** | 结构/内容/读者/主题 | 铺垫伏笔、照应标题、制造悬念、推动情节、深化主题 |
| **环境描写** | 自然环境/社会环境 | 渲染气氛、烘托心情、暗示命运、交代背景、推动情节 |
| **叙事技巧** | 视角/顺序/节奏/语言 | 第一人称真实感、全知视角全面性、倒叙设置悬念、插叙补充背景 |
| **标题含义** | 表层义/深层义/象征义 | 线索作用、点明对象、揭示主题、设置悬念 |

#### 散文阅读题型框架

| 题型 | 答题角度 | 核心术语 |
|------|----------|----------|
| **结构思路** | 线索/段落关系 | 以……为线索，贯穿全文；由……到……，逐层深入 |
| **语言特色** | 修辞/句式/用词/风格 | 清新自然/雄浑豪放/含蓄隽永/质朴平实/华丽典雅 |
| **情感主旨** | 情感词/议论句/卒章显志 | 表达了……情感；揭示了……哲理；寄托了……情怀 |

### 三、实用类文本阅读（已融入信息类文本，但保留答题术语）

| 考点 | 答题术语 |
|------|----------|
| **信息筛选** | 根据题干定位区间→比对关键词→概括整合 |
| **图文转换** | 图表类：先总述趋势/特征，再分述数据变化，最后得出结论 |
| **概念定义** | "XX是指……"（属概念+种差） |
| **概括要点** | 分层提炼→抓中心句→合并同类项→规范表述 |

## 备注

1. **广东卷特色**：广东新高考Ⅰ卷信息类文本近年呈现"双材料"特征（两则材料围绕同一话题），一则为学术/科普文本，一则为评论/时评，要求考生跨材料比较观点。需特别注意材料间的"互补"或"对立"关系。
2. **2024年风向**：现代文阅读II（文学类）出现"诗化小说"倾向（如2024年新课标Ⅰ卷《放牛记》），需关注散文化小说的叙事特征。
3. **答题规范**：信息类文本答案尽量使用原文关键词；文学类文本答案需"术语+文本分析"结合，避免只贴标签不分析。
4. **时间分配**：现代文阅读建议控制在35-40分钟（信息类19分钟+文学类16分钟），信息类文本比对选项时宁可慢不可错。
