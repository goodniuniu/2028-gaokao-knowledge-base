## 核心内容

### 一、知识网络（18个常见虚词及核心用法）

| 虚词 | 主要用法 | 课内/真题例句 |
|------|----------|---------------|
| **之** | ①代词（他/它）②结构助词"的"③主谓间取独④宾语前置标志 | 2023"不失君臣**之**礼"（助词"的"）；"是以先**之**"（代词） |
| **乎** | ①介词"于"②语气词③形容词词尾 | "生乎吾前"（于） |
| **者** | ①……的人/事②提顿③定语后置标志 | "不余欺**者**" |
| **也** | ①表判断②表语气 | "师**也**者，所以传道受业解惑也" |
| **以** | ①介词（用/凭/把/因）②连词（来/而）③通"已" | 2024"到兰干山南**以**分单于兵"（连，来）；2023"**是以**"（以+是，因此） |
| **于** | ①在②向③被（表被动）④比 | "不拘**于**时"（被） |
| **而** | ①顺承②转折③修饰④并列⑤通"尔" | 2023"骄侮**而**失礼"（转折）；2024"愧苏武**而**为之辞"（顺承） |
| **则** | ①就②那么③却 | "则**是**弃卫" |
| **为** | ①动词"做"②介词"替/被"③语气 | "**为**天下笑"（被） |
| **所** | ①所字结构②与"为"表被动 | "**所**畜积"（所字结构） |
| **其** | ①代词（他/它的）②语气副词 | "**其**莳也若子" |
| **乃** | ①于是②才③竟 | "**乃**以为西河守" |
| **且** | ①况且②暂且③将 | "**且**奈何"（将） |
| **因** | ①于是②趁③依靠 | "**因**归燕赵之兵" |
| **若** | ①像②如果③你 | "**若**甚怜焉" |
| **焉** | ①于此②语气③兼词 | "而卒以祸**焉**" |
| **何** | ①什么②怎么③宾语前置 | "**何**操"（操何） |
| **故** | ①所以②旧的③故意 | "**故**木受绳则直" |

### 二、核心结论
1. **虚词判定先看词性**：在句中作介词还是连词，决定译法（"以"表目的译"来"，表工具译"用"）。
2. **"之/是"为宾语前置标志**："是以"＝"以是"，须调序为"因此"。
3. **"于/为/所"表被动**：见此三标志优先考虑被动句式。
4. **通假优先**："乡"通"向"、"以"通"已"须先注通假。

### 三、易混点
- **而（转折 vs 顺承）**：前后意思相反用转折，先后相承用顺承。
- **以（介词 vs 连词）**：后接名词/代词为介词，连接分句为连词。
- **之（取独 vs 助词"的"）**：取消句子独立性时主谓间无"的"义。

### 四、真题母题印证
- **2023新高考Ⅰ卷·第13题**："是以先之"——"是以"宾语前置（以+是），"之"代词宾语，须"调+换"双得分点。
- **2024新高考Ⅰ卷·第13题**："到兰干山南以分单于兵"——"以"目的连词译"来"；"愧苏武而为之辞也"——"而"顺承连词。
