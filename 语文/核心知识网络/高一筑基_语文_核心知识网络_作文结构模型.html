作文结构模型

字段内容
来源广东高考作文命题偏好 + 新高考Ⅰ卷高分范文归纳
时间标签#高一筑基
难度★★★☆☆
状态⚠️待强化
试卷来源#新高考Ⅰ卷·广东
广东考情高频（作文60分，占全卷40%）；高档（思辨性作文要求逻辑严密、层次清晰）；广东卷偏好"二元/三元思辨"（如2021年"强与弱"、2023年"故事的力量"），要求考生在结构上就体现辩证思维

核心内容

一、基础结构：引-议-联-结 四段法

┌─────────────────────────────────────────┐
│  引（第1段）：引述材料 → 提出中心论点    │  ~100字
├─────────────────────────────────────────┤
│  议（第2-4段）：分析论证 → 并列/递进/对比│  ~500字
├─────────────────────────────────────────┤
│  联（第5段）：联系现实 → 拓展升华        │  ~150字
├─────────────────────────────────────────┤
│  结（第6段）：总结收束 → 回扣主题        │  ~100字
└─────────────────────────────────────────┘
                        总计：800-900字

二、议论主体段结构模型

模型A：并列式（分论点并列）

中心论点：XX需要A、B、C

分论点1：XX需要A（是什么/为什么）
  ↳ 论据1 + 分析
分论点2：XX需要B（是什么/为什么）
  ↳ 论据2 + 分析
分论点3：XX需要C（是什么/为什么）
  ↳ 论据3 + 分析

适用：观点明确、素材充足的题目

示例（2023年新高考Ⅰ卷"故事的力量"）：

  • 分论点1：好的故事，是个人成长的阶梯
  • 分论点2：好的故事，是民族精神的纽带
  • 分论点3：好的故事，是人类文明的桥梁

模型B：递进式（层层深入）

中心论点：XX的意义

是什么 → 为什么 → 怎么做
个人 → 社会 → 国家
过去 → 现在 → 未来
提出问题 → 分析问题 → 解决问题

适用：思辨性强、需要纵深分析的题目

示例（2021年新高考Ⅰ卷"强与弱"）：

  • 分论点1：何为强弱？（概念界定）
  • 分论点2：强弱为何转化？（辩证分析）
  • 分论点3：青年如何由弱变强？（实践路径）

模型C：对比式（正反对比）

中心论点：XX的重要性

正面论证：有XX则……（案例+分析）
反面论证：无XX则……（案例+分析）
辩证统一：XX不是……而是……

适用：涉及价值判断、有明显对立面的题目

示例（2020年新高考Ⅰ卷"疫情中的距离与联系"）：

  • 正面：物理距离无法阻隔心灵的联系（抗疫案例）
  • 反面：没有联系的距离是冷漠的隔离
  • 辩证：距离是形式，联系是本质

三、思辨性作文的辩证框架

广东卷高频思辨模型

思辨类型典型题目结构模板
二元思辨（A与B）强与弱、距离与联系A是B的基础，B是A的升华；二者辩证统一
三元思辨（A、B、C）本手妙手俗手、移用化用独创逐层推进：A→B→C，或A+B=C
情境思辨故事的力量由小及大：个人→民族→人类
时代思辨与时代同频回顾历史→立足当下→展望未来

思辨性段落公式

让步段公式：
诚然/不可否认，A有其合理性……（承认对方）
但是/然而，B才是根本/关键……（转折立论）
因此，我们应当在A的基础上追求B……（辩证统一）

示例：

诚然，技术的发展为故事的传播提供了前所未有的便利，短视频、VR技术让故事更加生动可感。然而，技术只是载体，真正打动人心的永远是故事内核中的真诚与价值。因此，我们既要善用技术之"器"，更要坚守内容之"道"。

四、高分结构加分技巧

技巧说明示例
标题对仗观点明确，形式工整《以故事之舟，渡文明之河》
段首排比三个分论点句式一致"于个人，故事是……；于民族，故事是……；于人类，故事是……"
过渡自然段落间用过渡句衔接"不仅于个人如此，放眼民族亦然"
回扣材料首尾呼应材料关键词结尾回到"故事的力量"，升华至"中国故事"

五、常见结构失误警示

失误类型表现对策
观点模糊中心论点不清或中途更换开篇明义，全文围绕一个核心
层次混乱分论点交叉重叠分论点之间互斥或互补，不重叠
头重脚轻开头过长，论证不足引论≤100字，本论≥500字
素材堆砌只有例子没有分析每例后必跟"这说明……"的分析
思辨缺失单向论证，没有辩证二元题目必须两边都谈

备注

  1. 广东卷特色：广东新高考Ⅰ卷作文近年呈现"弱化任务驱动、强化思辨空间"的趋势。2023年"故事的力量"、2022年"本手妙手俗手"、2021年"强与弱"都给了考生较大的立意空间，但高分作文必须在结构上体现思维的层次性。
  2. 时间分配：作文建议预留50-55分钟（构思5分钟+写作45分钟+检查5分钟）。
  3. 卷面要求：字数必须达到800字标记线，建议写到850-900字。段落以5-8段为宜，每段不宜过长。
  4. 2025-2028预测：关注AI与人文、生态文明、大湾区建设、青年责任等话题，结构上要能驾驭"科技+人文"的思辨框架。
  5. 检验标准：拿到任何作文题，能否在3分钟内确定中心论点和分论点框架。
