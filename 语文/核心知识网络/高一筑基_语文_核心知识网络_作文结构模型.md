| 字段 | 内容 |

| **来源** | 广东高考作文命题偏好 + 新高考Ⅰ卷高分范文归纳 |

| **难度** | ★★★☆☆ |

## 核心内容

### 一、基础结构：引-议-联-结 四段法

```
┌─────────────────────────────────────────┐
│  引（第1段）：引述材料 → 提出中心论点    │  ~100字
├─────────────────────────────────────────┤
│  议（第2-4段）：分析论证 → 并列/递进/对比│  ~500字
├─────────────────────────────────────────┤
│  联（第5段）：联系现实 → 拓展升华        │  ~150字
├─────────────────────────────────────────┤
│  结（第6段）：总结收束 → 回扣主题        │  ~100字
└─────────────────────────────────────────┘
                        总计：800-900字
```

---

### 二、议论主体段结构模型

#### 模型A：并列式（分论点并列）

```
中心论点：XX需要A、B、C

分论点1：XX需要A（是什么/为什么）
  ↳ 论据1 + 分析
分论点2：XX需要B（是什么/为什么）
  ↳ 论据2 + 分析
分论点3：XX需要C（是什么/为什么）
  ↳ 论据3 + 分析
```

**适用**：观点明确、素材充足的题目
**示例**（2023年新高考Ⅰ卷"故事的力量"）：
- 分论点1：好的故事，是个人成长的阶梯
- 分论点2：好的故事，是民族精神的纽带
- 分论点3：好的故事，是人类文明的桥梁

---

#### 模型B：递进式（层层深入）

```
中心论点：XX的意义

是什么 → 为什么 → 怎么做
个人 → 社会 → 国家
过去 → 现在 → 未来
提出问题 → 分析问题 → 解决问题
```

**适用**：思辨性强、需要纵深分析的题目
**示例**（2021年新高考Ⅰ卷"强与弱"）：
- 分论点1：何为强弱？（概念界定）
- 分论点2：强弱为何转化？（辩证分析）
- 分论点3：青年如何由弱变强？（实践路径）

---

#### 模型C：对比式（正反对比）

```
中心论点：XX的重要性

正面论证：有XX则……（案例+分析）
反面论证：无XX则……（案例+分析）
辩证统一：XX不是……而是……
```

**适用**：涉及价值判断、有明显对立面的题目
**示例**（2020年新高考Ⅰ卷"疫情中的距离与联系"）：
- 正面：物理距离无法阻隔心灵的联系（抗疫案例）
- 反面：没有联系的距离是冷漠的隔离
- 辩证：距离是形式，联系是本质

### 三、思辨性作文的辩证框架

#### 广东卷高频思辨模型

| 思辨类型 | 典型题目 | 结构模板 |
|----------|----------|----------|
| **二元思辨**（A与B） | 强与弱、距离与联系 | A是B的基础，B是A的升华；二者辩证统一 |
| **三元思辨**（A、B、C） | 本手妙手俗手、移用化用独创 | 逐层推进：A→B→C，或A+B=C |
| **情境思辨** | 故事的力量 | 由小及大：个人→民族→人类 |
| **时代思辨** | 与时代同频 | 回顾历史→立足当下→展望未来 |

#### 思辨性段落公式

```
让步段公式：
诚然/不可否认，A有其合理性……（承认对方）
但是/然而，B才是根本/关键……（转折立论）
因此，我们应当在A的基础上追求B……（辩证统一）
```

**示例**：
> 诚然，技术的发展为故事的传播提供了前所未有的便利，短视频、VR技术让故事更加生动可感。然而，技术只是载体，真正打动人心的永远是故事内核中的真诚与价值。因此，我们既要善用技术之"器"，更要坚守内容之"道"。

### 四、高分结构加分技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| **标题对仗** | 观点明确，形式工整 | 《以故事之舟，渡文明之河》 |
| **段首排比** | 三个分论点句式一致 | "于个人，故事是……；于民族，故事是……；于人类，故事是……" |
| **过渡自然** | 段落间用过渡句衔接 | "不仅于个人如此，放眼民族亦然" |
| **回扣材料** | 首尾呼应材料关键词 | 结尾回到"故事的力量"，升华至"中国故事" |

---

### 五、常见结构失误警示

| 失误类型 | 表现 | 对策 |
|----------|------|------|
| **观点模糊** | 中心论点不清或中途更换 | 开篇明义，全文围绕一个核心 |
| **层次混乱** | 分论点交叉重叠 | 分论点之间互斥或互补，不重叠 |
| **头重脚轻** | 开头过长，论证不足 | 引论≤100字，本论≥500字 |
| **素材堆砌** | 只有例子没有分析 | 每例后必跟"这说明……"的分析 |
| **思辨缺失** | 单向论证，没有辩证 | 二元题目必须两边都谈 |

## 题型识别标志

> **看到什么条件 → 立刻想到什么方法/模型**

| 题干关键条件 | 识别为 | 首选方法 |
|-------------|--------|----------|
| 材料给出两个/三个概念（如"强与弱""本手妙手俗手"） | 二元/三元思辨 | 递进式/对比式结构，先界定概念再辩证统一 |
| 材料是一句话或现象（如"故事的力量"） | 话题阐释型 | 引-议-联-结 + 由小及大（个人→民族→人类） |
| 要求"体现你的认识与思考" | 议论文，需明确中心论点 | 开篇明义，3分钟内定下分论点框架 |
| 材料有对立面（如"距离与联系""人·技术·时间"） | 辩证统一型 | 让步段公式：诚然…然而…因此 |
| 材料涉及时代/青年（如"跨越，再跨越"） | 时代思辨 | 回顾历史→立足当下→展望未来 |
| 材料给出比喻/寓言（围棋术语、红楼题额） | 隐喻引申型 | 由表及里，提炼本体主旨后再搭建结构 |
| 要求"明确文体，自拟标题" | 必须拟标题+5-8段 | 标题对仗技巧，段首排比，首尾回扣 |

## 解题路径（作文结构搭建四步法）

> 650分作文的底线是"结构清晰、层次分明、思辨到位"。四步法把50分钟拆成可控动作。

### 第一步：审题定体（3分钟）
- 圈出材料关键词，判断题型（思辨/话题/隐喻）
- 确定文体（材料多要求议论文），明确"写什么、不写什么"
- 常见陷阱：漏看"选准角度"导致立意偏；把隐喻当字面直写

### 第二步：立骨搭架（3分钟）
- 用"一句话中心论点 + 3个分论点"搭骨架
- 二元题用"A是基础、B是升华"；三元题用"A→B→C"递进
- 常见陷阱：分论点交叉重叠；只列例子不亮分论点

### 第三步：段落填充（40分钟）
- 引（≤100字）→ 议（≥500字，每段"分论点+论据+分析"）→ 联（≈150字）→ 结（≈100字）
- 每段用过渡句衔接，分论点可段首排比
- 常见陷阱：头重脚轻、素材堆砌无分析、字数不足800标记线

### 第四步：回扣升格（5分钟检查）
- 检查是否回扣材料关键词、是否体现思辨、字数是否≥850
- 升格动作：标题对仗、段首排比、结尾升华至时代/青年/中国

## 母题（2022 新课标Ⅰ卷·作文题，60分）

> 广东考生真题。三元递进式思辨结构的标准母题，对应解题路径"四步法"。

**题目**（2022年新课标Ⅰ卷作文）：
> "本手、妙手、俗手"是围棋的三个术语。本手是指合乎棋理的正规下法；妙手是指出人意料的精妙下法；俗手是指貌似合理，而从全局看通常会受损的下法。对于初学者而言，应该从本手开始，本手的功夫扎实了，棋力才会提高。一些初学者热衷于追求妙手，而忽视更为常用的本手。本手是基础，妙手是创造。一般来说，对本手理解深刻，才可能出现妙手；否则，难免下出俗手，水平也不易提升。
> 以上材料对我们颇具启示意义。请结合材料写一篇文章，体现你的感悟与思考。
> 要求：选准角度，确定立意，明确文体，自拟标题；不要套作，不得抄袭；不得泄露个人信息；不少于800字。

**结构搭建示范（四步法应用）**：
- **第一步定体**：三元概念（本手/妙手/俗手）→ 递进式思辨议论文。
- **中心论点**：夯实本手之基，方能成就妙手之创；舍本逐末，终落俗手之困。
- **第二步搭架（分论点互斥不重叠）**：
  - 分论点1（是什么）：本手是合乎棋理的"正"，是万事的根基。
  - 分论点2（为什么）：妙手源于对本手的深刻理解，无本则无妙。
  - 分论点3（怎么做+辩证）：警惕热衷妙手而忽视本手的"俗手"思维；基础与创新辩证统一。
- **第三步填充**：
  - 引：由围棋三术语引出——本手为基，妙手为创，俗手为戒（≤100字）。
  - 议：三段递进，每段"分论点+课内/时代论据+分析"（≥500字）。
  - 联：联系青年学习、科技创新——大国工匠亦从基本功始（≈150字）。
  - 结：回扣"本手—妙手"，升华至民族复兴需夯基与创新并重（≈100字）。
- **第四步升格**：标题拟《固本手之基，方妙手生花》；段首排比；结尾点"青年当以本手立身"。

> 💡 **关键技巧**：三元/二元思辨题，分论点必须"互斥不重叠"，并至少用一段做辩证统一（让步段公式），这是广东卷高分作文的硬指标。

## 备注

1. **广东卷特色**：广东新高考Ⅰ卷作文近年呈现"弱化任务驱动、强化思辨空间"的趋势。2023年"故事的力量"、2022年"本手妙手俗手"、2021年"强与弱"都给了考生较大的立意空间，但高分作文必须在结构上体现思维的层次性。
2. **时间分配**：作文建议预留50-55分钟（构思5分钟+写作45分钟+检查5分钟）。
3. **卷面要求**：字数必须达到800字标记线，建议写到850-900字。段落以5-8段为宜，每段不宜过长。
4. **2025-2028预测**：关注AI与人文、生态文明、大湾区建设、青年责任等话题，结构上要能驾驭"科技+人文"的思辨框架。
5. **检验标准**：拿到任何作文题，能否在3分钟内确定中心论点和分论点框架。
