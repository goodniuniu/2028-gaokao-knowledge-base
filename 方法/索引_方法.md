## 四、方法卡片与六科学习的关联使用指南

### 如何使用方法卡片？

| 学习场景 | 推荐方法卡片 | 使用方式 |
|----------|-------------|----------|
| 制定学习计划 | 学习六大定律 + 全盘经验 | 理解总时长约束，合理分配六科时间 |
| 遇到瓶颈卡壳 | 心理增强 + 学习六大定律（跃迁） | 识别低热情状态，用跃迁四步法集中突破 |
| 考前冲刺 | 应试提分策略 + 生理管理 | 调整作息、训练快速技巧、控制失分方差 |
| 成绩波动焦虑 | 心理增强（分数三参数） | 区分上限/方差/均值，找到真正的问题 |
| 倦怠不想学 | 心理增强 + 生理管理 | 检查睡眠/运动是否到位，用热机循环重启 |
| 周复盘时 | 学习六大定律（遗忘曲线） + 全盘经验 | 用间隔重复节奏检查知识掌握状态 |

### 方法卡片与知识库的周复盘结合

```
每周日复盘流程（方法版）：

① 身体检查（5分钟）
   → 本周睡眠是否充足？运动几次？
   → 参考：身体锻炼与精力管理卡片

② 心理状态检查（5分钟）
   → 本周学习热情高吗？有没有 burnout 信号？
   → 参考：心理增强卡片

③ 知识掌握检查（30-60分钟）
   → 按知识掌握状态表逐项检查
   → 用主动回忆法复习标记为⚠️和❌的卡片
   → 参考：学习六大定律（遗忘曲线）+ 全盘经验（13遍记忆法）

④ 考试策略检查（10分钟）
   → 本周有考试吗？失分原因分析（方差/上限/均值）
   → 参考：应试提分策略卡片

⑤ 下周计划调整（10分钟）
   → 确定下周主攻科目（一科一科攻克法）
   → 安排每日学习循环和休息日
```
