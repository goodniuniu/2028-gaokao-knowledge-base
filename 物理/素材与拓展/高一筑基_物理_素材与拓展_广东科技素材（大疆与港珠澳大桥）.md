### 情境一：大疆无人机（力学模型的"全家桶"）

#### 【情境背景】

大疆（DJI）总部在深圳，是全球民用无人机市场的领军企业。无人机看似高科技，其运动分析却完全落在中学生产力学范围内：悬停时受力平衡；起飞与降落可抽象为竖直方向的匀变速运动或竖直上抛；航拍中抛投物资是标准的平抛运动；螺旋桨向下推空气获得升力，则是牛顿第三定律与动量（反冲）的体现。

#### 【学科原理与考点链接】

**① 悬停 → 受力平衡（共点力平衡）**

- 悬停时加速度为零：升力 F = 重力 mg（竖直方向），水平方向合力为零
- 核心依据：牛顿第一定律/共点力平衡条件 `ΣF = 0`
- 变式：有风时悬停，机身倾斜，升力需分解——水平分量平衡风力，竖直分量平衡重力，考查力的分解与正交分解法
- 常见陷阱："悬停时无人机不受力"（错误，受平衡力）；"升力大于重力才能悬停"（错误，加速上升才需要 F > mg）

**② 起飞与降落 → 竖直方向匀变速运动 / 竖直上抛**

- 加速上升：`F − mg = ma`（超重，a 向上）；减速上升/加速下降：a 向下（失重）
- 竖直上抛模型：上升阶段 `v = v₀ − gt`，`h = v₀t − ½gt²`，最高点 v = 0，全程加速度恒为 g 向下
- 无人机断电坠落：自由落体 `h = ½gt²`，`v² = 2gh`
- 注意"上升过程中的减速"与"下降过程"不要混为一谈——分段建模是关键

**③ 航拍抛投物资 → 平抛运动**

- 平抛分解：水平方向匀速直线 `x = v₀t`；竖直方向自由落体 `y = ½gt²`
- 飞行时间只由高度决定：`t = √(2h/g)`，与水平速度无关（独立性原理）
- 合速度 `v = √(v₀² + g²t²)`，方向用 tanθ = gt/v₀ 描述
- 常考设问：无人机水平匀速飞行时投下物资，落地点相对投放点的水平位移；物资落地速度大小与方向
- 易错点：投放瞬间物资具有与无人机相同的水平初速度，不是从静止下落

**④ 螺旋桨升力 → 牛顿第三定律与动量（反冲）**

- 螺旋桨把空气向下推（对空气施加向下的力），空气对螺旋桨施加向上的反作用力——升力的来源
- 动量观点：单位时间内被加速向下的空气质量越大、速度越大，升力越大（`F = Δp/Δt`，高二动量模块）
- 悬停功率与旋翼下洗气流速度的关系可作信息题背景（给定公式代入即可，不要求推导）

#### 【命题角度】

- **受力分析型**：画出悬停、加速上升、倾斜抗风三种状态下无人机的受力图，比较升力与重力大小。
- **运动学计算型**：无人机从地面匀加速起飞，给定加速度与时间求高度；或悬停后断电自由下落求落地时间/速度。
- **平抛应用型**：无人机以水平速度 v₀ 在高度 h 处投放救援物资，求物资落地点距目标点的偏差，或求应提前多少水平距离投放。
- **动量定性型**（高二深化）：解释"旋翼转速提高，升力增大"的动量本质；比较载重前后维持悬停所需的气流速度。
- **超重失重型**：加速度传感器显示无人机竖直加速度方向，判断其处于超重还是失重状态。

> 例题风格：以大疆农业无人机喷洒作业为背景——满载时悬停求升力，匀加速爬升求升力增量，水平飞行中释放药箱求落点。一题串起受力平衡、牛顿第二定律、平抛运动三个模型，正是广东卷"一情境多考点"的典型风格。

### 情境二：港珠澳大桥与深中通道（超级工程中的力学）

#### 【情境背景】

港珠澳大桥2018年通车，全长约55公里，集桥、岛、隧于一体，是世界上最长的跨海大桥；深中通道（深圳—中山）2024年通车，同样采用"桥+岛+隧"组合方案。两大工程为力学命题提供了丰富素材：斜拉索桥对应力的分解与受力分析，桥塔对应力矩平衡，海底沉管隧道对应浮力与压强，车辆在桥面弯道和拱形桥上行驶对应圆周运动与超重失重。

#### 【学科原理与考点链接】

**① 斜拉桥 → 力的分解与受力分析**

- 斜拉索将桥面荷载传给桥塔：把索的拉力沿水平、竖直方向正交分解，竖直分量之和平衡桥面重力
- 核心方法：正交分解法、力的合成与分解、共点力平衡 `ΣFx = 0`、`ΣFy = 0`
- 定性结论：索与桥面夹角越小，提供同样竖直分量所需的拉力越大（T = G/(2sinθ) 一类关系），所以斜拉索不能太"平"
- 同一桥塔两侧对称布置的索，水平分量相互抵消，桥塔主要受压

**② 桥塔 → 力矩平衡初步**

- 桥塔可视为绕底部转动的杠杆：两侧索拉力的力矩相互平衡时桥塔稳定
- 核心公式：力矩 `M = F·L`（L 为力臂，转轴到力的作用线的垂直距离）
- 力矩平衡条件：`M顺 = M逆`
- 高考对力矩要求不高，常以信息题或定性判断形式出现（哪侧力臂大、风荷载产生的力矩方向等）

**③ 沉管隧道 → 浮力与压强**

- 沉管（预制好的隧道管节）浮运到位后注水下沉、坐底对接
- 阿基米德原理：`F浮 = ρ液·g·V排`；浮沉条件：F浮 > G 上浮，F浮 = G 悬浮，F浮 < G 下沉
- 沉管通过注水增大自身重力实现下沉，排水减重实现浮起——与潜水艇原理相同
- 液体压强：`p = ρgh`，深度 h 越大压强越大，沉管管壁须承受巨大海水压强
- 常考设问：沉管下沉过程中浮力如何变化（完全浸没后排开水的体积不变，浮力不变；靠增大重力下沉）

**④ 车辆过桥 → 圆周运动与超重失重**

- 汽车过凸形桥面（拱桥）：`mg − N = mv²/R`，N < mg，处于失重状态；速度越大支持力越小，v = √(gR) 时 N = 0（"飞桥"临界）
- 汽车过凹形桥面：`N − mg = mv²/R`，N > mg，处于超重状态
- 桥面弯道（水平面内圆周）：向心力由侧向摩擦力提供 `f = mv²/R`，雨天摩擦因数小，须限速——"限速牌"的物理依据
- 核心公式：向心加速度 `a = v²/R`，向心力 `F向 = mv²/R = mω²R`
- 易错点：向心力是效果力，由重力、支持力、摩擦力的合力（或分力）提供，受力图上不单独画"向心力"

#### 【命题角度】

- **力的分解型**：给斜拉索与桥面夹角和桥面重力，求单根索的拉力；或比较不同倾角索的拉力大小。
- **力矩定性型**：判断桥塔两侧索布置对称的原因；风从一侧吹来时桥塔底部的力矩平衡分析。
- **浮力压强型**：沉管注水下沉过程中重力、浮力、压强的变化判断；给沉管尺寸估算所受浮力或底部压强。
- **圆周运动型**：汽车以速度 v 过半径 R 的凸形桥面，求桥面支持力；求不脱离桥面的最大速度。
- **限速解释型**：用向心力公式解释弯道限速、雨天减速的原因（广东卷爱考"用物理知识解释交通规则"类简答）。

> 例题风格：以港珠澳大桥某桥段为背景——把斜拉索简化为两根对称的绳子吊起桥面，已知桥面质量与索的倾角，求索中拉力；再问若索与桥面夹角变小，拉力如何变化。——情境宏大，模型只是"力的分解+平衡"。

### 本素材的模型提取口诀

> 见"悬停、匀速"→ 想 ΣF = 0（受力平衡）；
> 见"起飞、爬升、坠落"→ 想竖直方向牛顿第二定律与匀变速；
> 见"抛投、投放"→ 想平抛：水平匀速 + 竖直自由落体；
> 见"螺旋桨、升力来源"→ 想牛顿第三定律与动量反冲；
> 见"斜拉索"→ 想力的正交分解；
> 见"桥塔、转动稳定"→ 想力矩平衡 M = FL；
> 见"沉管、注水下沉"→ 想 F浮 = ρgV排 与 p = ρgh；
> 见"过弯、拱桥"→ 想 F向 = mv²/R 与超重失重。

## 备注

- 本卡属"高一筑基"阶段的素材积累：动量（反冲）为高二内容，高一阶段理解"作用力与反作用力"层面即可。
- 两大工程的事实要点（公认事实即可，不必背数字细节）：
  - 港珠澳大桥：2018年通车，全长约55公里，桥—岛—隧组合，连接香港、珠海、澳门；
  - 深中通道：2024年通车，连接深圳与中山，同样含海底沉管隧道段。
- 广东卷情境题应对策略：
  1. 先找**运动状态关键词**（悬停/匀速/加速/转弯），它决定用平衡还是牛顿第二定律；
  2. 把工程结构**简化成几何图形**（索=绳、桥塔=杠杆、沉管=浸没物体、拱桥=圆弧）；
  3. 答题时回扣情境（"因此弯道要限速""因此索不能太平"），广东卷简答题看重情境回扣。
- 易错点提醒：
  1. 平抛飞行时间只由高度决定，与水平初速度无关；
  2. 沉管完全浸没后继续下沉，浮力不变、压强增大——"下沉是因为浮力变小"是常见错误；
  3. 过凸形桥是失重（N < mg），过凹形桥是超重（N > mg），方向记反是高频错误；
  4. 受力图上不得出现"向心力"，它由其他力的合力充当。
