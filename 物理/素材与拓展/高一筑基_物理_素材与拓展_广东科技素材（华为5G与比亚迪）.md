### 情境一：华为5G通信（电磁波与信号传输）

#### 【情境背景】

华为总部位于深圳，是全球5G技术的主要推动者之一。5G基站遍布广东各地，其使用的电磁波频率比4G更高（毫米波频段），波长更短，因此单基站覆盖范围更小、需要建设更多基站，但传输速率大幅提升。电磁波在真空和空气中传播速度约为光速 c。5G信号的调制（把声音、图像等信息"装载"到高频电磁波上）与解调（接收端把信息还原）是无线通信的基本过程。

#### 【学科原理】

- 5G信号本质上是**电磁波**，属于电磁场与电磁波模块（人教版必修三/选择性必修内容）。
- 电磁波谱：无线电波、微波、红外线、可见光、紫外线、X射线、γ射线，频率依次升高、波长依次变短。
- 电磁波在真空中传播速度 c 是定值，频率 f 与波长 λ 成反比：频率越高，波长越短，衍射（绕射）能力越弱，直线传播特性越明显——这正是5G基站密集布设、信号易被遮挡的物理原因。
- 调制：用低频信号（音频/视频）去改变高频载波的振幅（调幅AM）或频率（调频FM），以便天线高效发射。

#### 【考点链接】

- 电磁波传播公式：`c = λf`（c ≈ 3.0×10⁸ m/s）
- 电磁振荡与电磁波的产生条件：变化的电场产生磁场，变化的磁场产生电场（麦克斯韦电磁场理论）
- 电磁波谱排序与特性对比（穿透性、衍射性、能量）
- 调制与解调的基本概念辨析

#### 【命题角度】

- **直接计算型**：给出5G信号频率，求波长（或反求频率）；比较不同频段电磁波的波长数量级。
- **概念辨析型**：判断关于电磁波的说法正误，如"电磁波只能在真空中传播""频率越高的电磁波在真空中速度越大"（错误，真空中速度均为c）。
- **情境解释型**："5G基站比4G密集""电梯里手机信号变差"，要求用波长、衍射、电磁屏蔽等知识解释。
- **信息提取型**：长题干介绍华为某通信技术，实际考点仅为 c=λf 或电磁波谱排序，考查学生不被情境干扰、直取模型的能力。

> 例题风格：以"华为5G基站"为背景，给出信号频率数量级（如3.5 GHz），求空气中波长约为多少（答案量级：厘米级），并判断与4G信号（波长较长）相比衍射能力的强弱。

### 情境二：比亚迪新能源汽车（电学与力学的综合载体）

#### 【情境背景】

比亚迪总部在深圳，是全球新能源汽车销量领先的企业。新能源汽车是典型的"电学+力学"综合情境：动力电池充放电涉及恒定电流与电功率；电机驱动车轮涉及电磁感应与磁场对电流的作用；刹车时的"动能回收"系统把车轮的动能转化为电能存回电池，体现能量转化与守恒；宣传中常见的"百公里加速时间"则直接对应匀变速直线运动模型。

#### 【学科原理与考点链接】

**① 电池充放电 → 恒定电流与电功率**

- 电流定义：`I = q/t`；欧姆定律：`I = U/R`
- 电功与电功率：`W = UIt`，`P = UI`
- 焦耳定律：`Q = I²Rt`；纯电阻电路中 W = Q，非纯电阻电路（含电机、充电电池）中 W > Q
- 电池容量单位"千瓦时（kW·h）"是能量单位：1 kW·h = 3.6×10⁶ J，常作为隐含换算考点
- 充电效率 = 电池获得的化学能 / 电网消耗的电能 < 100%，损耗部分转化为内能（焦耳热）

**② 电机驱动 → 电磁感应与磁场**

- 通电导线在磁场中受安培力：`F = BIL`（B⊥I 时），方向用左手定则判断
- 电磁感应：闭合回路磁通量变化产生感应电动势，`E = n·ΔΦ/Δt`；导体切割磁感线 `E = BLv`
- 楞次定律：感应电流的效果总是阻碍引起感应电流的磁通量变化
- 电机转动时会在线圈中产生反电动势，这是"非纯电阻电路 P = UI ≠ I²R"的物理根源

**③ 动能回收（再生制动）→ 能量转化与守恒**

- 刹车时车轮带动电机变为发电机，动能 → 电能 → 化学能（存入电池）
- 能量转化链条：动能 -（发电机效率）→ 电能 -（充电效率）→ 化学能，每级都有损耗
- 核心公式：动能 `Ek = ½mv²`；动能定理 `W合 = ΔEk`；能量守恒：系统减少的机械能 = 回收的电能 + 摩擦等损耗的内能
- 常见设问：给定制动前后速度与回收效率，求回收的电能或能减少的耗电量

**④ 加速性能 → 匀变速直线运动**

- "百公里加速"（0→100 km/h 用时若干秒）≈ 匀加速直线运动模型
- 核心公式：`v = v₀ + at`，`x = v₀t + ½at²`，`v² − v₀² = 2ax`
- 注意单位换算：100 km/h ≈ 27.8 m/s
- 进阶：由 P = Fv 分析"恒定功率启动"（速度增大、牵引力减小、加速度减小的变加速过程，最终匀速时 F = f），这是电车/机车启动的经典模型

#### 【命题角度】

- **估算型**：给电车质量与百公里加速时间，估算平均加速度、平均牵引力（忽略阻力或给定阻力系数）。
- **电路计算型**：给充电桩电压电流与充电时间，求充电电能、电池充入的能量及充电效率。
- **能量转化型**：汽车刹车，动能一部分回收、一部分转化为内能，求回收效率或可回充的里程，常与动能定理结合。
- **图像型**：给出启动过程 v-t 图或 P-t 图，判断启动方式（恒定加速度 vs 恒定功率），求最大速度 v_m = P/f。
- **电磁综合型**（高二深化阶段）：以电机线圈为模型，结合安培力、电磁感应、反电动势考查非纯电阻电路。

> 例题风格：比亚迪某款车从静止开始以额定功率启动，质量 m、阻力恒为 f，额定功率 P₀，求（1）最大速度；（2）速度为某值时的加速度。——表面是电车广告情境，实为机车启动模型。

### 情境—模型—考点速查表

| 情境关键词 | 对应物理模型 | 核心公式/规律 | 所属模块 |
|------------|--------------|----------------|----------|
| 5G信号、频率、波长 | 电磁波的传播 | `c = λf` | 电磁场与电磁波 |
| 信号调制、载波 | 调幅/调频概念 | 定性辨析 | 电磁场与电磁波 |
| 充电桩、充电时间 | 电功与电功率 | `W = UIt`、`P = UI` | 恒定电流 |
| 充电损耗、发热 | 焦耳定律 | `Q = I²Rt` | 恒定电流 |
| 电机、驱动、线圈 | 安培力/电磁感应 | `F = BIL`、`E = BLv` | 磁场、电磁感应 |

| 百公里加速 | 匀变速直线运动 | `v² − v₀² = 2ax` 等三公式 | 匀变速直线运动 |
| 额定功率、最大速度 | 机车启动模型 | `P = Fv`、`v_m = P/f` | 功率、牛顿第二定律 |

### 本素材的模型提取口诀

> 见"信号、频率、波长"→ 想 `c = λf` 与电磁波谱；
> 见"充电、续航、电池"→ 想 `P = UI`、`W = UIt`、焦耳定律；
> 见"电机、驱动"→ 想安培力 F = BIL 与电磁感应；
> 见"动能回收、刹车"→ 想动能定理与能量守恒；
> 见"百公里加速"→ 想匀变速三公式与 P = Fv 启动模型。

### 自我检测（情境反应训练）

看到以下题干开头，3秒内说出应调用的模型与公式：

1. "某5G基站发射频率为 3.5 GHz 的电磁波……" → `c = λf`，先统一单位再求波长。
2. "快充桩以 200 kW 的功率为电池充电 30 分钟……" → `W = Pt`，注意 kW·h 与 J 的换算。
3. "汽车刹车时，动能回收系统将 40% 的动能转化为电能……" → 动能定理 + 效率定义，先算 ΔEk。
4. "新车百公里加速仅需 3.9 秒……" → 匀变速模型求平均加速度，再考虑是否升级为 P = Fv 变加速分析。
5. "电机线圈在磁场中转动，产生反电动势……" → 电磁感应 + 非纯电阻电路，UI = I²R + 机械功率。

## 备注

- 本卡属"高一筑基"阶段的素材积累：电磁感应、电磁场与电磁波为高二内容，高一先建立"情境→模块"的直觉，公式细节高二再深化。
- 广东卷情境题题干长、信息杂，训练时先划出题中的**物理量与过程动词**（加速、刹车、充电、发射），再决定调用哪个模型。
- 涉及企业数据（销量、频段等）以题面所给为准，不背具体数字；本卡只要求掌握模型映射。
- 易错点提醒：
  1. 电磁波在真空中速度恒为 c，与频率无关——频率只改变波长；
  2. 充电电路含电池（非纯电阻），不能用 P = I²R 算总功率；
  3. 能量回收不可能达到 100%，凡出现"全部回收"的选项必错。
