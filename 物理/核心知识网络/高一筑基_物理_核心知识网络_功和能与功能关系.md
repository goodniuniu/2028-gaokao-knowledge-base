## 核心内容

### 关键概念
- **功**：W = F·l·cosθ，力对空间的积累效果；单位焦耳(J)
- **功率**：P = W/t（平均功率），P = F·v（瞬时功率，力与速度同向时）
- **重力势能**：E_p = mgh，与零势能面选取有关；重力做功与路径无关
- **弹性势能**：E_p = ½kx²，与弹簧形变量有关
- **动能**：E_k = ½mv²，与参考系有关（通常以地面为参考系）
- **功能关系**：各种力做功与对应能量变化的关系
- **机械能守恒**：只有重力或弹力做功时，机械能守恒
- **能量守恒**：能量不会凭空产生或消失，只会从一种形式转化为另一种形式

### 核心公式/定理

#### 功的计算
```
恒力功：W = Fl cosθ
    θ = 0°  → W = Fl（正功）
    θ = 90° → W = 0（不做功）
    θ = 180°→ W = -Fl（负功）

变力功：
    ① 图像法：F-l图面积 = 功
    ② 平均力法：W = F̄·l（仅适用于线性变化的力）
    ③ 动能定理：W = ΔE_k（最通用方法）

功率：
    平均功率：P̄ = W/t
    瞬时功率：P = Fv cosθ（机车启动问题核心公式）
```
> **注意事项**：
> 1. 功是标量，但有正负；正功表示能量输入，负功表示能量输出
> 2. 计算功时，l 是力的作用点的位移（不一定是物体的位移）
> 3. 摩擦力做功与路径有关（非保守力），W_f = -f·s_路程

#### 机车启动问题（两种模型）
```
① 恒定功率启动：
   P = Fv 恒定，F = P/v，a = (F-f)/m = P/(mv) - f/m
   随着v增大，F减小，a减小，最终匀速时 F = f，v_max = P/f

② 恒定加速度启动：
   先匀加速：F = ma + f 恒定，P = Fv 逐渐增大
   匀加速结束时：P = P额，v₁ = P额/F = P额/(ma+f)
   之后功率恒定，变为变加速，最终匀速 v_max = P额/f
```
> **注意**：恒定加速度启动有两个转折点——匀加速结束（功率达额定值）和最终匀速（牵引力等于阻力）

#### 功能关系总结表
| 力做功 | 对应能量变化 | 关系式 |
|--------|-------------|--------|
| **重力做功** | 重力势能变化 | W_G = -ΔE_p = E_p初 - E_p末 |
| **弹力做功** | 弹性势能变化 | W_弹 = -ΔE_p弹 |
| **合外力做功** | 动能变化 | W_合 = ΔE_k = E_k末 - E_k初（动能定理） |
| **重力/弹力以外力做功** | 机械能变化 | W_其他 = ΔE_机 |
| **摩擦力做功** | 内能增加 | W_f = -Q = -f·s_相对（产生热量） |
| **电场力做功** | 电势能变化 | W_电 = -ΔE_p电 |
| **所有力做功** | 总能量变化 | 能量守恒：ΔE_k + ΔE_p + ΔE_内 + ... = 0 |

> **核心理解**：功能关系本质是"做功是能量转化的量度"，哪个力做功就对应哪种能量的转化

#### 动能定理
```
W_合 = ΔE_k = ½mv₂² - ½mv₁²
```
> **适用条件**：任何运动（直线/曲线、恒力/变力、单个/多个过程），只需要知道初末状态和总功
> **注意事项**：
> 1. 是标量方程，不能分方向列
> 2. 优先在全过程使用，避免分段求中间速度
> 3. 对多过程问题，可分段列也可全程列，灵活选择

#### 机械能守恒定律
```
条件：只有重力或弹力做功（其他力不做功或做功代数和为零）

表达式：
① E_k1 + E_p1 = E_k2 + E_p2（状态式）
② ΔE_k = -ΔE_p（转化式）
③ ΔE_A增 = ΔE_B减（转移式，系统内）
```
> **与动能定理对比**：
> | 对比项 | 动能定理 | 机械能守恒 |
> |--------|----------|------------|
> | 条件 | 无限制（适用于任何情况） | 只有重力/弹力做功 |
> | 研究对象 | 单个物体 | 单个物体或系统 |
> | 方程 | 功 = 动能变化 | 初态机械能 = 末态机械能 |
> | 优势 | 通用性强，求功/力方便 | 不涉及力的细节，过程简洁 |

### 方法步骤

#### 功能关系分析"五步法"
1. **确定研究对象**：单个物体还是系统？
2. **分析受力与做功**：哪些力做功？正功还是负功？
3. **分析能量变化**：动能、势能、内能、电势能等哪些在变化？
4. **选方法**：
   - 求功/力 → 优先动能定理
   - 只有重力/弹力做功 → 机械能守恒
   - 涉及摩擦生热 → 功能关系（W_f = -Q）
   - 涉及多种能量 → 能量守恒
5. **列方程求解**：注意正负号和零势能面选取

#### 摩擦力做功与内能计算
```
滑动摩擦力对系统做的总功：W_f = -f·s_相对
系统内能增加：Q = f·s_相对（s_相对为两物体相对滑动的路程）

对单个物体：W_f = -f·s_对地（注意是物体的对地位移）
```
> **重要区分**：摩擦力对单个物体做功用对地位移，系统内能增加用相对位移

### 记忆口诀/技巧
> **"功是能量转化的量度，哪种力做功对应哪种能量变"**
> 
> **"重力功负势能增，弹力功负弹性增，合外力功动能增，其他力功机械能增"**
> 
> **"动能定理最通用，机械能守恒条件严，摩擦生热相对程，功能关系连一线"**
> 
> **机车启动口诀**："恒功率启动加速度减，恒加速启动两转折点，最终都是P/f=v_max"

## 题型识别标志

> **看到什么条件 → 立刻想到什么方法/模型**

| 题干关键条件 | 识别为 | 首选方法 |
|-------------|--------|----------|
| "恒定功率 $P$、速度 $v$ 匀速行驶""牵引力 $F$" | 机车/功率问题 | $P=Fv$，匀速时 $F=f_{\text{阻}}$；克服摩擦做功 $W_f=F\cdot s$ |
| "克服摩擦力做功""摩擦生热" | 摩擦力做功与内能 | 单体 $W_f=f\cdot s_{\text{对地}}$；系统 $Q=f\cdot s_{\text{相对}}$ |
| "重力势能增加/减少""高度变化" | 重力做功 | $W_G=-\Delta E_p=-mg\Delta h$ |
| "从某高度下滑，阻力未知，末速度已知" | 动能定理求功 | $W_{\text{合}}=\Delta E_k\Rightarrow$ 阻力功 $=mgh-\frac12mv^2$ |
| "只有重力/弹力做功" | 机械能守恒 | $E_{k1}+E_{p1}=E_{k2}+E_{p2}$ |
| "多种能量同时存在（摩擦生热+势能变化）" | 功能关系综合 | 列能量守恒式 $\Delta E_k+\Delta E_p+\Delta E_{\text{内}}=0$ |

## 解题路径（功能关系五步法）

> 功能关系是广东卷力学综合题（新能源动能回收、过山车能量转化等）的核心方法，优先于逐过程动力学分析。

### 第一步：定对象、定过程
- 明确研究对象（单体 or 系统）与研究过程（一段 or 全段）。
- **"阻力未知"时优先用全过程动能定理**，避免分段求中间速度。

### 第二步：列受力与做功
- 逐个力判断做正功/负功/不做功。
- 摩擦力做功用**对地位移**；系统摩擦生热用**相对位移**。

### 第三步：对应能量变化
- 重力做功 ↔ 重力势能；合外力做功 ↔ 动能；其他力做功 ↔ 机械能。

### 第四步：选公式列方程
- 求功/力 → 动能定理 $W_{\text{合}}=\frac12mv_2^2-\frac12mv_1^2$
- 求高度/势能 → $W_G=-mg\Delta h$
- 求生热 → $Q=f\cdot s_{\text{相对}}$

### 第五步：代入求解，注意正负号
- 功率问题：$P=Fv$ 中 $v$ 为力作用点速度；匀速段 $F=f_{\text{阻}}=\dfrac{P}{v}$。
- **常见陷阱**：重力做功与路径无关，摩擦力做功与路径有关；$Q=f\cdot s_{\text{相对}}$ 的 $s_{\text{相对}}$ 是相对滑动路程。

## 母题（2022 广东选择性考试·第9题，6分）

> 广东卷高频题：以"无人驾驶小车"为情境，综合考查恒定功率、功与重力势能的功能关系（多选题）。

**题目**：如图所示，载有防疫物资的无人驾驶小车，在水平 $MN$ 段以恒定功率 $200\,\text{W}$、速度 $5\,\text{m/s}$ 匀速行驶，在斜坡 $PQ$ 段以恒定功率 $570\,\text{W}$、速度 $2\,\text{m/s}$ 匀速行驶。已知小车总质量为 $50\,\text{kg}$，$MN=PQ=20\,\text{m}$，$PQ$ 段的倾角为 $30^\circ$，重力加速度 $g$ 取 $10\,\text{m/s}^2$，不计空气阻力。下列说法正确的有（  ）
A. 从 $M$ 到 $N$，小车牵引力大小为 $40\,\text{N}$
B. 从 $M$ 到 $N$，小车克服摩擦力做功 $800\,\text{J}$
C. 从 $P$ 到 $Q$，小车重力势能增加 $1\times10^4\,\text{J}$
D. 从 $P$ 到 $Q$，小车克服摩擦力做功 $700\,\text{J}$

**解**：

**(1) 水平段 $MN$（匀速）**
由 $P=Fv$ 得牵引力 $F_1=\dfrac{P_1}{v_1}=\dfrac{200}{5}=40\,\text{N}$，故 A 正确。
匀速时牵引力等于摩擦力 $f_1=F_1=40\,\text{N}$，克服摩擦做功
$$W_{f1}=f_1\cdot MN=40\times20=800\,\text{J}$$
故 B 正确。

**(2) 斜坡段 $PQ$（匀速）**
牵引力 $F_2=\dfrac{P_2}{v_2}=\dfrac{570}{2}=285\,\text{N}$。
沿斜面方向平衡：$F_2-mg\sin30^\circ-f_2=0$
$$\Rightarrow f_2=285-50\times10\times0.5=35\,\text{N}$$
克服摩擦做功 $W_{f2}=f_2\cdot PQ=35\times20=700\,\text{J}$，故 D 正确。
重力势能增加 $\Delta E_p=mgh=mg\cdot PQ\sin30^\circ=50\times10\times(20\times0.5)=5000\,\text{J}=5\times10^3\,\text{J}\neq1\times10^4\,\text{J}$，故 C 错误。

**答**：选 **ABD**。

> 💡 关键：功能关系题先判"匀速→牵引力=阻力"，再用 $W=Fs$ 或 $W_G=mg\Delta h$ 分别算各类功与势能变化，逐项验证即可。

## 备注

- **广东情境化命题常见素材**：
  - 比亚迪新能源汽车动能回收系统（能量转化分析）
  - 广东水力发电站（重力势能→动能→电能）
  - 过山车/跳楼机（机械能守恒与临界条件）
  - 弹簧弹射装置（弹性势能→动能）
  - 广州塔摩天轮（重力势能周期性变化）
- **易错点**：
  1. 重力做功与路径无关，只与高度差有关；但摩擦力做功与路径有关
  2. 机械能守恒的条件是"只有重力或弹力做功"，不是"只受重力或弹力"
  3. 弹簧振子中，系统机械能守恒（包括弹性势能），但单个物体机械能不守恒
  4. 摩擦生热 Q = f·s_相对，s_相对是相对滑动的路程，不是位移
  5. 功率 P = Fv 中，v 是力F作用点的速度；如果F与v有夹角，需乘cosθ
  6. 恒力功 W = Fl cosθ 中，l 是力的作用点的位移，不是物体的位移（如绳子拉滑轮问题）
- **功的正负判断**：
  - 动力做正功（促进运动），阻力做负功（阻碍运动）
  - 重力做正功→重力势能减少；重力做负功→重力势能增加
- **等级赋分提示**：功能关系是力学综合题的核心，常出现在计算题第二、三问。建立清晰的"能量流动图"（哪种能量转化为哪种能量）是解题关键。基础题不丢分，中档题争取满分，压轴题尽量多写步骤分。
