## 核心内容

### 关键概念
- **电磁感应**：穿过闭合回路的磁通量发生变化时，回路中产生感应电流的现象
- **感应电动势**：产生感应电流的电动势，由磁通量变化率决定
- **感应电流**：由感应电动势驱动的电流，方向由楞次定律判断
- **自感**：由于线圈自身电流变化而产生感应电动势的现象
- **互感**：由于邻近线圈电流变化而产生感应电动势的现象

### 核心公式/定理

#### 1. 法拉第电磁感应定律
```
感应电动势大小：E = n|ΔΦ/Δt|（n为线圈匝数）
```
> 适用条件：任何因磁通量变化产生感应电动势的情形
> 注意事项：E与ΔΦ/Δt成正比，与ΔΦ本身大小无关；n匝线圈相当于n个电源串联

#### 2. 楞次定律
```
感应电流的磁场总要阻碍引起感应电流的磁通量变化
```
> 适用条件：所有电磁感应现象
> 注意事项："阻碍"不是"阻止"，只延缓变化；判断步骤：原磁场方向→原磁通量变化→感应电流磁场方向→感应电流方向

#### 3. 导体切割磁感线（动生电动势）
```
一般情况：E = BLvsinθ（θ为B与v夹角）
特殊情况：E = BLv（B⊥v，B⊥L，v⊥L，即三者两两垂直）
```
> 适用条件：导体在磁场中做切割磁感线运动
> 注意事项：L为有效切割长度（垂直于v和B的投影）；v为相对磁场的速度

#### 4. 感应电荷量
```
q = IΔt = (E/R)Δt = nΔΦ/R
```
> 适用条件：闭合回路
> 注意事项：q只与磁通量变化量ΔΦ和电阻R有关，与变化快慢无关

#### 5. 自感电动势
```
E_L = L|ΔI/Δt|
```
> 适用条件：线圈自身电流变化
> 注意事项：L为自感系数，与线圈匝数、铁芯有关；通电瞬间自感线圈相当于断路，稳定后相当于导线，断电瞬间相当于临时电源

### 方法步骤

#### 电磁感应问题分析三步法
1. **源分析**：确定是动生电动势（切割）还是感生电动势（磁场变化）
   - 动生：E = BLv（切割型）
   - 感生：E = nΔΦ/Δt = nSΔB/Δt（磁场变化型）
2. **路分析**：将导体棒视为电源，画出等效电路图
   - 电源内阻 = 导体棒电阻
   - 用右手定则或楞次定律判断电流方向
3. **力分析**：导体棒中有电流后受安培力，分析运动状态
   - F_安 = BIL = B²L²v/R（切割型典型结果）
   - 常见模型：加速度逐渐减小的加速运动，最终匀速

#### 单杆/双杆导轨模型通用分析框架
| 模型 | 电动势 | 安培力 | 最终状态 | 能量关系 |
|------|--------|--------|----------|----------|
| 单杆恒外力 | E = BLv | F_安 = B²L²v/R | a→0，匀速 | F做功 = Q + Ek |
| 单杆恒功率 | E = BLv | F_安 = B²L²v/R | a→0，匀速 | Pt = Q + Ek |
| 单杆无外力 | E = BLv | F_安 = B²L²v/R | a→0，静止 | Ek → Q |
| 双杆等长 | E = BL(v₁-v₂) | 系统动量守恒 | 共速 | 动能损失→Q |

### 记忆口诀/技巧
> **楞次定律速记**："增反减同"——磁通量增加，感应磁场与原磁场反向；磁通量减小，感应磁场与原磁场同向。
> **右手定则对比**：判断感应电流方向用右手（发电机），判断安培力方向用左手（电动机）。
> **能量守恒核心**：电磁感应过程必伴随能量转化，机械能→电能→焦耳热，或化学能→电能→机械能。
> **广东情境提示**：广东卷常考电磁感应与能量结合的情境题（如磁悬浮、无线充电、电磁制动），注意从能量守恒角度列方程，比动力学分析更简洁。

## 解题路径（电磁感应"源—路—力—能"四步法）

> 广东卷电磁感应多为压轴计算题（磁悬浮、无线充电、电磁制动），按步骤赋分，写对关键式即可拿可观分数。

### 第一步：源分析（定电动势）
- 动生（切割）：$E=BLv$（$B\perp v\perp L$ 两两垂直）。
- 感生（磁场变）：$E=n\dfrac{\Delta\Phi}{\Delta t}=nS\dfrac{\Delta B}{\Delta t}$。

### 第二步：路分析（画等效电路）
- 导体棒/线框视为电源，内阻=导体电阻。
- 用右手定则或楞次定律判电流方向（"增反减同"）。

### 第三步：力分析（运动状态）
- 安培力 $F_安=BIL=\dfrac{B^2L^2v}{R}$（切割型典型结果）。
- 单杆无外力：加速度减小，最终静止；恒外力：最终匀速 $F=F_安$。

### 第四步：能量分析（求热量/功）
- 能量守恒：机械能→电能→焦耳热。
- $Q=I^2Rt$，或对全过程 $W_{外}=Q+\Delta E_k$。

## 母题（2023 广东选择性考试·第14题，14分）

> 广东卷电磁感应压轴题：动生与感生综合，考查安培力、瞬时磁通量与焦耳热。

**题目**：光滑绝缘的水平面上有垂直平面的匀强磁场，磁场被分成区域Ⅰ和Ⅱ，宽度均为 $h$，俯视图如图(a)所示；两磁场磁感应强度随时间 $t$ 的变化如图(b)所示，$0\sim\tau$ 时间内两区域磁场恒定、方向相反，磁感应强度大小分别为 $2B_0$ 和 $B_0$。一电阻为 $R$、边长为 $h$ 的刚性正方形金属框 $abcd$ 平放在水平面上，$ab$、$cd$ 边与磁场边界平行。$t=0$ 时线框 $ab$ 边刚好跨过区域Ⅰ的左边界以速度 $v$ 向右运动。在 $\tau$ 时刻，$ab$ 边运动到距区域Ⅰ左边界 $\frac h2$ 处，线框速度近似为零，此时线框被固定，如图(a)中虚线框所示。随后 $\tau\sim2\tau$ 时间内，Ⅰ区磁感应强度线性减小到 0；$\tau\sim3\tau$ 时间内，Ⅱ区磁场也线性减小到 0。求：
(1) $t=0$ 时线框所受的安培力 $F$；
(2) $t=1.2\tau$ 时穿过线框的磁通量 $\Phi$；
(3) $2\tau\sim3\tau$ 时间内，线框中产生的热量 $Q$。

**解**：

**(1)** $t=0$ 时 $ab$ 边在Ⅰ区（$2B_0$）、$cd$ 边在Ⅱ区（$B_0$），两边均切割，由右手定则两电动势均顺时针：
$$E=2B_0hv+B_0hv=3B_0hv$$
$$I=\frac{E}{R}=\frac{3B_0hv}{R}$$
$ab$ 边安培力 $F_1=2B_0Ih$，$cd$ 边安培力 $F_2=B_0Ih$，由左手定则均向左，合力：
$$F=F_1+F_2=3B_0Ih=\frac{9B_0^2h^2v}{R}$$
方向水平向左。

**(2)** $t=1.2\tau$ 时Ⅰ区 $B=2B_0+\dfrac{0-2B_0}{2\tau}\cdot0.2\tau=1.6B_0$（向下），Ⅱ区仍为 $B_0$（向上）。线框固定时 $ab$ 边距Ⅰ左边界 $\frac h2$，即线框一半在Ⅰ区、一半在Ⅱ区：
$$\Phi=1.6B_0\cdot\frac{h^2}{2}-B_0\cdot\frac{h^2}{2}=0.3B_0h^2$$
方向垂直纸面向里。

**(3)** $2\tau\sim3\tau$ 仅Ⅱ区磁场由 $B_0$ 线性减到 0，磁通变化率：
$$\left|\frac{\Delta\Phi}{\Delta t}\right|=\frac{B_0\cdot h^2}{\tau}$$
$$E_1=\frac{B_0h^2}{\tau},\quad I_1=\frac{E_1}{R}=\frac{B_0h^2}{R\tau}$$
产生热量：
$$Q=I_1^2R\tau=\left(\frac{B_0h^2}{R\tau}\right)^2R\tau=\frac{B_0^2h^4}{R\tau}$$

**答**：(1) $F=\dfrac{9B_0^2h^2v}{R}$，水平向左；(2) $\Phi=0.3B_0h^2$（向里）；(3) $Q=\dfrac{B_0^2h^4}{R\tau}$。

> 💡 关键：双切割时两电动势同向相加；磁通量要带方向（一正一负相减）；感生电动势用 $E=\left|\Delta\Phi/\Delta t\right|$，热量用 $Q=I^2Rt$。

## 备注

- 电磁感应是广东卷物理压轴题的核心考点之一，常与力学、能量综合
- 图像问题：Φ-t图、B-t图、E-t图、I-t图、F-t图的斜率/面积物理意义
  - Φ-t图斜率 = E（感应电动势）
  - I-t图面积 = q（电荷量）
- 导轨模型中，若外力F恒定，最终匀速时 F = F_安 = B²L²v_m/R，可求 v_m
- 双杆模型：若导轨光滑且等长，系统动量守恒；若不等长，用动量定理分别分析
- 注意区分：感应电流I = E/R_total，安培力F = BIL = B²L²v/R_total，R_total是总电阻
