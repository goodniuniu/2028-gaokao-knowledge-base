核心内容

关键概念

  • 系统误差：由实验仪器或方法引起的，有规律可循，不能通过多次测量消除
  • 偶然误差：由随机因素引起，可通过多次测量取平均值减小
  • 绝对误差：测量值 - 真实值
  • 相对误差：绝对误差 / 真实值 × 100%
  • 有效数字：从第一个非零数字开始到最后一位可靠数字
  • 图像法处理数据：化曲为直，斜率和截距有物理意义
  • 等效替代法：用等效的简单实验替代复杂实验（如电阻箱替代电表）
  • 控制变量法：控制其他变量不变，研究某两个量的关系（如a与F、M的关系）
  • 逐差法：处理匀变速直线运动纸带数据，减小误差

核心公式/定理

力学实验核心公式

实验一：探究加速度与力、质量的关系

原理：F = ma → a = F/M（控制变量法）

平衡摩擦力：
    不挂砂桶，小车连纸带，调整斜面倾角，使小车能匀速下滑
    （纸带上点迹均匀，即匀速运动）

条件：
    ① 砂桶质量 m << 小车质量 M（保证拉力 ≈ mg）
    ② 细线与木板平行
    ③ 先接通电源，后释放小车

数据处理：
    a-F图：应过原点的直线（斜率 = 1/M）
    a-1/M图：应过原点的直线（斜率 = F）
    若a-F图有截距：正截距→平衡摩擦力过度；负截距→未平衡或平衡不足

实验二：验证机械能守恒定律

原理：mgh = ½mv²（自由落体）

器材：铁架台、打点计时器、重物、纸带、刻度尺
    （不需要天平！质量m可约去）
    （不需要秒表！时间由打点计时器记录）

数据处理：
    取下落点O，测各点高度h和速度v
    验证：gh ≈ ½v²（或比较 mgh 与 ½mv²）

误差来源：
    ① 空气阻力、纸带与打点计时器摩擦 → 系统误差（使 ½mv² < mgh）
    ② 测量h和v的误差 → 偶然误差

实验三：探究功与速度变化的关系

原理：W = ½mv²（橡皮筋做功，变力功）

方法：用n根相同橡皮筋（拉伸相同长度），做功分别为W、2W、3W...
    测小车获得的速度v，作W-v²图看是否为直线

实验四：验证动量守恒定律

原理：m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'

方法：平抛法（用水平位移代替速度，因为t相同）
    入射球质量 > 被碰球质量（防止反弹）
    两球半径相同（对心碰撞）

纸带数据处理公式

瞬时速度：v_n = (x_n + x_{n+1}) / (2T)（中间时刻速度 = 平均速度）
加速度（逐差法）：
    a = [(x₄+x₅+x₆) - (x₁+x₂+x₃)] / (9T²)  （6段纸带）
    或 a = [(x₃+x₄) - (x₁+x₂)] / (4T²)    （4段纸带）
    或 a = (x_{n+1} - x_n) / T²            （2段纸带，误差大）

T为相邻两点的时间间隔（通常为0.02s，若每5点取一个计数点则T=0.1s）
逐差法优势：充分利用所有数据，减小偶然误差

电学实验核心公式

实验五：伏安法测电阻

原理：R = U/I

两种接法：
    ① 内接法：电流表内接，测得 R测 = R_x + R_A > R_x（偏大）
       适用：R_x >> R_A（大电阻）
    ② 外接法：电流表外接，测得 R测 = (R_x·R_V)/(R_x+R_V) < R_x（偏小）
       适用：R_x << R_V（小电阻）

判断口诀："大内偏大，小外偏小"
    临界值：R_x ≈ √(R_A·R_V) 时内外接误差相近

实验六：测电源电动势和内阻

原理：E = U + Ir → U = E - Ir

方法：
    ① 伏安法：电压表测路端电压U，电流表测总电流I，作U-I图
       纵截距 = E（电动势）
       横截距 = I短 = E/r（短路电流）
       斜率绝对值 = r（内阻）
    ② 伏阻法：U = E - (U/R)·r → 1/U = 1/E + r/(E·R)
    ③ 安阻法：I = E/(R+r) → 1/I = (R+r)/E = R/E + r/E

误差分析：
    电流表内接法：E测 = E真，r测 = r真 + R_A（偏大）
    电流表外接法：E测 < E真，r测 < r真（系统误差来源于电压表分流）

实验七：电表改装

电流表改装为电压表：串联分压电阻 R
    量程扩大n倍：R = (n-1)R_g
    总内阻：R_V = nR_g

电流表改装为大量程电流表：并联分流电阻 R
    量程扩大n倍：R = R_g/(n-1)
    总内阻：R_A = R_g/n

其中 R_g 为表头内阻，I_g 为满偏电流

实验八：半偏法测电表内阻

测电流表内阻（半偏法）：
    ① 闭合S₁，调R₁使电流表满偏
    ② 闭合S₂，调R₂使电流表半偏
    ③ 则 R_g ≈ R₂（条件：R₁ >> R₂，电源电动势足够大）
    误差：R_测 < R_真（并联R₂后总电阻减小，总电流增大，R₂分流 > I_g/2）

测电压表内阻（半偏法）：
    ① 调R₁使电压表满偏
    ② 调R₂使电压表半偏
    ③ 则 R_V ≈ R₂（条件：R₂ << R₁）
    误差：R_测 > R_真

方法步骤

力学实验通用步骤

  1. 安装器材：按照实验要求安装，注意打点计时器限位孔与纸带对齐
  2. 平衡摩擦力/调零：如需要，先进行平衡摩擦力等准备工作
  3. 进行实验：先接通电源，后释放小车/重物；实验结束后先断电再取纸带
  4. 数据处理：用刻度尺测量数据，注意估读到毫米下一位（如用厘米刻度尺读到0.01cm）
  5. 作图分析：建立坐标系，描点，拟合直线（使点均匀分布在直线两侧）
  6. 误差分析：区分系统误差和偶然误差，提出减小误差的方法

电学实验通用步骤

  1. 选器材：根据待测量大小选择合适量程的电压表、电流表；选滑动变阻器（分压式/限流式）
  2. 画电路图：确定电流表内接/外接，确定滑动变阻器接法
  3. 连接实物：按电路图连接，注意电表极性和量程；开关断开，滑片置保护位置
  4. 进行实验：闭合开关，调节滑动变阻器，记录多组数据
  5. 数据处理：作图法（U-I图、1/U-1/R图等），求斜率截距
  6. 整理器材：实验完毕，断开开关，拆除连线

滑动变阻器接法选择

接法限流式分压式
电路串联在电路中一部分与待测电路并联
电压调节范围不能从零开始可从零开始
电流调节范围较小较大
适用场景测量电阻、测电源E和r描绘小灯泡伏安特性曲线、要求从零开始调节
节能性节能耗能大
选择原则：要求电压/电流从零开始调→分压式；测量电表内阻或校准电表→分压式；一般测电阻→限流式（节能）

备注

  • 广东实验题特色（探究性、设计性）：
  1. 广东卷实验题往往不是教材实验的简单复现，而是"变式实验"（如改变器材、改变测量方法）
  2. 常要求"设计实验步骤""补充实验器材""评价实验方案"
  3. 数据处理常要求用图像法（化曲为直），斜率/截距有特定物理意义
  4. 误差分析不仅要求判断偏大偏小，还要求提出改进方法
  • 广东情境化实验素材：
  • 深圳光电产业：LED伏安特性曲线测量、光电传感器实验
  • 佛山制造业：精密测量、传感器应用实验
  • 广东新能源汽车：电池电动势和内阻测量、电机效率实验
  • 大湾区桥梁：桥梁应力测量、振动频率实验（简化为单摆/弹簧振子）
  • 易错点：
  1. 打点计时器：电磁式用4-6V交流电，电火花式用220V交流电；频率50Hz，周期0.02s
  2. 纸带数据处理：注意区分计数点（每5点取一个，T=0.1s）和实际打点（T=0.02s）
  3. 平衡摩擦力：是让重力沿斜面的分力抵消摩擦力，不是让斜面完全水平
  4. 电表读数：电压表/电流表要估读（如0.6A量程读到0.01A，3A量程读到0.1A）
  5. 实物连线：先连主回路，再连并联支路；连线不能交叉；开关要断开
  6. 滑动变阻器分压接法：滑片初始位置应使待测电路电压为零（保护电路）
  7. 半偏法误差：电流表半偏法 R_测 < R_真；电压表半偏法 R_测 > R_真
  8. 图像法：作图时数据点要均匀分布在直线两侧，不是强行连线过所有点
  • 有效数字规则：
  • 测量结果的有效数字与测量工具精度一致
  • 运算结果保留到最小有效数字位（如乘除法取最少有效数字）
  • 广东卷常要求"保留三位有效数字"
  • 等级赋分提示：实验题是"步骤分大户"，即使最终结果算错，写出正确的实验原理、步骤、公式也能拿大部分分数。务必注意：实验器材选择、电路连接、数据处理的规范性。建议熟练掌握5-8个核心实验的原理、步骤、误差分析。
