## 核心内容

### 关键概念
- **动量 $p$**：描述物体运动状态的物理量，$p = mv$，矢量，方向与速度相同
- **冲量 $I$**：力对时间的累积效应，$I = Ft$，矢量，方向与力相同
- **动量定理**：物体动量的变化量等于合外力的冲量
- **动量守恒定律**：系统不受外力或所受外力之和为零时，系统总动量保持不变
- **碰撞**：相互作用时间极短，内力远大于外力，动量守恒
- **反冲**：系统内一部分向某方向运动，另一部分向相反方向运动

### 核心公式/定理

#### 1. 动量定理
```
Ft = Δp = mv₂ - mv₁
或：I_合 = p₂ - p₁
```
> 适用条件：任何力学过程（比牛顿定律适用范围更广，尤其适用于碰撞、爆炸、变力作用）
> 注意事项：是矢量式，需规定正方向；变力时可用平均力或积分；也可用于某一方向的分量式

#### 2. 动量守恒定律
```
系统总动量守恒：m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
或：p_初 = p_末
```
> 适用条件：
> - 系统不受外力
> - 系统所受外力之和为零
> - 内力远大于外力（如碰撞、爆炸）
> - 某一方向外力之和为零，则该方向动量守恒
> 注意事项：是矢量式，需规定正方向；只涉及初末状态，不涉及过程细节

#### 3. 碰撞分类与公式
| 碰撞类型 | 特征 | 公式 |
|----------|------|------|
| **弹性碰撞** | 动量守恒 + 动能守恒 | m₁v₁+m₂v₂ = m₁v₁'+m₂v₂'<br>½m₁v₁²+½m₂v₂² = ½m₁v₁'²+½m₂v₂'² |
| **完全非弹性碰撞** | 动量守恒，动能损失最大，共速 | m₁v₁+m₂v₂ = (m₁+m₂)v_共<br>ΔEk = ½μ(v_相)²（μ为约化质量） |
| **非完全弹性碰撞** | 动量守恒，动能有损失 | m₁v₁+m₂v₂ = m₁v₁'+m₂v₂'<br>动能不守恒，用恢复系数e描述 |

> 弹性碰撞速度公式（记忆版）：
> ```
> v₁' = [(m₁-m₂)v₁ + 2m₂v₂]/(m₁+m₂)
> v₂' = [(m₂-m₁)v₂ + 2m₁v₁]/(m₁+m₂)
> ```
> 特殊情况：m₁ = m₂ 时，速度交换；m₂ >> m₁ 且 v₂=0 时，v₁' ≈ -v₁（反弹）

#### 4. 反冲与爆炸
```
0 = m₁v₁' + m₂v₂'（系统初动量为零）
```
> 适用条件：内力作用，系统初动量为零或恒定
> 注意事项：人船模型、火箭发射、炮弹发射均属此类

#### 5. 人船模型
```
mx = Mx'，且 x + x' = L（相对位移）
```
> 适用条件：系统水平方向动量守恒，初动量为零
> 注意事项：位移与质量成反比；x为人位移，x'为船位移，L为船长

### 方法步骤

#### 动量守恒问题解题五步通法
1. **选系统**：明确研究对象（谁和谁组成系统）
2. **判守恒**：判断动量是否守恒（外力之和是否为零）
3. **定方向**：规定正方向（通常取初速度方向）
4. **列方程**：m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'（注意矢量符号）
5. **求结果**：解方程，验证物理合理性（速度方向、大小范围）

#### 碰撞问题判断流程
1. **动量守恒**：先验证 p_初 = p_末（必须满足）
2. **能量不增**：Ek_末 ≤ Ek_初（动能不可能增加）
3. **速度合理**：
   - 追碰后：v_前' ≥ v_后'（否则二次碰撞）
   - 被碰物体：v_被碰' ≥ 0（若原静止）

### 记忆口诀/技巧
> **动量守恒三条件**："无外力、外力零、内力大"——三满足其一即可。
> **弹性碰撞速记**：等质量，换速度；大撞小，同向走；小撞大，反向弹。
> **人船模型速记**：人走船退，人停船停，位移反比于质量。
> **广东情境提示**：广东卷常考动量与能量综合的情境题（如新能源汽车碰撞安全、体育比赛中的碰撞、大湾区交通流量），注意动量守恒是矢量守恒，必须画方向。

## 解题路径（动量守恒四步法 + 动量定理冲量法）

> 广东卷常以"新能源汽车碰撞安全、安全气囊、体育碰撞"为情境，正负号与方向是主要失分点。

### 第一步：选系统、判守恒
- 确定研究对象（谁和谁组成系统）。
- 判守恒：系统不受外力 / 外力之和为零 / 内力≫外力（碰撞、爆炸）。

### 第二步：规定正方向
- 通常取初速度方向为正；**矢量式必须带符号运算**。
- 单方向守恒时，该方向单独列式。

### 第三步：列式求解
- 动量守恒：$m_1v_1+m_2v_2=m_1v_1'+m_2v_2'$
- 动量定理（求冲量/平均力）：$I=\Delta p=F_{\text{平}}\Delta t$

### 第四步：验证合理性（碰撞三判据）
- 动量守恒（必须满足）；
- 动能不增（$E_{k末}\le E_{k初}$）；
- 速度合理（追碰后 $v_{\text{前}}'\ge v_{\text{后}}'$；原静止被碰体 $v'\ge0$）。

## 母题（2024 广东选择性考试·第14(2)题，约8分）

> 广东卷动量定理典型题：以"汽车安全气囊性能测试"为情境，考查 $F$-$t$ 图像面积求冲量及竖直上抛（综合动量定理与运动学）。

**题目**：在安全气囊的性能测试中，可视为质点的头锤从离气囊表面高度为 $H=3.2\,\text{m}$ 处做自由落体运动，与正下方气囊发生碰撞。以头锤到气囊表面为计时起点，气囊对头锤竖直方向作用力 $F$ 随时间 $t$ 的变化规律可近似用题图（三角形脉冲）描述。已知头锤质量 $M=30\,\text{kg}$，重力加速度取 $g=10\,\text{m/s}^2$。求：
① 碰撞过程中 $F$ 的冲量大小和方向；
② 碰撞结束后头锤上升的最大高度。

**解**：

**(1) 碰撞前速度**
自由落体：$v_0=\sqrt{2gH}=\sqrt{2\times10\times3.2}=8\,\text{m/s}$（向下）。

**(2) ① 冲量大小**
$F$-$t$ 图（底 $0.1\,\text{s}$、高 $6600\,\text{N}$ 的三角形）面积即冲量：
$$I_F=\frac12\times0.1\times6600=330\,\text{N·s}$$
方向竖直向上（气囊对头锤作用力向上）。

**(2) ② 碰撞后上升高度**
取向上为正，作用过程由动量定理：
$$I_F-Mgt=Mv-(-Mv_0)\quad (t=0.1\,\text{s})$$
$$330-30\times10\times0.1=30v+30\times8\Rightarrow 300=30v+240\Rightarrow v=2\,\text{m/s}$$
碰撞后竖直上抛，上升最大高度：
$$h=\frac{v^2}{2g}=\frac{2^2}{2\times10}=0.2\,\text{m}$$

**答**：① 冲量 $330\,\text{N·s}$，方向竖直向上；② 最大高度 $0.2\,\text{m}$。

> 💡 关键：$F$-$t$ 图像面积=冲量；动量定理必须规定正方向并对全过程列式；先求碰前速度再求碰后速度。

## 备注

- 动量定理的优势：不需要知道过程细节，只关心初末状态，特别适用于变力作用、曲线运动
- 动量守恒与能量守恒常联立使用：
  - 弹性碰撞：动量守恒 + 动能守恒
  - 完全非弹性碰撞：动量守恒 + 能量损失
- 注意区分：动量是矢量（p = mv），动能是标量（Ek = ½mv²），两者关系：Ek = p²/(2m)
- 广东卷高频陷阱：
  - 弹簧压缩到最短时两物体共速，是完全非弹性碰撞模型
  - 子弹打木块：若穿出，用动量定理；若未穿出，是完全非弹性碰撞
  - 多物体碰撞注意逐步分析，不要试图一次写全
