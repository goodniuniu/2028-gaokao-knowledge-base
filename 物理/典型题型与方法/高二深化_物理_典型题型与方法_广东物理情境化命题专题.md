## 核心内容

### 一、情境化命题的本质

> 情境化命题 = **真实情境包装** + **标准物理模型** + **核心物理规律**
>
> 考生能力要求：**从长题干中筛选有效信息，忽略干扰信息，建立物理模型。**

### 三、情境化审题三步法

```
Step 1：读问题 → 明确求什么（锁定物理模型方向）
    ↓
Step 2：回读题干 → 提取物理量（标出已知数据、忽略背景描述）
    ↓
Step 3：建模求解 → 转化为标准物理模型，用通法解答
```

#### 示例：2024广东物理（新能源汽车情境）

> 题干大意：某电动汽车在水平路面行驶，电机输出功率恒定，已知质量m、阻力f、额定功率P...求最大速度、加速时间等。

**提取过程**：
- 背景信息："电动汽车""比亚迪" → 忽略品牌信息
- 关键信息："功率恒定""水平路面""阻力恒定" → 机车启动模型（恒定功率启动）
- 物理量：P、m、f → 标准公式：$P = Fv$，$F - f = ma$，$v_{max} = P/f$

### 四、各模块情境化命题规律

#### 4.1 力学模块

| 标准模型 | 常见情境包装 | 提取要点 |
|----------|------------|----------|
| 匀变速直线运动 | 电梯升降、列车进站 | 初速度、加速度、位移 |
| 平抛运动 | 投篮、无人机投送、喷泉 | 水平初速、下落高度、射程 |
| 圆周运动 | 过山车、汽车转弯、洗衣机脱水 | 向心力来源、最高点临界 |
| 连接体/板块 | 拖挂车、叠放货物、汽车碰撞 | 摩擦力、加速度关系 |
| 传送带 | 工厂流水线、安检设备、分拣系统 | 带速、摩擦因数、长度 |
| 弹簧振子 | 汽车减震、蹦极、弹性碰撞 | 劲度系数、最大压缩量 |

#### 4.2 电磁学模块

| 标准模型 | 常见情境包装 | 提取要点 |
|----------|------------|----------|
| 带电粒子在电场中偏转 | 示波器、电子显微镜、喷墨打印 | 加速电压、偏转电压、板间距 |
| 带电粒子在磁场中运动 | 质谱仪、回旋加速器、磁约束 | 磁感应强度、入射速度、半径 |
| 电磁感应（单杆） | 磁悬浮、发电机、电磁制动 | 杆长、速度、磁场、电阻 |
| 变压器/远距离输电 | 特高压输电、充电桩、变电站 | 匝数比、功率损失、电压降 |

#### 4.3 热学/光学/原子物理模块

| 标准模型 | 常见情境包装 | 提取要点 |
|----------|------------|----------|
| 理想气体状态方程 | 潜水气瓶、轮胎充气、热气球 | P、V、T变化，注意等温/等压/等容 |
| 光的折射/全反射 | 光纤通信、海市蜃楼、钻石闪耀 | 折射率、入射角、临界角 |
| 光电效应 | 光电管、太阳能电池、光控开关 | 逸出功、截止频率、光电子动能 |
| 核反应 | 核电站、医用同位素、碳14测年 | 质量亏损、半衰期、核反应方程 |

### 六、信息干扰项识别

| 干扰类型 | 示例 | 应对策略 |
|----------|------|----------|
| **背景描述** | "某知名车企最新研发..." | 忽略品牌，提取物理参数 |
| **冗余数据** | 给出与实际解题无关的环保数据 | 先读问题，再按需提取 |
| **新颖概念** | 引入教材外的"新物理量" | 看题干是否给出定义式，若有则直接用 |
| **单位陷阱** | km/h、cm、g等非国际单位 | 统一换算为m/s、kg、m等国际单位 |
| **图像信息** | 给出复杂的装置图/流程图 | 关注图中标注的物理量和几何关系 |

### 七、情境化实验题特点

广东物理实验题近年趋向**探究性实验设计**：

1. **不考死记操作**，考**原理迁移**（如用电流传感器替代电流表）
2. **给出实验目的**，要求**选择器材、设计步骤**
3. **数据处理**：从图像中提取信息（斜率、截距的物理意义）
4. **误差分析**：系统误差 vs 偶然误差，改进方案

**2024广东实验题示例**：用DIS传感器探究加速度与力、质量的关系，要求分析传感器数据处理方法和误差来源。

## 备注

- **2026命题预测**：结合"新质生产力""碳中和""人工智能"等热点，广东物理可能增加"机器人运动控制""智能电网""光伏储能"等情境。建议关注广东省科技新闻。
- **等级赋分提示**：情境化命题的"新"主要体现在题干包装，物理本质不变。基础扎实的考生即使遇到陌生情境，也能通过建模拿到大部分分数。切忌因题干长而心态崩溃。
- **训练建议**：每周做1-2道广东情境化真题，限时训练"提取物理量→建模→求解"的完整流程，控制在5-8分钟内完成审题建模。
