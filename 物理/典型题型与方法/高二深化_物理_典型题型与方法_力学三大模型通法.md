## 核心内容

### 一、模型总览与决策树

```
力学压轴题 → 多物体+多过程 → 识别模型
  ├── 两物体叠放/相对滑动 → 板块模型
  ├── 物体在运动的带上运动 → 传送带模型
  └── 涉及弹簧连接/弹性势能 → 弹簧模型（类碰撞）
```

> **核心思想**：三大模型本质都是**牛顿定律 + 运动学 + 能量/动量**的综合。解题关键是**分段分析**、**找临界点**、**画v-t图**。

---

### 二、板块模型（滑块-木板）

#### 2.1 核心问题
两物体叠放，外力作用于其中一个，判断是否相对滑动、求相对位移/摩擦生热/分离时间。

#### 2.2 解题通法

**第一步：求临界力 F₀（判断是否相对滑动）**

假设相对静止，整体法求加速度：$a_{整体} = F/(M+m)$

对上方物块用隔离法：$f = ma_{整体}$

若 $f ≤ f_{max} = μmg$ → 相对静止，用整体法

若 $f > f_{max}$ → 相对滑动，分别隔离列方程

**第二步：相对滑动时的动力学方程**

| 对象 | 受力分析 | 方程 |
|------|----------|------|
| 小物块 m | 滑动摩擦力 $f = μmg$（方向与相对运动相反） | $a_m = μg$ |
| 木板 M | 外力 F + 地面摩擦力 + 物块给的摩擦力 | $a_M = (F ± μmg ± f_{地})/M$ |

**第三步：运动学关系**

- 相对位移：$Δx = x_{木板} - x_{物块}$（或反之，看参考系）
- 摩擦生热：$Q = f_{滑} · |Δx|$
- 共速条件：$v_m = v_M$ 时，判断是否还能相对滑动（若外力仍作用，需重新判断）

#### 2.3 易错点
- 摩擦力方向：**与相对运动方向相反**，不是与运动方向相反
- 地面光滑 vs 地面粗糙：地面有摩擦时木板加速度不同
- 共速后若外力 F 仍存在，可能再次相对滑动，需重新判断

### 三、传送带模型

#### 3.1 分类
| 类型 | 特点 | 关键分析点 |
|------|------|----------|
| **水平传送带** | 带匀速转动，物体初速v₀ | 判断共速前是否滑出 |
| **倾斜传送带** | 带有倾角θ | 比较 $μ$ 与 $tanθ$ 判断能否静止 |
| **平斜交接** | 水平段→倾斜段 | 交接处速度衔接，重新判断摩擦力方向 |

#### 3.2 水平传送带通法

**情景一：物体初速 v₀，带速 v，同向**

| 条件 | 运动过程 | 痕迹长度 |
|------|----------|----------|
| v₀ < v，带足够长 | 先加速 $a=μg$，再共速匀速 | $Δx = v·t - x_{物} = v²/(2μg) - v₀²/(2μg)$ |
| v₀ > v，带足够长 | 先减速 $a=μg$，再共速匀速 | 类似 |
| 带长 L < 加速距离 | 一直加速，末速 $v' = √(v₀²+2μgL)$ | 全程滑动 |

**情景二：物体初速 v₀，带速 v，反向**

- 先减速到0，再反向加速
- 若 $v₀ ≥ v$：返回速度 = v（与带共速后匀速）
- 若 $v₀ < v$：返回速度 = v₀（对称性）

#### 3.3 能量与摩擦生热

- 摩擦生热：$Q = f_{滑} · Δx_{相对}$（**注意是相对位移，不是对地位移**）
- 多消耗的电能 = 物块动能变化 + 摩擦生热（能量守恒）

#### 3.4 广东情境示例
- 2023广东模拟：地铁安检行李传送带（水平+倾斜交接）
- 2024广东模拟：工厂流水线工件传输（多物体依次放上传送带）

### 四、弹簧模型（连接体/类碰撞）

#### 4.1 两类问题

| 类型 | 特征 | 核心方法 |
|------|------|----------|
| **弹簧连接体** | 弹簧连接两物体，受外力作用 | 整体法 + 隔离法，注意弹簧弹力不可突变 |
| **弹簧类碰撞** | 弹簧作为相互作用媒介，求最大压缩/分离 | 动量守恒 + 能量守恒（弹性势能参与） |

#### 4.2 弹簧连接体通法

- 弹簧弹力：$F = kx$（x 为形变量，注意原长位置）
- 平衡时：对连接体整体分析，求共同加速度，再隔离求弹簧形变量
- **关键**：弹簧弹力不可突变，但形变量可以渐变

#### 4.3 弹簧类碰撞（滑块-弹簧-滑块）

- **系统动量守恒**：$m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'$
- **系统能量守恒**：$rac{1}{2}m₁v₁² + rac{1}{2}m₂v₂² = rac{1}{2}m₁v₁'² + rac{1}{2}m₂v₂'² + E_p$

**临界条件**：两滑块速度相等时，弹簧压缩/伸长最大，弹性势能最大。

**求解方法**：
1. 动量守恒求共同速度：$m₁v₁ + m₂v₂ = (m₁+m₂)v_{共}$
2. 能量守恒求最大弹性势能：$E_{p,max} = E_{k初} - E_{k共}$

### 五、三大模型通用解题步骤

```
① 审题画情境图 → 标出已知量、未知量、各物体初速度
② 分段分析 → 按"速度变化点"分段（摩擦突变点、共速点、分离点）
③ 选对象列方程 → 整体法/隔离法交替使用
④ 运动学联立 → v-t图辅助分析，避免代数错误
⑤ 验证结果 → 检查物理意义（速度方向、位移正负、能量是否守恒）
```
