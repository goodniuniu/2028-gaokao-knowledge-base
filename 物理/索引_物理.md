## 五、重点专题规划（预设计）

### 核心知识网络（建议建设）
- [ ] 运动学：匀变速直线运动公式体系（含v-t图、x-t图）
- [ ] 力学：牛顿三定律+受力分析方法体系
- [ ] 曲线运动：平抛、圆周运动公式与临界条件
- [ ] 万有引力：卫星轨道、宇宙速度、双星问题
- [ ] 能量：功和能、动能定理、机械能守恒、功能关系
- [ ] 动量：动量定理、动量守恒、碰撞/反冲模型
- [ ] 电场：电场强度、电势、电容公式体系
- [ ] 磁场：安培力、洛伦兹力、带电粒子运动
- [ ] 电磁感应：法拉第定律、楞次定律、导轨模型
- [ ] 热学/光学/原子物理：选修模块核心公式

### 典型题型与方法（建议建设）
- [ ] 平衡问题：整体法与隔离法、动态平衡图解法
- [ ] 牛顿定律：连接体问题、传送带模型、板块模型
- [ ] 能量动量：滑块-木板、弹簧、圆弧轨道综合
- [ ] 电场：带电粒子在电场中的加速与偏转
- [ ] 磁场：带电粒子在磁场中的圆周运动（找圆心定半径）
- [ ] 电磁感应：单杆/双杆导轨、线框进出磁场
- [ ] 实验：力学实验（纸带、光电门、碰撞）+电学实验（伏安法、半偏法）
- [ ] **广东情境题**：从生活/科技情境中提取物理模型（如新能源汽车、5G通信、大湾区的桥梁）
- [ ] **广东实验题**：探究性实验设计、数据处理方法

### 易错警示与辨析（建议建设）
- [ ] 运动学：刹车陷阱、往返运动、追及相遇临界
- [ ] 受力分析：摩擦力方向突变、弹簧弹力不可突变
- [ ] 圆周运动：轻绳vs轻杆模型、最高点临界条件
- [ ] 万有引力：黄金代换的适用条件、同步卫星特点
- [ ] 能量：摩擦力做功与路程关系、内能增加量计算
- [ ] 电场：电场线方向与电势高低的关系、电势能正负
- [ ] 电磁感应：感应电量与感应电流的区别、自感现象
- [ ] **广东情境陷阱**：情境信息干扰、非物理条件迷惑
- [ ] **等级赋分**：原始分与赋分的差异、基础题不丢分策略

### 素材与拓展（建议建设）
- [ ] 物理前沿：量子计算、引力波、黑洞研究
- [ ] **广东科技素材**：华为5G、比亚迪新能源汽车、大疆无人机、港珠澳大桥
- [ ] 物理学家：牛顿、爱因斯坦、麦克斯韦、费曼
- [ ] 实验拓展：DIS实验、传感器应用
- [ ] 生活物理：体育运动中的力学分析（跳高、跳水、排球）
- [ ] **大湾区科技**：粤港澳科技创新、深海探测、航天发射（文昌发射场关联）
