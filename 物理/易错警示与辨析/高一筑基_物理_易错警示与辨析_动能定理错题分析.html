错题1：斜面往返动能问题（单选）

题目：物体以100J的初动能从斜面底端沿斜面向上运动，当它向上通过斜面上某一点M时，其动能减少了80J，克服摩擦力做功32J，则物体返回到斜面底端时的动能为（　　）

A. 20J　　B. 48J　　C. 60J　　D. 68J

错因分析：

  • 误把"动能减少80J"直接等同于"重力势能增加80J"，忽略了摩擦力也在消耗能量
  • 没有建立动能减少量 = 重力势能增加量 + 摩擦生热的能量分配关系
  • 对往返过程中摩擦力做功的特点不清晰

正确解法：

从底端 → M点：

  • 动能减少 $\Delta E_k = 80\text{J}$，即 $W_{合} = -80\text{J}$
  • 克服摩擦力做功 $W_{f1} = 32\text{J}$（摩擦力做负功 $-32\text{J}$）
  • 由动能定理：$W_G + W_f = \Delta E_k$
  • 即 $-mgh_M + (-32) = -80$
  • 解得：重力势能增加 $mgh_M = 80 - 32 = 48\text{J}$

关键比例发现：

  • 在上升过程中，每减少80J动能，48J转化为重力势能，32J转化为摩擦热
  • 势能增量 : 摩擦损耗 = $48 : 32 = 3 : 2$

从底端 → 最高点（设总动能减少100J）：

  • 按 $3:2$ 比例分配：势能增量 = $100 \times \frac{3}{5} = 60\text{J}$，摩擦损耗 = $100 \times \frac{2}{5} = 40\text{J}$
  • 即上升过程克服摩擦力做功 40J

往返全过程：

  • 上升克服摩擦力做功 40J，下降克服摩擦力做功 40J（路程相同）
  • 总摩擦损耗 = $40 + 40 = 80\text{J}$
  • 返回底端时动能：$E_k = 100 - 80 = \boxed{20\text{J}}$，选 A

方法提炼：

比例法：当斜面倾角不变、摩擦系数不变时，上升过程中重力做功与摩擦力做功的比例恒定。利用某一点的能量分配比例，可推算全程。

错题2：斜轨道+圆弧轨道模型（单选）

题目：斜轨道ab的下端与半径为 $r = 40\text{cm}$ 的圆轨道平滑连接。现在使质量为 $m = 100\text{g}$ 的小球从斜轨道上端距地面 $h = 2\text{m}$ 的a点由静止滑下，运动到圆形轨道的最高点c处时，对轨道的压力大小等于小球的重力大小，$g$ 取 $10\text{m/s}^2$，则小球从a运动到c的过程中克服摩擦力所做的功为（　　）

A. 0.2J　　B. 0.4J　　C. 0.6J　　D. 0.8J

错因分析：

  • 对"最高点c处压力等于重力"的向心力分析不熟练，列错方程
  • 忘记或错误计算c点相对于a点的高度差（$h - 2r$ 而非 $h$）
  • 动能定理方程中功和动能的符号混乱

正确解法：

第一步：分析最高点c的受力与速度

小球在圆轨道最高点c，对轨道压力等于重力，即 $N = mg$。

由牛顿第三定律，轨道对小球支持力 $N' = mg$（方向向下，与重力同向）。

向心力方程（指向圆心向下）：

$$mg + N' = \frac{mv_c^2}{r}$$

$$mg + mg = \frac{mv_c^2}{r}$$

$$v_c^2 = 2gr = 2 \times 10 \times 0.4 = 8\text{ m}^2/\text{s}^2$$

第二步：从a到c列动能定理

初态（a点）：$v_a = 0$，$E_{ka} = 0$

末态（c点）：$E_{kc} = \frac{1}{2}mv_c^2 = \frac{1}{2} \times 0.1 \times 8 = 0.4\text{J}$

重力做功：小球下降高度为 $h - 2r = 2 - 0.8 = 1.2\text{m}$

$$W_G = mg(h - 2r) = 0.1 \times 10 \times 1.2 = 1.2\text{J}$$

设摩擦力做功为 $W_f$（负值），动能定理：

$$W_G + W_f = E_{kc} - E_{ka}$$

$$1.2 + W_f = 0.4 - 0$$

$$W_f = 0.4 - 1.2 = -0.8\text{J}$$

克服摩擦力所做的功 = $|W_f| = \boxed{0.8\text{J}}$，选 D

方法提炼：

"先找临界速度，再算能量损失"：涉及圆周轨道最高点的题目，通常先通过向心力方程求出临界速度，再通过动能定理或机械能守恒（含摩擦）计算能量损失。

共性知识点总结

考点要点
动能定理的灵活选取可以全程用，也可以分段用；优先全程
摩擦力做功与路径有关，往返运动要乘2；$W_f = -f \cdot s_{路程}$
圆周运动最高点先写 $F_{向} = \frac{mv^2}{r}$，再确定各力方向
能量分配动能减少 = 势能增加 + 内能增加（摩擦热）

高考真题变式训练

命题趋势：动能定理是高考5年4考的高频考点。2024广东卷考查多过程动能定理应用（选择题），2024新课标卷/辽宁卷以计算题形式考查，2025广东卷第2道计算题明确要求"用动能定理求解外力做功和初末速度"。以下变式题均来自近年高考真题同类型。

变式1（2024广东卷同类型 · 多过程+图像）：

一物体从固定斜面底端以一定初速度沿斜面向上滑动，然后返回底端。以底端为原点，沿斜面向上为正方向建立 $x$ 轴。物体动能 $E_k$ 随位移 $x$ 变化的图像如图所示（上升过程为斜向下的直线，下降过程为斜向上的直线，两直线斜率不同）。已知上升最大位移为 $x_0$，上升过程末动能为0，返回底端时动能为 $\frac{1}{5}E_{k0}$。则（　　）

A. 上升过程摩擦力做功为 $\frac{2}{5}E_{k0}$

B. 斜面倾角 $\theta$ 满足 $\sin\theta = \frac{3E_{k0}}{5mgx_0}$

C. 物体与斜面间的动摩擦因数 $\mu = \frac{2}{3}\tan\theta$

D. 上升与下降过程所用时间之比为 $1:2$

解题思路：

  • 上升过程：$E_k$ 从 $E_{k0}$ 减到0，$W_{合} = -E_{k0} = -(mg\sin\theta + \mu mg\cos\theta)x_0$
  • 下降过程：$E_k$ 从0增到 $\frac{1}{5}E_{k0}$，$W_{合} = \frac{1}{5}E_{k0} = (mg\sin\theta - \mu mg\cos\theta)x_0$
  • 联立解得：$mg\sin\theta \cdot x_0 = \frac{3}{5}E_{k0}$，$\mu mg\cos\theta \cdot x_0 = \frac{2}{5}E_{k0}$
  • 故 $\mu = \frac{2}{3}\tan\theta$，选 C

变式2（2022福建卷同类型 · 斜面往返）：

一物块以初速度 $v_0$ 自固定斜面底端沿斜面向上运动，一段时间后回到斜面底端。已知斜面倾角为 $\theta$，物块与斜面间的动摩擦因数为 $\mu$（$\mu < \tan\theta$），重力加速度为 $g$。则物块（　　）

A. 上升过程中加速度大小为 $g(\sin\theta + \mu\cos\theta)$

B. 上升的最大高度为 $\frac{v_0^2}{2g(1 + \mu\cot\theta)}$

C. 返回底端时的速度大小为 $v_0\sqrt{\frac{\sin\theta - \mu\cos\theta}{\sin\theta + \mu\cos\theta}}$

D. 整个过程中摩擦生热为 $\frac{\mu mv_0^2\cos\theta}{2(\sin\theta + \mu\cos\theta)}$

解题思路：

  • 上升：$a_{上} = g(\sin\theta + \mu\cos\theta)$，A正确
  • 最大位移：$x = \frac{v_0^2}{2a_{上}} = \frac{v_0^2}{2g(\sin\theta + \mu\cos\theta)}$
  • 最大高度：$h = x\sin\theta = \frac{v_0^2\sin\theta}{2g(\sin\theta + \mu\cos\theta)} = \frac{v_0^2}{2g(1 + \mu\cot\theta)}$，B正确
  • 全程动能定理：$-2\mu mg\cos\theta \cdot x = \frac{1}{2}mv^2 - \frac{1}{2}mv_0^2$
  • 解得 $v = v_0\sqrt{\frac{\sin\theta - \mu\cos\theta}{\sin\theta + \mu\cos\theta}}$，C正确
  • 摩擦生热：$Q = 2\mu mg\cos\theta \cdot x = \frac{\mu mv_0^2\cos\theta}{\sin\theta + \mu\cos\theta}$，D错误（系数应为1不是1/2）

变式3（2025广东卷计算题同类型 · 动能定理+功率）：

一质量为 $m = 2\text{kg}$ 的物体在水平拉力 $F$ 作用下沿粗糙水平面从静止开始做匀加速直线运动，运动 $t = 4\text{s}$ 后撤去拉力，物体又滑行 $x = 8\text{m}$ 后停止。已知水平面动摩擦因数 $\mu = 0.2$，$g = 10\text{m/s}^2$。求：

（1）撤去拉力时物体的速度大小；

（2）拉力 $F$ 的大小；

（3）拉力 $F$ 的平均功率。

解题思路：

  • （1）撤去拉力后：$-\mu mgx = 0 - \frac{1}{2}mv^2$，$v = \sqrt{2\mu gx} = \sqrt{32} = 4\sqrt{2}\text{m/s}$
  • （2）全过程：$Fs_1 - \mu mg(s_1 + x) = 0$，其中 $s_1 = \frac{v}{2}t = 2\sqrt{2} \times 4 = 8\sqrt{2}\text{m}$... 或用牛顿定律：$F - \mu mg = ma$，$v = at$，$a = \sqrt{2}\text{m/s}^2$，$F = 2\sqrt{2} + 4 = (2\sqrt{2} + 4)\text{N}$
  • （3）$P = F \cdot \bar{v} = F \cdot \frac{v}{2} = ...$（略）
方法归纳：近年高考动能定理考查三大模式：

  1. 斜面往返模式：全程优先，注意摩擦力往返做功与单程的关系
  2. 圆弧临界模式：先向心力方程求速度，再动能定理求能量损失
  3. 多过程+功率模式：分阶段列动能定理，结合功率公式 $P = Fv$ 或 $P = W/t$
