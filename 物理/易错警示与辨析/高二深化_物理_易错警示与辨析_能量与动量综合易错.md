## 核心内容

### 易错点总览表

| 易错点 | 常见错解 | 正确认知 |
|--------|----------|----------|
| **动量守恒条件** | 以为"有外力就不守恒" | 系统**合外力为 0** 才守恒；某方向合外力为 0 则**该方向**守恒 |
| **动量 vs 机械能守恒** | 两守恒混为一谈 | 动量守恒看外力、与参考系无关；机械能守恒看**只有重力/弹力做功** |
| **碰撞类型判定** | 见碰就当弹性 | 弹性（动量和动能均守）、非弹性、完全非弹性（**共速**，动能损失最大） |
| **动量定理方向** | 标量代换、丢方向 | $I=\Delta p$ 是**矢量**，先建坐标正方向 |
| **反冲/爆炸** | 忽略内力远大于外力 | 内力≫外力时近似**动量守恒**（如爆竹、火箭） |
| **人船模型** | 误用单物体动量 | 初动量为 0 则末动量也为 0：$m_1v_1=m_2v_2$（方向相反） |
| **能量损失计算** | 与动量守恒混淆 | 损失的动能 $\Delta E_k=E_{k初}-E_{k末}$，完全非弹性时最大 |

### 易错点 1：动量守恒的"系统"与"方向"

- 守恒条件：**系统所受合外力为零**。注意是"系统"不是"单物体"。
- 若合外力不为零但**某一方向**合外力为零，则**该方向**动量分量守恒（如爆炸瞬间水平方向、斜面光滑时水平方向）。
- 易错：把"地面摩擦"等外力一律视为破坏守恒——若题目明确"光滑""碰撞时间极短（外力冲量可忽略）"，则近似守恒。

### 易错点 2：三类碰撞的能量命运

| 碰撞类型 | 动量 | 动能 | 末态特征 |
|----------|------|------|----------|
| **弹性碰撞** | 守恒 | 守恒 | 可解出 $v_1',v_2'$ 公式 |
| **非弹性碰撞** | 守恒 | 有损失 | — |
| **完全非弹性碰撞** | 守恒 | 损失**最大** | 两物体**共速** $v_1'=v_2'$ |

> 一维弹性碰撞结论（质量 $m_1,m_2$，初速 $v_1,v_2$）：
> $$v_1'=\frac{m_1-m_2}{m_1+m_2}v_1+\frac{2m_2}{m_1+m_2}v_2,\quad v_2'=\frac{2m_1}{m_1+m_2}v_1+\frac{m_2-m_1}{m_1+m_2}v_2$$
> 特例：$m_1=m_2$ 弹性碰撞 → **速度交换**。

### 易错点 3：多过程"分段"衔接

综合题（板块、弹簧、圆弧+碰撞）标准顺序：
1. **判段**：哪一段动量守恒？（看合外力/方向）
2. **定类型**：碰撞属哪类？是否共速？
3. **算能量**：哪一段机械能守恒/有损失？用动能定理或能量守恒补刀。
4. **衔接**：上一段末态 = 下一段初态。

> 关键：动量守恒与机械能守恒**不是同时自动成立**——弹性碰撞才两者皆守；非弹性只守动量。

### 经典易错例（碰撞 + 能量）

**题**：质量 $m$ 的子弹以速度 $v_0$ 水平射入静止于光滑水平面的木块（质量 $M$），子弹留在木块中（完全非弹性）。求系统损失的机械能。

**错解**：直接用弹性碰撞公式，或认为动能守恒。
**正解**：
- 水平面光滑 → 系统动量守恒：$mv_0=(m+M)v$，得共同速度 $v=\dfrac{m}{m+M}v_0$。
- 初动能：$E_{k0}=\dfrac12 mv_0^2$。
- 末动能：$E_k=\dfrac12(m+M)v^2=\dfrac12(m+M)\left(\dfrac{m}{m+M}\right)^2v_0^2=\dfrac{m^2}{2(m+M)}v_0^2$。
- 损失：$\Delta E_k=E_{k0}-E_k=\dfrac12 mv_0^2\left(1-\dfrac{m}{m+M}\right)=\boxed{\dfrac{M}{2(m+M)}mv_0^2}$。
- 说明：完全非弹性碰撞动能损失最大，且全部转化为内能（子弹与木块摩擦生热）。

### 题型识别标志（陷阱信号）

| 题干/选项信号 | 触发陷阱 | 立即动作 |
|------|------|------|
| "光滑水平面 / 碰撞时间极短 / 内力远大于外力" | 误判动量不守恒 | 先判系统合外力是否为零；某方向合外力为零则该方向守恒；碰撞瞬间外力冲量可忽略≈守恒 |
| "两物体相碰" | 见碰就当弹性（动能守恒） | 先定类型：弹性（动量和动能均守）/非弹性/完全非弹性（共速、动能损最大） |
| "子弹打木块 / 板块 / 人船" | 动能损失与动量守恒混淆 | Δ E_k = E_k初 − E_k末；完全非弹性共速时损失最大，全转内能 |
| "斜面上 / 爆炸 / 反冲" | 忽略某方向守恒 | 爆炸/反冲内力≫外力≈守恒；斜面光滑时水平方向分量守恒 |
| "多过程：先碰后滑 / 弹簧+碰撞" | 一段判错全崩 | 固定顺序：先判守恒段→定碰撞类型→算能量损失→上段末态=下段初态 |
| "动量定理 I=Δp" | 标量代换丢方向 | 先建坐标正方向，动量是矢量，按分量列式 |

### 标准化纠错路径（先判守恒 · 再定类型 · 后算能量）

**一、判守恒段**
1. **定系统**：明确研究对象（两物体及以上），不拿单物体当系统。
2. **看外力**：合外力为零→系统动量守恒；某方向合外力为零→该方向分量守恒。
3. **抓瞬间**：碰撞/爆炸时间极短，外力冲量可忽略，近似守恒。

**二、定碰撞类型**
1. **看末态**：是否共速？共速→完全非弹性（动能损最大）；速度可解且动能守→弹性；否则非弹性。
2. **套公式**：弹性用一维结论（m₁=m₂ 则速度交换）；完全非弹性用共速 v=(m₁v₁+m₂v₂)/(m₁+m₂)。

**三、算能量损失**
1. 初动能 E_k初、末动能 E_k末 分别算；Δ E_k = E_k初 − E_k末（完全非弹性最大）。
2. **衔接**：上一段末态速度 = 下一段初态，逐段推进，避免跳步。

### 真题母题（2021 广东卷 T13）

**题**：算盘是我国古老的计算工具，中心带孔的相同算珠可在算盘的固定导杆上滑动，使用前算珠需要归零。如图所示，水平放置的算盘中有甲、乙两颗算珠未在归零位置，甲靠边框 b，甲、乙相隔 s₁ = 3.5×10⁻² m，乙与边框 a 相隔 s₂ = 2.0×10⁻² m，算珠与导杆间的动摩擦因数 μ = 0.1。现用手指将甲以 0.4 m/s 的初速度拨出，甲、乙碰撞后甲的速度大小为 0.1 m/s，方向不变，碰撞时间极短且不计，重力加速度 g 取 10 m/s²。
(1) 通过计算，判断乙算珠能否滑动到边框 a；
(2) 求甲算珠从拨出到停下所需的时间。

**错解**：(1) 直接用弹性碰撞公式求乙碰后速度（本题非弹性，动能不守恒），或漏用动能定理求甲碰前速度，凭感觉答"能/不能"；(2) 只算碰前时间漏算碰后滑行时间，或用位移公式代错。
**正解**：
- 算珠加速度 a = μg = 1 m/s²。
- (1) 甲碰前：由动能定理（或 v²−v₀²=−2as₁）得 v₁ = √(0.4²−2×1×0.035) = √0.09 = **0.3 m/s**。碰撞动量守恒（质量相同）：0.3 = 0.1 + v_乙' ⇒ v_乙' = **0.2 m/s**。乙碰后减速到停位移 s = v_乙'²/(2a) = 0.04/2 = 0.02 m = s₂，**恰好滑到边框 a**，答：能。
- (2) 甲碰前时间 t₁ = (v₀−v₁)/a = (0.4−0.3)/1 = **0.1 s**；碰后甲减速时间 t₂ = v_甲'/a = 0.1/1 = **0.1 s**；总时 t = t₁+t₂ = **0.2 s**。
**避错要点**：本题把"动能定理求碰前速度 + 动量守恒判碰撞 + 再算摩擦滑行"串成一条线，正是本卡"先判守恒段→定碰撞类型→算能量"的顺序。关键易错：①碰撞后甲仍 0.1 m/s、乙 0.2 m/s，动能不守恒 ⇒ 是非弹性碰撞，不能用弹性公式；②求末速度必须用动能定理/运动学（含摩擦做功），不能直接套动量；③第(2)问时间是"碰前+碰后"两段之和，漏一段即错。

## 备注
- **广东卷命题落点**：能量与动量综合几乎必以计算题出现，且常"嵌套"板块/弹簧/圆弧（见典型题型卡片）。解题顺序固定为"先判守恒段→定碰撞类型→算能量损失"。
- **避错口诀**：守恒先看系统合外力；碰撞先分型（弹/非弹/完全非弹）；完全非弹性必共速、动能损最大；动量定理带方向。
- **等级赋分**：该类题是拉开 650 分档的关键压轴，单处守恒条件判错即连锁崩盘；建议把"分段衔接"写成显性步骤，避免跳步。
