### 情境一：广东红树林 —— 湿地生态系统的群落结构与演替

【情境背景】
广东拥有全国面积最大的红树林，湛江红树林国家级自然保护区是我国红树林面积最大的自然保护区，深圳福田红树林位于城市腹地、与香港米埔湿地隔湾相望。红树林是生长在热带、亚热带海岸潮间带的木本植物群落，兼具陆地与海洋生态特征，是广东卷生态情境的头号素材。

【学科原理】
- 生态系统组成成分：红树植物（生产者）→ 底栖动物、鱼类、鸟类（消费者）→ 微生物（分解者）；潮间带淤泥、海水、阳光属非生物的物质和能量。
- 群落结构：垂直结构（乔木层红树—林下幼苗—底栖生物分层分布，提高对阳光等资源的利用能力）；水平结构（随潮位高低呈带状分布，由地形变化、土壤湿度和盐碱度差异造成）。
- 群落演替：光滩 → 先锋红树植物定居 → 红树林成熟群落，属于原生演替（从无植被的裸滩开始）；人为砍伐后红树林的恢复属于次生演替。

【考点链接】
- 群落的物种组成是区别不同群落的重要特征；物种丰富度指群落中物种数目的多少。
- 种间关系：原始合作、互利共生、种间竞争、捕食、寄生。
- 演替类型比较：原生演替（起点无植被无土壤条件，速度慢）与次生演替（起点有土壤条件和繁殖体，速度快）。
- 人类活动会使群落演替按照不同于自然演替的速度和方向进行。

【命题角度】
- 给出红树林不同潮位带的物种分布表，判断体现群落的哪种结构并说明意义。
- 长句表达："红树林退化为光滩后，人工修复为什么比自然恢复更快？"（人为选择适生树种种植、改善环境条件，使演替按更快方向进行）。
- 判断演替类型：围垦弃塘后红树林的恢复属于次生演替（保留了土壤条件和繁殖体）。

### 情境二：防风消浪与促淤保滩 —— 生物多样性的间接价值

【情境背景】
红树林发达的根系和密集的植株能削减风浪、固岸护堤、促进泥沙淤积，是抵御台风暴潮的"海岸卫士"。广东台风多发，红树林的防护功能在生态保护决策中常被作为"不开发而保护"的理由，这正是生物多样性间接价值的典型例证。

【学科原理】
- 生物多样性的价值（人教版2019教材口径）：
  - **直接价值**：食用、药用、工业原料、旅游观赏、科学研究、文学艺术创作等。
  - **间接价值**：调节生态系统功能方面，如防风消浪、固碳释氧、涵养水源、净化水质等——**间接价值明显大于直接价值**。
  - **潜在价值**：目前人类尚不清楚的价值。
- 红树林的防风消浪、促淤保滩、净化海水属于间接价值，而非"生态功能"这种笼统说法。

【考点链接】
- 生物多样性三个层次：遗传（基因）多样性、物种多样性、生态系统多样性。
- 价值判断口诀：**"吃穿用游是间接不对——吃穿用游是直接；调节环境是间接；尚不知是潜在。"**
- 就地保护（建立自然保护区，最有效）与易地保护。

【命题角度】
- 选择题高频：给出红树林若干功能（提供木材、防风消浪、为鸟类提供栖息地、科研价值、未知药用价值），要求归类为直接/间接/潜在价值。
- 长句表达："为什么保护红树林的生态意义远大于直接砍伐利用？"（红树林防风消浪、促淤保滩、维持生物多样性等间接价值远大于木材等直接价值）。
- 易错警示：为鸟类提供栖息地、维持种群 → 偏向间接价值（调节生态功能），不要误判为直接价值。

### 情境三：蓝碳与碳汇 —— 物质循环与碳循环

【情境背景】
红树林、盐沼、海草床被称为"蓝碳"生态系统，其固碳能力突出——红树林单位面积固碳能力强，且碳长期储存在潮间带沉积物中。广东提出红树林保护修复专项行动，蓝碳交易也是大湾区绿色发展的热点。该情境直接对接碳循环与"碳达峰、碳中和"国家战略。

【学科原理】
- 碳循环形式：碳在生物群落与非生物环境（无机环境）之间主要以**CO₂**形式循环；在生物群落内部以**含碳有机物**形式沿食物链（网）传递。
- 碳循环关键环节：①光合作用（及化能合成作用）把CO₂固定为有机物进入生物群落；②呼吸作用、③分解者的分解作用、④化石燃料燃烧把碳以CO₂形式返回大气。
- 红树林固碳机制：红树植物通过光合作用固定CO₂，部分碳储存在植株体内，大量碳随凋落物进入沉积物长期封存。
- 物质循环特点：全球性、循环往复运动（可反复利用）；能量流动则单向流动、逐级递减——两者同时进行、相互依存。

【考点链接】
- 碳循环图解：大气CO₂库 ⇌ 生产者 ⇌ 消费者 → 分解者；化石燃料燃烧是大气CO₂增加的重要原因。
- 碳中和：通过植树造林、节能减排等形式，抵消自身产生的CO₂排放，实现正负相抵。
- 易错点：碳进入生物群落的途径是生产者的光合作用（和化能合成作用），**不是**消费者摄食；碳返回大气以CO₂形式，不是有机物。

【命题角度】
- 流程图补全：给出含红树林生态系统的碳循环模式图，标注各环节（光合作用、呼吸作用、分解作用、捕食）。
- 长句表达："种植红树林为什么有助于实现碳中和？"（红树植物通过光合作用吸收CO₂，将碳固定在生物量与沉积物中，减少大气CO₂含量）。
- 与能量流动对比设问：碳可以循环利用，能量不能——说明原因（能量以热能形式散失，不能再被生产者固定利用）。

### 情境四：红树林生物多样性保护 —— 保护措施与生态工程

【情境背景】
广东通过建立自然保护区、退塘还湿（把围垦养殖塘恢复为红树林湿地）、禁止破坏性开发等措施保护红树林，深圳在城市中心区保留福田红树林保护区是"人与自然和谐共生"的典型案例。

【学科原理】
- 生物多样性丧失原因：生存环境的改变和破坏（栖息地丧失，主要原因）、掠夺式开发利用、环境污染、外来物种入侵等。
- 保护措施：就地保护（建立自然保护区及国家公园，最有效的保护）；易地保护（植物园、动物园、濒危物种繁育中心）；利用生物技术保护（建立精子库、种子库）；加强立法、执法和宣传教育。
- 生态工程基本原理（人教版2019选择性必修2）：**自生、循环、协调、整体**。
  - 自生：生态系统具有自我组织和自我优化能力（合理选择组分、创造有利条件）。
  - 循环：促进物质迁移与转化，减少废弃物。
  - 协调：生物与环境、生物与生物的协调适应，考虑环境容纳量。
  - 整体：兼顾生态、经济、社会效益。

【考点链接】
- 环境容纳量（K值）：一定环境条件所能维持的种群最大数量；引入物种须考虑当地环境容纳量（协调原理）。
- 外来物种入侵危害：缺乏天敌、竞争力强 → 破坏本地生物多样性（如互花米草入侵挤占红树林滩涂——互花米草治理正是广东红树林保护的真实议题）。
- 退塘还湿体现：整体原理（兼顾养殖户转产的生计与生态修复）。

【命题角度】
- 情境问答："红树林修复优先选用本地树种"体现了生态工程的什么原理？（协调原理：本地树种适应当地气候与盐碱环境，避免外来物种入侵）。
- 措施类长句："提出保护红树林生物多样性的两条合理措施"（建立自然保护区就地保护；清理互花米草等外来入侵物种；退塘还湿恢复栖息地）。
- 判断保护类型：将勺嘴鹬等珍稀候鸟的越冬地划为保护区属于就地保护。

### 情境五：南海珊瑚礁 —— 生态系统稳定性

【情境背景】
南海诸岛及周边海域分布有珊瑚礁生态系统。珊瑚礁被誉为"海洋中的热带雨林"，物种丰富度极高，但受海水升温、海洋酸化和人类活动影响，珊瑚白化（珊瑚虫排出共生虫黄藻、失去颜色甚至死亡）问题突出。

【学科原理】
- 生态系统稳定性：生态系统维持或恢复自身结构与功能处于相对平衡状态的能力，包括**抵抗力稳定性**（抵抗干扰、保持原状）和**恢复力稳定性**（遭到破坏后恢复原状）。
- 一般规律：生态系统的组分越多、食物网越复杂（营养结构复杂），自我调节能力越强，抵抗力稳定性越高；但珊瑚礁对环境条件要求苛刻，一旦遭到严重破坏，恢复非常缓慢。
- 自我调节能力的基础：负反馈调节（结果抑制或减弱最初发生的变化），是生态系统维持稳定性的关键机制。

【考点链接】
- 抵抗力稳定性与恢复力稳定性往往呈相反关系（特例：冻原等两者都低）。
- 珊瑚白化的生态学解读：珊瑚虫与虫黄藻互利共生 → 环境胁迫导致共生关系破裂 → 珊瑚失去主要营养来源 → 抵抗力稳定性下降、群落退化。
- 提高生态系统稳定性的措施：控制干扰程度、合理利用；对人类利用强度大的生态系统给予相应物质能量投入。

【命题角度】
- 长句表达："珊瑚礁生态系统抵抗力稳定性较强的原因？"（生物种类多、营养结构复杂、自我调节能力强）。
- 分析题：海水温度异常升高 → 珊瑚白化 → 植食性鱼类减少 → 藻类过度繁殖，要求用负反馈失调/食物网角度分析连锁反应。
- 辨析：不要笼统说"物种多所以恢复力也强"——珊瑚礁遭严重破坏后恢复缓慢，体现恢复力稳定性低的另一面。

### 情境六：海洋牧场 —— 能量流动与种群数量

【情境背景】
广东是海洋大省，在粤东、粤西沿海建设现代化海洋牧场（深水网箱养殖、人工鱼礁增殖放流），是"海上粮仓"战略的一部分。海洋牧场的科学管理直接对应种群数量变化与生态系统能量流动知识。

【学科原理】
- 种群数量"S"形（逻辑斯谛）增长：资源和空间有限时，种群数量先增加后趋于稳定，环境容纳量（K值）附近波动；**K/2时种群增长速率最大**。
- 渔业捕捞原理：捕捞后使种群数量维持在K/2左右，可获得最大持续捕获量（增长速率最大、恢复最快）；过度捕捞使种群低于K/2甚至破坏繁殖能力，资源衰退。
- 能量流动：沿食物链单向流动、逐级递减，相邻营养级间能量传递效率约为10%~20%；人工鱼礁增殖放流可提高初级生产者利用率、优化食物网结构。

【考点链接】
- "J"形增长条件：食物和空间充裕、气候适宜、没有天敌和其他竞争物种（理想条件）；S形增长曲线各阶段特点。
- 能量流动计算：摄入量 = 同化量 + 粪便量；同化量 = 呼吸消耗量 + 用于生长、发育和繁殖的能量。
- 研究能量流动的意义：科学规划、设计人工生态系统，使能量得到最有效的利用；调整能量流动关系，使能量持续高效地流向对人类最有益的部分。

【命题角度】
- 计算题：给出海洋牧场某食物链各营养级同化量，计算传递效率并判断是否合理。
- 长句表达："为什么海洋牧场要在捕捞后保持种群数量在K/2左右？"（K/2时种群增长速率最大，种群恢复最快，可获得最大持续产量）。
- 曲线分析：给出放流后某经济鱼类种群数量变化曲线，标注K值、K/2及合理捕捞时机。

### 情境七：珠江口水质 —— 水体富营养化与生态治理

【情境背景】
珠江口是珠江水系入海口，承接流域内生产生活排水，N、P等营养元素输入较多，局部海域曾出现富营养化并诱发赤潮（有害藻华）。广东持续推进近岸海域污染防治与流域协同治理，是"绿水青山就是金山银山"理念在海洋生态中的实践。

【学科原理】
- 水体富营养化：N、P等无机盐大量排入水体 → 藻类及其他浮游生物大量繁殖 → 表层藻类遮蔽阳光、死亡后被分解者分解消耗大量溶解氧 → 水体缺氧 → 鱼类等动物大量死亡 → 水质恶化（正反馈加剧）。
- 赤潮：海洋中的富营养化现象，某些赤潮藻类还会产生毒素，通过食物链富集危害人类健康。
- 治理思路：控制污染源（减少N、P排放，如污水处理、控制养殖密度）→ 生态修复（种植红树林/海草吸收N、P，投放滤食性贝类）→ 恢复生态系统自我调节能力。

【考点链接】
- 生物富集：难以降解的物质（重金属、某些毒素）沿食物链逐级浓缩，营养级越高体内浓度越高。
- 正反馈与负反馈：富营养化恶化过程属正反馈（结果加强最初变化），是生态系统稳定性被破坏的表现。
- 分解者在物质循环中的作用：将动植物遗体和排遗物中的有机物分解为无机物。

【命题角度】
- 因果链长句："解释富营养化水体中鱼类大量死亡的原因"（藻类大量繁殖，死亡后被需氧微生物分解，消耗水中溶解氧，鱼类因缺氧死亡；同时部分藻类产生毒素）。
- 措施类："从生态工程角度提出珠江口水质治理的措施"（控制N、P排放——协调与整体；构建人工湿地/恢复红树林吸收营养盐——循环与自生）。
- 曲线/数据分析：给出某断面溶解氧、N、P浓度随时间变化，判断富营养化程度并推断原因。

### 广东长句表达答题示范（生态题专项）

> 广东卷生态大题的失分点不在"不会"，而在"说不全"。以下为三类高频设问的示范答法，答题时按"**原理 + 情境对接 + 结论**"三段展开。

**示范1 · 原因类**
问：红树林退化为光滩后，自然恢复为成熟红树林需要很长时间，请从群落演替角度说明原因。
答：红树林退化后若土壤条件和繁殖体严重破坏，演替需从较低级的群落阶段重新开始（原理）；光滩缺乏红树幼苗等繁殖体来源，潮间带高盐、潮汐冲刷的环境使先锋物种定居困难（情境对接）；因此演替速度慢、恢复到成熟群落需要较长时间（结论）。

**示范2 · 依据类**
问：海洋牧场对某经济鱼类实行"限额捕捞、捕捞后种群数量维持在K/2左右"，其生物学依据是什么？
答：该鱼类种群数量呈"S"形增长（原理）；种群数量为K/2时增长速率最大，捕捞后种群能最快恢复（情境对接）；因此维持在K/2左右可实现最大持续捕获量，避免资源衰退（结论）。

**示范3 · 措施类**
问：为提升珠江口近岸海域水质，从生态系统物质循环角度提出两条措施，并说明理由。
答：①控制含N、P污水排放——减少进入水体的营养盐，从源头抑制藻类过度繁殖；②修复红树林等滨海湿地——红树植物可吸收利用水中N、P，促进营养物质向生物量转化、减少水体营养盐负荷。

> **扣分警示**：只写"因为要保护环境""因为红树林好"不给分；必须出现生态学术语（演替、环境容纳量/K值、增长速率、物质循环、负反馈等）并与情境事实对应。

- 易错提醒：
  1. 生物多样性价值归类是高频失分点：防风消浪、固碳、净化水质 → 间接价值；"间接价值大于直接价值"是教材明确结论。
  2. 生态工程四原理"自生、循环、协调、整体"不要写成旧教材术语（如"整体性原理""协调与平衡原理"——按2019版教材表述为准，注意教材版本变化，以学校使用教材为准）。

  4. K/2对应"增长速率最大"而非"增长速率最大时种群数量最多"。
