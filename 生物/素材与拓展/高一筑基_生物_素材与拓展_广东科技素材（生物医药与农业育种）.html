广东科技素材（生物医药与农业育种）

⚠️待强化★★★☆☆#高一筑基#广东选择性考试来源：广东生物医药产业（深圳、广州）/ 华南农业育种实践 / 教材选择性必修3《生物技术与工程》、选择性必修1《稳态与调节》
广东考情：考查频率：高频（基因工程、免疫调节、育种几乎每年在广东选择性考试中出现，常以科技或农业情境为载体）；难度定位：中档为主，基因工程与实验设计结合时可至中高档；广东卷特色：情境化命题突出，偏好以本地生物医药企业、华南农作物（水稻、荔枝）为素材，要求从产业背景中提取生物学模型，并常以长句表达形式考查原理阐述；等级赋分提示：生物为等级赋分科目，中档情境题是原始分拉开差距的主战场，务必保证原理表述准确、术语规范，避免"会做但说不全"的隐性失分。

核心内容

素材卡定位：本卡不是知识点讲解，而是"广东产业情境 → 生物学原理 → 考点链接 → 命题角度"的映射库。看到题目中出现"深圳某生物公司""华南稻田""荔枝果园"等字眼时，按本卡快速对位提取模型。

情境一：深圳/广州生物医药产业 —— 基因工程与单克隆抗体

【情境背景】 深圳、广州是粤港澳大湾区生物医药产业核心城市，聚集了大量基因工程药物、抗体药物研发企业。单克隆抗体药物（如抗肿瘤、抗自身免疫病抗体）是该产业的重要产品，其制备技术正是教材中"动物细胞融合与单克隆抗体"的直接应用，是广东卷情境化命题的高频素材。

【学科原理】

  • 动物细胞融合：诱导因素有灭活的病毒、聚乙二醇（PEG）、电激等；细胞膜具有流动性是细胞融合的结构基础。
  • 单克隆抗体制备：将免疫小鼠的B淋巴细胞与骨髓瘤细胞融合，形成杂交瘤细胞；杂交瘤细胞既能无限增殖，又能产生特异性抗体。
  • 两次筛选：第一次用选择培养基（HAT培养基）筛选出杂交瘤细胞；第二次用克隆化培养和抗体检测筛选出能产生所需特异性抗体的杂交瘤细胞。

【考点链接】

  • 单克隆抗体的优点：特异性强、灵敏度高，并可大量制备。
  • 单克隆抗体制备流程：注射特定抗原免疫小鼠 → 取脾细胞（含已免疫的B淋巴细胞）→ 与骨髓瘤细胞融合 → 选择培养基筛选 → 克隆化培养与专一抗体检测 → 体外培养或注入小鼠腹腔获取单克隆抗体。
  • 相关概念：抗原、抗体、体液免疫；细胞膜的流动性。

【命题角度】

  • 给出"某公司利用杂交瘤技术生产抗××单克隆抗体"的情境，要求补全流程图、指出两次筛选的目的与方法。
  • 长句表达："为什么要先向小鼠注射特定抗原？"（使小鼠体内产生能分泌所需抗体的B淋巴细胞）。
  • 辨析：B淋巴细胞与骨髓瘤细胞各自提供什么特性；为什么不能直接用B淋巴细胞大量培养。

情境二：疫苗研发 —— 免疫调节

【情境背景】 广东是疫苗研发与生产大省，新冠疫情期间广东企业参与灭活疫苗、重组蛋白疫苗等研发。疫苗是免疫调节知识最经典的应用情境，广东卷常以"某疫苗接种后机体的免疫过程"为背景命题。

【学科原理】

  • 疫苗的本质：通常是灭活或减毒的病原体、病原体的抗原成分（如蛋白质、mRNA），能刺激机体产生特异性免疫应答但不致病或致病性极弱。
  • 初次免疫与二次免疫：接种疫苗后，B细胞增殖分化为浆细胞和记忆B细胞，浆细胞产生抗体；再次接触相同抗原时，记忆细胞迅速增殖分化，产生更快、更强的免疫反应（抗体浓度更高、反应更快）。
  • 体液免疫与细胞免疫的协作：辅助性T细胞分泌细胞因子，促进B细胞和细胞毒性T细胞的增殖分化。

【考点链接】

  • 体液免疫过程：抗原 → 抗原呈递细胞摄取处理 → 辅助性T细胞 → B细胞 → 浆细胞（产生抗体）+ 记忆B细胞。
  • 二次免疫应答曲线特点：潜伏期短、抗体峰值高、维持时间长。
  • 相关概念：抗原、抗体、记忆细胞、免疫活性物质（抗体、细胞因子、溶菌酶）。

【命题角度】

  • 以"接种某疫苗后，间隔一段时间需加强针"为情境，用二次免疫原理解释加强针的作用（长句表达高频问法）。
  • 曲线分析：给出血清抗体浓度随时间变化曲线，判断初次应答与二次应答。
  • 辨析：疫苗预防（主动免疫）与注射抗体治疗（被动免疫）的区别。

情境三：CAR-T细胞疗法 —— 细胞工程与免疫

【情境背景】 CAR-T（嵌合抗原受体T细胞）疗法是肿瘤免疫治疗的前沿技术，国内多家生物医药企业（含广东企业）开展相关研发与临床试验。该疗法把"基因工程改造细胞"与"细胞免疫杀伤肿瘤"结合，是考查细胞免疫与基因工程的综合性新情境。

【学科原理】

  • 基本思路：从患者体内分离T细胞，通过基因工程技术将编码嵌合抗原受体（CAR）的基因导入T细胞，使其能特异性识别肿瘤细胞表面抗原，再扩增后回输患者体内，定向杀伤肿瘤细胞。
  • 免疫学基础：细胞毒性T细胞能识别被病原体感染的宿主细胞、肿瘤细胞等靶细胞，并使其裂解死亡（细胞免疫）。
  • 基因工程基础：目的基因（CAR基因）需构建基因表达载体后导入受体细胞（T细胞）。

【考点链接】

  • 细胞免疫过程：细胞毒性T细胞的活化、增殖分化、识别并裂解靶细胞。
  • 基因工程基本步骤：目的基因的筛选与获取 → 基因表达载体的构建 → 将目的基因导入受体细胞 → 目的基因的检测与鉴定。
  • 免疫系统功能：免疫监视（清除突变细胞，防止肿瘤发生）。

【命题角度】

  • 以CAR-T为情境，问"CAR-T细胞能特异性识别肿瘤细胞的结构基础是什么"（CAR与肿瘤表面抗原特异性结合，体现细胞膜信息交流功能）。
  • 分析"用患者自身T细胞而不用他人T细胞"的原因（避免免疫排斥/移植物抗宿主反应）。
  • 注意：该情境超出教材深度，题目信息会给足，关键是把信息对接到细胞免疫与基因工程的教材框架上，不要脑补教材外细节。

情境四：杂交水稻 —— 杂交育种与杂种优势

【情境背景】 华南地区是杂交水稻的重要推广与育种区域，袁隆平团队及华南农业大学等科研机构长期开展水稻育种研究。杂交水稻"三系配套"育种是我国农业科技自主创新的标志，也是广东卷农业情境的经典素材。

【学科原理】

  • 杂交育种原理：基因重组。将两个或多个品种的优良性状通过交配集中在一起，再经选择和培育获得新品种。
  • 杂种优势：基因型不同的亲本杂交，F₁在生长势、产量、抗逆性等方面优于双亲的现象。
  • 杂种优势的特点：F₁代优势最强，F₂代发生性状分离、优势衰退，因此生产上杂交种需年年制种，不能留种自留。

【考点链接】

  • 杂交育种流程：杂交 → 自交 → 选优 → 连续自交（纯化）；若利用杂种优势则每年杂交制种。
  • 孟德尔自由组合定律：非同源染色体上非等位基因的自由组合是基因重组的主要来源之一。
  • 育种方法比较：杂交育种（基因重组）、诱变育种（基因突变）、单倍体育种（染色体数目变异，明显缩短育种年限）、多倍体育种（染色体数目变异）。

【命题角度】

  • 给出两个纯合亲本（如抗病矮秆 × 感病高秆），要求写出选育纯合新品种的遗传图解并说明原理。
  • 长句表达："为什么杂交水稻的F₂代不宜留种？"（F₂发生性状分离，杂种优势衰退，产量下降）。
  • 育种方案评价：给情境选最合理的育种途径（如需快速获得纯合子 → 单倍体育种）。

情境五：华南双季稻 —— 光合作用与育种

【情境背景】 广东属亚热带季风气候，热量充足、无霜期长，水稻一年两熟（早稻+晚稻），是我国重要的双季稻产区。双季稻生产中的"提高光能利用率""选育耐弱光/耐高温品种"等问题，是光合作用与育种知识的天然情境。

【学科原理】

  • 水稻是C₃植物，光合速率受光照强度、CO₂浓度、温度、水分、矿质元素（如N、Mg）等因素影响。
  • 光能利用率与产量：合理密植可充分利用光能、增大光合面积，但过密导致下部叶片光照不足、呼吸消耗增大，净光合速率下降。
  • 双季稻的生育期安排本质是"抢积温"，育种上需选育感光性、感温性适宜的早熟高产品种。

【考点链接】

  • 光合作用总反应式：CO₂ + H₂O →(光能、叶绿体） (CH₂O) + O₂。
  • 净光合速率 = 真正（总）光合速率 − 呼吸速率；有机物积累量取决于净光合速率。
  • 影响光合速率的环境因素曲线：光补偿点、光饱和点；CO₂补偿点、CO₂饱和点。
  • 提高农作物产量的措施：合理密植、间作套种、增施有机肥（提供CO₂和矿质元素）、合理灌溉。

【命题角度】

  • 以"早稻抽穗期遇连续阴雨导致减产"为情境，从光照强度影响光合速率的角度分析原因（长句表达）。
  • 曲线分析：给出不同密度下水稻群体光合速率与呼吸速率曲线，判断最适种植密度。
  • 实验设计：探究"增施CO₂对大棚水稻产量的影响"，考查自变量、因变量、无关变量控制。

情境六：荔枝/龙眼 —— 植物激素与无子果实

【情境背景】 广东是荔枝、龙眼的主产区（茂名、增城等地闻名）。荔枝的保花保果、催熟保鲜、无子化处理等生产技术，直接对应植物激素调节知识，是广东卷极具本土特色的命题素材。

【学科原理】

  • 生长素作用特点：低浓度促进生长，高浓度抑制生长（作用具有两重性）；促进子房发育为果实（无子果实原理：未受粉的雌蕊柱头上涂抹一定浓度生长素类似物，子房壁发育成果皮，形成无子果实）。
  • 乙烯：促进果实成熟（荔枝采后催熟与保鲜贮藏原理）。
  • 无子果实两种类型辨析：生长素诱导的无子番茄/无子荔枝属于不可遗传变异（遗传物质未改变，只影响当代）；无子西瓜属于可遗传变异（三倍体，染色体数目变异，减数分裂时联会紊乱不能正常形成配子）。

【考点链接】

  • 五种植物激素：生长素、赤霉素（促进细胞伸长、促进种子萌发和果实发育）、细胞分裂素（促进细胞分裂）、脱落酸（抑制细胞分裂、促进叶和果实衰老脱落）、乙烯（促进果实成熟）。
  • 生长素两重性实例：顶端优势、根的向地性。
  • 植物生长调节剂：人工合成的对植物生长发育有调节作用的化学物质（如NAA、2,4-D）。

【命题角度】

  • 以"果农对荔枝喷施生长素类似物保果"为情境，说明原理并解释浓度不能过高的原因（两重性）。
  • 辨析题：无子荔枝与无子西瓜的成因及可遗传性比较（高频易错点）。
  • 长句表达："为什么用乙烯利处理可使荔枝提早上市？"（乙烯利释放乙烯，乙烯促进果实成熟）。

情境七：转基因抗虫作物 —— 基因工程

【情境背景】 转基因抗虫作物（如含Bt毒蛋白基因的抗虫棉）是基因工程在农业上的标志性应用，华南地区农业科研中也常涉及抗虫、抗病基因工程育种。该情境是基因工程大题最经典的命题模板。

【学科原理】

  • 原理：将苏云金芽孢杆菌的Bt毒蛋白基因（目的基因）导入植物细胞，使植物表达Bt毒蛋白，害虫取食后中毒死亡，从而减少农药使用。
  • 基因工程的理论基础：不同生物共用一套遗传密码，DNA是主要的遗传物质且结构相同（双螺旋），故目的基因可在受体细胞中表达。
  • 安全性与检测：需检测目的基因是否插入、是否转录出mRNA、是否翻译成蛋白质（个体水平还需做抗虫接种实验）。

【考点链接】

  • 基因工程操作工具：限制性内切核酸酶（"分子手术刀"）、DNA连接酶（"分子缝合针"）、载体（"分子运输车"，常用质粒）。
  • 操作四步：目的基因的筛选与获取 → 基因表达载体的构建（核心步骤）→ 将目的基因导入受体细胞（植物常用农杆菌转化法）→ 目的基因的检测与鉴定。
  • 检测方法：DNA分子杂交/PCR检测基因插入；分子杂交检测转录；抗原—抗体杂交检测翻译；个体生物学水平鉴定抗虫性。

【命题角度】

  • 流程补全：给出抗虫基因工程流程图，要求填工具酶、方法、检测手段。
  • 长句表达："为什么Bt毒蛋白基因能在棉花细胞中表达？"（不同生物共用一套遗传密码）。
  • 辨析：转基因作物带来的收益（减少农药）与潜在风险（基因漂移、抗性进化）的辩证分析。

备注

  • 本卡为生物"素材与拓展"维度破零批次之一，高一阶段以"见到情境能说出对应教材章节"为达标线，不要求掌握全部细节；基因工程、免疫细节到高二学选择性必修时再深化。
  • 使用建议：每次月考/期中考前浏览一遍，遇到情境题先问三个问题——①这是哪个产业/农业情境？②对应教材哪个原理？③题目要我"解释原因"还是"补全流程"？
  • 易错提醒：
    1. 无子果实的两种成因（不可遗传 vs 可遗传）是高频陷阱，务必区分"当代性状"与"遗传物质改变"。
    2. 单克隆抗体制备的"两次筛选"目的不同，答题时不能合并表述。
    3. 杂种优势≠杂交育种能稳定遗传，F₂衰退是必考点。
  • 等级赋分提示：情境题中档为主，长句表达若术语不规范（如把"浆细胞"写成"效应B细胞"在旧新教材混用、把"杂交瘤细胞"写成"融合细胞"）会被扣分，原始分差距在赋分后可能放大。
