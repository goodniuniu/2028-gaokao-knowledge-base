代谢与调节易错点辨析（光合计算 + 神经体液调节）

⚠️待强化★★★★☆#高二深化#广东选择性考试来源：人教版必修1第5章 + 选择性必修1第2~3章 + 2023 广东选择性考试·第18题、第8题
广东考情：考查频率：高频（代谢与调节是广东卷选择题+非选择题的常年主场，2023 第7、8、18 题连续命中）；难度：代谢计算偏中档、调节机制偏复杂；特色：广东卷重"净光合/总光合辨析""光饱和点/光补偿点图表分析"，调节题常结合真实情境（血糖、运动、神经肌肉接头）考查概念精准度；赋分提示：代谢与调节约占卷面 30+ 分，概念混淆一分全丢，等级赋分下基础题务必零失误。

核心内容

一句话区分代谢与调节的易错本质：代谢错在"量"（净光合还是总光合、呼吸算没算），调节错在"向"（兴奋/抑制、升糖/降糖、交感/副交感）。

一、代谢（光合作用与细胞呼吸）易错辨析表

易混概念正确含义典型误判
净光合速率光照下测得的"表观"速率 = 总光合 − 呼吸把"光照下释放 O₂/吸收 CO₂"当成总光合
总（真正）光合速率叶绿体实际固定量 = 净光合 + 呼吸漏加呼吸，直接用净光合当总光合
"产生 O₂ / 固定 CO₂"指总光合（叶绿体产生量）与"释放/吸收"混淆
"释放 O₂ / 吸收 CO₂"指净光合（叶肉对外净变化）与"产生/固定"混淆
光补偿点光合速率 = 呼吸速率时的光照强度与光饱和点颠倒
光饱和点光合速率不再随光强增加时的光强误认为"光合最大处光强无限大"
密闭容器"CO₂ 先降后稳"初期光合 > 呼吸，后期光合 = 呼吸误判"光合速率一直升高"
整株干重 vs 叶肉整株有根部等不光合器官，仍呼吸耗有机物用"叶片净光合=0"推"整株干重不变"

二、调节（神经—体液—免疫）易错辨析表

易混概念正确含义典型误判
胰岛 A 细胞 / 胰岛 B 细胞A 泌胰高血糖素（升糖），B 泌胰岛素（降糖）细胞与激素张冠李戴
交感神经兴奋使血糖升高（动员能量）误以为"放松"会降血糖
"空腹"指未进食，但胰岛等仍在分泌激素误判"空腹时不分泌激素"
兴奋性递质 / 抑制性递质前者使突触后膜兴奋，后者使其抑制见"释放 Glu"就判兴奋（需看作用）
神经递质作用对象可作用于神经元、肌肉、腺体误限于"神经元之间"
激素"作用于靶器官"需靶细胞有对应受体误认为激素无差别作用于全身

三、典型错例集锦

错例错因正解
"光照下测得的 O₂ 释放量就是总光合"混淆净光合与总光合总光合 = 净光合（释放量）+ 黑暗呼吸量
"温度 a 与 c 净光合相等 → 有机物积累速率相等"忽略两温度呼吸速率不同积累速率 = 净光合 − 该温度呼吸，呼吸不同则积累不同
"叶片光合=呼吸时整株干重不变"忘根部不光合仍呼吸整株干重会减少（根部持续耗有机物的呼吸）
"空腹时糖尿病患者胰岛细胞不分泌激素"误解"空腹"含义空腹仍分泌胰岛素/胰高血糖素维持稳态
"紧张时交感神经兴奋使血糖降低"交感应激升糖记忆错交感神经兴奋→升血糖，供能应对应激
"Glu 是递质就使后膜兴奋"未看具体突触功能需结合图示判断是兴奋还是抑制性作用

题型识别标志（陷阱信号）

题干/选项信号触发陷阱立即动作
"光照下释放 O₂ / 吸收 CO₂"问是总光合还是净光合代谢陷阱："净"vs"总"混淆释放/吸收 = 净光合；产生/固定 = 总光合；总 = 净 + 呼吸
图表"光补偿点/光饱和点""温度 a 与 c 净光合相等"代谢陷阱：看错轴/忽略呼吸先标坐标轴含义（净/总、光强/温度），再下笔；呼吸不同则积累不同
"整株干重 vs 叶肉""叶片光合=呼吸"代谢陷阱：漏根部呼吸整株有根等不光合器官仍呼吸，干重会减少
"胰岛 A/B 细胞""交感神经兴奋"问升糖/降糖调节陷阱：细胞与激素张冠李戴A 泌胰高血糖素（升糖），B 泌胰岛素（降糖）；交感神经兴奋→升血糖
"空腹时""Glu 是递质"调节陷阱：误解范围/作用空腹仍分泌激素；见 Glu 需看突触功能判兴奋/抑制

标准化纠错路径（代谢/调节 双线分流法）

一、代谢线（先辨"率"）

  1. 题干出现"释放/吸收"→ 锁定净光合；出现"产生/固定"→ 锁定总光合。
  2. 关系：总光合 = 净光合 + 呼吸；密闭容器"CO₂ 先降后稳"= 后期光合 = 呼吸。
  3. 图表题铁律：先看坐标轴（净/总？光强/温度？），再看曲线，绝不"看错轴"。

二、调节线（先判"向"）

  1. 见到腺体/激素 → 判细胞与激素对应关系（A 升糖、B 降糖）。
  2. 见到神经 → 判兴奋/抑制、交感/副交感（交感应激升血糖）。
  3. 画箭头比死记更稳：升糖↑/降糖↓、兴奋↑/抑制↓。

真题母题（2023 广东选择性考试 T8·T18）

（1）2023 广东选择性考试·第18题（节选，光合计算）

水稻野生型（WT）与黄绿叶突变体（ygl）相关生理特征表（叶绿素、类胡萝卜素及比值）。
(2) …比较两者的光饱和点，可得 ygl 高于 WT（填"高于/低于/等于"）。ygl 有较高的光补偿点，可能的原因是叶绿素含量较低和 呼吸速率较高。
(3) 与 WT 相比，ygl 叶绿素含量低，高密度栽培条件下更多光可到达下层叶片，且 ygl 群体净光合速率较高，表明该群体 光能利用率较高。
易错点：光饱和点=最大净光合所需最小光强，ygl 呼吸高→需更强光才达最大实际光合，故光饱和点更高；光补偿点同理受"叶绿素低+呼吸高"双重拉高。

（2）2023 广东选择性考试·第8题（选择，调节）

空腹血糖是糖尿病筛查常用指标，但易受运动和心理状态干扰。下列叙述正确的是（ ）
A. 空腹时健康人血糖水平保持恒定 B. 空腹时糖尿病患者胰岛细胞不分泌激素
C. 运动时血液中的葡萄糖只消耗没有补充 D. 紧张时交感神经兴奋使血糖水平升高
答：D。易错点：A 错在"恒定"（血糖动态平衡非绝对恒定）；B 错在"不分泌"（空腹仍分泌激素）；C 错在"只消耗无补充"（肝糖原分解等补充）；D 正确——交感神经兴奋升血糖。

错解 → 正解（两题合并）

  • T18 错解：把"光饱和点"等同于"最大净光合所需最小光强"的反面，或只盯叶绿素低、漏掉"呼吸速率较高"也会拉高光补偿点；正解：ygl 呼吸高 → 需更强光才达最大实际光合，故光饱和点更高，光补偿点也更高（受"叶绿素低 + 呼吸高"双重拉高）。
  • T8 错解：凭直觉选 A（"恒定"）或 C（"只消耗无补充"）；正解：血糖是动态平衡非绝对恒定，运动时肝糖原分解等持续补充，紧张时交感神经兴奋升血糖，故选 D。

避错要点：代谢题失分 90% 在"量"（净/总、呼吸算没算），调节题失分 90% 在"向"（升/降、兴奋/抑制、交感/副交感）。做题先用本卡"双线分流"——代谢线先定净/总再算，调节线先画箭头定方向，可同时规避 T18 的"光补偿点双因子"与 T8 的"交感神经升糖"两类高频陷阱。
