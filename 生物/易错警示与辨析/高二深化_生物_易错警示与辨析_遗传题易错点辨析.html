遗传题易错点辨析（概率计算与条件陷阱）

⚠️待强化★★★★☆#高二深化#广东选择性考试来源：人教版必修2第1~2章 + 2022 广东选择性考试·第16题（镰状细胞贫血家系基因检测）
广东考情：考查频率：高频（广东卷遗传大题/选择题几乎年年出现，2022 第16、2024 第18 家系判定）；难度：中高档，失分主因在"概率条件"而非"遗传规律本身"；特色：广东卷常以真实遗传病研究/基因检测电泳图为情境，把"系谱判定+概率计算+生命伦理"打包；赋分提示：遗传题原始分高（选择 2~4 分/空、大题 12~14 分），一处条件概率算错满盘皆输，等级赋分下是拉分主战场。

核心内容

一句话点破易错本质：遗传题 90% 的失分不是不会"分离/自由组合"，而是没看清"条件"与"范围"——谁的基因型已知、谁的概率要加权、算的是"所有孩子中"还是"男孩中"。

一、五大高频易错点辨析表

易错点正解典型误判
"表型正常"≠"不携带"隐性病中正常双亲生患病子女 → 双亲必为杂合（Aa）以为父母正常就不带致病基因
"已知表型正常"要加权正常后代基因型为 1/3 AA、2/3 Aa（在患者同胞中）直接当 Aa（概率 1）算
传递/生育致病基因概率先判自身基因型概率，再 ×1/2；如女儿 = 2/3×1/2 = 1/3想当然写 1/2
"患病男孩" vs "男孩患病"患病男孩 = 所有孩子中患病且男 = 患病概率×1/2；男孩患病 = 男孩中患病概率（常染色体病=患病概率）两者混为一谈
"携带致病基因概率"范围指含一个 a 即可（Aa 或 aa 都算"携带/带有"），常 = 3/4只算 Aa 漏掉 aa

二、概率计算的"三步防错法"

第一步：先定遗传方式

  • 无中生有 → 隐性；有中生无 → 显性；伴性看"男多女少/女患父子必病"等。
  • 隐性病 + 正常双亲生患病子 → 双亲必杂合（这是广东卷最高频隐含条件）。

第二步：列出已知个体的基因型概率（不是直接写基因型）

  • 患者同胞中"表型正常者"：AA 概率 $1/3$，Aa 概率 $2/3$（不是在算 1/2！）。
  • 若题干已用电泳/基因检测确定某人为 Aa，则该人概率就是 1，不再加权。

第三步：按"范围"算目标概率

  • "传递给下一代"：自身为 Aa 才算，故 = $P(\text{Aa}) \times 1/2$。
  • "生育一个患病子代"：依遗传方式用配子法。
  • "男孩/女孩"条件：常染色体病中加 ×1/2；伴性病中范围限定在对应性别。

三、典型错例集锦

错例错因正解
"父母正常，女儿正常，故女儿基因型为 Aa"忽略正常同胞有 1/3 AA 可能女儿 = 1/3 AA、2/3 Aa
"女儿把致病基因传给下一代概率 1/2"把女儿当确定 Aa2/3×1/2 = 1/3
"第三胎携带致病基因概率 = 1/2"只算 Aa，漏 aaAa+aa = 2/4+1/4 = 3/4
"患病男孩概率 = 男孩患病概率"未区分"所有孩子"与"男孩中"常染色体病中相差 ×1/2
"父母正常→不携带致病基因"混淆表型与基因型隐性病双亲生患儿必为杂合携带者

题型识别标志（陷阱信号）

题干/选项信号触发陷阱立即动作
"表型正常"却问基因型/携带概率易错点：表型≠不携带隐性病中正常双亲生患病子→双亲必杂合(Aa)
"已知表型正常"要算传递/携带概率易错点：忽略加权正常同胞基因型 1/3 AA、2/3 Aa，不是直接当 Aa
"传递给下一代/生患病子代"易错点：漏×1/2自身为 Aa 才算，概率 = P(Aa)×1/2
"患病男孩" vs "男孩患病"易错点：范围混淆患病男孩=所有孩子中患病且男；男孩患病=男孩中患病概率
"携带致病基因概率"易错点：范围漏 aa含一个 a 即可（Aa 或 aa），常 = 3/4

标准化纠错路径（定方式—列概率—算范围 三步法）

一、定遗传方式

  1. 无中生有→隐性；有中生无→显性；伴性看"男多女少/女患父子必病"。
  2. 隐性病 + 正常双亲生患病子 → 双亲必杂合（广东卷最高频隐含条件）。

二、列已知个体基因型概率（不是直接写基因型）

  1. 患者同胞中"表型正常者"：AA 1/3、Aa 2/3。
  2. 若题干已用检测确定某人为 Aa，则该人概率 = 1，不再加权。

三、按"范围"算目标概率

  1. "传递给下一代"：自身为 Aa 才算 → P(Aa)×1/2。
  2. "生育患病子代"：依遗传方式用配子法。
  3. "男孩/女孩"条件：常染色体病加 ×1/2；伴性病范围限定对应性别。

真题母题（2022 广东选择性考试 T16）

（2022 广东选择性考试·第16题，单选）

一对表型正常的夫妇，生育了一个表型正常的女儿和一个患镰状细胞贫血症的儿子（致病基因位于 11 号染色体，单对碱基突变）。家庭成员进行了基因检测（电泳图，略）。下列叙述错误的是（ ）
A. 女儿和父母基因检测结果相同的概率是 2/3
B. 若父母生育第三胎，此孩携带该病致病基因的概率是 3/4
C. 女儿将该病致病基因传递给下一代的概率是 1/2
D. 该家庭的基因检测信息应受到保护，避免基因歧视

分析与答案：C

  • 该病为常染色体隐性（设 A/a），父母均 Aa，儿子 aa，女儿表型正常 → 基因型 1/3 AA、2/3 Aa。
  • A 对：父母均为杂合 Aa（检测结果相同），女儿为 Aa 的概率 2/3 → "相同概率 2/3"正确。
  • B 对：第三胎基因型 1/4 AA、2/4 Aa、1/4 aa，携带致病基因（含 a）= Aa+aa = 3/4。
  • C 错：女儿传递致病基因概率 = $P(\text{Aa}) \times 1/2 = 2/3 \times 1/2 = 1/3$，并非 1/2。
  • D 对：基因隐私应受保护。
  • 答：C。本题核心陷阱正是 C——把"女儿"当成确定杂合，漏掉 1/3 AA 的加权。

错解 → 正解

  • 错解：把"表型正常的女儿"直接当成杂合子 Aa，于是"女儿传递致病基因概率 = 1/2"，掉进 C 选项的陷阱；同理易把"第三胎携带概率"只算 Aa 漏掉 aa。
  • 正解：儿子 aa → 父母均 Aa；女儿表型正常但其基因型为 1/3 AA、2/3 Aa（在患者同胞中加权）。故女儿传递 = 2/3×1/2 = 1/3（C 错）；第三胎 = 1/4 AA + 2/4 Aa + 1/4 aa，携带(含 a) = 3/4（B 对）；女儿与父母检测结果相同(均 Aa)概率 = 2/3（A 对）。

避错要点：遗传题 90% 失分在"条件与范围"而非规律本身。本题核心陷阱即本卡"已知表型正常要加权"——绝不在脑中默认 Aa，先写"谁的基因型概率是多少"再代入。C 选项故意把女儿当确定杂合，正是广东卷"条件概率"设坑的高频手法；做完用配子法回验一次可防翻车。
