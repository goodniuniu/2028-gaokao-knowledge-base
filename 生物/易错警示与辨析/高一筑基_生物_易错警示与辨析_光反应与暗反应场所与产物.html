光反应与暗反应场所与产物辨析

⚠️待强化★★★☆☆#高一筑基#广东选择性考试来源：人教版必修1第5章第4节"光合作用与能量转化" + 近5年广东选择性考试光合作用高频考点
广东考情：高频，近5年广东选择性考试几乎每卷必考光合作用，常以"场所/物质变化/C3、C5含量变化"设问，并常与呼吸作用综合。难度定位中档。广东卷特色：爱用**真实实验情境**（如"夜间测定密闭大棚CO2变化""遮阴处理对产量的影响"）包装考点，并以**长句表达**要求说明"C3含量上升的原因"等，只背结论不会表述照样丢分。等级赋分提示：光合作用属基础主干，原始分中这类题必须全拿，错一题赋分后差距会被放大。

核心内容

本质理解：光合作用两阶段的辨析错误，几乎都可归结为三个根源——场所记串、[H]和ATP的来源去向搞混、C3/C5变化分析方向反了。记住一句话："光反应供能供氢（[H]、ATP），暗反应耗能耗氢（固定还原CO2）；场所一个类囊体薄膜，一个叶绿体基质。"

一、光反应 vs 暗反应对比总表（核心记忆框架）

对比维度光反应暗反应（碳反应）
场所类囊体薄膜（基粒）叶绿体基质
条件必须有光、色素、酶有光无光均可（但依赖光反应产物）
物质变化①水的光解：H2O → [H] + O2；②ATP的合成：ADP + Pi + 光能 → ATP①CO2的固定：CO2 + C5 → 2C3；②C3的还原：2C3 + [H] + ATP → (CH2O) + C5
能量变化光能 → ATP中活跃的化学能ATP中活跃的化学能 → 有机物中稳定的化学能
[H]的来源/去向来源：水的光解；去向：供暗反应还原C3消耗[H]（由光反应供给）
ATP的来源/去向光能驱动合成；去向：供暗反应（C3还原）消耗ATP（由光反应供给）
联系为暗反应提供[H]和ATP为光反应提供ADP、Pi和NADP+，保证光反应持续进行

特别提醒：光反应合成的ATP只用于暗反应，不用于细胞其他生命活动；细胞呼吸产生的ATP才是全细胞通用"货币"。

二、陷阱逐条拆解

陷阱1："产生O2的场所"答成"叶绿体"或"线粒体"

  • 正解：O2产生于光反应，场所是类囊体薄膜（基粒）。
  • 答题精度分三级：叶绿体（粗）→ 基粒（中）→ 类囊体薄膜（准）。选择题问"场所"给"叶绿体基质"是错，给"类囊体薄膜"才对；长句题写"类囊体薄膜上"最稳。
  • 同理："固定CO2的场所"→ 叶绿体基质；"消耗ATP的场所"→ 叶绿体基质。

陷阱2："无光时暗反应立即停止"——错误！

  • 正解：无光时光反应立即停止，但暗反应可利用细胞中已积累的[H]和ATP短暂进行，随后因[H]、ATP耗尽而停止。
  • 典型设问："突然停止光照，短时间内C3含量如何变化？"→ 见下方C3/C5分析表。
  • 避坑口诀："光停暗不停（短暂），氢尽暗才停。"

陷阱3：把暗反应写成"只在黑暗中进行"

  • 暗反应有光无光都能进行，"暗"只是指不直接需要光。正常光照下，光反应、暗反应同时进行。
  • 选择题常见干扰项："暗反应只能在夜间进行"——直接排除。

陷阱4：[H]在不同场景指代不同物质，混为一谈

场景[H]的化学本质来源去向
光合作用NADPH（还原型辅酶Ⅱ）水的光解暗反应还原C3
有氧呼吸NADH（还原型辅酶Ⅰ）葡萄糖分解、丙酮酸和水彻底分解（第一、二阶段）第三阶段与O2结合生成水
  • 避坑：看到"[H]"先判断是光合还是呼吸，再谈来源去向。

陷阱5：C3/C5含量变化分析方向搞反（最高频失分点）

分析总原则：条件突变 → 找受直接影响的环节 → 顺推。牢记两条供给链：
- 光照变化 → 直接影响[H]、ATP → 影响C3的还原
- CO2浓度变化 → 直接影响CO2的固定

条件突变[H]、ATPC3C5(CH2O)合成
停止光照（CO2供应不变）减少增加（还原减弱，固定照常）减少减少
增强光照增加减少增加增加
停止/降低CO2供应（光照不变）积累增加减少（固定减弱，还原照常）增加减少
增加CO2供应消耗增加增加减少增加
  • 分析方法（三步）：①变化影响哪个环节？②该环节被促进还是抑制？③短期内该物质"生成vs消耗"谁占上风？
  • 例：停止光照→C3还原受抑（消耗↓），CO2固定照常（生成不变）→C3积累↑；C5因生成减少而消耗不变→C5减少↓。
  • 避坑口诀："断光C3升C5降，断碳(CO2)C3降C5升——两者恰好相反。"

三、联系图解式记忆

          光                          [H]、ATP
光反应（类囊体薄膜） ──────────────────→ 暗反应（叶绿体基质）
  H2O → O2 + [H]                         CO2 + C5 → 2C3
  ADP+Pi → ATP                           2C3 →(还原)→ (CH2O) + C5
          ↑                                │
          └────────── ADP、Pi、NADP+ ────────┘
                    （暗反应"反哺"光反应原料）
  • 记忆抓手：一个"供"（光反应供[H]、ATP），一个"耗"（暗反应耗[H]、ATP），一个"还"（暗反应还ADP、Pi）。
  • 叶绿体结构联想：基粒（类囊体叠成）像"发电站"产能量，基质像"加工厂"造有机物。

四、典型错例与避坑方法

错例错误原因正确表述
"光反应产生的ATP可用于主动运输"混淆两类ATP光反应ATP专供暗反应；主动运输用的是呼吸作用产生的ATP
"晚上植物只进行暗反应"误解"暗"字无光则光反应停止，暗反应短暂进行后也停止；晚上植物只进行呼吸作用
"停止光照后C5增多"方向分析反了停止光照→[H]、ATP减少→C3还原减弱→C5生成减少→C5减少
"CO2固定的场所是类囊体薄膜"场所记串CO2固定在叶绿体基质中进行
"光合作用产生的O2来自CO2"元素来源记错O2全部来自水的光解；CO2中的氧进入有机物和水

避坑方法（广东长句表达题）：回答"原因类"问题时用固定句式——"因为……（条件变化），导致……（直接环节变化），从而使……（物质含量/速率变化）"。例如："停止光照后，光反应减弱，[H]和ATP供应减少，C3的还原受阻，而CO2的固定仍在进行，故C3含量升高。"

五、训练计划（2周）

阶段内容方法
第1-3天对比总表默写每天默写对比表1遍，重点场所、物质变化、能量变化三列
第4-7天C3/C5变化专项每天10道条件突变判断题，强制写出三步分析过程，不写过程不算对
第8-10天光合-呼吸综合做"密闭容器CO2变化曲线"类题，分清净光合/总光合/呼吸速率
第11-14天广东风格长句训练用"因为…导致…从而…"句式完整表述5个原因类问题，对照标准答案打磨措辞
