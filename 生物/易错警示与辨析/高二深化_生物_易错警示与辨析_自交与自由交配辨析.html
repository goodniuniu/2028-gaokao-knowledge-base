自交与自由交配辨析（含计算模板）

⚠️待强化★★★★☆#高二深化#广东选择性考试来源：人教版必修2第1章"遗传因子的发现" + 近5年广东选择性考试遗传计算高频考点
广东考情：高频，近5年广东选择性考试遗传大题与选择题多次涉及"逐代自交淘汰"与"自由交配（随机交配）"计算，是区分度最高的计算点之一。难度定位中高档。广东卷特色：常把两种交配方式放在**同一情境的不同小问**中对比设问（如"F2自交一代"与"F2随机交配一代"各占一问），审题时一字之差全盘皆错；情境化命题中常以育种、遗传病筛查为背景。等级赋分提示：遗传计算是拉开原始分差距的关键题，高手过招在此处，务必做到方法模板化、零计算失误。

核心内容

本质理解：两类题的分水岭只有一句话——自交是"自己和自己"，基因型频率按分离定律逐代递推；自由交配是"群体内部随机结合"，必须先算配子（基因）频率，再用棋盘/公式合成子代。 拿到题先问：合子是"自交产生"还是"随机结合产生"？

一、五种交配方式概念辨析表

交配方式含义典型用途关键特征
自交基因型相同的个体交配（植物自花传粉、同株异花传粉）纯合子鉴定、提高纯合度、逐代淘汰育种群体中每个个体只与"自己"结合
杂交基因型不同的个体交配显隐性判断、获得杂种优势范围最广的统称
测交待测个体与隐性纯合子交配判断待测个体基因型（AA还是Aa）后代比例直接反映待测个体产生的配子种类与比例
正交/反交父本母本互换的两组杂交（如♀AA×♂aa 与 ♀aa×♂AA）判断核遗传还是细胞质遗传、伴性还是常染色体遗传两组结果相同→核遗传（且常染色体）；不同→细胞质或伴性
自由交配（随机交配）群体内所有个体随机结合群体遗传计算必须用配子法（基因频率）计算

避坑：自交≠近交的同义词混用没问题，但自交绝不是"群体随机交配"。题目出现"自然状态""随机传粉""自由传粉"→自由交配；出现"自交""自花传粉""闭花受粉""逐代自留种自种"→自交。

二、识别标志：一眼判断用哪种方法

题干关键词判定方法
自交、自花传粉、自留种连续种植、豌豆自然状态（自花闭花）自交分离定律逐代递推
自由交配、随机交配、随机传粉、相互传粉、玉米自然状态（异花）自由交配配子法（基因频率）
玉米"同株异花"需注意可自交也可自由交配，看题干限定按题干字眼判定

三、自交计算模板：杂合子逐代自交（含淘汰隐性个体）

1. 基础模型：Aa连续自交n代

代数AaAA + aa杂合子比例公式
F11/21/2(1/2)¹
F21/43/4(1/2)²
Fn(1/2)ⁿ1 − (1/2)ⁿ(1/2)ⁿ
  • 推论：Fn中 AA = aa = [1 − (1/2)ⁿ] / 2。

2. 进阶模型：每代淘汰aa后存活群体继续自交（广东高频）

操作要点：每代先算出后代比例 → 剔除aa → 把剩余 AA:Aa 重新归一化 → 再进入下一代自交。

示例：Aa自交，每代淘汰aa，求F3中Aa的比例

  • F1：1/4 AA : 1/2 Aa : 1/4 aa → 淘汰aa → 存活 1/3 AA : 2/3 Aa
  • 自交一代：AA后代全AA；Aa后代 1/4 AA : 1/2 Aa : 1/4 aa
    • AA = 1/3 + 2/3×1/4 = 1/2；Aa = 2/3×1/2 = 1/3；aa = 2/3×1/4 = 1/6
    • 淘汰aa → 存活 AA : Aa = 1/2 : 1/3 = 3/5 AA : 2/5 Aa（即F2存活群体）
  • 再自交一代：Aa = 2/5×1/2 = 1/5（占全体）；淘汰aa后归一化……按题目所问代数逐步推。

避坑警示：

  • ❌ 最常见的错误：淘汰aa后忘记归一化，直接用含aa的原始比例进入下一代计算。
  • ❌ 把"F2中淘汰aa再自交"理解为"F2群体自由交配"——两字之差，方法全错。
  • ✅ 口诀："先剔除，再归一，逐代推。"

四、自由交配计算模板：配子法（基因频率法）

三步模板：①求群体中各基因型比例 → ②计算配子频率（A频率 = AA + ½Aa；a频率 = aa + ½Aa）→ ③用棋盘/公式合成子代：AA = p²，Aa = 2pq，aa = q²。

示例：群体中 1/3 AA : 2/3 Aa，自由交配一代，求子代基因型比例

  • ① 配子频率：p(A) = 1/3 + 2/3×1/2 = 2/3；q(a) = 2/3×1/2 = 1/3
  • ② 子代：AA = (2/3)² = 4/9；Aa = 2×2/3×1/3 = 4/9；aa = (1/3)² = 1/9

含淘汰的变式：若上述群体先淘汰aa再自由交配——先归一化存活群体，再算配子频率，步骤不变。

避坑警示：

  • ❌ 直接用基因型比例做棋盘（把 1/3 AA 与 2/3 Aa 两两组合列6种交配再相加）——不是不能做，但极易漏项、重复计数，考试只推荐配子法。
  • ❌ 雌雄配子频率不同的题（如伴性遗传、某性别中淘汰）——必须雌雄分别算配子频率再棋盘，不能共用一套p、q。
  • ❌ 自由交配一代后群体即达遗传平衡（基因频率不变、基因型频率 = p²:2pq:q²），之后各代比例不再变化——若题目说"连续自由交配n代"，算一代即可。
  • ✅ 口诀："自由交配算配子，p²、2pq、q²；自交递推看（1/2）ⁿ。"

五、两类题对照：同一群体，两种结果（经典对比）

群体 1/3 AA : 2/3 Aa：

处理方式子一代结果
自交AA = 1/3 + 2/3×1/4 = 1/2；Aa = 1/3；aa = 1/6
自由交配AA = 4/9；Aa = 4/9；aa = 1/9

结果明显不同（自交 3:2:1 vs 自由交配 4:4:1），审题时圈出交配方式关键词是第一步。

六、典型错例集锦

错例错因正解
"Aa自交两代，杂合子占1/2"公式记错杂合子 = (1/2)² = 1/4
"淘汰aa后群体1/3AA、2/3Aa直接自交，aa = 2/3×1/4 = 1/6"这步本身对，但若忘记这是"占全体"而非"占存活"会直接当答案要看清题目问"占存活个体"还是"占全部后代"
"自由交配用基因型两两组合，漏掉Aa♂×AA♀与AA♂×Aa♀是两份"组合漏项用配子法，p²:2pq:q² 一次到位
"连续自由交配3代，每代都重新算一次"不懂遗传平衡无选择无突变时，自由交配一代即平衡，以后各代相同
"测交后代1:1说明是自由交配的结果"概念混淆测交是特定杂交方式，与自由交配无关

七、训练计划（2周）

阶段内容方法

第4-7天自交专项(1/2)ⁿ公式推导默写 + 逐代淘汰隐性个体计算5题，强制写"剔除→归一"步骤
