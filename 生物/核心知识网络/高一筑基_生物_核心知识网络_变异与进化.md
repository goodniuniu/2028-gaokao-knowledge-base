## 核心内容

### 关键概念

#### 一、可遗传变异的三大来源

| 变异类型 | 本质 | 发生时期 | 特点 | 意义 |
|----------|------|----------|------|------|
| **基因突变** | 碱基对的替换、增添或缺失 → 基因结构改变 | 主要：DNA复制时（有丝分裂间期/减Ⅰ间期） | ①普遍性 ②随机性 ③不定向性 ④低频性 ⑤多害少利性 | ①新基因产生的途径 ②生物变异的根本来源 ③生物进化的原始材料 |
| **基因重组** | 原有基因的重新组合 | ①减Ⅰ前期（交叉互换）②减Ⅰ后期（自由组合）③基因工程（体外重组） | 只产生新基因型，不产生新基因 | 生物变异的重要来源，生物多样性原因之一 |
| **染色体变异** | 染色体结构或数目改变 | 有丝/减数分裂过程中 | 可在显微镜下观察 | 往往导致严重后果 |

#### 二、染色体结构变异

| 类型 | 图示简记 | 结果 | 实例 |
|------|----------|------|------|
| **缺失** | 染色体断片丢失 | 基因数目减少 | 猫叫综合征（5号染色体短臂缺失） |
| **重复** | 染色体片段增加 | 基因数目增加 | 果蝇棒眼 |
| **易位** | 非同源染色体间片段交换 | 基因排列改变 | 慢性粒细胞白血病（费城染色体） |
| **倒位** | 染色体片段180°翻转 | 基因排列顺序改变 | 某些昆虫遗传 |

> ⚠️ **易位 ≠ 交叉互换**：易位发生在**非同源染色体**之间；交叉互换发生在**同源染色体的非姐妹染色单体**之间。

#### 三、染色体数目变异

- **个别染色体增减**：21三体综合征（唐氏综合征）、Turner综合征（XO）
- **染色体组倍增减**（整倍体变异）：
  - 单倍体：由配子直接发育而来，含本物种配子染色体数（未必只含1个染色体组）
  - 二倍体：由受精卵发育，含2个染色体组
  - 多倍体：由受精卵发育，含3个及以上染色体组

> 判断染色体组数：找同源染色体，有几条相同形态的染色体就有几个染色体组。

#### 四、人类遗传病

| 类型 | 遗传特点 | 常见实例 |
|------|----------|----------|
| **单基因遗传病** | 孟德尔遗传规律 | 白化病（常隐）、血友病（X隐）、抗VD佝偻病（X显） |
| **多基因遗传病** | 易受环境影响，群体发病率高 | 青少年型糖尿病、冠心病、唇腭裂 |
| **染色体异常遗传病** | 往往造成严重后果，流产率高 | 21三体综合征、猫叫综合征 |

> 监测预防：遗传咨询 + 产前诊断（羊水检查、B超检查、基因诊断）

#### 五、现代生物进化理论（达尔文自然选择学说 + 基因频率视角）

**核心公式：哈迪-温伯格定律（理想条件下）**

```
(p + q)² = p² + 2pq + q² = 1

p：显性基因A的频率
q：隐性基因a的频率
p²：AA基因型频率
2pq：Aa基因型频率
q²：aa基因型频率
```

> 适用条件：①种群大 ②随机交配 ③无突变 ④无自然选择 ⑤无迁入迁出。实际种群中基因频率**必然变化**。

**进化实质**：种群基因频率的**定向改变**

**自然选择**：定向改变基因频率，决定进化方向

**物种形成三环节**：

```
突变和基因重组 → 自然选择 → 隔离
   （原材料）      （定向）    （物种形成必要条件）
```

- **地理隔离**：同一种生物由于地理障碍分成不同种群 → 阻断基因交流
- **生殖隔离**：不同种群不能交配，或交配后不能产生可育后代 → 新物种形成的**标志**

> 地理隔离 ≠ 生殖隔离：地理隔离是过程，生殖隔离是结果。生殖隔离是物种形成的**真正标志**。

#### 六、共同进化与生物多样性

- **共同进化**：不同物种之间、生物与无机环境之间在相互影响中不断进化和发展
- **生物多样性**：遗传多样性（基因）→ 物种多样性 → 生态系统多样性

### 核心方法

#### 基因频率计算方法

**方法一：定义法**
```
某基因频率 = (该基因纯合子个体数×2 + 杂合子个体数) / (总个体数×2) × 100%
```

**方法二：基因型频率法**
```
A基因频率 = AA基因型频率 + 1/2 × Aa基因型频率
a基因频率 = aa基因型频率 + 1/2 × Aa基因型频率
```

### 广东长句表达模板

| 设问类型 | 答题模板 |
|----------|----------|
| 解释基因突变产生新基因的原因 | 基因突变是DNA分子中发生碱基对的替换、增添或缺失，引起基因结构的改变，从而产生新基因 |
| 解释自然选择决定进化方向 | 自然选择使种群中适应环境的基因频率定向增加，不适应环境的基因频率定向减少，从而决定生物进化的方向 |
| 解释隔离导致新物种形成 | 隔离阻断不同种群间的基因交流，使种群各自在不同环境下积累变异，最终形成生殖隔离，产生新物种 |
| 分析遗传病发病率高的原因 | 该地区人群中携带致病基因的个体比例高，且近亲结婚（或随机交配）使隐性致病基因纯合的概率增加 |

## 解题路径（变异类型判定与进化分析五步法）

> 广东卷"变异与进化"板块常以"华南生物多样性保护""外来物种入侵""育种实践"为情境，要求先准确定性变异类型，再用进化理论解释原因，长句表达逻辑链完整度直接决定赋分档次。

### 第一步：判定变异层级（分子/细胞水平）
- 先看是否改变 DNA 碱基序列：改变 → 基因突变或染色体变异；不改变但表现型变 → 表观遗传。
- ⚠️ 常见陷阱：把"基因重组产生新基因型"误说成"产生新基因"（新基因只来自基因突变）。

### 第二步：染色体变异精判定
- 结构变异（缺失/重复/易位/倒位） vs 数目变异（个别/整倍体）。
- 易位（非同源染色体间）≠ 交叉互换（同源染色体非姐妹染色单体间）。
- 显微镜可见：染色体变异；不可见：基因突变、基因重组。

### 第三步：系谱/遗传病定性
- 无中生有 → 隐性；有中生无 → 显性。
- 隐性看女病（父正子正非伴性）；显性看男病（母正女正非伴性）。

### 第四步：进化实质与基因频率
- 进化标志 = 种群基因频率定向改变；新物种标志 = 生殖隔离（地理隔离≠生殖隔离）。
- 哈迪-温伯格适用条件检查：大种群 / 随机交配 / 无突变 / 无自然选择 / 无迁入迁出。

### 第五步：长句作答（广东卷核心赋分点）
- 用"因为……所以……"因果链；解释突变、选择、隔离的机理要完整，避免只写结论。

## 母题（2024 广东选择性考试·第11题，选择题）

> 题号与年份已核实；单题分值按广东卷结构（选择题第 1–12 题每题 2 分）标注，精确分值以原卷为准。

**题目**：EDAR 基因的一个碱基替换与东亚人有更多汗腺等典型体征有关。用 $M$、$m$ 分别表示突变前后的 EDAR 基因，研究发现，$m$ 的频率从末次盛冰期后开始明显升高。下列推测合理的是（ ）
- A. $m$ 的出现是自然选择的结果
- B. $m$ 不存在于现代非洲和欧洲人群中
- C. $m$ 的频率升高是末次盛冰期后环境选择的结果
- D. $MM$、$Mm$ 和 $mm$ 个体的汗腺密度依次下降

**解**：

**答案：C**

| 选项 | 分析 |
|------|------|
| A | 一个碱基替换导致 $M \to m$，这属于**基因突变**，不是自然选择的结果，A 错 |
| B | 题干仅说明该突变与东亚人体征有关，无法据此判断 $m$ 是否存在于非洲、欧洲人群，B 错 |
| C | 自然选择使基因频率发生定向改变；$m$ 频率在末次盛冰期后明显升高，是环境定向选择的结果，C 对 |
| D | $m$ 频率升高、$M$ 频率下降，说明含 $m$ 的个体（更多汗腺）更适应环境，故 $MM$、$Mm$、$mm$ 汗腺密度应**依次上升**，D 错 |

> 💡 **关键技巧**：进化类题先锁定"变的是 DNA 序列还是基因频率"——突变产生新等位基因，选择改变其频率；二者因果关系不能颠倒。

## 备注

- **广东卷高频陷阱**：
  1. "基因突变产生新基因型" → 错误！基因突变产生**新基因**，基因重组产生**新基因型**
  2. "生物进化一定形成新物种" → 错误！进化是基因频率改变，新物种形成需要**生殖隔离**
  3. 哈迪-温伯格定律适用条件常作为判断依据，不满足条件时基因频率一定改变
- **广东情境化命题**：常以"华南地区生物多样性保护""外来物种入侵""农作物育种"为情境，需结合进化理论分析
- 单倍体育种和多倍体育种在现代农业中应用广泛，注意区分原理（染色体数目变异）
