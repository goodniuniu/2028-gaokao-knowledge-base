核心内容

关键概念

一、种群特征

特征类型含义调查/计算方法应用
种群密度最基本数量特征植物/活动范围小的动物：样方法；活动能力强动物：标记重捕法害虫监测、渔业资源评估
出生率/死亡率直接决定种群密度—人口预测、资源管理
迁入率/迁出率直接决定种群密度—种群动态分析
年龄结构增长型（幼多老少）/稳定型/衰退型年龄金字塔预测种群发展趋势
性别比例影响出生率—害虫防治（诱杀雄性）

种群密度调查方法选择：
- 样方法：植物、昆虫卵、蚜虫、跳蝻（随机取样、求平均值）
- 标记重捕法：鸟类、哺乳类（公式：N = 初次标记数×重捕总数 / 重捕中标记数）
- 黑光灯诱捕法：趋光性昆虫

二、种群数量变化

J型曲线（理想条件）

Nt = N0 × λ^t
N0：起始数量
λ：增长倍数（λ > 1）
t：时间

适用：实验室培养初期、外来物种入侵早期、理想环境

S型曲线（有限环境/自然条件）

K值（环境容纳量）：环境条件不受破坏时，一定空间所能维持的种群最大数量

特点：种群增长率先增大后减小，K/2时增长率最大（渔业捕捞最佳时机）

曲线对比J型S型
前提条件食物空间充足、无天敌、气候适宜资源空间有限、存在种内/种间斗争
数学模型Nt = N0λ^t逻辑斯蒂增长
K值无K值有K值
增长率不变先增后减

K值变动：改善环境→K值增大；破坏环境→K值减小。K值不是固定不变的。

三、群落结构

结构类型含义影响因素生态意义
垂直结构群落在垂直方向上的分层现象光照（植物）、食物/栖息空间（动物）提高资源利用效率
水平结构不同地段分布不同种群，呈镶嵌分布地形、土壤湿度、光照强度、生物自身生长特点、人类活动空间异质性

垂直结构 ≠ 垂直地带性：前者是同一群落内部的分层；后者是不同海拔出现不同群落。

四、种间关系

关系特点图示记忆实例
互利共生双方有利，彼此依存"同生共死"地衣（藻+菌）、根瘤菌与豆科植物
捕食一方获益，一方受害"先增先减的是被捕食者"狼和兔、猫头鹰和田鼠
竞争双方争夺资源，常一方占优"你死我活"牛和羊、水稻和杂草
寄生一方获益，一方受害（不致死）"寄生者小，宿主大"蛔虫与人、菟丝子与大豆

捕食关系曲线辨识：先增先减的是被捕食者，后增后减的是捕食者。

五、生态系统成分

成分组成作用地位
非生物的物质和能量阳光、热能、水、空气、无机盐为生物提供物质和能量生态系统的基础
生产者绿色植物、光合细菌、化能合成细菌（如硝化细菌）将无机物→有机物，将光能/化学能→化学能生态系统的基石
消费者各种动物加快物质循环，帮助植物传粉和传播种子最活跃部分
分解者腐生细菌、真菌、蚯蚓、秃鹫等将有机物→无机物，供生产者重新利用物质循环的关键

⚠️ 不是所有植物都是生产者：菟丝子是消费者；不是所有细菌都是分解者：硝化细菌是生产者，寄生细菌是消费者。

六、能量流动（核心考点）

特点：

  1. 单向流动：从低营养级→高营养级，不可逆转，不可循环
  2. 逐级递减：传递效率约 10%~20%

能量传递效率计算：

能量传递效率 = 下一营养级同化量 / 上一营养级同化量 × 100%

同化量 = 摄入量 - 粪便量（粪便中的能量属于上一营养级的同化量，流向分解者）

能量去向（某营养级同化量的4个去向）：

  1. 呼吸作用散失（热能）
  2. 流向下一营养级（最高营养级无此项）
  3. 流向分解者（遗体残骸、粪便）
  4. 未利用（暂时储存，如下一年生长）

七、物质循环——碳循环

大气CO2库 ←→ 生产者 ←→ 消费者
              ↓
            分解者
过程方式生理作用
CO2进入生物群落光合作用、化能合成作用生产者固定CO2
CO2返回无机环境呼吸作用、分解作用、化石燃料燃烧生物→CO2

碳循环特点是全球性、循环性，在生物群落与无机环境之间以CO2形式循环，在生物群落内部以有机物形式传递。

八、信息传递

类型来源实例
物理信息光、声、温度、湿度、磁力等萤火虫闪光、植物向光性、候鸟迁徙
化学信息生物代谢产物昆虫信息素、植物次生代谢物（如生物碱）
行为信息动物特殊行为蜜蜂舞蹈、孔雀开屏

信息传递意义：①维持生命活动正常进行 ②维持生物种群繁衍 ③调节种间关系，维持生态系统稳定

九、生态稳定性

类型含义基础特点
抵抗力稳定性抵抗外界干扰，保持原状生态系统组分复杂、食物网复杂与营养结构复杂程度正相关
恢复力稳定性受到干扰后恢复原状生态系统自我调节能力与抵抗力稳定性负相关（一般）

生态系统的自我调节能力是有限的（负反馈调节是基础）。超过限度，生态系统会崩溃。

广东生态案例专题

珠江口湿地生态

  • 生态功能：涵养水源、净化水质、调节气候、为候鸟提供栖息地
  • 威胁因素：围海造地、水污染、外来物种入侵
  • 保护措施：建立自然保护区、退耕还湿、控制污染排放

红树林生态

  • 分布：广东沿海（深圳福田、珠海淇澳岛、湛江红树林保护区）
  • 生态价值：
    1. 防浪护岸（"海岸卫士"）
    2. 为鱼类、鸟类提供栖息地
    3. 固碳能力强（"蓝碳"生态系统）
    4. 净化海水
  • 保护现状：广东红树林面积全国前列，建有多个国家级自然保护区

广东卷情境化命题：常以"红树林保护""珠江口生态治理""外来物种（如福寿螺、薇甘菊）入侵"为情境，考查生态系统稳定性、能量流动、生物多样性保护等内容。

广东长句表达模板

设问类型答题模板
解释种群数量S型增长的原因因为资源和空间是有限的，随着种群密度增加，种内斗争加剧、天敌数量增加，使种群增长率下降，最终达到K值
分析能量单向流动的原因因为食物链中各营养级的顺序是长期自然选择形成的，不可逆转；且每个营养级的能量通过呼吸作用以热能形式散失，无法重新利用
说明保护生物多样性的意义生物多样性具有直接价值（食用、药用）、间接价值（生态功能）和潜在价值（未发现的价值），保护生物多样性有利于维持生态系统稳定性
解释广东沿海建立红树林保护区的原因红树林具有防浪护岸、提供栖息地、固碳释氧等生态功能，建立保护区可以有效保护生物多样性，维持珠江口湿地生态系统的稳定性

备注

  • 广东卷高频陷阱：
    1. "能量传递效率 = 下一营养级摄入量 / 上一营养级摄入量" → 错误！应为同化量/同化量
    2. "粪便中的能量属于该营养级的同化量" → 错误！粪便能量属于上一营养级
    3. "分解者不参与能量流动" → 错误！分解者分解有机物，能量从各营养级流向分解者
    4. "生产者一定是植物" → 错误！蓝藻（蓝细菌）、硝化细菌也是生产者
  • 能量流动计算易错点：
    • 求"最多"传递多少能量 → 用20%（最短食物链）
    • 求"最少"传递多少能量 → 用10%（最长食物链）
