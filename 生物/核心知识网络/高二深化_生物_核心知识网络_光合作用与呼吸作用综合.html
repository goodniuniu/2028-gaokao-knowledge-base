光合作用与呼吸作用综合（速率计算·物质联系）

⚠️待强化★★★★☆#高二深化#广东选择性考试来源：人教版必修1第5章 / 2023广东卷T18、2024广东卷T20(1)、2022广东卷T18
广东考情：考查频率：高频（近5年广东卷年年考，T18/T20稳定出现）；难度：中档偏难，失分在"净光合=总光合−呼吸"关系不清；特色：常以密闭容器CO₂变化、遮阴/密度栽培、湖泊沉水植物为情境，把"光补偿点/光饱和点""净光合与呼吸的关系"包装成曲线与长句；试卷来源：广东选择性考试。

核心内容

一、知识网络（两过程关系总图）

                         O₂、[H]、ATP
   光反应（类囊体膜） ───────────→ 暗反应（叶绿体基质）
     H₂O → O₂ + [H]                    CO₂ + C₅ → 2C₃ →(还原)→ (CH₂O)
     ADP+Pi → ATP                      消耗[H]、ATP，再生C₅、ADP、Pi
          ↑                                  │
          └──────── ADP、Pi、NADP⁺ ───────────┘

   呼吸作用（细胞质基质+线粒体）：
     葡萄糖 →(有氧)→ CO₂ + H₂O + 能量(ATP)
     场所：第一阶段 细胞质基质；第二、三阶段 线粒体
   光合"产"的O₂可供呼吸；呼吸"产"的CO₂可供光合；二者在细胞内循环利用。

二、核心过程与结论

项目光合作用呼吸作用
本质合成有机物、储能分解有机物、放能
原料/产物CO₂+H₂O → (CH₂O)+O₂(CH₂O)+O₂ → CO₂+H₂O
能量光能→化学能（贮）化学能→ATP/热能（放）
[H]NADPH（水光解）NADH（糖分解、丙酮酸分解）
ATP仅供暗反应全细胞通用

关键结论：

  1. 净光合速率 = 总(真正)光合速率 − 呼吸速率
    • 表观光合（净）= 环境CO₂吸收量 = O₂释放量 = 有机物积累量
    • 真正光合 = 净光合 + 呼吸 = 叶绿体内CO₂固定量 = O₂产生量
  2. 光补偿点：光合吸收CO₂ = 呼吸释放CO₂ 时的光照强度（此时净光合=0）。
  3. 光饱和点：光合速率不再随光强增加而增加的拐点。
  4. 密闭容器/自然条件下：白天净光合积累，夜间只呼吸消耗；全天有机物积累 = Σ净光合 − 夜间呼吸。

三、易混点辨析（广东卷高频错区）

易混点正解典型误判
"光合作用产生的O₂供全身"仅植物自身呼吸可用，释放到大气的O₂才对外以为光合O₂直接供动物
"净光合=总光合"净光合已扣除呼吸消耗，二者不等把CO₂吸收量当总光合
"光补偿点时植物不代谢"光合=呼吸，仍有旺盛代谢，净积累为0以为此时植物"停止"
"遮阴一定减产"阴生/下层叶在弱光下净光合可能更高（光补偿点低）一概认为弱光不利
"[H]、ATP减少C₃一定减"须同时看生成端（CO₂固定）与消耗端（C₃还原）只看一端必错

四、计算模板（四步）

  1. 辨"率"：题目给的是净光合还是总光合（看单位是"吸收/释放"还是"产生/固定"）。
  2. 定关系：总 = 净 + 呼吸。
  3. 算昼夜：全天积累 = 白天净光合总量 − 夜间呼吸总量。
  4. 查单位：统一"mg/h·单位叶面积"，注意"24h"与"光照h"的区别。

五、真题母题（广东卷，已核实）

（2023 广东选择性考试·第18题，13分） 水稻野生型(WT)与黄绿叶突变体(ygl)在高密度栽培下产量比较……

  • (2) ygl有较高的光补偿点，可能的原因是叶绿素含量较低和______。（答：呼吸速率较强）——直接考查"光补偿点=光合吸收CO₂=呼吸释放CO₂"的概念综合。
  • (4) 在0~50 μmol·m⁻²·s⁻¹低光强下分析WT与ygl净光合速率变化并绘制曲线，提出一个科学问题。（综合光合-呼吸曲线与长句表达）

（2024 广东选择性考试·第20题(1)，15分） 湖泊富营养化时原有沉水植物因光合作用合成的有机物少于______的有机物而衰退消亡。（答：呼吸作用消耗）——净光合<0即消亡，直击"光合-呼吸综合"。
