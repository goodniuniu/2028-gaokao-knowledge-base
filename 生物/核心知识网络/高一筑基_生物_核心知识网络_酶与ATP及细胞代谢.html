酶与ATP及细胞代谢

⚠️待强化★★★☆☆#高一筑基#广东选择性考试来源：人教版必修第一册 / 2023-2025广东选择性考试·生物代谢专题
广东考情：考查频率：高频（近5年广东卷每年必考，光合与呼吸综合常作为非选择题大题出现）；难度定位：中档（酶的特性曲线偏基础，光合呼吸综合分析可达中高档）；特色描述：广东卷常结合农业生产情境命题，如温室大棚增产措施、岭南水果贮藏保鲜、曲线图分析光合速率影响因素，注重实验探究与坐标曲线解读；赋分提示：光合作用与细胞呼吸的物质能量对应关系是长句表达重灾区，"解释曲线变化原因"类设问逻辑链完整度直接影响赋分档次

核心内容

关键概念

  • 酶：活细胞产生的具有催化作用的有机物，绝大多数是蛋白质，少数是RNA
  • ATP：三磷酸腺苷，细胞的直接能源物质，结构简式A-P~P~P
  • 细胞呼吸：有机物在细胞内经过一系列氧化分解，释放能量的过程
  • 光合作用：绿色植物利用光能将CO₂和H₂O转化为有机物并释放O₂的过程
  • 细胞代谢：细胞内所有有序化学反应的总称，包括物质代谢和能量代谢

核心知识体系

1. 酶的本质与特性

本质：蛋白质（绝大多数）或 RNA（少数，如核酶）

特性：
① 高效性：催化效率是无机催化剂的10⁷~10¹³倍
② 专一性：一种酶只能催化一种或一类化学反应（锁钥学说/诱导契合学说）
③ 作用条件温和：需要适宜的温度和pH

影响因素：
- 温度：低温抑制活性（可恢复），高温使酶变性失活（不可恢复）
- pH：过酸或过碱使酶变性失活，最适pH因酶而异（胃蛋白酶1.5，胰蛋白酶8~9）
- 底物浓度、酶浓度、抑制剂、激活剂

2. ATP结构与功能

结构：A-P~P~P
- A：腺苷（腺嘌呤+核糖）
- P：磷酸基团
- ~：高能磷酸键（断裂释放30.54kJ/mol能量）

ATP与ADP的转化：
ATP → ADP + Pi + 能量（水解，供能）
ADP + Pi + 能量 → ATP（合成，储能）

功能：细胞的直接能源物质（"能量通货"）
注意：ATP在细胞中含量很少，但转化非常迅速；ATP与ADP的转化不可逆（酶不同、能量来源不同、场所不同）

3. 细胞呼吸（有氧+无氧）

##### 有氧呼吸三阶段

阶段场所反应物产物释放能量O₂参与
第一阶段细胞质基质葡萄糖丙酮酸 + [H] + 少量ATP少量否
第二阶段线粒体基质丙酮酸 + H₂OCO₂ + [H] + 少量ATP少量否
第三阶段线粒体内膜[H] + O₂H₂O + 大量ATP大量是
总反应式：C₆H₁₂O₆ + 6H₂O + 6O₂ → 6CO₂ + 12H₂O + 能量（2870kJ，其中1161kJ储存在ATP中）
注意事项：O₂只在第三阶段参与反应；CO₂在第二阶段产生；H₂O在第三阶段产生、第二阶段消耗；[H]是NADH和FADH₂的简化写法

##### 无氧呼吸两类型

类型场所反应式产物常见生物
乳酸发酵细胞质基质C₆H₁₂O₆ → 2C₃H₆O₃ + 少量能量乳酸动物、乳酸菌、马铃薯块茎、甜菜块根
酒精发酵细胞质基质C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂ + 少量能量酒精+CO₂植物、酵母菌
注意事项：无氧呼吸只在第一阶段释放少量能量，第二阶段不释放能量；动物细胞无氧呼吸产物是乳酸，植物细胞可以是酒精或乳酸

4. 光合作用两阶段

##### 光反应（类囊体薄膜）

条件：光、色素、酶
物质变化：H₂O → [H] + O₂（光解）
           ADP + Pi + 能量 → ATP
能量变化：光能 → ATP中活跃化学能
产物：O₂、[H]（NADPH）、ATP

##### 暗反应（叶绿体基质）

条件：酶、[H]、ATP
物质变化：CO₂ + C₅ → 2C₃（CO₂固定）
           2C₃ + [H] + ATP → (CH₂O) + C₅（C₃还原）
能量变化：ATP中活跃化学能 → 有机物中稳定化学能
产物：糖类(CH₂O)、C₅（再生）
总反应式：6CO₂ + 12H₂O → C₆H₁₂O₆ + 6H₂O + 6O₂（条件：光、叶绿体）

注意事项：CO₂中的O进入有机物和H₂O（产物）；H₂O中的O进入O₂（产物）；O₂全部来自H₂O的光解

5. 影响光合作用和呼吸作用的因素

因素对光合作用的影响对呼吸作用的影响
温度有最适温度（酶活性），高温抑制随温度升高而增强（有最适温度）
CO₂浓度低浓度时限制，饱和后不再增加高浓度抑制（一般不考虑）
光照强度光饱和点以下随光照增强而增强无直接影响
O₂浓度高浓度促进光呼吸（抑制）低浓度抑制有氧呼吸，促进无氧呼吸
水分缺水导致气孔关闭，CO₂供应不足自由水比例高促进代谢

方法步骤

光合与呼吸综合计算（三率法）

总光合速率（真正光合速率）= 净光合速率（表观光合速率）+ 呼吸速率

- 总光合：叶绿体固定CO₂或产生O₂的总量
- 净光合：植物从外界吸收CO₂或释放O₂的量
- 呼吸速率：黑暗中CO₂释放量或O₂吸收量

常用表示：
CO₂：总固定 = 净吸收 + 呼吸释放
O₂：总产生 = 净释放 + 呼吸消耗
有机物：总合成 = 净积累 + 呼吸消耗

C₃、C₅含量变化分析（CO₂浓度骤变时）

条件变化C₃变化C₅变化原因
CO₂供应增加↑↓CO₂固定增强，C₅消耗多，C₃生成多
CO₂供应停止↓↑CO₂固定停止，C₃不生成，C₅积累
光照增强↓↑[H]和ATP增多，C₃还原快，C₅再生多
光照停止↑↓[H]和ATP减少，C₃还原慢，C₅不再生
记忆口诀：CO₂增则C₃增C₅减，光增则C₃减C₅增；CO₂停则C₃减C₅增，光停则C₃增C₅减。

记忆口诀/技巧

酶特性口诀：高效专一条件温，高温过酸变性失，低温抑制可恢复。

有氧呼吸口诀：一糖丙酮细胞质，二丙酮酸线粒体，三氢氧合内膜上；CO₂二产，H₂O三产，能量逐步放出来。

光合作用口诀：水光解在类囊体，CO₂固定在基质；光产ATP和NADPH，暗用ATP和NADPH；C₃被还原，C₅再循环。

三率关系口诀：总光=净光+呼吸，常用CO₂或O₂来表示。

广东情境提示：广东卷代谢题常以广东农业为情境（如荔枝保鲜、水稻增产、大棚种植、红树林耐盐），注意从情境中提取影响光合/呼吸的因素（温度、CO₂、光照等）。

备注

  • 光合与呼吸的综合是广东卷生物大题高频考点，常与实验设计结合
  • 高频陷阱：
  • 光反应产生的ATP只用于暗反应，不能用于其他生命活动（细胞其他生命活动所需ATP来自呼吸作用）
  • 呼吸作用产生的CO₂中的O来自葡萄糖和H₂O，不是O₂
  • 光合作用产物中O₂的O全部来自H₂O，不是CO₂
  • 无氧呼吸只在第一阶段产生ATP，第二阶段不产生ATP
  • 植物夜间只进行呼吸作用，白天同时进行光合和呼吸
  • 净光合为零时的光照强度 = 光补偿点；净光合最大时的光照强度 = 光饱和点
  • 夏季中午高温导致气孔关闭，CO₂减少，光合速率下降（"午休"现象）
  • 广东卷常考：环境因素对光合速率的影响曲线分析、C₃/C₅含量变化、光合与呼吸综合计算
