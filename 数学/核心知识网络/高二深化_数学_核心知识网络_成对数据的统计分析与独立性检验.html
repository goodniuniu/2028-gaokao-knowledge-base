成对数据的统计分析与独立性检验

⚠️待强化★★★☆☆#高二深化#新高考Ⅰ卷·广东来源：53科学备考《高中知识清单》数学知识图谱 / 人教A版选择性必修第三册第八章
广东考情：考查频率：中频；难度：中档；回归分析和独立性检验是概率统计大题的另一大方向，近年广东卷中交替出现，计算量大但套路固定

核心内容

一、成对数据的统计相关性

  • 变量的相关关系：正相关（$r > 0$）、负相关（$r < 0$）
  • 样本相关系数：

$$r = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^n (x_i - \bar{x})^2} \sqrt{\sum_{i=1}^n (y_i - \bar{y})^2}}$$

  • $|r| \in [0, 1]$：$|r|$ 越接近1，线性相关程度越强；越接近0，越弱

二、一元线性回归模型及其应用

  • 模型：$Y = bx + a + e$，其中 $E(e) = 0$，$D(e) = \sigma^2$
  • 最小二乘估计（经验回归方程 $\hat{y} = \hat{b}x + \hat{a}$）：

$$\hat{b} = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n (x_i - \bar{x})^2}, \quad \hat{a} = \bar{y} - \hat{b}\bar{x}$$

  • 拟合效果评价：
  • 残差平方和：越小越好
  • 决定系数：$R^2 = 1 - \frac{\sum_{i=1}^n (y_i - \hat{y}_i)^2}{\sum_{i=1}^n (y_i - \bar{y})^2}$，$R^2$ 越接近1，拟合效果越好

三、列联表与独立性检验

#### $2 \times 2$ 列联表

$Y = 0$$Y = 1$合计
$X = 0$$a$$b$$a + b$
$X = 1$$c$$d$$c + d$
合计$a + c$$b + d$$n = a + b + c + d$

#### 独立性检验（$\chi^2$ 检验）

$$\chi^2 = \frac{n(ad - bc)^2}{(a+b)(c+d)(a+c)(b+d)}$$

  • 将计算得到的 $\chi^2$ 与临界值比较，判断独立性
  • 等高堆积条形图：直观判断两个分类变量是否相关

备注
