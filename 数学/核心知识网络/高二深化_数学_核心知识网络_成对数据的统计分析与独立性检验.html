成对数据的统计分析与独立性检验

⚠️待强化★★★☆☆#高二深化#新高考Ⅰ卷·广东来源：53科学备考《高中知识清单》数学知识图谱 / 人教A版选择性必修第三册第八章
广东考情：考查频率：中频；难度：中档；回归分析和独立性检验是概率统计大题的另一大方向，近年广东卷中交替出现，计算量大但套路固定

核心内容

一、成对数据的统计相关性

  • 变量的相关关系：正相关（$r > 0$）、负相关（$r < 0$）
  • 样本相关系数：

    • $$r = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^n (x_i - \bar{x})^2} \sqrt{\sum_{i=1}^n (y_i - \bar{y})^2}}$$

  • $|r| \in [0, 1]$：$|r|$ 越接近1，线性相关程度越强；越接近0，越弱

二、一元线性回归模型及其应用

  • 模型：$Y = bx + a + e$，其中 $E(e) = 0$，$D(e) = \sigma^2$
  • 最小二乘估计（经验回归方程 $\hat{y} = \hat{b}x + \hat{a}$）：

    • $$\hat{b} = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n (x_i - \bar{x})^2}, \quad \hat{a} = \bar{y} - \hat{b}\bar{x}$$

  • 拟合效果评价：
    • 残差平方和：越小越好
    • 决定系数：$R^2 = 1 - \frac{\sum_{i=1}^n (y_i - \hat{y}_i)^2}{\sum_{i=1}^n (y_i - \bar{y})^2}$，$R^2$ 越接近1，拟合效果越好

三、列联表与独立性检验

$2 \times 2$ 列联表

$Y = 0$$Y = 1$合计
$X = 0$$a$$b$$a + b$
$X = 1$$c$$d$$c + d$
合计$a + c$$b + d$$n = a + b + c + d$

独立性检验（$\chi^2$ 检验）

$$\chi^2 = \frac{n(ad - bc)^2}{(a+b)(c+d)(a+c)(b+d)}$$

  • 将计算得到的 $\chi^2$ 与临界值比较，判断独立性
  • 等高堆积条形图：直观判断两个分类变量是否相关

题型识别标志

看到什么条件 → 立刻想到什么方法

题干关键条件识别为首选方法
给 $2\times2$ 列联表，问"是否有把握认为有关"独立性检验算 $\chi^2$ 与临界值比较
问"有 $99\%$ 把握"取临界值查表 $k_{0.010}=6.635$
求经验回归方程 $\hat y=\hat b x+\hat a$回归分析最小二乘 $\hat b=\frac{\sum(x_i-\bar x)(y_i-\bar y)}{\sum(x_i-\bar x)^2}$
判断线性相关强弱相关系数看 $|r|$ 接近 $1$ 的程度、正负表方向
给条件概率比值 $R$比值化简用 $P(B|A)=\frac{P(AB)}{P(A)}$ 展开为比值式
"正相关/负相关"方向判断$r>0$ 正，$r<0$ 负

解题路径（独立性检验四步）

第一步：列出 $2\times2$ 列联表

整理 $a,b,c,d$ 及各行、列合计与总计 $n$。

第二步：套 $\chi^2$ 公式

$$\chi^2=\frac{n(ad-bc)^2}{(a+b)(c+d)(a+c)(b+d)}.$$

第三步：与临界值比较

$\chi^2>6.635$ 有 $99\%$ 把握有关；$\chi^2>3.841$ 有 $95\%$ 把握有关。

第四步：条件概率比值 $R$

用条件概率定义把 $R$ 化为列联表比值，代入频率估计值计算。

易错提醒：$\chi^2$ 中 $n$ 是总样本量；临界值记错（99% 对应 6.635）；比值 $R$ 的方向与分子分母别颠倒。

母题（2022 新课标Ⅰ卷·第20题，12分）

题目：一医疗队为研究某地一种地方性疾病与当地居民卫生习惯（分为良好和不够良好两类）的关系，在已患该疾病的人群中随机调查了 $100$ 例（病例组），在未患该疾病的人群中随机调查了 $100$ 人（对照组），得到如下数据：

不够良好良好
病例组$40$$60$
对照组$10$$90$

(1) 能否有 $99\%$ 的把握认为患该疾病群体与未患该疾病群体的卫生习惯有差异？ (2) 从该地人群中随机选一人，$A$ 表示"选到的人卫生习惯不够良好"，$B$ 表示"选到的人患有该疾病"。记 $$R=\frac{P(B|A)/P(\overline B|A)}{P(B|\overline A)/P(\overline B|\overline A)}$$ 为卫生习惯不够良好对患该疾病风险程度的度量指标。 (i) 证明：$R=\dfrac{P(A|B)P(\overline A|\overline B)}{P(A|\overline B)P(\overline A|B)}$； (ii) 利用调查数据，给出 $P(A|B),P(A|\overline B)$ 的估计值，并利用 (i) 的结果给出 $R$ 的估计值。 附：$\chi^2$ 临界值 $k_{0.05}=3.841,\ k_{0.01}=6.635,\ k_{0.001}=10.828$。

解答： (1) 列联表合计：不够良好 $50$、良好 $150$、总计 $200$。 $$\chi^2=\frac{200\times(40\times90-60\times10)^2}{100\times100\times50\times150}=\frac{200\times3000^2}{75\,000\,000}=24.$$ 因为 $24>6.635$，所以有 $99\%$ 的把握认为两者卫生习惯有差异。

(2)(i) 由 $P(B|A)=\dfrac{P(AB)}{P(A)}$ 等， $$R=\frac{P(AB)P(\overline A\,\overline B)}{P(A\overline B)P(\overline A B)} =\frac{P(A|B)P(B)\cdot P(\overline A|\overline B)P(\overline B)}{P(A|\overline B)P(\overline B)\cdot P(\overline A|B)P(B)} =\frac{P(A|B)P(\overline A|\overline B)}{P(A|\overline B)P(\overline A|B)}.$$

(ii) 估计值：$P(A|B)=\frac{40}{100}=0.4$，$P(A|\overline B)=\frac{10}{100}=0.1$， $P(\overline A|B)=\frac{60}{100}=0.6$，$P(\overline A|\overline B)=\frac{90}{100}=0.9$。 $$R\text{ 的估计值}=\frac{0.4\times0.9}{0.1\times0.6}=6.$$
