## 核心内容

### 一、等差数列
| 项目 | 公式/性质 |
|------|----------|
| 定义 | $a_n - a_{n-1} = d$（$n \geq 2, n \in \mathbb{N}^*$） |
| 等差中项 | $2A = a + b$（$A$ 为 $a, b$ 的等差中项） |
| 通项公式 | $a_n = a_1 + (n-1)d$ |
| 前 $n$ 项和 | $S_n = \frac{(a_1 + a_n)n}{2} = na_1 + \frac{n(n-1)}{2}d = \frac{d}{2}n^2 + (a_1 - \frac{d}{2})n$ |
| 单调性 | $d > 0$ 递增；$d = 0$ 常数列；$d < 0$ 递减 |

**重要性质**：
- 若 $m + n = p + q$（$m, n, p, q \in \mathbb{N}^*$），则 $a_m + a_n = a_p + a_q$
- 性质1：$S_k, S_{2k} - S_k, S_{3k} - S_{2k}, \dots$ 组成公差为 $k^2 d$ 的等差数列
- 性质2：项数为 $2n$ 时，$S_{2n} = n(a_n + a_{n+1})$，$S_{偶} - S_{奇} = nd$，$\frac{S_{偶}}{S_{奇}} = \frac{a_{n+1}}{a_n}$
- 性质3：$\{a_n\}$ 为等差数列 $\Rightarrow \left\{\frac{S_n}{n}\right\}$ 为等差数列
- 性质4：若 $\{a_n\}, \{b_n\}$ 都为等差数列，前 $n$ 项和分别为 $S_n, T_n$，则 $\frac{a_m}{b_m} = \frac{S_{2m-1}}{T_{2m-1}}$

### 二、等比数列
| 项目 | 公式/性质 |
|------|----------|
| 定义 | $\frac{a_n}{a_{n-1}} = q$（$n \geq 2, n \in \mathbb{N}^*$） |
| 等比中项 | $G^2 = ab$（$G = \pm\sqrt{ab}$，等比中项有两个） |
| 通项公式 | $a_n = a_1 q^{n-1}$（$a_1, q \neq 0$） |
| 前 $n$ 项和 | $S_n = \begin{cases} na_1 & (q = 1) \\ \frac{a_1(1 - q^n)}{1 - q} = \frac{a_1 - a_n q}{1 - q} & (q \neq 1) \end{cases}$ |
| 单调性 | $a_1 > 0, q > 1$ 或 $a_1 < 0, 0 < q < 1$ 时递增；$a_1 > 0, 0 < q < 1$ 或 $a_1 < 0, q > 1$ 时递减 |

**重要性质**：
- 若 $m + n = p + q$（$m, n, p, q \in \mathbb{N}^*$），则 $a_m \cdot a_n = a_p \cdot a_q$
- 性质1：$S_{m+n} = S_n + q^n S_m$
- 性质2：$S_k, S_{2k} - S_k, S_{3k} - S_{2k}, \dots$（均不为0）成等比数列，公比为 $q^k$
- 性质3：项数为 $2n$ 时，$\frac{S_{偶}}{S_{奇}} = q$；项数为 $2n+1$ 时，$\frac{S_{奇} - a_1}{S_{偶}} = q$

## 题型识别标志

> **看到什么条件 → 立刻想到什么方法**

| 题干关键条件 | 识别为 | 首选方法 |
|------|------|------|
| 给 $S_n$ 与 $a_n$ 的混合关系 | 消去 $S_n$ | $a_n=S_n-S_{n-1}\ (n\ge2)$ |
| $\{S_n/a_n\}$ 等差/等比 | 辅助数列 | 设 $b_n=S_n/a_n$ 先求 $b_n$ 通项 |
| 求通项 | 累乘/累加 | $\frac{a_n}{a_{n-1}}=f(n)$ 用累乘法 |
| 求和证不等式 | 裂项相消 | $\frac1{a_n}$ 裂成差分 |
| 等比数列求和 | 讨论 $q$ | 必须讨论 $q=1$ 与 $q\ne1$ |
| 前 $n$ 项和最值 | 二次/转折 | $S_n$ 是 $n$ 的二次函数或看正负项 |

## 解题路径（由 $S_n/a_n$ 关系求通五步）

### 第一步：设辅助数列
令 $b_n=\dfrac{S_n}{a_n}$，利用已知公差（或公比）写出 $b_n$ 通项。

### 第二步：建立 $S_n$ 与 $a_n$ 关系
由 $S_n=b_n a_n$，对 $n\ge2$ 用 $a_n=S_n-S_{n-1}$ 代入。

### 第三步：化为比式
整理出 $\dfrac{a_n}{a_{n-1}}=f(n)$ 的形式。

### 第四步：累乘求通
$$a_n=a_1\prod_{k=2}^n f(k),$$
化简时注意分子分母大量相消。

### 第五步：裂项证不等式
将 $\dfrac1{a_n}$ 裂项为差分，求和后放缩。

> **易错提醒**：忽略 $n=1$ 的验证；累乘时上下标约错；等比数列求和忘讨论 $q=1$。

## 母题（2022 新课标Ⅰ卷·第17题，10分）

**题目**：记 $S_n$ 为数列 $\{a_n\}$ 的前 $n$ 项和，已知 $a_1=1$，$\left\{\dfrac{S_n}{a_n}\right\}$ 是公差为 $\dfrac13$ 的等差数列。
(1) 求 $\{a_n\}$ 的通项公式；
(2) 证明：$\dfrac1{a_1}+\dfrac1{a_2}+\cdots+\dfrac1{a_n}<2$。

**解答**：
(1) 设 $b_n=\dfrac{S_n}{a_n}$，则 $b_1=\dfrac{S_1}{a_1}=1$，
$$b_n=1+\frac13(n-1)=\frac{n+2}{3},\quad\text{即 }S_n=\frac{n+2}{3}a_n.$$
对 $n\ge2$，$a_n=S_n-S_{n-1}=\frac{n+2}{3}a_n-\frac{n+1}{3}a_{n-1}$，
整理得 $a_n(1-n)=-(n+1)a_{n-1}$，即
$$\frac{a_n}{a_{n-1}}=\frac{n+1}{n-1}.$$
累乘得
$$a_n=a_1\prod_{k=2}^n\frac{k+1}{k-1}=\frac{3\cdot4\cdots(n+1)}{1\cdot2\cdots(n-1)}=\frac{n(n+1)}{2}.$$
当 $n=1$ 时 $a_1=1$ 也满足，故 $a_n=\dfrac{n(n+1)}{2}$。

(2) $\dfrac1{a_n}=\dfrac{2}{n(n+1)}=2\left(\dfrac1n-\dfrac1{n+1}\right)$，
$$\sum_{k=1}^n\frac1{a_k}=2\left(1-\frac1{n+1}\right)=2-\frac{2}{n+1}<2.$$
证毕。
