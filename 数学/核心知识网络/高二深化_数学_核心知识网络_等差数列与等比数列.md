## 核心内容

### 一、等差数列
| 项目 | 公式/性质 |
|------|----------|
| 定义 | $a_n - a_{n-1} = d$（$n \geq 2, n \in \mathbb{N}^*$） |
| 等差中项 | $2A = a + b$（$A$ 为 $a, b$ 的等差中项） |
| 通项公式 | $a_n = a_1 + (n-1)d$ |
| 前 $n$ 项和 | $S_n = \frac{(a_1 + a_n)n}{2} = na_1 + \frac{n(n-1)}{2}d = \frac{d}{2}n^2 + (a_1 - \frac{d}{2})n$ |
| 单调性 | $d > 0$ 递增；$d = 0$ 常数列；$d < 0$ 递减 |

**重要性质**：
- 若 $m + n = p + q$（$m, n, p, q \in \mathbb{N}^*$），则 $a_m + a_n = a_p + a_q$
- 性质1：$S_k, S_{2k} - S_k, S_{3k} - S_{2k}, \dots$ 组成公差为 $k^2 d$ 的等差数列
- 性质2：项数为 $2n$ 时，$S_{2n} = n(a_n + a_{n+1})$，$S_{偶} - S_{奇} = nd$，$\frac{S_{偶}}{S_{奇}} = \frac{a_{n+1}}{a_n}$
- 性质3：$\{a_n\}$ 为等差数列 $\Rightarrow \left\{\frac{S_n}{n}\right\}$ 为等差数列
- 性质4：若 $\{a_n\}, \{b_n\}$ 都为等差数列，前 $n$ 项和分别为 $S_n, T_n$，则 $\frac{a_m}{b_m} = \frac{S_{2m-1}}{T_{2m-1}}$

### 二、等比数列
| 项目 | 公式/性质 |
|------|----------|
| 定义 | $\frac{a_n}{a_{n-1}} = q$（$n \geq 2, n \in \mathbb{N}^*$） |
| 等比中项 | $G^2 = ab$（$G = \pm\sqrt{ab}$，等比中项有两个） |
| 通项公式 | $a_n = a_1 q^{n-1}$（$a_1, q \neq 0$） |
| 前 $n$ 项和 | $S_n = \begin{cases} na_1 & (q = 1) \\ \frac{a_1(1 - q^n)}{1 - q} = \frac{a_1 - a_n q}{1 - q} & (q \neq 1) \end{cases}$ |
| 单调性 | $a_1 > 0, q > 1$ 或 $a_1 < 0, 0 < q < 1$ 时递增；$a_1 > 0, 0 < q < 1$ 或 $a_1 < 0, q > 1$ 时递减 |

**重要性质**：
- 若 $m + n = p + q$（$m, n, p, q \in \mathbb{N}^*$），则 $a_m \cdot a_n = a_p \cdot a_q$
- 性质1：$S_{m+n} = S_n + q^n S_m$
- 性质2：$S_k, S_{2k} - S_k, S_{3k} - S_{2k}, \dots$（均不为0）成等比数列，公比为 $q^k$
- 性质3：项数为 $2n$ 时，$\frac{S_{偶}}{S_{奇}} = q$；项数为 $2n+1$ 时，$\frac{S_{奇} - a_1}{S_{偶}} = q$

## 备注
- 易错点：等比数列求和时必须讨论 $q = 1$ 的情况！$q = 1$ 时 $S_n = na_1$
- 等差数列前 $n$ 项和 $S_n = An^2 + Bn$ 是无常数项的二次函数，对称轴 $n = \frac{1 - 2a_1}{2d}$，可用于求最值
