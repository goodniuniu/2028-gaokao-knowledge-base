一元二次函数、方程与不等式

⚠️待强化★★☆☆☆#高一筑基#新高考Ⅰ卷·广东来源：53科学备考《高中知识清单》数学知识图谱 / 人教A版必修第一册第二章
广东考情：考查频率：高频；难度：基础~中档；基本不等式常与最值问题结合，是广东卷多选和填空压轴的常见考点

核心内容

一、不等式性质

性质内容
对称性$a > b \Leftrightarrow b < a$
传递性$a > b, b > c \Rightarrow a > c$
可加性$a > b \Rightarrow a + c > b + c$
可乘性$a > b, c > 0 \Rightarrow ac > bc$；$a > b, c < 0 \Rightarrow ac < bc$
同向可加性$a > b, c > d \Rightarrow a + c > b + d$
同向同正可乘性$a > b > 0, c > d > 0 \Rightarrow ac > bd$
可乘方性$a > b > 0 \Rightarrow a^n > b^n$（$n \in \mathbb{N}, n \geq 2$）

二、基本不等式

$$\sqrt{ab} \leq \frac{a+b}{2} \quad (a > 0, b > 0)$$

当且仅当 $a = b$ 时取等号

重要变形：

  • $a^2 + b^2 \geq 2ab$（$a, b \in \mathbb{R}$）
  • $\frac{a^2 + b^2}{2} \geq \left(\frac{a+b}{2}\right)^2$
  • $\left(\frac{a+b}{2}\right)^2 \geq ab$
  • $a + b \geq 2\sqrt{ab}$（$a, b > 0$）
  • $ab \leq \left(\frac{a+b}{2}\right)^2$（$a, b > 0$）

应用条件（一正二定三相等）：

  1. 一正：各项均为正
  2. 二定：和或积为定值
  3. 三相等：能取到等号

三、一元二次不等式解法（三步法）

  1. 第一步：求对应方程 $ax^2 + bx + c = 0$（$a > 0$）的根 $x_1, x_2$
  2. 第二步：画出 $y = ax^2 + bx + c$ 的抛物线图象
  3. 第三步：由图象写出解集
    • $ax^2 + bx + c > 0$：取两边（$x < x_1$ 或 $x > x_2$）
    • $ax^2 + bx + c < 0$：取中间（$x_1 < x < x_2$）

四、一元二次不等式恒成立问题

条件等价转化
$ax^2 + bx + c > 0$ 在 $\mathbb{R}$ 上恒成立$a > 0$ 且 $\Delta < 0$
$ax^2 + bx + c < 0$ 在 $\mathbb{R}$ 上恒成立$a < 0$ 且 $\Delta < 0$
在某区间上恒成立分离参数求最值，或端点效应分析

五、根的分布问题

影响二次函数根分布的因素：

  • 图象开口方向（$a$ 的符号）
  • 判别式 $\Delta = b^2 - 4ac$
  • 对称轴位置 $x = -\frac{b}{2a}$
  • 端点处的函数值

题型识别标志

看到什么条件 → 立刻想到什么方法

题干关键条件识别为首选方法
"和/积为定值，求另一量的最值"基本不等式一正二定三相等，构造 $a+b$ 或 $ab$ 后用 $ab\le\left(\frac{a+b}{2}\right)^2$
"椭圆/双曲线上动点，求 $\|MF_1\|\cdot\|MF_2\|$ 类乘积"圆锥曲线+基本不等式用定义 $\|MF_1\|+\|MF_2\|=2a$，再用均值不等式
"解含参一元二次不等式 $ax^2+bx+c>0$"二次不等式先求方程根，画抛物线写解集（注意 $a<0$ 时图象翻折）
"含参不等式恒成立"恒成立求参分离参数求最值，或等价转为 $a>0$ 且 $\Delta<0$
"讨论方程根的分布"根的分布抓四要素：开口、$\Delta$、对称轴、端点函数值
"求二次函数在某区间最值"最值看对称轴 $x=-\frac{b}{2a}$ 与区间的位置关系

解题路径（基本不等式求最值三步法）

650分导向：基本不等式是广东卷选填压轴与解答题放缩的常客，等号能否取到是扣分重灾区。

第一步：定模型

识别是"和定求积最大"还是"积定求和最小"，并列出已知定值（如圆锥曲线中的 $2a$）。

第二步：凑形式

通过变形（配凑、换元、常数 $1$ 的代换）使各项为正且和/积为定值。例如 $ab$ 与 $a+b$ 互化、$(a+b)^2$ 展开。

第三步：验等号

检查取等条件是否可取（如 $a=b$ 是否在定义域内），再下结论。避免"正而不等"或"等而不可取"。

母题（2021 新课标Ⅰ卷·第5题，5分）

广东考生做的是同一张新课标Ⅰ卷，此题用椭圆定义+基本不等式求最值，是不等式与解析几何的交汇典型。

题目：已知 $F_1,F_2$ 是椭圆 $C:\dfrac{x^2}{9}+\dfrac{y^2}{4}=1$ 的两个焦点，点 $M$ 在 $C$ 上，则 $\|MF_1\|\cdot\|MF_2\|$ 的最大值为（ ）

A. $13$　B. $12$　C. $9$　D. $6$

解： 由椭圆定义，$\|MF_1\|+\|MF_2\|=2a=6$。

由基本不等式： $$\|MF_1\|\cdot\|MF_2\|\le\left(\frac{\|MF_1\|+\|MF_2\|}{2}\right)^2=\left(\frac{6}{2}\right)^2=9$$

当且仅当 $\|MF_1\|=\|MF_2\|=3$，即 $M$ 为短轴端点时取等号（此时确在椭圆上）。

答：最大值为 $9$，选 C。
