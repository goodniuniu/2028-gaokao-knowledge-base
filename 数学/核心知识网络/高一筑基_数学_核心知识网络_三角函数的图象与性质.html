三、函数 $y = A\sin(\omega x + \varphi)$（$A > 0, \omega > 0$）

参数意义影响
$A$振幅$A > 1$ 纵坐标伸长为 $A$ 倍；$0 < A < 1$ 缩短为 $A$ 倍
$\omega$角频率$\omega > 1$ 横坐标缩短为 $\frac{1}{\omega}$；$0 < \omega < 1$ 伸长为 $\frac{1}{\omega}$ 倍
$\varphi$初相$\varphi > 0$ 图象向左平移 $\frac{\varphi}{\omega}$；$\varphi < 0$ 向右平移 $\frac{|\varphi|}{\omega}$
周期$T = \frac{2\pi}{\omega}$决定图象重复的频率
频率$f = \frac{1}{T} = \frac{\omega}{2\pi}$单位时间内完整周期的次数

变换顺序：$y = \sin x \rightarrow y = \sin(x + \varphi) \rightarrow y = \sin(\omega x + \varphi) \rightarrow y = A\sin(\omega x + \varphi)$

四、三角函数的应用（简谐运动）

  • 解析式：$y = A\sin(\omega x + \varphi)$（$A > 0, \omega > 0, x \in [0, +\infty)$）
  • 振幅：$A$
  • 周期：$T = \frac{2\pi}{\omega}$
  • 频率：$f = \frac{1}{T} = \frac{\omega}{2\pi}$
  • 相位：$\omega x + \varphi$
  • 初相：$\varphi$

解题路径（$y=A\sin(\omega x+\varphi)$ 参数求解三步法）

650分导向：广东卷三角大题第一/二问几乎必考"由条件定参"，步骤固定、不跳步。

第一步：抓周期

由已知周期 $T$（或范围）求 $\omega=\frac{2\pi}{T}$，注意 $\omega>0$ 且常需结合范围锁定唯一值。

第二步：用对称/最值定参

代入对称中心或最值点列方程求 $\varphi,b$。中心点满足正弦部分为 $0$、纵坐标为 $b$；最值点满足正弦部分为 $\pm1$。

第三步：代回验解

将求得的 $\omega,\varphi,b$ 代回原式，计算目标值并核对取值范围。

母题（2022 新课标Ⅰ卷·第6题，5分）

广东考生真题。由周期范围+中心对称定参，是 $y=A\sin(\omega x+\varphi)+b$ 的经典考法。

题目：记函数 $f(x)=\sin\left(\omega x+\dfrac{\pi}{4}\right)+b\;(\omega>0)$ 的最小正周期为 $T$。若 $\dfrac{2\pi}{3}

A. $1$　B. $\dfrac{3}{2}$　C. $\dfrac{5}{2}$　D. $3$

解： 由 $\frac{2\pi}{3}

中心 $\left(\frac{3\pi}{2},2\right)$ 在图象上且为正弦的平衡位置，故 $b=2$，且 $$\omega\cdot\frac{3\pi}{2}+\frac{\pi}{4}=k\pi\quad(k\in\mathbb Z)$$

即 $\omega=-\frac{1}{6}+\frac{2}{3}k$。结合 $2<\omega<3$，取 $k=4$ 得 $\omega=\frac{5}{2}$。

于是 $f(x)=\sin\left(\frac{5}{2}x+\frac{\pi}{4}\right)+2$， $$f\left(\frac{\pi}{2}\right)=\sin\left(\frac{5\pi}{4}+\frac{\pi}{4}\right)+2=\sin\frac{3\pi}{2}+2=-1+2=1$$

答：选 A。
