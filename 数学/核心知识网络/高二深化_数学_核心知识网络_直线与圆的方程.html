直线与圆的方程

⚠️待强化★★★☆☆#高二深化#新高考Ⅰ卷·广东来源：53科学备考《高中知识清单》数学知识图谱 / 人教A版选择性必修第一册第二章
广东考情：考查频率：中频；难度：中档；直线与圆常以选填形式出现，圆与圆锥曲线的综合是解析几何大题的前置基础

核心内容

一、直线的交点坐标与距离公式

  • 两直线交点：联立 $A_1 x + B_1 y + C_1 = 0$ 与 $A_2 x + B_2 y + C_2 = 0$ 求解
  • 两点间距离：$|P_1 P_2| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$
  • 点到直线距离：点 $P(x_0, y_0)$ 到直线 $Ax + By + C = 0$ 的距离 $d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$
  • 两平行线距离：$Ax + By + C_1 = 0$ 与 $Ax + By + C_2 = 0$ 的距离 $d = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$（$C_1 \neq C_2$）

二、圆的方程

  • 标准方程：$(x - a)^2 + (y - b)^2 = r^2$（$r > 0$），圆心 $(a, b)$，半径 $r$
  • 一般方程：$x^2 + y^2 + Dx + Ey + F = 0$（$D^2 + E^2 - 4F > 0$）
    • 圆心：$(-\frac{D}{2}, -\frac{E}{2})$，半径：$r = \frac{\sqrt{D^2 + E^2 - 4F}}{2}$

三、点与圆的位置关系

点 $M(x_0, y_0)$ 与圆 $C: (x - a)^2 + (y - b)^2 = r^2$：

  • 点在圆内：$(x_0 - a)^2 + (y_0 - b)^2 < r^2$
  • 点在圆上：$(x_0 - a)^2 + (y_0 - b)^2 = r^2$
  • 点在圆外：$(x_0 - a)^2 + (y_0 - b)^2 > r^2$

四、直线与圆的位置关系

设圆心到直线距离为 $d$，半径为 $r$：

  • $d < r$：相交（弦长 $= 2\sqrt{r^2 - d^2}$）
  • $d = r$：相切
  • $d > r$：相离

五、圆与圆的位置关系

设两圆圆心距为 $d$，半径分别为 $r_1, r_2$：

  • $d > r_1 + r_2$：外离
  • $d = r_1 + r_2$：外切
  • $|r_1 - r_2| < d < r_1 + r_2$：相交
  • $d = |r_1 - r_2|$：内切
  • $d < |r_1 - r_2|$：内含

关联卡片

  • 椭圆双曲线抛物线 — 圆锥曲线综合
  • 空间向量与立体几何 — 坐标法思想

备注

  • 直线与圆相切问题：利用圆心到直线距离等于半径，或联立方程判别式等于0
  • 弦长问题：弦长 $= 2\sqrt{r^2 - d^2}$，其中 $d$ 为圆心到直线距离
  • 广东卷常考：圆的切线方程、切点弦方程、两圆公共弦方程
