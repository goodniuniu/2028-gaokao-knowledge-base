三角函数的概念与诱导公式

核心内容

一、任意角与弧度制

  • 终边相同角：$S = \{\beta \mid \beta = \alpha + k \cdot 360°\}$ 或 $\beta = \alpha + 2k\pi$（$k \in \mathbb{Z}$）
  • 弧度制：$|\alpha| = \frac{l}{r}$，$1 \text{ rad} \approx 57.3°$
  • 换算：$\pi = 180°$，$1° = \frac{\pi}{180} \text{ rad}$

二、三角函数定义

单位圆定义：设角 $\alpha$ 终边与单位圆交于点 $P(x, y)$，则

  • $\sin\alpha = y$，$\cos\alpha = x$，$\tan\alpha = \frac{y}{x}$（$x \neq 0$）

一般定义：终边上一点 $P(x, y)$，$r = \sqrt{x^2 + y^2}$，则

  • $\sin\alpha = \frac{y}{r}$，$\cos\alpha = \frac{x}{r}$，$\tan\alpha = \frac{y}{x}$

三、三角函数值在各象限的符号

口诀：一全正，二正弦，三正切，四余弦

四、同角三角函数的基本关系式

$$\sin^2\alpha + \cos^2\alpha = 1, \quad \tan\alpha = \frac{\sin\alpha}{\cos\alpha}\quad(\alpha \neq k\pi + \frac{\pi}{2})$$

  • 应用：求值、化简、证明（知一求二）

五、诱导公式

公式正弦余弦正切记忆口诀
公式一：$2k\pi + \alpha$$\sin\alpha$$\cos\alpha$$\tan\alpha$周期不变
公式二：$\pi + \alpha$$-\sin\alpha$$-\cos\alpha$$\tan\alpha$同名
公式三：$-\alpha$$-\sin\alpha$$\cos\alpha$$-\tan\alpha$同名
公式四：$\pi - \alpha$$\sin\alpha$$-\cos\alpha$$-\tan\alpha$同名
公式五：$\frac{\pi}{2} - \alpha$$\cos\alpha$$\sin\alpha$$\cot\alpha$互余互换
公式六：$\frac{\pi}{2} + \alpha$$\cos\alpha$$-\sin\alpha$$-\cot\alpha$互余互换

统一口诀：奇变偶不变，符号看象限

  • "奇偶"：指 $\frac{\pi}{2}$ 的奇数倍或偶数倍
  • "变"：函数名改变（正弦 $\leftrightarrow$ 余弦，正切 $\leftrightarrow$ 余切）
  • "符号"：把 $\alpha$ 看成锐角，判断原函数在相应象限的符号

终边对称关系：

  • $\pi - \alpha$ 与 $\alpha$：终边关于 $y$ 轴对称
  • $\pi + \alpha$ 与 $\alpha$：终边关于原点对称
  • $2\pi - \alpha$ 与 $\alpha$：终边关于 $x$ 轴对称

关联卡片

  • 三角函数的图象与性质 — 诱导公式是图象分析的基础

备注

  • 易错点：诱导公式使用时"符号看象限"容易记错，建议用单位圆辅助判断
  • 给值求角问题时注意角的范围讨论，这是广东卷三角大题的第一问常考点
  • 化简原则：负化正、大化小、化到锐角
