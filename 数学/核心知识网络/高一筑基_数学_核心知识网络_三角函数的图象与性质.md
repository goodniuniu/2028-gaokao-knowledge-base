## 核心内容

### 一、基本三角函数图象与性质

| 性质 | $y = \sin x$ | $y = \cos x$ | $y = \tan x$ |
|------|-------------|-------------|-------------|
| 定义域 | $\mathbb{R}$ | $\mathbb{R}$ | $\{x \mid x \neq k\pi + \frac{\pi}{2}, k \in \mathbb{Z}\}$ |
| 值域 | $[-1, 1]$ | $[-1, 1]$ | $\mathbb{R}$ |
| 周期 | $2\pi$ | $2\pi$ | $\pi$ |
| 奇偶性 | 奇函数 | 偶函数 | 奇函数 |
| 递增区间 | $[-\frac{\pi}{2} + 2k\pi, \frac{\pi}{2} + 2k\pi]$ | $[(2k-1)\pi, 2k\pi]$ | $(-\frac{\pi}{2} + k\pi, \frac{\pi}{2} + k\pi)$ |
| 递减区间 | $[\frac{\pi}{2} + 2k\pi, \frac{3\pi}{2} + 2k\pi]$ | $[2k\pi, (2k+1)\pi]$ | 无 |
| 对称轴 | $x = \frac{\pi}{2} + k\pi$ | $x = k\pi$ | 无 |
| 对称中心 | $(k\pi, 0)$ | $(\frac{\pi}{2} + k\pi, 0)$ | $(\frac{k\pi}{2}, 0)$ |
| 最大值点 | $x = \frac{\pi}{2} + 2k\pi$ 时 $y = 1$ | $x = 2k\pi$ 时 $y = 1$ | 无 |
| 最小值点 | $x = -\frac{\pi}{2} + 2k\pi$ 时 $y = -1$ | $x = \pi + 2k\pi$ 时 $y = -1$ | 无 |

### 二、五点法作图（$y = \sin x, x \in [0, 2\pi]$）
| $x$ | $0$ | $\frac{\pi}{2}$ | $\pi$ | $\frac{3\pi}{2}$ | $2\pi$ |
|-----|-----|------------------|--------|-------------------|--------|
| $y$ | $0$ | $1$ | $0$ | $-1$ | $0$ |

### 三、函数 $y = A\sin(\omega x + \varphi)$（$A > 0, \omega > 0$）
| 参数 | 意义 | 影响 |
|------|------|------|
| $A$ | 振幅 | $A > 1$ 纵坐标伸长为 $A$ 倍；$0 < A < 1$ 缩短为 $A$ 倍 |
| $\omega$ | 角频率 | $\omega > 1$ 横坐标缩短为 $\frac{1}{\omega}$；$0 < \omega < 1$ 伸长为 $\frac{1}{\omega}$ 倍 |
| $\varphi$ | 初相 | $\varphi > 0$ 图象向左平移 $\frac{\varphi}{\omega}$；$\varphi < 0$ 向右平移 $\frac{|\varphi|}{\omega}$ |
| 周期 | $T = \frac{2\pi}{\omega}$ | 决定图象重复的频率 |
| 频率 | $f = \frac{1}{T} = \frac{\omega}{2\pi}$ | 单位时间内完整周期的次数 |

**变换顺序**：$y = \sin x \rightarrow y = \sin(x + \varphi) \rightarrow y = \sin(\omega x + \varphi) \rightarrow y = A\sin(\omega x + \varphi)$

### 四、三角函数的应用（简谐运动）
- 解析式：$y = A\sin(\omega x + \varphi)$（$A > 0, \omega > 0, x \in [0, +\infty)$）
- 振幅：$A$
- 周期：$T = \frac{2\pi}{\omega}$
- 频率：$f = \frac{1}{T} = \frac{\omega}{2\pi}$
- 相位：$\omega x + \varphi$
- 初相：$\varphi$

## 备注
- 求三角函数单调区间时，注意 $\omega$ 的符号：若 $\omega < 0$，先用诱导公式化为正
- 对称轴处取最值，对称中心处函数值为 $0$
- 广东卷常考：给定部分图象求解析式 $y = A\sin(\omega x + \varphi)$，注意最高点/最低点定 $A$，周期定 $\omega$，特殊点定 $\varphi$
