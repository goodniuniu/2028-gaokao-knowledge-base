统计与概率基础（抽样·分布·概率计算）

⚠️待强化★★★☆☆#高一筑基#广东名校模拟（广州各区期末、九区联考）来源：人教A版必修第二册第九章（统计）+第十章（概率） / 2024~2025学年广州各区高一下期末试卷（单选第7-8题、多选第10-11题、解答第17题）
广东考情：高一下期末**解答题固定板块**。广州各区期末卷解答题第17题（12-14分）几乎固定考查统计与概率的综合应用，通常是"频率分布直方图 + 概率计算"或"分层抽样 + 古典概型"的组合。选填中也有1-2道基础题（互斥事件、独立事件、抽样方法）。难度定位**中档**，核心在于准确读取图表信息和规范的概率计算步骤。

核心内容

本质理解：统计是"由样本推断总体"，概率是"随机事件发生的可能性度量"。核心思想：用数据说话，用频率估计概率。

一、统计基础：三种抽样方法

方法特点适用场景操作步骤广东卷考法
简单随机抽样每个个体等可能入选总体容量小抽签法/随机数法直接概念判断
分层抽样按层按比例抽取总体有明显分层各层抽样比 = 总体抽样比高频！计算各层样本数
系统抽样等间隔抽取总体容量大且有序编号→分段→定起点→等距抽取概念判断+计算

分层抽样公式：某层样本数 = 该层个体数 × 总体抽样比
示例：某校高一800人，高二700人，高三500人，共抽取100人。则高一抽取 $800 \times \frac{100}{2000} = 40$ 人。

二、频率分布直方图（解答题核心！）

1. 读图三要素

要素含义公式
纵坐标频率/组距$\frac{频率}{组距}$，不是频率！
矩形面积频率$面积 = \frac{频率}{组距} \times 组距 = 频率$
总面积1所有矩形面积之和 = 1

关键提醒：频率分布直方图的纵坐标是频率/组距，不是频率！计算频率必须乘以组距。

2. 由图计算数字特征

特征计算方法
频率矩形面积 = 纵坐标 × 组距
频数频率 × 样本容量
众数最高矩形中点的横坐标
中位数左右面积各为0.5时的横坐标（需插值）
平均数各矩形中点 × 对应频率，再求和
方差各矩形中点与平均数差的平方 × 频率，再求和

中位数计算步骤：
1. 从左到右累加频率，找到中位数所在组
2. 设中位数为$x$，利用比例关系：$\frac{x - 组下限}{组距} = \frac{0.5 - 前面积}{该组频率}$

三、概率基础：事件关系与运算

1. 事件关系

关系定义符号概率公式
包含A发生则B必发生$A \subseteq B$$P(A) \leq P(B)$
相等A、B同时发生或不发生$A = B$$P(A) = P(B)$
并事件A或B发生$A \cup B$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$
交事件A且B发生$A \cap B$—
互斥A、B不能同时发生$A \cap B = \varnothing$$P(A \cup B) = P(A) + P(B)$
对立互斥且必有一个发生$\bar{A}$$P(A) + P(\bar{A}) = 1$

关键区分：互斥 ≠ 对立。对立一定互斥，互斥不一定对立。

2. 概率基本性质

性质内容说明
非负性$P(A) \geq 0$概率不可能为负
规范性$P(\Omega) = 1$必然事件概率为1
可加性互斥事件：$P(A_1 \cup A_2 \cup ... \cup A_n) = P(A_1) + ... + P(A_n)$互斥才能直接相加
对立公式$P(\bar{A}) = 1 - P(A)$正难则反的数学思想

四、古典概型（广东卷解答题核心！）

1. 定义与公式

$$P(A) = \frac{事件A包含的基本事件数}{样本空间的基本事件总数} = \frac{n(A)}{n(\Omega)}$$

适用条件：① 有限个基本事件；② 每个基本事件等可能发生。

2. 常用计数方法

方法适用场景公式
列举法基本事件数少（≤20）一一列出
排列数有序选取$A_n^m = n(n-1)...(n-m+1)$
组合数无序选取$C_n^m = \frac{n!}{m!(n-m)!}$
分步乘法分步完成各步方法数相乘
分类加法分类完成各类方法数相加

广东卷特点：古典概型题常与频率分布直方图结合，先由直方图得频率，再用频率估计概率，最后计算复合事件概率。

五、事件独立性（高频！）

1. 定义

$$A, B 独立 \Leftrightarrow P(A \cap B) = P(A) \cdot P(B)$$

2. 独立事件的性质

性质内容
对称性A与B独立 ⟺ B与A独立
与对立独立A与B独立 ⟺ A与$\bar{B}$独立 ⟺ $\bar{A}$与B独立 ⟺ $\bar{A}$与$\bar{B}$独立
乘法公式$P(A_1 \cap A_2 \cap ... \cap A_n) = P(A_1)P(A_2)...P(A_n)$（各事件独立）

与互斥的区别：
- 互斥：$A \cap B = \varnothing$，$P(A \cup B) = P(A) + P(B)$
- 独立：$P(A \cap B) = P(A)P(B)$，$A$发生不影响$B$的概率
- 互斥与独立一般不共存（除非其中一个概率为0）

六、频率估计概率（大题答题规范）

标准表述：

"由频率分布直方图可知，事件A的频率为$\frac{频数}{样本容量} = 0.XX$，故用频率估计概率，得$P(A) \approx 0.XX$。"

广东卷评分要点：必须写明"用频率估计概率"这句话，不能直接写$P(A) = 频率$！

七、记忆口诀

统计口诀：
- "频率分布直方图，纵轴频率除以距"
- "面积才是频率值，总面积必等于一"
- "中位数找左右半，平均数用中点乘"
- "分层按比例，系统按间隔"

概率口诀：
- "古典概型两条件：有限等可能"
- "互斥加，独立乘，对立用一减"
- "正难则反是良策，对立事件来帮忙"

备注

  • 广东高一下期末命题规律：解答题第17题固定是统计与概率综合，通常分为两问：第1问求频率分布直方图中的参数（如$X$的值）、中位数或平均数；第2问用频率估计概率，计算某复合事件的概率（如"至少有一人合格"）。
  • 2024-2025学年广州越秀区期末第17题：频率分布直方图 + 古典概型 + 独立事件，是广东卷"统计与概率融合"的典型考法。
