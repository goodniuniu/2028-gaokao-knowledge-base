## 核心内容

### 一、直线的交点坐标与距离公式
- **两直线交点**：联立 $A_1 x + B_1 y + C_1 = 0$ 与 $A_2 x + B_2 y + C_2 = 0$ 求解
- **两点间距离**：$|P_1 P_2| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$
- **点到直线距离**：点 $P(x_0, y_0)$ 到直线 $Ax + By + C = 0$ 的距离 $d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$
- **两平行线距离**：$Ax + By + C_1 = 0$ 与 $Ax + By + C_2 = 0$ 的距离 $d = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$（$C_1 \neq C_2$）

### 二、圆的方程
- **标准方程**：$(x - a)^2 + (y - b)^2 = r^2$（$r > 0$），圆心 $(a, b)$，半径 $r$
- **一般方程**：$x^2 + y^2 + Dx + Ey + F = 0$（$D^2 + E^2 - 4F > 0$）
  - 圆心：$(-\frac{D}{2}, -\frac{E}{2})$，半径：$r = \frac{\sqrt{D^2 + E^2 - 4F}}{2}$

### 三、点与圆的位置关系
点 $M(x_0, y_0)$ 与圆 $C: (x - a)^2 + (y - b)^2 = r^2$：
- 点在圆内：$(x_0 - a)^2 + (y_0 - b)^2 < r^2$
- 点在圆上：$(x_0 - a)^2 + (y_0 - b)^2 = r^2$
- 点在圆外：$(x_0 - a)^2 + (y_0 - b)^2 > r^2$

### 四、直线与圆的位置关系
设圆心到直线距离为 $d$，半径为 $r$：
- $d < r$：相交（弦长 $= 2\sqrt{r^2 - d^2}$）
- $d = r$：相切
- $d > r$：相离

### 五、圆与圆的位置关系
设两圆圆心距为 $d$，半径分别为 $r_1, r_2$：
- $d > r_1 + r_2$：外离
- $d = r_1 + r_2$：外切
- $|r_1 - r_2| < d < r_1 + r_2$：相交
- $d = |r_1 - r_2|$：内切
- $d < |r_1 - r_2|$：内含

| 题干关键条件 | 识别为 | 首选方法 |

## 解题路径（圆上点与直线·数形结合五步）

### 第一步：写圆与直线方程
写出圆心 $M$、半径 $r$；直线化为一般式 $Ax+By+C=0$。

### 第二步：求圆心到直线距离
$$d=\frac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}.$$

### 第三步：定距离范围
圆上点到该直线的距离 $\in[d-r,\ d+r]$，据此判断最值选项。

### 第四步：角的极值
过定点 $B$ 作圆的切线，切点处 $\angle PBA$ 取得最小或最大。

### 第五步：勾股定理求切线长
$$|PB|=\sqrt{|BM|^2-r^2}.$$

> **易错提醒**：把圆心到直线距离当成了点到直线距离的最值；忘记切线长公式而直接猜；未判断选项是否"一定成立"。

## 母题（2021 新课标Ⅰ卷·第11题，5分）

**题目**：已知点 $P$ 在圆 $(x-5)^2+(y-5)^2=16$ 上，点 $A(4,0)$，$B(0,2)$，则（　）
A. 点 $P$ 到直线 $AB$ 的距离小于 $10$
B. 点 $P$ 到直线 $AB$ 的距离大于 $2$
C. 当 $\angle PBA$ 最小时，$|PB|=3\sqrt2$
D. 当 $\angle PBA$ 最大时，$|PB|=3\sqrt2$

**解答**：
直线 $AB$：$\frac{x}{4}+\frac{y}{2}=1$，即 $x+2y-4=0$。
圆心 $M(5,5)$，半径 $r=4$，圆心到直线距离
$$d=\frac{|5+10-4|}{\sqrt{1^2+2^2}}=\frac{11}{\sqrt5}>4.$$
故圆上点到 $AB$ 的距离范围为 $\left[\frac{11}{\sqrt5}-4,\ \frac{11}{\sqrt5}+4\right]$。
因 $\frac{11}{\sqrt5}<5$，所以 $\frac{11}{\sqrt5}-4<1$，$\frac{11}{\sqrt5}+4<10$，故 A 正确、B 错误。

当 $PB$ 与圆相切时，$\angle PBA$ 取得最小或最大。此时
$$|BM|=\sqrt{(5-0)^2+(5-2)^2}=\sqrt{34},\quad |PB|=\sqrt{|BM|^2-r^2}=\sqrt{34-16}=3\sqrt2,$$
故 C、D 正确。

综上选 **ACD**。
