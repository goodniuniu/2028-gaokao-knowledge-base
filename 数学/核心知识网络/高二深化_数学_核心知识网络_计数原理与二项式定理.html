计数原理与二项式定理

⚠️待强化★★★☆☆#高二深化#新高考Ⅰ卷·广东来源：53科学备考《高中知识清单》数学知识图谱 / 人教A版选择性必修第三册第六章
广东考情：考查频率：中频；难度：中档；排列组合与二项式定理是选填常考点，注意分类讨论和排除法的运用

核心内容

一、排列与组合

  • 排列：从 $n$ 个不同元素中取出 $m$（$m \leq n$）个元素，按一定顺序排成一列
  • 排列数：$A_n^m = n(n-1)(n-2)\cdots(n-m+1) = \frac{n!}{(n-m)!}$（$m, n \in \mathbb{N}^*, m \leq n$）
  • 组合：从 $n$ 个不同元素中取出 $m$（$m \leq n$）个元素作为一组
  • 组合数：$C_n^m = \frac{A_n^m}{A_m^m} = \frac{n(n-1)(n-2)\cdots(n-m+1)}{m!} = \frac{n!}{m!(n-m)!}$（$m, n \in \mathbb{N}^*, m \leq n$）

二、组合数性质

  • $C_n^m = C_n^{n-m}$（对称性）
  • $C_{n+1}^m = C_n^m + C_n^{m-1}$（递推公式）

三、二项式定理

$$(a + b)^n = C_n^0 a^n + C_n^1 a^{n-1}b + C_n^2 a^{n-2}b^2 + \cdots + C_n^k a^{n-k}b^k + \cdots + C_n^n b^n$$

  • 通项公式：$T_{k+1} = C_n^k a^{n-k} b^k$（$k = 0, 1, 2, \dots, n$）
  • 二项式系数：第 $k+1$ 项的二项式系数为 $C_n^k$

四、二项式系数的性质

  • 对称性：$C_n^k = C_n^{n-k}$
  • 增减性与最大值：
  • $n$ 为偶数时，中间一项 $C_n^{\frac{n}{2}}$ 最大
  • $n$ 为奇数时，中间两项 $C_n^{\frac{n-1}{2}}$ 和 $C_n^{\frac{n+1}{2}}$ 相等且最大
  • 各二项式系数和：$C_n^0 + C_n^1 + C_n^2 + \cdots + C_n^n = 2^n$
  • 奇数项与偶数项的二项式系数和相等：$C_n^0 + C_n^2 + C_n^4 + \cdots = C_n^1 + C_n^3 + C_n^5 + \cdots = 2^{n-1}$
