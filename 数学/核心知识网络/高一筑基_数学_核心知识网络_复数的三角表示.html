复数的三角表示

⚠️待强化★★★☆☆#高一筑基#新高考Ⅰ卷·广东来源：53科学备考《高中知识清单》数学知识图谱 / 人教A版必修第二册第七章
广东考情：考查频率：低频；难度：基础；选填基础题，主要考查三角形式与代数形式的互化及几何意义

核心内容

一、复数的三角形式

设 $z = a + bi$（$a, b \in \mathbb{R}$），则 $a = r\cos\theta$，$b = r\sin\theta$，$r = \sqrt{a^2 + b^2}$。

$$z = r(\cos\theta + i\sin\theta)$$

其中：

  • $r = |z| = \sqrt{a^2 + b^2}$：复数的模
  • $\theta$：辐角，满足 $\tan\theta = \frac{b}{a}$
  • 辐角主值：$0 \leq \theta < 2\pi$

二、代数形式与三角形式的互化

  • 代数 $\rightarrow$ 三角：求模 $r = \sqrt{a^2 + b^2}$，求辐角 $\tan\theta = \frac{b}{a}$（根据象限确定）
  • 三角 $\rightarrow$ 代数：$a = r\cos\theta$，$b = r\sin\theta$

三、复数乘除法的三角表示

设 $z_1 = r_1(\cos\theta_1 + i\sin\theta_1)$，$z_2 = r_2(\cos\theta_2 + i\sin\theta_2)$：

  • 乘法：$z_1 z_2 = r_1 r_2[\cos(\theta_1 + \theta_2) + i\sin(\theta_1 + \theta_2)]$
  • 模相乘，辐角相加
  • 除法：$\frac{z_1}{z_2} = \frac{r_1}{r_2}[\cos(\theta_1 - \theta_2) + i\sin(\theta_1 - \theta_2)]$
  • 模相除，辐角相减

四、几何意义

  • 复数乘法：对应向量旋转和伸缩
  • 复数除法：对应向量反向旋转和伸缩
  • 乘以 $i$：逆时针旋转 $90°$
  • 乘以 $-i$：顺时针旋转 $90°$

备注

  • 易错点：求辐角时必须结合复数所在象限，不能仅由 $\tan\theta = \frac{b}{a}$ 确定
  • 复数三角形式的标准形式：$r(\cos\theta + i\sin\theta)$，注意中间是 "$+$"，不是 "$-$"
