## 核心内容

### 一、复数的三角形式
设 $z = a + bi$（$a, b \in \mathbb{R}$），则 $a = r\cos\theta$，$b = r\sin\theta$，$r = \sqrt{a^2 + b^2}$。

$$z = r(\cos\theta + i\sin\theta)$$

其中：
- $r = |z| = \sqrt{a^2 + b^2}$：复数的模
- $\theta$：辐角，满足 $\tan\theta = \frac{b}{a}$
- **辐角主值**：$0 \leq \theta < 2\pi$

### 二、代数形式与三角形式的互化
- **代数 $\rightarrow$ 三角**：求模 $r = \sqrt{a^2 + b^2}$，求辐角 $\tan\theta = \frac{b}{a}$（根据象限确定）
- **三角 $\rightarrow$ 代数**：$a = r\cos\theta$，$b = r\sin\theta$

### 三、复数乘除法的三角表示
设 $z_1 = r_1(\cos\theta_1 + i\sin\theta_1)$，$z_2 = r_2(\cos\theta_2 + i\sin\theta_2)$：

- **乘法**：$z_1 z_2 = r_1 r_2[\cos(\theta_1 + \theta_2) + i\sin(\theta_1 + \theta_2)]$
  - 模相乘，辐角相加
- **除法**：$\frac{z_1}{z_2} = \frac{r_1}{r_2}[\cos(\theta_1 - \theta_2) + i\sin(\theta_1 - \theta_2)]$
  - 模相除，辐角相减

### 四、几何意义
- 复数乘法：对应向量旋转和伸缩
- 复数除法：对应向量反向旋转和伸缩
- 乘以 $i$：逆时针旋转 $90°$
- 乘以 $-i$：顺时针旋转 $90°$

## 解题路径（复数三角形式乘除三步法）

> 650分导向：三角形式是复数运算的"几何化"利器，处理旋转、开方、乘方尤其简洁。

### 第一步：取模定角
将每个复数写成 $r(\cos\theta+i\sin\theta)$，确定模 $r$ 与辐角 $\theta$（按象限）。

### 第二步：模角运算
乘法：$r_1r_2$、角 $\theta_1+\theta_2$；除法：$r_1/r_2$、角 $\theta_1-\theta_2$。

### 第三步：回代数（如需）
由 $a=r\cos\theta,\;b=r\sin\theta$ 写回 $a+bi$。

**题目（代表性）**：设复数 $z_1=2\left(\cos\dfrac{\pi}{6}+i\sin\dfrac{\pi}{6}\right)$，$z_2=\sqrt2\left(\cos\dfrac{\pi}{4}+i\sin\dfrac{\pi}{4}\right)$，求 $z_1z_2$ 的代数形式。

**解**：
由三角形式乘法，
$$z_1z_2=2\sqrt2\left[\cos\left(\frac{\pi}{6}+\frac{\pi}{4}\right)+i\sin\left(\frac{\pi}{6}+\frac{\pi}{4}\right)\right]
=2\sqrt2\left(\cos\frac{5\pi}{12}+i\sin\frac{5\pi}{12}\right)$$

其中 $\cos\frac{5\pi}{12}=\cos75^\circ=\frac{\sqrt6-\sqrt2}{4}$，$\sin\frac{5\pi}{12}=\sin75^\circ=\frac{\sqrt6+\sqrt2}{4}$，故
$$z_1z_2=2\sqrt2\cdot\frac{\sqrt6-\sqrt2}{4}+i\cdot2\sqrt2\cdot\frac{\sqrt6+\sqrt2}{4}
=(\sqrt3-1)+i(\sqrt3+1)$$

## 备注
- 易错点：求辐角时必须结合复数所在象限，不能仅由 $\tan\theta = \frac{b}{a}$ 确定
- 复数三角形式的标准形式：$r(\cos\theta + i\sin\theta)$，注意中间是 "$+$"，不是 "$-$"
