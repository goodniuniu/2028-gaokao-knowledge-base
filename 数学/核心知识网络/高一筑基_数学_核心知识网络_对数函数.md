## 核心内容

### 一、对数的概念与性质
- **定义**：若 $a^x = N$（$a > 0$ 且 $a \neq 1$），则 $x = \log_a N$
- **基本性质**：
  - 负数和 $0$ 没有对数
  - $\log_a 1 = 0$
  - $\log_a a = 1$
- **对数恒等式**：$a^{\log_a N} = N$（$a > 0, a \neq 1, N > 0$）

### 二、对数运算性质（$a > 0, a \neq 1, M > 0, N > 0$）
- $\log_a(MN) = \log_a M + \log_a N$（积的对数 = 对数之和）
- $\log_a\frac{M}{N} = \log_a M - \log_a N$（商的对数 = 对数之差）
- $\log_a M^n = n\log_a M$（$n \in \mathbb{R}$）

### 三、换底公式
$$\log_a b = \frac{\log_c b}{\log_c a} \quad (a > 0, a \neq 1; c > 0, c \neq 1; b > 0)$$

**常用推论**：
- $\log_a b = \frac{1}{\log_b a}$
- $\log_{a^n} b^m = \frac{m}{n}\log_a b$
- $\log_a b \cdot \log_b c = \log_a c$

### 四、对数函数 $y = \log_a x$（$a > 0$ 且 $a \neq 1$）
| 性质 | $a > 1$ | $0 < a < 1$ |
|------|---------|-------------|
| 图象 | 上升，过 $(1, 0)$ | 下降，过 $(1, 0)$ |
| 定义域 | $(0, +\infty)$ | $(0, +\infty)$ |
| 值域 | $\mathbb{R}$ | $\mathbb{R}$ |
| 单调性 | 在 $(0, +\infty)$ 上递增 | 在 $(0, +\infty)$ 上递减 |

### 五、反函数关系
- 同底的指数函数与对数函数互为反函数
- 互为反函数的两个函数图象关于直线 $y = x$ 对称

## 题型识别标志

> **看到什么条件 → 立刻想到什么方法**

| 题干关键条件 | 识别为 | 首选方法 |
|------|------|------|
| 出现 $f(x)=\log_a x$ 与指数式混合 | 指对互化 | 用 $a^{\log_a N}=N$、$\log_a a^x=x$ 统一形式 |
| 比较 $\log_a M$ 与 $\log_a N$ 大小 | 同底对数比大小 | 利用单调性（看 $a>1$ 或 $0<a<1$） |
| 不同底对数比较（如 $\log_2 3$ 与 $\log_3 4$） | 异底比大小 | 换底公式或中间值 $0,1$ |
| 真数含变量求定义域 | 对数定义域 | 令真数 $>0$，且底数 $>0$ 且 $\ne 1$ |
| 含 $|\log_a x|$ 或 $|2x-1|$ | 绝对值分段 | 以符号变化点（如 $x=1$、$x=\frac12$）分段讨论 |
| 求 $f(x)$ 最小值/零点 | 函数最值 | 分段求导 $f'(x)$，比较分段点与驻点 |

## 解题路径（对数最值·分段讨论四步法）

### 第一步：定定义域与分界点
真数 $x>0$；绝对值/对数符号变化点作为分段界（如 $|2x-1|$ 以 $x=\frac12$ 为界，$\log_a x$ 以 $x=1$ 为界）。

### 第二步：分段去绝对值、化简
在各段内去掉绝对值，写出 $f(x)$ 的显式表达式；注意 $\ln 1=0$ 常是关键的零点。

### 第三步：分段求导定单调性
对每段求导 $f'(x)$，解 $f'(x)=0$ 找驻点；判断每段内增减，列出各段的最小值候选（驻点、分段点）。

### 第四步：比较得全局最值
取各段最小候选中的最小者；**必须**核对分段点处函数值，否则易漏解。

> **易错提醒**：忘记 $x>0$ 定义域；误以为绝对值整体单调；直接用单调性却跨段比较。

## 母题（2021 新课标Ⅰ卷·第15题，5分）

**题目**：已知函数 $f(x)=|2x-1|-2\ln x$，则 $f(x)$ 的最小值为 ________。

**解答**：
函数定义域为 $x>0$，以 $x=\frac12$ 为界分段：

- 当 $0<x<\frac12$ 时，$f(x)=1-2x-2\ln x$，
  $$f'(x)=-2-\frac{2}{x}<0,$$
  故在 $(0,\frac12)$ 上单调递减，且 $f(x)>f(\frac12)=2\ln 2>1$。

- 当 $x\ge \frac12$ 时，$f(x)=2x-1-2\ln x$，
  $$f'(x)=2-\frac{2}{x}=\frac{2(x-1)}{x}.$$
  当 $\frac12\le x<1$ 时 $f'(x)<0$；当 $x>1$ 时 $f'(x)>0$，故在 $x=1$ 处取最小值：
  $$f(1)=|2-1|-2\ln 1=1.$$

综上，$f(x)$ 的最小值为 **$1$**（在 $x=1$ 处取得）。
