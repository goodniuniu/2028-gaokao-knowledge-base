核心内容

一、集合的概念与表示

  • 定义：研究对象称为元素，元素组成的总体叫做集合
  • 元素特性：确定性、互异性、无序性
  • 表示法：
  • 列举法：$\{a_1, a_2, a_3, \dots\}$
  • 描述法：$\{x \in A \mid P(x)\}$
  • 图示法：Venn图
  • 常用数集：$\mathbb{N}^*$（正整数）$\subset \mathbb{N}$（自然数）$\subset \mathbb{Z}$（整数）$\subset \mathbb{Q}$（有理数）$\subset \mathbb{R}$（实数）$\subset \mathbb{C}$（复数）

二、集合间的基本关系

  • 子集：$A \subseteq B$（$\forall x \in A, x \in B$）
  • 真子集：$A \subsetneqq B$（$A \subseteq B$ 且 $A \neq B$）
  • 集合相等：$A = B \Leftrightarrow A \subseteq B$ 且 $B \subseteq A$
  • 空集：$\varnothing$ 是任何集合的子集；是任何非空集合的真子集
  • 含 $n$ 个元素的集合：子集个数 $2^n$，真子集个数 $2^n-1$，非空真子集 $2^n-2$

三、集合的基本运算

运算定义Venn图特征
并集 $A \cup B$$\{x \mid x \in A$ 或 $x \in B\}$两圆覆盖的全部区域
交集 $A \cap B$$\{x \mid x \in A$ 且 $x \in B\}$两圆重叠区域
补集 $\complement_U A$$\{x \mid x \in U$ 且 $x \notin A\}$全集中圆外区域

运算律：

  • $A \cup B = B \cup A$；$A \cap B = B \cap A$
  • $(A \cup B) \cup C = A \cup (B \cup C)$
  • $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
  • $\complement_U(A \cup B) = \complement_U A \cap \complement_U B$（德摩根律）
  • $\complement_U(A \cap B) = \complement_U A \cup \complement_U B$

四、充分条件与必要条件

设 $A = \{x \mid p(x)\}$，$B = \{x \mid q(x)\}$：

条件关系集合关系口诀
$p$ 是 $q$ 的充分不必要条件$A \subsetneqq B$小推大
$p$ 是 $q$ 的必要不充分条件$B \subsetneqq A$大推小
$p$ 是 $q$ 的充要条件$A = B$等价
$p$ 是 $q$ 的既不充分也不必要条件$A \not\subseteq B$ 且 $B \not\subseteq A$无关

五、全称量词与存在量词

  • 全称量词命题：$\forall x \in M, p(x)$，否定为 $\exists x \in M, \neg p(x)$
  • 存在量词命题：$\exists x \in M, p(x)$，否定为 $\forall x \in M, \neg p(x)$
  • 规律：全称 $\leftrightarrow$ 存在互换，结论取反

关联卡片

  • 函数的定义与三要素 — 集合是函数定义域、值域的基础

备注

  • 易错点：空集是任何集合的子集，但 $\varnothing \subseteq \varnothing$ 成立，$\varnothing \in \varnothing$ 不成立
  • 广东卷常考：集合与不等式解集的综合运算，注意端点检验
  • 充分必要条件判断优先用集合法，直观不易错
