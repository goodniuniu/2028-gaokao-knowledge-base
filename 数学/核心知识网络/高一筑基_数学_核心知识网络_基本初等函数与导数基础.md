## 核心内容

> **本质理解**：基本初等函数是"原材料"，导数是"加工工具"。先认清每个基本函数的图与性，再掌握求导四则，才能应对压轴。

---

### 一、知识网络

```
基本初等函数
 ├─ 幂函数 y=x^a（过(1,1)，a符号定象限增减）
 ├─ 指数 y=a^x (a>0,a≠1) — 过(0,1)，a>1增
 ├─ 对数 y=log_a x — 过(1,0)，与指数互逆
 ├─ 三角 y=sin/cos/tan — 周期、有界
 └─ 导数基础
      ├─ 求导公式：(x^n)'=nx^{n-1}, (e^x)'=e^x, (lnx)'=1/x
      ├─ 四则：(uv)'=u'v+uv', (u/v)'=(u'v-uv')/v²
      ├─ 复合：链式法则
      └─ 几何意义：f'(x0)=切线斜率；切线 y-f(x0)=f'(x0)(x-x0)
```

### 二、核心公式/结论

| 公式 | 内容 | 注意 |
|------|------|------|
| 求导公式 | $(e^x)'=e^x,\ (\ln x)'=\frac1x,\ (x^n)'=nx^{n-1}$ | 常背常新 |
| 公切线 | 两曲线共点共斜率：$f'(x_0)=g'(x_1)$ 且切线方程同 | 切点可不同 |
| 分段单调 | 全 R 单调 ⇒ 各段单调且**衔接处不降/不增** | 端点值比较不可省 |
| 对数恒等式 | $a^{\log_a x}=x,\ \log_a a^x=x$ | 与指数互逆 |

### 三、易混点

| 易混点 | 正解 | 错法 |
|--------|------|------|
| 分段函数在分界点单调 | 需左段末值 ≤ 右段初值（递增） | 只看各段导数>0 |
| 公切线切点 | 两曲线切点**可不同** | 误设同一点 |
| $(a^x)'$ | $a^x\ln a$ | 误为 $xa^{x-1}$ |
| $\ln(x+1)$ 定义域 | $x>-1$ | 忽略对数定义域 |

### 真题母题（2024 新课标Ⅰ卷·第13题，5分）

**题目**：若曲线 $y=e^x+x$ 在点 $(0,1)$ 处的切线也是曲线 $y=\ln(x+1)+a$ 的切线，则 $a=$___。

**解析**：
$y=e^x+x$ 在 $(0,1)$：$y'=e^x+1$，斜率 $k=2$，切线 $y=2x+1$。
对 $y=\ln(x+1)+a$，设切点 $(t,\ln(t+1)+a)$，斜率 $y'|_{t}=\frac1{t+1}=2\Rightarrow t+1=\frac12\Rightarrow t=-\frac12$。
切点纵坐标 $\ln\frac12+a=-\ln2+a$，须满足切线：$2(-\frac12)+1=0=-\ln2+a\Rightarrow a=\ln2$。
**答案：$\ln2$**。

**关键提炼**：公切线题是"求导+点斜式"的标准调用；T6分段单调则考"衔接处不降"，均属导数基础。
