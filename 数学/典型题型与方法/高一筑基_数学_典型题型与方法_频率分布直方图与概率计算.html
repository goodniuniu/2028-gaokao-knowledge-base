核心内容

本质理解：频率分布直方图是"用样本的频率分布估计总体的概率分布"的可视化工具。核心操作：读图→算频率→估概率→求复合事件概率。

一、大题标准解题模板（五步式）

#### 第1步：求直方图参数（如$a$的值）

原理：所有矩形面积之和 = 1

标准写法：

由频率分布直方图可知，各组频率/组距分别为 [列出已知值]，
组距为 [值]。

∵ 所有矩形面积之和为1
∴ (a_1 + a_2 + ... + a_n) × 组距 = 1
即 [方程]
解得：a = [值]
广东卷评分要点：必须写"所有矩形面积之和为1"的理由，不能凭空列方程。

#### 第2步：计算频率、频数

标准写法：

[某组]的频率 = [该组纵坐标] × 组距 = [值]
[某组]的频数 = 频率 × 样本容量 = [值] × [样本容量] = [值]

#### 第3步：计算中位数（高频！）

标准写法：

设中位数为x。

∵ 中位数左右两侧面积各为0.5
左侧累加面积：[计算过程]
可知中位数落在[第k组]内。

列方程：前(k-1)组面积 + (x - 下限) × 该组纵坐标 = 0.5
即 [方程]
解得：x = [值]

∴ 中位数为 [值]
关键：中位数不是最高矩形中点！而是使左右面积各为0.5的横坐标。

#### 第4步：计算平均数（或估计总体平均数）

标准写法：

样本平均数 = Σ(各组中点 × 该组频率)
           = [组1中点]×[组1频率] + [组2中点]×[组2频率] + ...
           = [计算过程]
           = [值]

∴ 估计总体平均数为 [值]
注意：用样本平均数估计总体平均数时，必须写"估计"二字。

#### 第5步：概率计算（大题第2问核心！）

标准写法：

由（1）可知，[某事件A]的频率为 [值]。
用频率估计概率，得 P(A) ≈ [值]。

[根据题意，分情况计算]

情况1：...
P(情况1) = ...

情况2：...
P(情况2) = ...

∴ 所求概率 P = P(情况1) + P(情况2) = [值]
广东卷必写语句："用频率估计概率"——不写扣1分！

二、高频概率计算类型

#### 类型1：古典概型（列举法）

适用：样本空间小（基本事件数≤20）

标准步骤：

  1. 设事件："设事件A为..."
  2. 列举所有基本事件：用坐标、编号或树状图列出$n(\Omega)$
  3. 找出事件A包含的基本事件：列出$n(A)$
  4. 计算概率：$P(A) = \frac{n(A)}{n(\Omega)} = \frac{[值]}{[值]}$
示例：从分层抽样的5人中选2人，求恰有1男1女的概率。 - 设5人为$M_1, M_2, F_1, F_2, F_3$ - 所有基本事件：$C_5^2 = 10$种 - 事件A（1男1女）：$2 \times 3 = 6$种 - $P(A) = \frac{6}{10} = \frac{3}{5}$

#### 类型2：独立事件的复合概率

适用：多次试验，各次结果互不影响

公式：

  • $P(A \cap B) = P(A) \cdot P(B)$（A、B独立）
  • $P(A \cup B) = P(A) + P(B) - P(A)P(B)$（A、B独立）
  • $P(\text{至少一个发生}) = 1 - P(\text{都不发生}) = 1 - P(\bar{A})P(\bar{B})$
广东卷高频："至少有一人合格""至少有一次成功"——用对立事件$1 - P(\text{全不合格})$。

#### 类型3：条件概率（高一拓展）

公式：$P(B|A) = \frac{P(A \cap B)}{P(A)}$（$P(A) > 0$）

缩小样本空间法：$P(B|A) = \frac{n(A \cap B)}{n(A)}$

适用场景：已知某条件发生，求另一事件的概率。如"已知选中的男生，求是优秀生的概率"。

三、大题答题规范（广东卷评分细则）

项目要求分值常见错误
设事件必须设"设事件A为..."1分不设事件，直接写$P = ...$
列总数写明样本空间总数$n(\Omega) = ...$1分不写总数，直接写概率
列事件数写明事件包含的基本事件数$n(A) = ...$1分不写事件数
写公式$P(A) = \frac{n(A)}{n(\Omega)}$1分直接写结果
计算结果分数或小数（保留要求）1分计算错误
统计语句"用频率估计概率"1分不写此句
广东卷评分要点：统计大题共12-14分，概率计算部分通常占6-8分。按上述规范书写，可确保不丢步骤分。

四、高频易错点速查

易错点正确做法错误示例
混淆频率与频率/组距频率 = 纵坐标 × 组距直接把纵坐标当频率
中位数计算错误中位数是面积平分点，不是最高矩形中点取最高矩形中点横坐标
平均数用组端点平均数用组中点，不是组端点用组下限或上限
概率语句不写必须写"用频率估计概率"直接写$P = 0.3$
互斥与独立混淆互斥用加法，独立用乘法互斥事件直接相乘
"至少"问题直接算正难则反，用$1 - P(\text{对立})$逐个情况相加，漏算

五、记忆口诀

统计大题口诀： - "频率分布直方图，面积等于频率值" - "中位数找左右半，平均数用中点乘" - "概率计算三步走：设事件、列总数、套公式" - "用频率估计概率，这句话必须写" - "至少问题正难反，一减对立最方便" - "独立事件乘法连，互斥事件加法连"
