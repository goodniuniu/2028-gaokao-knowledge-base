## 核心内容

### 题型识别标志
> **看到什么条件 → 立刻想到什么方法**

| 题干关键条件 | 识别为 | 首选方法 |
|-------------|--------|----------|
| "每次由其中一人投篮/抽奖，命中则继续、不中则换人" | 马尔可夫链 | 设 aₙ=P(第n次状态)，建递推 |
| "第n次处于某状态的概率" | 递推数列 | aₙ = p·aₙ₋₁ + q·(1-aₙ₋₁) 构造等比 |
| "已知 Xi 两点分布，E(ΣXi)=Σqi" | 期望线性性 | 直接求和 Σaᵢ |
| 随机变量取值为 0/1（是否发生） | 两点分布 | 用指示变量把计数转化为 ΣXi |
| 多个互斥状态转移 | 全概率公式 | 按上一状态分类 |

### 解题路径（概率递推大题四步法）
#### 第一步：建模设变量
设 aₙ = "第 n 次处于目标状态"的概率，写出 a₁（初始概率）。
#### 第二步：建递推
用全概率：aₙ = P(本状态命中留)+P(他状态未中换)，化为 aₙ = p·aₙ₋₁ + q（一阶线性递推）。
#### 第三步：求通项
构造常数列 aₙ + c = k(aₙ₋₁ + c) 化为等比，得 aₙ = A·k^(n-1) + B。
#### 第四步：算期望
利用两点分布期望线性性 E(Y) = Σaᵢ，分组（等比求和+常数求和）得结果。

### 母题（2023 新课标Ⅰ卷·第21题，12分）
**题目**：甲、乙两人投篮，每次由其中一人投篮，规则：若命中则此人继续，若未命中则换对方。甲命中率0.6，乙命中率0.8，第一次由抽签决定，甲、乙各0.5。
(1) 求第2次投篮的人是乙的概率；
(2) 求第 i 次投篮的人是甲的概率；
(3) 记前 n 次中甲投篮次数为 Y（Xi 为两点分布、P(Xi=1)=aᵢ），求 E(Y)。

**解析**：
(1) P(第2次是乙) = P(甲先投且未中) + P(乙先投且命中) = 0.5×0.4 + 0.5×0.8 = 0.6 = **3/5**。
(2) 设 aᵢ = P(第i次甲投)，则 aᵢ = 0.6·aᵢ₋₁ + 0.2·(1-aᵢ₋₁) = (2/5)aᵢ₋₁ + 1/5（i≥2），a₁=1/2。
构造：aᵢ - 1/3 = (2/5)(aᵢ₋₁ - 1/3)，故 {aᵢ-1/3} 是首项 1/6、公比 2/5 的等比，
得 **aᵢ = (1/6)(2/5)^(i-1) + 1/3**。
(3) 由两点分布期望线性性，E(Y) = Σ₁ⁿ aᵢ = Σ₁ⁿ[(1/6)(2/5)^(i-1) + 1/3]
= (1/6)·[1-(2/5)ⁿ]/(1-2/5) + n/3 = **n/3 + 5/18 - (1/9)(2/5)^(n-1)**。

**关键提炼**：马尔可夫链本质是一阶线性递推数列；期望用"指示变量+线性性"一步到位，避免逐一列分布列。
