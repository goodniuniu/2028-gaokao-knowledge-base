## 核心内容

> **本质理解**：立体几何证明题是"逻辑推理的书面表达"。核心要求：**条件充分、定理正确、逻辑链条完整**。广东卷评分标准中，证明步骤的规范性占分比重高达30%。

---

### 一、证明题通用结构模板

#### 标准答题格式（四段式）

```
【第一步：申明已知条件】
∵ [题目给出的条件]

【第二步：推导中间结论】
∴ [由条件推出的中间结论]（可引用定理或性质）

【第三步：验证定理条件】
又∵ [验证定理所需的全部条件]

【第四步：得出结论】
∴ [最终结论]（写明所用定理名称）
```

> **广东卷评分要点**：每一步必须有"∵"和"∴"，不能直接写结论。使用定理时必须写明定理名称（如"由线面垂直的判定定理得..."）。

---

### 二、四大证明类型标准步骤

#### 类型1：证明线面平行（广州期末高频！）

**目标**：证明 $l \parallel \alpha$

**标准步骤**：

```
【方法一：线线平行 → 线面平行】（最常用）

第一步：在平面α内找一条直线m
第二步：证明 l ∥ m
     ∵ [条件]
     ∴ [中间结论] → l ∥ m
第三步：验证l在平面α外
     ∵ l ⊄ α，m ⊂ α
第四步：应用判定定理
     ∴ l ∥ α（线面平行判定定理）
```

> **找平行线的常用技巧**：
> - **中位线**：取中点，连中位线
> - **平行四边形**：证明四边形是平行四边形，对边平行
> - **比例线段**：$\frac{AE}{EB} = \frac{AF}{FC}$ → $EF \parallel BC$
> - **线面平行性质**：已知$l \parallel \beta$，$\alpha \cap \beta = m$，则$l \parallel m$

---

#### 类型2：证明线面垂直（广州期末高频！）

**目标**：证明 $l \perp \alpha$

**标准步骤**：

```
【方法一：线线垂直 → 线面垂直】（最常用）

第一步：在平面α内找两条相交直线m和n
     ∵ m ⊂ α，n ⊂ α，且 m ∩ n = P
第二步：证明 l ⊥ m
     ∵ [条件]
     ∴ l ⊥ m
第三步：证明 l ⊥ n
     ∵ [条件]
     ∴ l ⊥ n
第四步：应用判定定理
     ∴ l ⊥ α（线面垂直判定定理）
```

> **证明线线垂直的常用技巧**：
> - **等腰三角形三线合一**：底边中线 ⊥ 底边
> - **菱形/正方形对角线**：对角线互相垂直
> - **勾股定理逆定理**：$a^2 + b^2 = c^2$ → 夹角为90°
> - **已知线面垂直**：若$l \perp \beta$，$m \subset \beta$，则$l \perp m$
> - **面面垂直性质**：若$\alpha \perp \beta$，$l \subset \alpha$，$l \perp$交线，则$l \perp \beta$

---

#### 类型3：证明面面平行

**目标**：证明 $\alpha \parallel \beta$

**标准步骤**：

```
【方法一：两相交线平行 → 面面平行】

第一步：在平面α内找两条相交直线a和b
     ∵ a ⊂ α，b ⊂ α，且 a ∩ b = P
第二步：证明 a ∥ β
     ∵ [条件]
     ∴ a ∥ β
第三步：证明 b ∥ β
     ∵ [条件]
     ∴ b ∥ β
第四步：应用判定定理
     ∴ α ∥ β（面面平行判定定理）
```

---

#### 类型4：证明面面垂直

**目标**：证明 $\alpha \perp \beta$

**标准步骤**：

```
【方法一：线面垂直 → 面面垂直】（最常用）

第一步：在一个平面（如α）内找一条直线l
第二步：证明 l ⊥ β
     ∵ [条件]
     ∴ l ⊥ β（用线面垂直判定）
第三步：验证l在另一个平面α内
     ∵ l ⊂ α
第四步：应用判定定理
     ∴ α ⊥ β（面面垂直判定定理）
```

> **高频场景**：已知$PA \perp$底面$ABCD$，求证：面$PAB \perp$面$ABCD$。
> - ∵ $PA \perp$面$ABCD$，$PA \subset$面$PAB$
> - ∴ 面$PAB \perp$面$ABCD$（面面垂直判定）

### 三、证明题中的常用辅助线作法

| 辅助线类型 | 作法 | 适用场景 |
|-----------|------|----------|
| **中位线** | 连接两边中点 | 证平行、证线段相等 |
| **平行线** | 过某点作已知线的平行线 | 构造平行四边形、转移比例 |
| **垂线** | 过某点向平面/直线作垂线 | 证垂直、求距离/高 |
| **交线** | 作两平面的交线 | 面面关系问题 |
| **延长线** | 延长某线段与平面相交 | 找交点、构造新平面 |
| **补形** | 补全几何体（如补成柱体） | 不规则几何体的对称分析 |

> **广东卷提示**：广州名校联考的证明题常需要作辅助线，辅助线描述必须准确（如"取$BC$中点$M$，连接$AM$、$DM$"）。

### 四、证明题书写规范（广东卷评分标准）

#### 1. 必须写全的条件

| 定理 | 容易遗漏的条件 | 遗漏后果 |
|------|---------------|----------|
| 线面平行判定 | $l \not\subset \alpha$（线在面外） | 无法判定 |
| 线面垂直判定 | $m \cap n \neq \varnothing$（两线相交） | 无法判定 |
| 面面平行判定 | $a \cap b \neq \varnothing$（两线相交） | 无法判定 |
| 面面垂直性质 | $l \subset \alpha$且$l \perp$交线 | 无法得出线面垂直 |

> **广东卷典型扣分**：证明线面平行时只写"$l \parallel m$，$m \subset \alpha$"，漏写"$l \not\subset \alpha$"，扣1-2分。

#### 2. 定理名称必须写

| 正确写法 | 错误写法 |
|----------|----------|
| ∴ $l \parallel \alpha$（**线面平行判定定理**） | ∴ $l \parallel \alpha$ |
| ∴ $l \perp \alpha$（**线面垂直判定定理**） | ∴ $l \perp \alpha$ |
| ∴ $\alpha \perp \beta$（**面面垂直判定定理**） | ∴ $\alpha \perp \beta$ |

> **广东卷评分**：不写定理名称扣1分/处。

#### 3. 符号使用规范

| 正确 | 错误 |
|------|------|
| 点 $A$、直线 $l$、平面 $\alpha$ | 点 $a$、直线 $A$、平面 $A$ |
| $l \subset \alpha$（线在面内） | $l \in \alpha$ |
| $P \in l$（点在直线上） | $P \subset l$ |
| $l \cap \alpha = P$ | $l + \alpha = P$ |

### 五、名校联考拔高：存在性问题

**题型**：在棱$PC$上是否存在点$M$，使得$BM \parallel$面$PAD$？

**通法**：
1. **假设存在**：设$PM = \lambda PC$（$0 \leq \lambda \leq 1$）
2. **找平行关系**：构造过$BM$且与面$PAD$相交的平面
3. **利用性质**：若$BM \parallel$面$PAD$，则$BM$平行于交线
4. **建立方程**：用向量法或比例法求$\lambda$
5. **验证**：若$0 \leq \lambda \leq 1$，则存在；否则不存在

> **2024-2025广州五校联考第16题**：动点存在性证明，用**向量法**或**比例法**均可。

### 六、记忆口诀

> **证明步骤口诀**：
> - "线面平行：找线证平行，别忘了线在面外"
> - "线面垂直：找两相交线，分别证垂直"
> - "面面平行：找两相交线，都平行于另一平面"
> - "面面垂直：找一垂线，它在另一平面内"
> - "定理名称必须写，条件一个不能漏"

## 备注

- **广东高一下期末命题规律**：解答题第15-16题每题12-14分，共两问。第1问（6-7分）通常是线面平行或垂直的证明；第2问（6-7分）是另一面面关系的证明或体积计算。名校联考可能增加第3问（存在性/最值）。
- **2024-2025学年广州越秀区期末第15题**：第1问证明线面平行（用中位线），第2问证明线面垂直（用等腰三角形三线合一+勾股定理）。
- **训练建议**：先熟背八大定理（每天默写），再按标准步骤写10道基础证明题，最后做5道名校联考综合题（限时15分钟/题）。
- **书写自查清单**：每写完一道证明题，检查：①定理名称写了没？②线在面外/两线相交等条件写了没？③符号使用对不？
