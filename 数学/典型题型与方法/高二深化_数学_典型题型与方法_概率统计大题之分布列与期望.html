【卡片标题】概率统计大题之分布列与期望

⚠️待强化★★★★☆#高二深化#新高考Ⅰ卷·广东来源：专题归纳 / 2023新课标Ⅰ卷·第21题
广东考情：高频（新高考Ⅰ卷概率压轴常考）· 中档 · 状态转移递推+期望线性性 · 真题锚定：2023新高考Ⅰ卷T21（马尔可夫链/递推期望）

核心内容

题型识别标志

看到什么条件 → 立刻想到什么方法

题干关键条件识别为首选方法
"每次由其中一人投篮/抽奖，命中则继续、不中则换人"马尔可夫链设 aₙ=P(第n次状态)，建递推
"第n次处于某状态的概率"递推数列aₙ = p·aₙ₋₁ + q·(1-aₙ₋₁) 构造等比
"已知 Xi 两点分布，E(ΣXi)=Σqi"期望线性性直接求和 Σaᵢ
随机变量取值为 0/1（是否发生）两点分布用指示变量把计数转化为 ΣXi
多个互斥状态转移全概率公式按上一状态分类

解题路径（概率递推大题四步法）

第一步：建模设变量

设 aₙ = "第 n 次处于目标状态"的概率，写出 a₁（初始概率）。

第二步：建递推

用全概率：aₙ = P(本状态命中留)+P(他状态未中换)，化为 aₙ = p·aₙ₋₁ + q（一阶线性递推）。

第三步：求通项

构造常数列 aₙ + c = k(aₙ₋₁ + c) 化为等比，得 aₙ = A·k^(n-1) + B。

第四步：算期望

利用两点分布期望线性性 E(Y) = Σaᵢ，分组（等比求和+常数求和）得结果。

母题（2023 新课标Ⅰ卷·第21题，12分）

题目：甲、乙两人投篮，每次由其中一人投篮，规则：若命中则此人继续，若未命中则换对方。甲命中率0.6，乙命中率0.8，第一次由抽签决定，甲、乙各0.5。 (1) 求第2次投篮的人是乙的概率； (2) 求第 i 次投篮的人是甲的概率； (3) 记前 n 次中甲投篮次数为 Y（Xi 为两点分布、P(Xi=1)=aᵢ），求 E(Y)。

解析： (1) P(第2次是乙) = P(甲先投且未中) + P(乙先投且命中) = 0.5×0.4 + 0.5×0.8 = 0.6 = 3/5。 (2) 设 aᵢ = P(第i次甲投)，则 aᵢ = 0.6·aᵢ₋₁ + 0.2·(1-aᵢ₋₁) = (2/5)aᵢ₋₁ + 1/5（i≥2），a₁=1/2。 构造：aᵢ - 1/3 = (2/5)(aᵢ₋₁ - 1/3)，故 {aᵢ-1/3} 是首项 1/6、公比 2/5 的等比， 得 aᵢ = (1/6)(2/5)^(i-1) + 1/3。 (3) 由两点分布期望线性性，E(Y) = Σ₁ⁿ aᵢ = Σ₁ⁿ[(1/6)(2/5)^(i-1) + 1/3] = (1/6)·[1-(2/5)ⁿ]/(1-2/5) + n/3 = n/3 + 5/18 - (1/9)(2/5)^(n-1)。

关键提炼：马尔可夫链本质是一阶线性递推数列；期望用"指示变量+线性性"一步到位，避免逐一列分布列。

关联卡片

  • 数列放缩与不等式证明 — 递推求通项、等比求和的同款运算

备注

  • 广东卷概率压轴近3年连续考递推（2022传球、2023投篮、2024骰子），"识别状态→建递推→求通项→算期望"是通用打法。
  • 易错：第(3)问误把 Y 当单一分布列硬算；忘记用期望线性性。
  • 提速：直接记递推 aₙ=paₙ₋₁+q 通解 aₙ=(a₁-q/(1-p))(p)^(n-1)+q/(1-p)。
