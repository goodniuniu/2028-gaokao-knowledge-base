【卡片标题】导数大题之双变量与同构问题

⚠️待强化★★★★☆#高二深化#新高考Ⅰ卷·广东来源：专题归纳 / 2020新高考Ⅰ卷(山东)·第21题（同构视角）
广东考情：高频（同构法是新课标Ⅰ卷导数压轴最常用技巧）；看到 e^x 与 lnx 混合的恒成立/不等式，优先尝试同构化为同一函数

核心内容

题型识别标志

看到什么条件 → 立刻想到什么方法

题干关键条件识别为首选方法
式子含 e^{x}+x 与 lnx+x 形式同构构造 g(t)=e^t+t，化归 g(A)≥g(B)
"f(x)≥1 恒成立"且含 ae^{x-1} 与 lnx同构写成 e^{ln a+x-1}+lna+x-1 ≥ lnx+x
指数与对数并存、难直接解同构优先移项配成 g(u)=g(v) 利用单调性
同构不易时转隐零点见"导数大题之隐零点问题"卡

解题路径（同构三步法）

第一步：观察结构

将式中的 e^{...} 与 ln(...) 配成"指数+指数内"或"对数+对数内"的对称形式。

第二步：构造辅助函数

令 g(t)=e^t+t（或 g(t)=t+ln t 等），把原不等式化为 g(u) ≥ g(v)。

第三步：用单调性消参

因 g(t) 单调增，g(u)≥g(v) ⇔ u≥v，从而把超越不等式化为代数不等式求参。

母题（2020 新高考Ⅰ卷(山东)·第21题，12分）

题目：已知函数 f(x)=ae^{x-1} - lnx + lna。(1) 当 a=e 时，求曲线在点(1,f(1))处切线与坐标轴围成三角形面积；(2) 若 f(x)≥1，求a的取值范围。

解析（同构视角）： (2) f(x) = e^{ln a + x - 1} - lnx + ln a = e^{ln a + x - 1} + (ln a + x - 1) - (lnx + x)。 令 g(t)=e^t + t（在R上严格增），则 f(x) = g(ln a + x - 1) - g(ln x)。 f(x)≥1 即 g(ln a + x - 1) - g(ln x) ≥ 1。 注意到 g(ln x) + 1 = e^{ln x} + ln x + 1 = x + ln x + 1，而 g(ln a + x - 1) ≥ g(ln x) + 1 不易直接。 改为：由 g 单调增，f(x)≥1 等价于 ln a + x - 1 ≥ ln x（即 g(ln a+x-1)≥g(ln x)），即 ln a ≥ ln x - x + 1。 令 h(x)=lnx - x + 1，h'(x)=1/x - 1，h(x) 在(0,1)增、(1,+∞)减，最大值 h(1)=0。 故 ln a ≥ 0 ⇒ a ≥ 1。

关键提炼：同构本质"把指数与对数配成同一个增函数 g 的复合，用单调性把超越不等式降为代数不等式"。

  • 提速：熟记 e^x+x、x+lnx 这两类标准同构母函数。
