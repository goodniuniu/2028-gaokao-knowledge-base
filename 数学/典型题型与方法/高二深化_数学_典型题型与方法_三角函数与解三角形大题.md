## 核心内容

### 题型识别标志
> **看到什么条件 → 立刻想到什么方法**

| 题干关键条件 | 识别为 | 首选方法 |
|-------------|--------|----------|
| "A+B=3C" 或角之间倍数关系 | 内角和消元 | A+B+C=π 联立求某角 |
| "2sin(A-C)=sinB" | 和差化积/展开 | 用 sinB=sin(A+C) 展开 |
| 求单角 sinA/cosA | 同角平方关系 | 由关系得 sinA=k·cosA 再平方 |
| "AB=5，求AB边上的高" | 高/面积 | h=AC·sinA 或 等面积法 |
| 边角混合 | 正弦定理/余弦定理 | 统一为边或统一为角 |

### 解题路径（解三角形大题四步法）
#### 第一步：消元求角
用 A+B+C=π 与已知角关系，解出某个具体角（如 C=π/4）。
#### 第二步：化简求单角函数值
把含 B 的式用 sin(A+C) 替换，展开后得 sinA 与 cosA 的线性关系，结合同角平方关系求 sinA。
#### 第三步：用正弦定理求边长
由已知边与已求角，正弦定理 a/sinA = b/sinB = c/sinC 求未知边。
#### 第四步：求高/面积
h = 邻边长 × sin(夹角)；或等面积 S = 1/2·ab·sinC 双算。

### 母题（2023 新课标Ⅰ卷·第17题，10分）
**题目**：在 △ABC 中，A+B=3C，2sin(A-C)=sinB。
(1) 求 sinA；
(2) 设 AB=5，求 AB 边上的高。

**解析**：
(1) 由 A+B=3C 且 A+B+C=π，得 4C=π，故 **C=π/4**。
2sin(A-C)=sinB=sin(A+C) → 2sinAcosC - 2cosAsinC = sinAcosC + cosAsinC
→ sinAcosC = 3cosAsinC → sinA = 3cosA（因 cosC=sinC=√2/2）。
结合同角：sin²A + cos²A = 1，且 sinA>0，得 **sinA = 3√10/10**（cosA=√10/10）。
(2) 法一：sinB = sin(A+C) = sinAcosC + cosAsinC = (3√10/10)(√2/2)+(√10/10)(√2/2) = 2√5/5。
由正弦定理 AC/sinB = AB/sinC → AC = 5·(2√5/5)/(√2/2) = 2√10。
AB 边上的高 h = AC·sinA = 2√10·(3√10/10) = **6**。
（法二：等面积 S=1/2·AB·h = 1/2·AC·BC·sinC，亦得 h=6。）

**关键提炼**：第一问"消元→和差展开→单角平方关系"是送分解法；第二问"正弦定理求边→h=边×sin夹角"或等面积法，二选一即可。

## 备注
- 广东卷解三角形第一问几乎必考"角的倍关系 + 一个三角方程"，套路固定。
- 易错：把 sinB 写成 sin(A-C) 而非 sin(A+C)；第二问忘记 h=AC·sinA 直接求错边。
- 提速：先锁定一个角（如 C），再把所有 B 用 A+C 展开，单角 sinA 一步到位。
