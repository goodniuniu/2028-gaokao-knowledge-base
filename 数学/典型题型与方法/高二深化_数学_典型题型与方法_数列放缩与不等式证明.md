## 核心内容

### 题型识别标志
> **看到什么条件 → 立刻想到什么方法**

| 题干关键条件 | 识别为 | 首选方法 |
|-------------|--------|----------|
| "证明 1/a₁+1/a₂+…+1/aₙ < C（C为整数）" | 放缩证明 | 裂项相消后整体放缩 |
| "{Sₙ/aₙ} 是等差数列" / "{naₙ}等差" | 累乘法求通项 | 用 Sₙ-Sₙ₋₁ 构造 aₙ/aₙ₋₁ 比，累乘 |
| 通项为 n(n+1)/2、n² 等二次型 | 1/aₙ 可裂项 | 1/aₙ = 常数·(1/f(n)-1/f(n+1)) |
| 出现 1/[n(n+1)]、1/[n(n+k)] | 标准裂项 | 1/[n(n+k)] = (1/k)(1/n - 1/(n+k)) |
| 要证 Σ < C 但逐项求和发散 | 放缩后再裂项 | 先放缩通项再求和 |

### 解题路径（数列放缩证明四步法）
#### 第一步：求通项
由题干递推/和项关系，用累乘法、构造常数列或待定系数求出 aₙ 的显式。广东卷第17题几乎必考"和项关系→累乘法"。
#### 第二步：裂项变形
将 1/aₙ 写成常数×(1/f(n) - 1/f(n+1)) 的形式。关键是配出"后项减前项"的分母结构。
#### 第三步：求和并放缩
Σ1/aₙ = 常数×(首项 - 末项)。因末项 >0，故 Σ < 常数×首项 = 目标上界 C。
#### 第四步：验证严格不等
说明等号无法取到（n 有限时末项 >0），即得严格小于。若问"≤"则补 n→∞ 极限情形。

### 母题（2022 新课标Ⅰ卷·第17题，10分）
**题目**：记 Sₙ 为数列 {aₙ} 的前 n 项和，已知 a₁=1，{Sₙ/aₙ} 是公差为 1/3 的等差数列。
(1) 求 {aₙ} 的通项公式；
(2) 证明：1/a₁ + 1/a₂ + … + 1/aₙ < 2。

**解析**：
(1) 由题 Sₙ/aₙ = 1 + (n-1)·(1/3) = (n+2)/3，故 Sₙ = (n+2)aₙ/3。
当 n≥2 时 Sₙ₋₁ = (n+1)aₙ₋₁/3，两式相减得 aₙ = (n+2)aₙ/3 - (n+1)aₙ₋₁/3，
整理：(n-1)aₙ = (n+1)aₙ₋₁，即 aₙ/aₙ₋₁ = (n+1)/(n-1)。
累乘：aₙ/a₁ = [(n+1)/(n-1)]·[n/(n-2)]·…·(3/1) = n(n+1)/2，又 a₁=1，故 **aₙ = n(n+1)/2**（首项也满足）。
(2) 1/aₙ = 2/[n(n+1)] = 2(1/n - 1/(n+1))，
Σ₁ⁿ 1/aₖ = 2[(1-1/2)+(1/2-1/3)+…+(1/n-1/(n+1))] = 2(1 - 1/(n+1)) < 2。证毕。

**关键提炼**：和项关系→累乘法求通项是广东卷数列第一问的固定套路；第二问"裂项相消+末项正数放缩"是放缩证明题的标准模板。
