核心内容

关键概念

  • 方向（战略）：面对问题时，先确定从哪个数学分支切入——代数、分析、几何、概率等。方向是"宏观思路"，决定解题的顶层框架。
  • 解法（战术）：在同一方向内，选择具体的定理、工具或步骤链。解法是"微观步骤"，可以在同一方向内灵活切换。
  • 致命误区：跳过方向选择，直接跳到某个解法（如拿到题先想"求导"），导致思维混乱、计算量暴增。

题目原型

已知 $a, b > 0$ 且 $a + b = 1$，求证：$a^2 + b^2 \geq \dfrac{1}{2}$
本题结构简单，但恰好是训练"方向选择"的绝佳载体。三种方向均可独立完成证明，且各有优劣。

方向一：代数方向

宏观思路：把 $a,b$ 看作代数符号，通过恒等变形、消元、配方或经典不等式，将目标式转化为显然成立的形式。

解法 A1：消元 + 配方

  1. 由 $a + b = 1$，得 $b = 1 - a$（$0 < a < 1$）
  2. 代入：$a^2 + b^2 = a^2 + (1-a)^2 = 2a^2 - 2a + 1$
  3. 配方：$= 2\left(a - \dfrac{1}{2}\right)^2 + \dfrac{1}{2}$
  4. 平方项非负，故 $a^2 + b^2 \geq \dfrac{1}{2}$，当 $a = b = \dfrac{1}{2}$ 取等
优点：最朴素，不需要额外知识储备；缺点：仅适用于二次型，推广性有限。

解法 A2：对称恒等式 + AM-GM

  1. 恒等式：$a^2 + b^2 = (a+b)^2 - 2ab = 1 - 2ab$
  2. AM-GM：$ab \leq \left(\dfrac{a+b}{2}\right)^2 = \dfrac{1}{4}$
  3. 所以 $a^2 + b^2 = 1 - 2ab \geq 1 - 2 \times \dfrac{1}{4} = \dfrac{1}{2}$
优点：一步到位，展示对称式的威力；缺点：需识别 $a^2+b^2$ 与 $ab$ 的关系。

解法 A3：柯西不等式

  1. $(a^2 + b^2)(1^2 + 1^2) \geq (a \cdot 1 + b \cdot 1)^2 = (a+b)^2 = 1$
  2. 即 $2(a^2 + b^2) \geq 1$
  3. 所以 $a^2 + b^2 \geq \dfrac{1}{2}$
优点：最优雅，向量形式的推广极强；缺点：需熟练掌握柯西结构识别。

方向二：分析方向

宏观思路：将 $a,b$ 视为连续变量，构造辅助函数，利用微积分（导数、极值）来证明。

解法 B1：一元函数求导

  1. 消元设 $f(a) = a^2 + (1-a)^2 = 2a^2 - 2a + 1$，$a \in (0,1)$
  2. 求导：$f'(a) = 4a - 2$
  3. 令 $f'(a) = 0$，得临界点 $a = \dfrac{1}{2}$
  4. 二阶导 $f''(a) = 4 > 0$，确认极小值点
  5. 最小值 $f\left(\dfrac{1}{2}\right) = \dfrac{1}{2}$，故 $f(a) \geq \dfrac{1}{2}$
优点：普适性极强，几乎可处理任何可微函数；缺点：计算量可能大，且需验证边界。

解法 B2：拉格朗日乘数法

  1. 构造 $L(a,b,\lambda) = a^2 + b^2 - \lambda(a+b-1)$
  2. $\dfrac{\partial L}{\partial a} = 2a - \lambda = 0$，$\dfrac{\partial L}{\partial b} = 2b - \lambda = 0$
  3. 得 $a = b$，结合 $a + b = 1$，得 $a = b = \dfrac{1}{2}$
  4. 代入得最小值 $\dfrac{1}{2}$
优点：不消元，直接处理多元条件极值，结构对称时极快；缺点：高中阶段较少直接使用，需结合导数背景理解。

方向三：几何方向

宏观思路：将 $a^2 + b^2$ 和 $a+b=1$ 翻译成几何量（距离、直线、向量），借助图形直观完成证明。

解法 C1：点到直线距离

  1. $a^2 + b^2$ 是点 $P(a,b)$ 到原点 $O(0,0)$ 距离的平方
  2. 约束 $a+b=1$（$a,b>0$）是第一象限内线段
  3. 问题转化为：求原点 $O$ 到线段 $a+b=1$ 的最短距离
  4. 原点到直线 $a+b=1$ 的距离 $d = \dfrac{|0+0-1|}{\sqrt{1^2+1^2}} = \dfrac{1}{\sqrt{2}}$
  5. 垂足 $\left(\dfrac{1}{2}, \dfrac{1}{2}\right)$ 恰好在线段上
  6. 最小距离平方 $d^2 = \dfrac{1}{2}$，即 $a^2 + b^2 \geq \dfrac{1}{2}$
优点：直观，培养数形结合能力；缺点：对几何直觉要求高，复杂约束难以可视化。

解法 C2：向量投影（柯西-施瓦茨几何意义）

  1. 设向量 $\vec{u} = (a,b)$，则 $|\vec{u}|^2 = a^2 + b^2$
  2. 约束即 $\vec{u} \cdot (1,1) = 1$
  3. 投影不等式：$|\vec{u} \cdot \vec{n}| \leq |\vec{u}| \cdot |\vec{n}|$
  4. 即 $1 \leq |\vec{u}| \cdot \sqrt{2}$，所以 $|\vec{u}|^2 \geq \dfrac{1}{2}$
优点：连接代数与几何，体现柯西不等式的几何本质；缺点：需向量基础。

方法步骤

#### 一题多解训练的正确流程

  1. 先锁定方向：读题30秒，判断题目结构更适合代数/分析/几何
  • 二次齐次式 → 优先代数（配方、柯西）
  • 任意函数、复杂约束 → 优先分析（求导、极值）
  • 距离、面积、角度暗示 → 优先几何（向量、图形）
  1. 再选具体解法：在同一方向内，选择最熟练或计算量最小的工具
  2. 卡壳时切换：同一方向内换解法（如代数配方卡壳，换AM-GM试试）
  3. 彻底卡壳再换方向：一个方向3-5分钟无进展，果断换另一个方向

记忆口诀/技巧

方向选择口诀：代数先看变形路，分析构造函数图，几何找距找角找向量；三方向不混搭，混搭必迷路。 解题节奏口诀：三十秒定方向，三分钟选解法，五分钟不出换战术，十分钟不出换战略。 新高考Ⅰ卷提示：压轴题常可用多种方向解决，但考场上时间有限，优先选择自己最熟练的方向（通常是代数方向），而非"最 elegant 的解法"。
