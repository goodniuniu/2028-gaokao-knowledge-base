恒成立与存在性问题的题型通法

⚠️待强化★★★★☆#高二深化#新高考Ⅰ卷·广东 / #新高考Ⅰ卷·江苏来源：人教A版选择性必修二 / 新高考Ⅰ卷近5年解答题真题归纳 / 2024新课标Ⅰ卷第18题（导数压轴）
广东考情：高频（新高考Ⅰ卷导数压轴核心）· 压轴 · "当且仅当"型充要证明+双变量存在性 · 真题锚定：2024新高考Ⅰ卷T18（恒成立）、2023新高考Ⅰ卷T20（双变量存在性）

核心内容

本质理解：恒成立与存在性问题，是全称量词∀与存在量词∃在函数最值上的翻译。核心口诀：恒成立求"最值边界"，存在性求"最值可达"。

一、单变量问题：最值转化对照表

题干表述数学翻译转化目标记忆口诀
∀x∈D, f(x) < a 恒成立全称量词f(x)ₘₐₓ < a大值小于a（所有值都小）
∀x∈D, f(x) ≤ a 恒成立全称量词f(x)ₘₐₓ ≤ a大值不大于a
∀x∈D, f(x) > a 恒成立全称量词f(x)ₘᵢₙ > a小值大于a（所有值都大）
∀x∈D, f(x) ≥ a 恒成立全称量词f(x)ₘᵢₙ ≥ a小值不小于a
∃x∈D, f(x) < a 有解存在量词f(x)ₘᵢₙ < a小值小于a（至少有一个小）
∃x∈D, f(x) ≤ a 有解存在量词f(x)ₘᵢₙ ≤ a小值不大于a
∃x∈D, f(x) > a 有解存在量词f(x)ₘₐₓ > a大值大于a（至少有一个大）
∃x∈D, f(x) ≥ a 有解存在量词f(x)ₘₐₓ ≥ a大值不小于a

关键提醒：恒成立与存在性问题的最值取法恰好相反！这是考生最容易混淆的地方。

二、三大解题通法

方法1：分离参数法（首选，避免讨论）

适用条件：参数易分离，且分离后的函数最值可求。

操作步骤：

  1. 将不等式整理为：参数 ≤ g(x) 或 参数 ≥ g(x)
  2. 求g(x)在定义域上的最值
  3. 参数范围 = 最值的"边界值"

典型例题（2024新课标Ⅰ卷风格）：

已知f(x)=eˣ-ax，若∀x>0, f(x)≥0恒成立，求a的范围。

解析：

  • 分离参数：a ≤ eˣ/x （x>0）
  • 设g(x)=eˣ/x，求g(x)ₘᵢₙ
  • g'(x)=eˣ(x-1)/x²，极小值点x=1
  • g(x)ₘᵢₙ=g(1)=e
  • ∴ a ≤ e

可直接使用：分离参数法的结论可直接用于解答题，无需额外证明。

方法2：不分离参数法（直接构造函数）

适用条件：参数不易分离，或区间端点代入后不等式两边相等。

操作步骤：

  1. 构造F(x)=f(x)-目标值（使不等式变为F(x)≥0或F(x)≤0）
  2. 求F'(x)，分析F(x)的单调性
  3. 求F(x)的最值，确保F(x)ₘᵢₙ≥0（或F(x)ₘₐₓ≤0）
  4. 若含参数，对参数分类讨论

典型例题：

已知f(x)=eˣ-1-x-ax²，若∀x≥0, f(x)≥0恒成立，求a的范围。

解析：

  • 观察：f(0)=0，不等式两边在x=0处相等 → 不分离参数
  • 构造F(x)=eˣ-1-x-ax²，需F(x)ₘᵢₙ≥0
  • F'(x)=eˣ-1-2ax，F'(0)=0
  • 再求F''(x)=eˣ-2a，需对a分类讨论：
    • 若a≤1/2：F''(x)≥e⁰-2a=1-2a≥0，F'(x)单调增，F'(x)≥F'(0)=0，F(x)单调增，F(x)≥F(0)=0 ✓
    • 若a>1/2：存在x₀使F''(x₀)=0，F'(x)先减后增，可能F'(x)<0，导致F(x)<0 ✗
  • ∴ a ≤ 1/2

可直接使用：不分离参数法是新高考Ⅰ卷标准解法，可直接使用。

方法3：必要性探路法（减少讨论范围）

适用条件：含参函数在区间端点或特殊点有必然约束。

操作步骤：

  1. 找到必要条件（如端点值满足不等式）
  2. 缩小参数范围
  3. 在该范围内证明充分性

典型例题：

已知f(x)=eˣ-ax²-x-1，若∀x≥0, f(x)≥0，求a的范围。

解析：

  • 必要性：f(0)=0，f'(0)=0，f''(0)=1-2a
  • 若f''(0)<0（即a>1/2），则在0附近f(x)<0，不满足 ✗
  • ∴ 必要条件：a≤1/2
  • 充分性：当a≤1/2时，f''(x)=eˣ-2a≥1-2a≥0，f'(x)单调增，f'(x)≥f'(0)=0，f(x)单调增，f(x)≥f(0)=0 ✓
  • ∴ a ≤ 1/2

可直接使用：必要性探路法在解答题中可直接使用，是减少分类讨论的高效策略。

三、双变量问题：分步转化策略

题干表述数学翻译转化结果记忆口诀
∀x₁,∀x₂, f(x₁)≥g(x₂)两个全称f(x)ₘᵢₙ ≥ g(x)ₘₐₓ最值交叉比
∀x₁,∃x₂, f(x₁)≥g(x₂)全称+存在f(x)ₘᵢₙ ≥ g(x)ₘᵢₙ小对小
∃x₁,∀x₂, f(x₁)≥g(x₂)存在+全称f(x)ₘₐₓ ≥ g(x)ₘₐₓ大对大
∃x₁,∃x₂, f(x₁)≥g(x₂)两个存在f(x)ₘₐₓ ≥ g(x)ₘᵢₙ大对小

双变量核心逻辑：先固定一个变量，再处理另一个变量；或转化为两个函数最值的比较。

四、高频陷阱与避坑指南

陷阱说明避坑方法
混淆量词将"存在"当作"恒成立"处理，最值取反写出∀/∃符号，对照转化表确认
忽略定义域分离参数后未考虑x的范围限制每一步标注定义域，最后取交集
端点效应漏用未利用区间端点值缩小参数范围先代入端点，获取必要条件
分类讨论遗漏导函数零点与区间位置关系未穷尽画数轴，标注零点与区间端点
最值点不可达函数在开区间上无最值，仅有确界用极限值代替，注意等号是否可取
