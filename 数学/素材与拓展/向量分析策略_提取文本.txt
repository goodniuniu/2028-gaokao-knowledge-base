B2. 平行与垂直的坐标条件
引入
章前总览： 在掌握了向量的基本坐标运算（ B1 ）之后，本章聚焦于两个最
核心的几何关系 ——平行与垂直。核心任务是：如何利用向量的坐标，快
速、准确地判定两向量是否平行（共线）或垂直？这是考试中的必考题型，
也是解决后续动态问题、角度计算和几何建模的基石。
逻辑链条：
从定义到坐标：平行由  a = λb 导出坐标比例关系  (C1) ；垂直由  a·b =
0 导出坐标点乘为零  (C2) 。
规范书写：区分草稿纸上的简便比例式与大题中的严谨乘积式。
变式应用：通过含参数、混合运算（如向量的线性组合）的例题，强化两
个条件的运用，并提醒共线别名、分母为零等易错点。
= C1. 平行（共线）条件 ——x₁y₂ = x₂y₁
平行  (Parallel) / 共线  (Collinear) ： 两非零向量  a = (x₁, y₁) 与  b =
(x₂, y₂) 方向相同或相反，则称它们平行（也叫做共线）。存在唯一实数  λ 使
得  a = λ b。
坐标条件推导：
• 由  a = λ b 得  (x₁, y₁) = λ(x₂, y₂) ⇒ x₁ = λ x₂, y₁ = λ y₂ 。
• 消去  λ ：若  x₂≠0, y₂≠0 ，则  λ = x₁/x₂ = y₁/y₂ ，即对应坐标成比例。
• 为避免分母为零（如其中一个坐标为 0 ），采用交叉相乘形式：x₁ y₂ = x₂ y₁（或  x₁ y₂ -
x₂ y₁ = 0 ）。这是严格等价且适用于所有情况的通用条件。

C2. 垂直条件 ——x₁x₂ + y₁y₂ = 0
✑实用写法
直接套用公式
直接套用公式  x₁x₂ + y₁y₂ = 0，没有分母限制，是最简单可靠的判定
方式。
垂直  (Perpendicular) ： 两非零向量  a = (x₁, y₁) 与  b = (x₂, y₂) 的夹角
为  90° ，此时它们的数量积（点积）为零。
坐标条件： 由点积的坐标定义，a·b = x₁x₂ + y₁y₂。因此垂直的充要条件
是：x₁x₂ + y₁y₂ = 0。
推导注解
垂直的几何定义：|a||b|cos90° = 0，与数量积定义一致。坐标形式直接给出，无需额外推
导，但应牢记与平行条件的区别：平行用乘积差，垂直用和为零。

 例题应用
例 3 （ 2022 年全国卷改编）
已知  a=(0,1), b=(2,x) ，且  b ⊥ (b - 4a) ，求  x 。
解法：
先算  4a = (0,4)，b - 4a = (2, x-4)。
垂直条件：b·(b-4a) = 0
⇒ 2×2 + x×(x-4) = 0
⇒ 4 + x² - 4x = 0
⇒ x² - 4x + 4 = 0
⇒ (x-2)² = 0
⇒ x = 2。
  注意：本题中  b-4a 的纵坐标  x-4 是由向量减法直接得到，切勿与  x-16 混淆（若a=
(0,4) 则不同，但题设a为 (0,1) ）。
易错点 / 辨析
⚠零向量： 零向量与任何向量垂直（因为点积 =0 ），但在常规题目中向量默
认为非零，无需特判。
⚠区分钝角 / 锐角： 垂直是点积 =0 ；点积 <0 为钝角，点积 >0 为锐角。但判
断钝角时还需排除反向平行（即点积 <0 且不共线），这将在 B5 中系统处

 本节小结
◆ 平行判定： 核心公式为  x₁y₂ = x₂y₁（或  x₁y₂ - x₂y₁ = 0）。草稿可用比例式，大题必须用
乘积式，需警惕分母为零的情况。
◆ 垂直判定： 核心公式为  x₁x₂ + y₁y₂ = 0。该公式无分母限制，是最直接可靠的判定工具。
◆ 概念辨析： 向量中的 “ 平行 ” 与 “ 共线 ” 是同一概念。零向量与任何向量既平行也垂直，但常规题目
默认向量非零。
◆ 应用关键： 解含参数的平行 / 垂直方程时，注意检验解的合理性（如是否为零向量）。对于向量线
性组合（如  b + λc）的问题，先进行坐标运算，再代入条件。

⚠C1. 钝角 / 锐角判定 —— 点乘符号 + 排除共线
点乘的正负可以直接反映夹角是锐角（ >0 ）还是钝角（ <0 ），但这里隐藏
着一个致命陷阱：当两向量共线（平行）时，点乘同样满足正负条件，但
夹角却是 0° 或 180° ，都不属于锐角或钝角。
--- PAGE 22 ---
 核心概念：夹角范围与 cos θ 符号
• 锐角：θ ∈ (0°, 90°) → cos θ > 0 → 点乘  > 0
• 钝角：θ ∈ (90°, 180°) → cos θ < 0 → 点乘  < 0
• 直角：θ = 90° → cos θ = 0 → 点乘  = 0
• 平角  / 零角：θ = 0° 或 180° → cos θ = 1 或 -1 ，点乘  ≠ 0 ，但既
不锐也不钝。
易错点 / 辨析（考试必考）
 很多同学只记得 “ 钝角 ⇔ 点乘 <0” ，但忘记了当两向量反向（夹角
180° ）时，点乘也是负的，而 180° 不属于钝角（钝角严格小于
180° ）。
 同理，锐角对应点乘 >0 ，但两向量同向（夹角 0° ）时点乘也
>0 ， 0° 也不属于锐角。
 核心规则：判定钝角或锐角时，必须先求出点乘满足符号条件，
再排除两向量共线（平行）的情况。
钝角：点乘  < 0 且 两向量不共线（排除反向平行，即排除比例
为负的情况）。
锐角：点乘  > 0 且 两向量不共线（排除同向平行，即排除比例
为正的情况）。
--- PAGE 23 ---
总结步骤（答题模板）
1 写出两向量的坐标（若含参）。
2 计算点乘，令其大于 0 （锐角）或小于 0 （钝角），解出参数范
围。
3 令两向量平行（即对应坐标成比例），解出使共线的参数值。
4 排除该共线值（若为锐角，排除同向；若为钝角，排除反向 ——
但更稳妥的做法是直接排除所有共线情况）。
5 最后写出参数取值范围（注意写成区间或集合形式）。
