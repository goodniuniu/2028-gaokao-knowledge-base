## 核心内容

> **本质理解**：立体几何证明与计算的失误，80%不是"不会"，而是"不细"。核心对策：**定理条件逐一核对、计算分步进行、符号规范书写**。

---

### 一、证明题五大失分点

#### 失分点1：定理条件遗漏——"线在面外"被遗忘

**高频场景**：证明线面平行时，只写"$l \parallel m$，$m \subset \alpha$"，漏写"$l \not\subset \alpha$"。

**正确定理**：线面平行判定定理要求：**平面外**一条直线与平面内一条直线平行，则线面平行。

**三个条件缺一不可**：
1. $l \parallel m$（线线平行）
2. $m \subset \alpha$（线在面内）
3. $l \not\subset \alpha$（线在面外）← **最容易遗漏！**

**典型错误写法**：
```
∵ EF ∥ AB，AB ⊂ 面 PAB
∴ EF ∥ 面 PAB  ← 错误！漏了EF ⊄ 面PAB
```

**正确写法**：
```
∵ EF ∥ AB，AB ⊂ 面 PAB，且 EF ⊄ 面 PAB
∴ EF ∥ 面 PAB（线面平行判定定理）
```

> **广东卷扣分**：漏写"$l \not\subset \alpha$"扣1-2分/处。一道证明题若有两处遗漏，可能扣3-4分。

---

#### 失分点2：定理条件遗漏——"两线相交"被遗忘

**高频场景**：证明线面垂直时，只写"$l \perp m$，$l \perp n$，$m, n \subset \alpha$"，漏写"$m \cap n = P$"（两线相交）。

**正确定理**：线面垂直判定定理要求：一条直线与平面内**两条相交直线**都垂直，则线面垂直。

**四个条件缺一不可**：
1. $l \perp m$（线线垂直1）
2. $l \perp n$（线线垂直2）
3. $m \subset \alpha$，$n \subset \alpha$（线在面内）
4. $m \cap n = P$（两线相交）← **最容易遗漏！**

> **广东卷扣分**：漏写"$m \cap n \neq \varnothing$"扣1-2分/处。

---

#### 失分点3：定理名称不写——"由...得"代替定理名

**错误写法**：
```
∵ PA ⊥ 面 ABCD，PA ⊂ 面 PAB
∴ 面 PAB ⊥ 面 ABCD  ← 错误！没写定理名称
```

**正确写法**：
```
∵ PA ⊥ 面 ABCD，PA ⊂ 面 PAB
∴ 面 PAB ⊥ 面 ABCD（面面垂直判定定理）
```

> **广东卷评分**：不写定理名称扣1分/处。一道12分的证明题，若3处不写定理名，扣3分，直接降档。

> **必须记熟的定理名称**：
> - 线面平行判定定理 / 线面平行性质定理
> - 线面垂直判定定理 / 线面垂直性质定理
> - 面面平行判定定理 / 面面平行性质定理
> - 面面垂直判定定理 / 面面垂直性质定理

---

#### 失分点4：面面垂直性质定理的误用

**错误认知**：两平面垂直，一个平面内的任意直线都垂直于另一平面。

**正确定理**：面面垂直性质定理要求：**一个平面内垂直于交线的直线**才垂直于另一平面。

**两个条件缺一不可**：
1. 直线$l \subset \alpha$（线在面内）
2. $l \perp$交线（线垂直于交线）← **最容易遗漏！**

**典型错误**：
> 已知面$PAB \perp$面$ABCD$，交线为$AB$。在面$PAB$内取$PA$，直接说$PA \perp$面$ABCD$。
> - **错误**：$PA$不一定垂直于交线$AB$！
> - **正确**：只有当$PA \perp AB$时，才有$PA \perp$面$ABCD$。

> **广东卷高频**：2024-2025广州天河区期末第16题第2问，面面垂直性质的应用，得分率仅45%。

---

#### 失分点5：辅助线不描述或描述不清

**错误写法**：
```
作DE ⊥ BC，连接AE。
∴ ...
```

**正确写法**：
```
取BC中点E，连接DE、AE。
∵ D是PC中点，E是BC中点
∴ DE ∥ PB（三角形中位线定理）
```

> **广东卷要求**：辅助线必须说明作法（"取...中点""过...作...垂线"），连接哪些线段也要写清楚。

---

### 二、计算题四大失误

#### 失误1：混淆表面积与侧面积

**错误**：求"圆锥的表面积"时，只算$\pi r l$（侧面积），漏加底面积$\pi r^2$。

**正确**：表面积 = 侧面积 + 底面积（圆锥只有一个底）。

> **快速区分**：
> - "侧面积"或"全面积"：不含底（柱体不含两个底，锥体不含底）
> - "表面积"：含所有面（柱体含两个底，锥体含底）
> - "全面积"在某些教材中 = 表面积（含所有面），需根据上下文判断

---

#### 失误2：圆锥/圆台中混淆母线、高、半径

**错误关系**：把圆锥母线$l$当作高$h$代入体积公式$V = \frac{1}{3}\pi r^2 h$。

**正确关系**：$l^2 = r^2 + h^2$（母线、半径、高构成直角三角形）。

**典型错误**：
> 已知圆锥底面半径$r = 3$，母线$l = 5$，求体积。
> - 错误：$V = \frac{1}{3}\pi \times 9 \times 5 = 15\pi$（把母线当高）
> - 正确：$h = \sqrt{25-9} = 4$，$V = \frac{1}{3}\pi \times 9 \times 4 = 12\pi$

> **广东卷高频**：2024-2025广州番禺区期末第3题、越秀区第14题均涉及此混淆。

---

#### 失误3：圆台体积公式漏中间项

**错误公式**：$V = \frac{1}{3}\pi h(r_1^2 + r_2^2)$（漏了$r_1r_2$项）

**正确公式**：$V = \frac{1}{3}\pi h(r_1^2 + r_1r_2 + r_2^2)$

**记忆口诀**："圆台体积三兄弟，上底平方、下底平方、加上上下乘积"

---

#### 失误4：球的直径与半径混淆

**错误**：题目给"球的直径为4"，直接代入$R = 4$计算$V = \frac{4}{3}\pi \times 64$。

**正确**：$R = \frac{d}{2} = 2$，$V = \frac{4}{3}\pi \times 8 = \frac{32\pi}{3}$。

> **审题技巧**：看到"直径"二字，立即画圈提醒自己先除以2。

### 三、符号与书写混乱

#### 混乱1：点、线、面的符号混用

| 正确 | 错误 | 说明 |
|------|------|------|
| 点 $A$ | 直线 $A$ | 点用大写，直线用小写 |
| 直线 $l$ | 点 $l$ | 直线用小写，点用大写 |
| 平面 $\alpha$ | 直线 $\alpha$ | 平面用希腊字母，直线用小写 |
| $P \in l$ | $P \subset l$ | 点在直线上用"∈"，不是"⊂" |
| $l \subset \alpha$ | $l \in \alpha$ | 直线在平面内用"⊂"，不是"∈" |

---

#### 混乱2：证明过程缺乏逻辑连接词

**错误写法**：
```
PA ⊥ 面 ABCD
PA ⊂ 面 PAB
面 PAB ⊥ 面 ABCD
```

**正确写法**：
```
∵ PA ⊥ 面 ABCD，且 PA ⊂ 面 PAB
∴ 面 PAB ⊥ 面 ABCD（面面垂直判定定理）
```

> **广东卷要求**：每一步必须有"∵"和"∴"，不能省略逻辑连接词。

### 四、快速自查清单（证明题写完后必查）

**检查条件完整性**：
- [ ] 线面平行：写了"线在面外"吗？
- [ ] 线面垂直：写了"两线相交"吗？
- [ ] 面面垂直性质：写了"线垂直于交线"吗？
- [ ] 辅助线：描述清楚作法了吗？

**检查定理名称**：
- [ ] 每一步结论都写了定理名称吗？
- [ ] 定理名称写对了吗？（判定/性质别混淆）

**检查符号规范**：
- [ ] 点用大写、直线用小写、平面用希腊字母了吗？
- [ ] "∈"和"⊂"用对了吗？
- [ ] 逻辑连接词"∵""∴"写了吗？

**检查计算**：
- [ ] "表面积"和"侧面积"区分了吗？
- [ ] 圆锥/圆台的母线、高、半径关系理清了吗？
- [ ] 圆台体积公式中间项$r_1r_2$写了吗？
- [ ] 直径转半径了吗？

### 标准化纠错路径（「定理三查·计算四核」纠错法）

**一、证明题纠错路径（写完后必查）**
1. **查条件**：线面平行补"线在面外"；线面垂直补"两线相交"；面面垂直性质确认"直线⊥交线"。
2. **查名称**：每一步结论后写定理名称（判定/性质分清，不写扣 1 分/处）。
3. **查逻辑**：辅助线说明作法，"∵…∴…"连接词齐全。

**二、计算题纠错路径（动笔前必核）**
1. **核面数**：表面积↔侧面积区分（含不含底）。
2. **核关系**：圆锥/圆台先用 $l^2=r^2+h^2$ 求高 $h$，再代体积。
3. **核公式**：台体体积 $V=\dfrac{h}{3}(S_1+S_2+\sqrt{S_1S_2})$，中间项 $\sqrt{S_1S_2}$ 不可漏。
4. **核半径**：见"直径"先 $÷2$ 得半径 $R$。

### 真题母题（2022 新高考Ⅰ卷 T4）

**题**：南水北调工程缓解了北方一些地区水资源短缺问题，其中一部分水蓄入某水库。已知该水库水位为海拔 $148.5\,\text{m}$ 时，相应水面的面积为 $140.0\,\text{km}^2$；水位为海拔 $157.5\,\text{m}$ 时，相应水面的面积为 $180.0\,\text{km}^2$。将该水库在这两个水位间的形状看作一个棱台，则该水库水位从海拔 $148.5\,\text{m}$ 上升到 $157.5\,\text{m}$ 时，增加的水量约为（$\sqrt7\approx2.65$）（ ）
A. $1.0\times10^9\,\text{m}^3$  B. $1.2\times10^9\,\text{m}^3$  C. $1.4\times10^9\,\text{m}^3$  D. $1.6\times10^9\,\text{m}^3$

**错解**：记错/漏记台体体积公式，写成 $V=\dfrac{h}{3}(S_1+S_2)$（漏掉中间项 $\sqrt{S_1S_2}$），得 $V=\dfrac{9}{3}\times(140+180)\times10^6=9.6\times10^8\approx1.0\times10^9\,\text{m}^3$，误选 **A**；或把棱台当棱柱用单底面积 $V=Sh$，得 $140\times10^6\times9\approx1.26\times10^9$，误选 **B**。

**正解**：棱台体积 $V=\dfrac{h}{3}(S_1+S_2+\sqrt{S_1S_2})$。高 $h=157.5-148.5=9\,\text{m}$，$S_1=140\times10^6\,\text{m}^2$，$S_2=180\times10^6\,\text{m}^2$。
$$V=\frac{9}{3}\times\bigl[140+180+\sqrt{140\times180}\bigr]\times10^6=3\times(320+60\sqrt7)\times10^6\approx3\times(320+159)\times10^6=1437\times10^6\approx1.4\times10^9\,\text{m}^3$$
故选 **C**。

**避错要点**：本题直接命中本卡**失误3（圆台/棱台体积公式漏中间项）**——台体体积是"三项和"：上底面积 + 下底面积 + 上下底面积几何平均 $\sqrt{S_1S_2}$，$1/3$ 系数与中间项缺一不可。2022 年新高考Ⅰ卷把它放在第 4 题（5 分基础题），正是检验"冷门公式是否记全"。计算类失误属于"会做但丢分"，建议把台体体积、圆锥侧面积、球直径转半径等作成"计算自查卡"，考前一眼扫过即可排雷。
