数列n=1验证与等比q=1讨论易错警示（通项求法·分段通项·分类讨论）

⚠️待强化★★★☆☆#高二深化#新高考Ⅰ卷·广东来源：新高考Ⅰ卷数列真题错题归纳 / 人教A版选择性必修第二册第四章
广东考情：考查频率：近5年新高考Ⅰ卷数列每年必考（2021-2023年为第17题解答题+选填各1道，2024年结构改革后8单选+3多选+3填空+5解答共19题，数列多以中低档解答题或选填出现，2025年第17题为数列，占12分左右）；难度定位：基础~中档，属"必须满分"板块；特色描述：广东考区数列解答题第一问求通项几乎是固定送分点，评卷严格按步骤给分，漏验证n=1或漏讨论q=1属于"会做却丢分"的典型扣分点，多选题也常以"以下说法正确的是"考查分段通项与q=1特例，少选得部分分、错选得0分；提分提示：求通项后强制"代回n=1检验"，等比求和前先问"q可能等于1吗"，两问习惯可挽回3-5分。

核心内容

本质理解：这两大易错点都源于"公式的适用条件被忽略"。核心对策：用公式前先看前提，算完后代回检验。

关键概念

  • $a_n$与$S_n$关系公式：$a_n = S_n - S_{n-1}$ 仅当$n \geq 2$时成立；$n=1$时$a_1 = S_1$，必须单独算、单独验。
  • 分段通项：当$n=1$不满足$n \geq 2$求出的通项时，通项必须写成分段形式

    • $$a_n = \begin{cases} a_1, & n = 1 \\ f(n), & n \geq 2 \end{cases}$$

  • 等比数列求和公式：$S_n = \dfrac{a_1(1-q^n)}{1-q}$ 仅当$q \neq 1$；$q = 1$时$S_n = n a_1$（常数列）。
  • 等比中项陷阱：$b^2 = ac$ 是$a, b, c$成等比的必要不充分条件（$a=b=c=0$不成等比，等比数列各项均不为0）。

核心方法/步骤

一、已知$S_n$求$a_n$的标准三步

  1. 第一步：$n=1$时，$a_1 = S_1$，单独算出；
  2. 第二步：$n \geq 2$时，$a_n = S_n - S_{n-1}$，化简得通项$f(n)$；
  3. 第三步：代回检验——把$n=1$代入$f(n)$，看是否等于$a_1$。相等则合并写$a_n = f(n)$；不等则必须写分段，缺一扣分。

二、含参数等比问题的讨论三步

  1. 第一步：识别公比是否含参数（如$q = t$、$q = a+1$、由条件解出的$q$）；
  2. 第二步：用求和公式前，先单独讨论$q = 1$（常数列，$S_n = n a_1$），再讨论$q \neq 1$；
  3. 第三步：汇总两种情况，检验是否满足题设（如"递增""$S_n$存在"等），舍去矛盾情形。

答题模板/技巧

求通项模板（解答题书写规范）：
当$n=1$时，$a_1 = S_1 = \cdots$；
当$n \geq 2$时，$a_n = S_n - S_{n-1} = \cdots$；
检验：当$n=1$时，上式$= \cdots = a_1$（成立/不成立），
故$a_n = \cdots$（合并式 / 分段式）。
少写"检验"这一步，广东卷评卷扣1-2分步骤分。

等比讨论模板：
① 若$q = 1$，则$\{a_n\}$为常数列，$S_n = n a_1 = \cdots$；
② 若$q \neq 1$，则$S_n = \dfrac{a_1(1-q^n)}{1-q} = \cdots$；
综上，……（结合题意取舍）。

多选题技巧：选项中出现"$\{S_n\}$为等差数列""$a_n$为常数列"等表述，优先检验$q=1$（或$d=0$）的特例；拿不准的选项不选，少选得部分分。

标准化纠错路径（两步法）

一、已知 Sₙ 求 aₙ 的标准路径

  1. n=1：a₁=S₁，单独算出并记下。
  2. n≥2：aₙ=Sₙ−Sₙ₋₁，化简得 f(n)。
  3. 代回检验：将 n=1 代入 f(n)，等于 a₁ → 合并写 aₙ=f(n)；不等 → 必须写分段，缺一扣分。

二、含参等比求和标准路径

  1. 识别公比是否含参数。
  2. 先讨论 q=1（Sₙ=na₁），再 q≠1（Sₙ=a₁(1−qⁿ)/(1−q)）。
  3. 汇总并检验是否满足题设（递增/Sₙ存在等），舍去矛盾。

真题母题（2022 新高考Ⅰ卷 T17）

题：记 Sₙ 为数列 {aₙ} 的前 n 项和，已知 a₁=1，{Sₙ/aₙ} 是公差为 1/3 的等差数列。 (1) 求 {aₙ} 的通项公式；(2) 证明：1/a₁+1/a₂+…+1/aₙ < 2。

错解：由 {Sₙ/aₙ} 公差 1/3 得 Sₙ=(n+2)aₙ/3，n≥2 时 aₙ=Sₙ−Sₙ₋₁ 化简得 aₙ=n(n+1)/2；直接落笔"aₙ=n(n+1)/2"，跳过 n=1 验证书写——广东卷评卷该步占 1~2 分步骤分，漏写即失。

正解（关键步骤）：

  • 由 Sₙ/aₙ = 1+(n−1)/3 = (n+2)/3，得 Sₙ=(n+2)aₙ/3，Sₙ₋₁=(n+1)aₙ₋₁/3。
  • n≥2：aₙ=Sₙ−Sₙ₋₁=(n+2)aₙ/3−(n+1)aₙ₋₁/3 ⇒ aₙ/aₙ₋₁=(n+1)/(n−1)，累乘得 aₙ=n(n+1)/2。
  • 代回检验：当 n=1 时，上式=1×2/2=1=a₁，成立 ⇒ 合并写 aₙ=n(n+1)/2（n∈N*）。
  • (2) 由 aₙ=n(n+1)/2 得 1/aₙ=2(1/n−1/(n+1))，裂项求和=2(1−1/(n+1))<2。

避错要点：本题 n=1 恰好满足，属"验证通过"型；但书写上必须显式写出"当 n=1 时，上式=a₁，成立"，否则按广东卷评分细则扣步骤分。若遇 n=1 不满足（如 Sₙ−Sₙ₋₁ 化简后 n=1 不成立），则必须写分段通项——这正是本卡"三步法"第三步的意义。
