广东考情高一下期末选填失分重灾区。广州各区期末试卷中，向量与解三角形的选择填空题（通常占15-20分）平均失分率高达40%。主要错误集中在：向量夹角范围、SSA多解取舍、投影概念、模长公式误用。名校联考（五校联考）中，解三角形第3问的多解讨论是区分中等生与优等生的关键。

核心内容

本质理解：向量与解三角形的易错点源于"概念混淆"和"范围忽视"。核心对策：每一步标注范围，每一个公式验证条件。

一、平面向量六大陷阱

#### 陷阱1：向量夹角范围——必须是"同起点"

错误认知：任意两个向量的夹角就是它们所在直线的夹角。

正确定义：向量$\vec{a}$与$\vec{b}$的夹角，必须将两向量的起点放在同一点后，所成的最小正角，范围是$[0, \pi]$。

典型错误：

在$\triangle ABC$中，$\vec{AB}$与$\vec{BC}$的夹角是$\angle B$？ - 错误：认为是$\angle B = 60°$ - 正确：将$\vec{BC}$平移至与$\vec{AB}$同起点，夹角为$180° - B = 120°$

避坑方法：画图！将两个向量平移到同起点，再读夹角。

广东卷高频：2024-2025广州越秀区期末第5题，$\vec{AB}$与$\vec{CA}$的夹角，答案不是$\angle A$而是$180° - A$。

#### 陷阱2：投影向量 vs 投影数量

概念混淆：

  • 投影数量（标量）：$|\vec{b}|\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}|}$，可正可负可零
  • 投影向量（向量）：$\frac{\vec{a} \cdot \vec{b}}{|\vec{a}|^2}\vec{a}$，方向与$\vec{a}$相同或相反

典型错误：

求$\vec{b}$在$\vec{a}$方向上的投影——答"$\frac{\vec{a} \cdot \vec{b}}{|\vec{a}|}$"（这是投影数量）还是"$\frac{\vec{a} \cdot \vec{b}}{|\vec{a}|^2}\vec{a}$"（这是投影向量）？ - 审题不清导致全错！

避坑方法：看到"投影"二字，先确认问的是数量还是向量。

广东卷提示：广州九区联考曾考查投影向量，许多考生答成投影数量。

#### 陷阱3：$\vec{a} \cdot \vec{b} > 0$ ⟹ 夹角为锐角？

错误认知：$\vec{a} \cdot \vec{b} > 0$ ⟹ 夹角是锐角。

正确结论：$\vec{a} \cdot \vec{b} > 0$ ⟹ 夹角是锐角或零角（即$0 \leq \theta < 90°$）。

必须排除同向共线：

  • 若$\vec{a} = k\vec{b}$（$k > 0$），则$\vec{a} \cdot \vec{b} = k|\vec{b}|^2 > 0$，但夹角为$0°$（不是锐角）
  • 若$\vec{a} \cdot \vec{b} > 0$且$\vec{a}$与$\vec{b}$不共线，则夹角为锐角

同理：$\vec{a} \cdot \vec{b} < 0$ ⟹ 夹角是钝角或平角（$180°$）。

避坑方法：判断"锐角"时，必须加条件"$\vec{a}$与$\vec{b}$不共线"。

#### 陷阱3+：钝角/锐角判定的完整答题模板（PDF补充·5步法）

来源补充：03.【第3讲】平面向量中的分析策略

广东卷大题中，判定钝角/锐角必须按以下步骤书写，否则扣步骤分：

【答题模板】

步骤1：写出两向量的坐标（若含参数）
步骤2：计算点乘，令其>0（锐角）或<0（钝角），解出参数范围
步骤3：令两向量平行（对应坐标成比例），解出使共线的参数值
步骤4：排除该共线值（若为锐角，排除同向；若为钝角，排除反向）
步骤5：最后写出参数取值范围（写成区间或集合形式）

典型例题（2022全国卷风格）：

已知$\vec{a}=(-3,2)$，$\vec{b}=(3,-1)$，$\vec{a}-t\vec{b}$与$\vec{a}$的夹角为钝角，求$t$的取值范围。 步骤1：$\vec{a}-t\vec{b} = (-3-3t, 2+t)$ 步骤2：点乘$<0$：$(-3)(-3-3t) + 2(2+t) = 9+9t+4+2t = 11t+13 < 0$ → $t < -\frac{13}{11}$ 步骤3：令平行：$(-3)(2+t) = 2(-3-3t)$ → $-6-3t = -6-6t$ → $t = 0$。此时$\vec{a}-t\vec{b} = \vec{a}$，与$\vec{a}$同向，不是钝角。 步骤4：$t=0$时两向量同向（$0°$），需排除。 步骤5：答案：$t < -\frac{13}{11}$且$t \neq 0$。
广东卷评分要点：不写"排除共线"步骤，扣1-2分。许多考生只写$t < -\frac{13}{11}$，漏了$t \neq 0$，直接丢分。

#### 陷阱4：模长公式与平方关系

错误用法：$|\vec{a} + \vec{b}| = |\vec{a}| + |\vec{b}|$（仅当同向时成立）

正确公式：

$$|\vec{a} + \vec{b}|^2 = |\vec{a}|^2 + |\vec{b}|^2 + 2\vec{a} \cdot \vec{b}$$

$$|\vec{a} - \vec{b}|^2 = |\vec{a}|^2 + |\vec{b}|^2 - 2\vec{a} \cdot \vec{b}$$

典型错误：

已知$|\vec{a}| = 2$，$|\vec{b}| = 3$，夹角$60°$，求$|\vec{a} + \vec{b}|$。 - 错误：$2 + 3 = 5$ - 正确：$|\vec{a} + \vec{b}|^2 = 4 + 9 + 2 \times 2 \times 3 \times \frac{1}{2} = 19$，$|\vec{a} + \vec{b}| = \sqrt{19}$

避坑方法：看到"求模长"，先想"平方"。

#### 陷阱5：平行与垂直的坐标条件混淆

关系坐标条件常见错误
垂直$x_1x_2 + y_1y_2 = 0$写成$x_1x_2 = y_1y_2$（漏符号）
平行$x_1y_2 = x_2y_1$（或$\vec{a} = k\vec{b}$）写成$\frac{x_1}{x_2} = \frac{y_1}{y_2}$（未讨论$x_2 = 0$）
更安全：平行用$x_1y_2 = x_2y_1$，避免分式（防止分母为0）。

#### 陷阱5+：大题中平行条件的书写规范（PDF补充·草稿比例式 vs 大题乘积式）

来源补充：03.【第3讲】平面向量中的分析策略

错误写法（草稿纸可用，但大题中写会被扣分）：

∵ a ∥ b
∴ x₁/x₂ = y₁/y₂

正确写法（大题规范）：

∵ a ∥ b
∴ x₁y₂ = x₂y₁（或 x₁y₂ - x₂y₁ = 0）

原因：

  • 比例式$x₁/x₂ = y₁/y₂$隐含"分母不为零"的限制，当$x₂=0$或$y₂=0$时失效
  • 乘积式$x₁y₂ = x₂y₁$适用于所有情况，包括坐标为0的情形
  • 高考评卷要求严谨性，大题必须用乘积式

典型例题：

已知$\vec{a}=(t-1, 3)$，$\vec{b}=(2, t)$，且$\vec{a} \parallel \vec{b}$，求$t$。 草稿纸写法：$(t-1)/2 = 3/t$ → $(t-1)t = 6$ → $t² - t - 6 = 0$ → $t = 3$或$t = -2$。 大题规范写法： - $∵ \vec{a} \parallel \vec{b}$ - $∴ (t-1)t = 2×3$（即$x₁y₂ = x₂y₁$） - $∴ t² - t - 6 = 0$ - $∴ (t-3)(t+2) = 0$ - $∴ t = 3$ 或 $t = -2$。
广东卷评分要点：大题中写比例式，如果恰好分母为0，会导致整个推导失效。即使分母不为0，不写乘积式也可能被扣1分"书写不规范"。

#### 陷阱5++：单位向量"平行"与"同向"的区分（PDF补充）

来源补充：03.【第3讲】平面向量中的分析策略

错误认知：求"与$\vec{a}$平行的单位向量"，答案只有一个。

正确结论：

  • 同向单位向量：只有一个，$\frac{\vec{a}}{|\vec{a}|}$
  • 平行单位向量：有两个，$\pm\frac{\vec{a}}{|\vec{a}|}$（包含同向和反向）

典型错误：

已知$\vec{a}=(3,6)$，求与$\vec{a}$平行的单位向量。 - 错误：只答$\frac{\vec{a}}{|\vec{a}|} = (\frac{\sqrt{5}}{5}, \frac{2\sqrt{5}}{5})$ - 正确：$|\vec{a}| = 3\sqrt{5}$，同向单位向量$(\frac{\sqrt{5}}{5}, \frac{2\sqrt{5}}{5})$，反向单位向量$(-\frac{\sqrt{5}}{5}, -\frac{2\sqrt{5}}{5})$。题目问"平行"，答案必须包含两个！

避坑方法：看到"平行"，立即画圈提醒自己考虑正负；看到"同向"，只有一个答案。

广东卷高频：广州九区联考、名校联考中常考此点，多选题中漏选反向单位向量直接丢一半分。

二、解三角形四大陷阱

#### 陷阱6：SSA（两边一对角）的多解取舍

错误认知：已知两边一对角，三角形唯一确定。

正确结论：可能无解、一解或两解，需用"大边对大角"判断。

判断流程：

已知$a, b, A$，求$B$：

  1. $a < b\sin A$ → 无解
  2. $a = b\sin A$ → 一解（直角三角形）
  3. $b\sin A < a < b$ → 两解（$B$为锐角或钝角）
  4. $a \geq b$ → 一解（$A \geq B$，$B$只能为锐角）

典型错误：

已知$a = 2$，$b = 3$，$A = 30°$，求$B$。 - $b\sin A = 3 \times \frac{1}{2} = 1.5 < a = 2 < b = 3$ → 两解！ - 许多考生只算出一个锐角解，漏了钝角解。

避坑方法：算出$\sin B$后，先判断有几个解，再分别求出所有可能的$B$值，并验证$A + B < 180°$。

广东卷高频：2024-2025广州五校联考第18题第3问考查SSA多解，是压轴区分题。

#### 陷阱7：角的范围与内角和约束

错误认知：算出$\cos A = -\frac{1}{2}$后，直接写$A = 120°$（虽然这个是对的）。

更隐蔽的错误：

已知$A = 30°$，$B = 40°$，求$C$ → $C = 110°$。但如果题目是求$\sin C$，则$\sin C = \sin 110° = \sin 70°$。若后续求$\cos C$，则必须是$\cos 110° = -\cos 70°$（负数！），许多考生写成正数。

避坑方法：求出角后，先判断是锐角还是钝角，再计算其三角函数值。在三角形中，$A + B + C = 180°$是"隐形约束"，必须随时检查。

#### 陷阱8：面积最值中的等号条件

错误认知：用基本不等式求面积最值时，不写取等条件。

正确做法：

由$S = \frac{1}{2}bc\sin A$和$b^2 + c^2 - a^2 = 2bc\cos A$， 得$bc \leq \frac{a^2}{2(1-\cos A)}$（当且仅当$b = c$时取等号） ∴ $S_{max} = \frac{a^2\sin A}{4(1-\cos A)}$（当$b = c$时取得）

避坑方法：用基本不等式求最值，必须写"当且仅当...时取等号"。否则广东卷扣1分。

三、综合陷阱：向量与解三角形的交叉

#### 陷阱9：向量模长与三角形边长的混淆

错误：在$\triangle ABC$中，$\vec{AB} + \vec{BC} = \vec{AC}$，于是$AB + BC = AC$（边长相加）。

正确：$\vec{AB} + \vec{BC} = \vec{AC}$是向量等式，但$AB + BC > AC$（三角形两边之和大于第三边）。向量等式不能直接转化为边长相加！

四、快速自查清单

做向量题时问自己：

  • [ ] 夹角是同起点吗？
  • [ ] 问的是投影数量还是投影向量？
  • [ ] 判断"锐角"时排除了共线吗？（五步模板：坐标→点乘→平行→排除→范围）
  • [ ] 求模长时先平方了吗？
  • [ ] 平行条件用$x_1y_2 = x_2y_1$了吗？（大题中写乘积式，不写比例式）
  • [ ] 单位向量问的是"平行"还是"同向"？平行有两个答案！
  • [ ] 建系时找直角了吗？坐标轴垂直了吗？
  • [ ] 动点参数$\lambda$的范围限制了吗？（线段上$[0,1]$）

做解三角形题时问自己：

  • [ ] 已知条件是SSA吗？需要讨论多解吗？
  • [ ] 所有角都在$(0, 180°)$范围内吗？
  • [ ] 内角和等于180°验证了吗？
  • [ ] 用基本不等式时写取等条件了吗？
  • [ ] 钝角的余弦值是负数吗？
